### [extrawurst gitui](https://github.com/extrawurst/gitui)  ![GitHub Repo stars](https://img.shields.io/github/stars/extrawurst/gitui?style=social)

gitui 是一款基于 Rust 的终端 Git 图形界面工具，旨在提供 GUI 的操作便利性与终端的高效性。它支持纯键盘交互，核心功能包括代码检查、提交与修改、暂存/恢复（支持文件、块及行级别）、分支管理、远程同步、日志搜索及 Stash 操作。工具针对大型仓库进行了性能优化，采用异步 API，具备启动速度快、内存占用低且不卡顿的优势，同时支持子模块、GPG 签名及自定义主题和快捷键。
