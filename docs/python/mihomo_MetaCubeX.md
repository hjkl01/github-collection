### [MetaCubeX mihomo](https://github.com/MetaCubeX/mihomo)  ![GitHub Repo stars](https://img.shields.io/github/stars/MetaCubeX/mihomo?style=social)

该项目是一个基于 Pydantic 的 Python 库，用于解析崩坏：星穹铁道（Honkai: Star Rail）的玩家数据。
主要功能如下：
1. 通过 Mihomo API 获取玩家基本信息及角色详情，支持 V1 和 V2 两种数据模型，提供类型提示和自动补全。
2. 支持将图标文件名替换为资源 URL。
3. 提供辅助工具，包括去除重复角色和合并角色数据。
4. 支持使用 Pickle 或 JSON 格式进行数据持久化存储。
