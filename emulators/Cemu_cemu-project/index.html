cemu-project Cemu ¶

Cemu 是一款用 C/C++ 编写的开源 Wii U 模拟器，能够在 64 位 Windows、Linux 及 macOS 系统上以可玩状态运行大多数 Wii U 游戏和自制软件。项目处于活跃开发中，采用 Mozilla Public License 2.0 协议，其中 macOS 版本目前为实验性阶段。
