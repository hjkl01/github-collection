### [LunarBar-app LunarBar](https://github.com/LunarBar-app/LunarBar)

**核心内容总结：**  
LunarBar 是一款免费开源的 macOS 状态栏极简日历应用，支持农历、公共假日、系统日历及提醒功能。用户可通过下载 `LunarBar.dmg` 文件并拖拽至 `Applications` 文件夹安装，或使用 Homebrew 命令 `brew install --cask lunarbar` 安装。应用具备自动更新功能，兼容 macOS 13/14 版本，且为沙盒应用，已通过 Apple 签名和认证。其设计以极简为核心，不提供 iOS 版本及小组件功能，开发团队表示此为基于自身需求开发的工具，旨在满足用户对简洁日历功能的需求。
