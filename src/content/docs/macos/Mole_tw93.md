### [tw93 Mole](https://github.com/tw93/Mole)  ![GitHub Repo stars](https://img.shields.io/github/stars/tw93/Mole?style=social)

Mole 是一款专为 macOS 设计的命令行深度清理与优化工具，旨在整合多个系统管理应用的功能。

**核心功能：**
1. **深度清理**：清除系统缓存、日志、浏览器残留及废纸篓文件，释放大量磁盘空间。
2. **智能卸载**：移除应用程序的同时，自动清理关联的启动项、偏好设置及隐藏残留文件。
3. **系统优化**：重建系统数据库、重置网络服务、刷新 Finder 与 Dock 界面及 Spotlight 索引。
4. **磁盘分析**：可视化展示磁盘使用情况，支持查找大文件、分析外部硬盘。
5. **实时监控**：提供 CPU、内存、磁盘、网络及硬件温度的实时仪表盘与性能指标。
6. **专项清理**：清理项目构建产物（如 node_modules、target）及分散在各处的安装包文件。

**使用特性：**
* 支持 Homebrew 及脚本安装，提供交互式菜单与独立命令操作。
* 提供 `dry-run` 安全预览模式，防止误删，所有操作均有日志记录。
* 支持 Shell 补全、Raycast/Alfred 快速启动及配置白名单保护。
