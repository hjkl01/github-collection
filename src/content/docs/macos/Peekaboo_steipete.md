### [steipete Peekaboo](https://github.com/steipete/Peekaboo)  ![GitHub Repo stars](https://img.shields.io/github/stars/steipete/Peekaboo?style=social)

Peekaboo 是一款面向 macOS 的 GUI 自动化工具，通过高精度屏幕捕捉与 AI 分析实现屏幕操作。主要功能包括：
1. **精准捕捉**：支持窗口、屏幕及菜单栏的像素级截图，含 Retina 2x 缩放。
2. **智能代理**：支持自然语言指令，自动编排“查看、点击、输入、滚动、快捷键”等工具链完成任务。
3. **多模型支持**：兼容 OpenAI、Anthropic、Grok、Google 及本地 Ollama 等多种 AI 模型。
4. **接口集成**：提供原生 CLI 命令行工具及 MCP 服务器，支持集成至 Claude Desktop 和 Cursor 等编辑器。
5. **系统控制**：支持应用生命周期管理、窗口调整、Dock 交互、菜单查找及对话框处理。
使用前需配置 macOS 屏幕录制与辅助功能权限。
