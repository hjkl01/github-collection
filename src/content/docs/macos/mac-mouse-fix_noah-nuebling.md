**项目核心内容总结：**

Mac Mouse Fix 是一款用于优化鼠标操作的驱动程序，支持自定义鼠标按钮功能、调整滚轮行为，并解决与特定软件的兼容性问题。主要功能包括：

1. **鼠标按钮自定义**：可为不同鼠标按钮分配快捷键或触发特定操作（如点击、拖拽等）。
2. **滚轮设置**：支持调整滚轮滚动速度、方向及兼容性（如网页/文档中的滚动行为）。
3. **临时禁用功能**：通过菜单栏快速关闭某些功能，避免在特定应用中冲突。
4. **跨平台支持**：当前专注于 macOS，未来可能扩展至其他平台（非当前重点）。
5. **开源与付费模式**：项目开源，提供 30 天免费试用，后续需购买许可证（价格约 2 美元），但开源代码可自由使用。

**使用方法**：安装后通过图形界面配置按钮和滚轮行为，启用后无需保持应用前台运行，设置会持久生效。  

**主要特性**：支持多种鼠标型号，提供临时禁用功能以适配不同场景，开源社区可参与代码、翻译或反馈优化。
