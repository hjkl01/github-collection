### [noah-nuebling mac-mouse-fix](https://github.com/noah-nuebling/mac-mouse-fix)  ![GitHub Repo stars](https://img.shields.io/github/stars/noah-nuebling/mac-mouse-fix?style=social)

Mac Mouse Fix 是一款开源的 macOS 鼠标增强应用，旨在将普通鼠标体验提升至媲美苹果触控板的水平。其核心功能包括：提供自然流畅的多指手势与滚动体验；支持鼠标按键深度自定义，可分配点击、拖拽、导航及媒体控制等操作；允许通过鼠标调用系统快捷键以控制亮度和音量。应用原生支持 Apple Silicon，兼容 macOS 11 及以上版本，其中 Mac Mouse Fix 2 永久免费，Mac Mouse Fix 3 提供免费试用。
