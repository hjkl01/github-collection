### [lwouis alt-tab-macos](https://github.com/lwouis/alt-tab-macos)  ![GitHub Repo stars](https://img.shields.io/github/stars/lwouis/alt-tab-macos?style=social)

AltTab 是一个为 macOS 带来类似 Windows Alt+Tab 切换窗口体验的工具。通过快捷键，用户可以快速在打开的应用和窗口之间切换，提升工作效率。

**使用方法**：下载安装后，默认使用 Alt+Tab 快捷键即可进行窗口切换。

**主要特性**：
- 支持类似 Windows 的窗口切换体验
- 支持多种切换模式（如按应用、窗口分组等）
- 高度可定制，支持快捷键和样式调整
- 安装简单，使用方便
