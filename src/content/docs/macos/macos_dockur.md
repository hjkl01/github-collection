### [dockur macos](https://github.com/dockur/macos)  ![GitHub Repo stars](https://img.shields.io/github/stars/dockur/macos?style=social)

该项目允许在 Docker 容器中运行 macOS 操作系统。核心功能包括 KVM 硬件加速、Web 查看器及系统自动下载。支持通过 Docker Compose、CLI、Kubernetes 及 GitHub Codespaces 部署。用户可自定义 macOS 版本（11 至 15）、磁盘大小、CPU 和内存配置，并支持网络模式（macvlan、DHCP）、设备透传（磁盘、USB）及文件共享。项目声明代码开源且仅允许在 Apple 硬件上合法使用。
