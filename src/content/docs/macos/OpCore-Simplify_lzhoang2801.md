### [lzhoang2801 OpCore-Simplify](https://github.com/lzhoang2801/OpCore-Simplify)  ![GitHub Repo stars](https://img.shields.io/github/stars/lzhoang2801/OpCore-Simplify?style=social)

OpCore Simplify 是一款自动化 OpenCore EFI 创建工具，旨在简化 Hackintosh 搭建流程。

1. **广泛兼容**：支持 Intel（1-15 代）与 AMD CPU，涵盖多种 GPU 架构及 macOS High Sierra 至 Tahoe 系统。
2. **自动配置**：根据硬件自动检测并集成 ACPI 补丁与 Kexts（集成 SSDTTime），处理 CPU 拓扑、设备禁用及睡眠状态修复。
3. **智能更新**：构建前自动同步 OpenCorePkg 及相关驱动至最新版本。
4. **EFI 定制**：自动优化配置（如 ID 欺骗、NVRAM、SMBIOS 选择），并提供手动自定义接口。
5. **跨平台运行**：支持 Windows、macOS 及 Linux 脚本，涵盖硬件检测、EFI 构建及 USB 映射全流程辅助。

注：工具仅用于辅助搭建，不保证一次安装成功，用户需具备基础 Hackintosh 知识及故障排查能力。
