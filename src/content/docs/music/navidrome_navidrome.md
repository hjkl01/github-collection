### [navidrome navidrome](https://github.com/navidrome/navidrome)

Navidrome 是一个开源的基于 Web 的音乐服务器和流媒体工具，允许用户通过浏览器或移动设备访问个人音乐库，功能类似个人版 Spotify。  

**核心功能**：  
- 支持大型音乐库，兼容几乎所有音频格式；  
- 多用户管理，支持独立播放记录、歌单和收藏；  
- 自动监控音乐库变化，实时更新元数据；  
- 提供主题化 Web 界面（基于 Material UI）；  
- 兼容 Subsonic/Madsonic/Airsonic 客户端；  
- 支持转码（如 Opus 编码），可按用户/设备自定义设置。  

**使用方法**：  
- 提供 Docker 镜像、预编译二进制文件（含 Raspberry Pi 版本）及源码编译方式；  
- 可通过云托管服务 [PikaPods](https://www.pikapods.com) 部署；  
- 安装与配置文档见 [项目官网](https://www.navidrome.org/docs)。  

**特性**：  
- 跨平台运行（macOS、Linux、Windows）；  
- 低资源占用；  
- 支持合集（Various Artists）和多碟套装（Box Sets）；  
- 多语言支持（通过 POEditor 管理翻译）。
