### [jeffvli feishin](https://github.com/jeffvli/feishin)  ![GitHub Repo stars](https://img.shields.io/github/stars/jeffvli/feishin?style=social)

Feishin 是 Sonixd 的重写版本，一款现代化的音乐播放器客户端。它支持 MPV 和 Web 双后端播放，提供现代 UI 界面、播放状态同步 (Scrobble)、歌词支持（含同步/异步及内置获取）以及智能播放列表编辑功能。项目提供桌面客户端（推荐）、Web 在线版和 Docker 镜像三种部署方式，兼容支持 Navidrome、Jellyfin 或 OpenSubsonic 协议的音乐服务器。
