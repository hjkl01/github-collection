### [jitsi docker-jitsi-meet](https://github.com/jitsi/docker-jitsi-meet)  ![GitHub Repo stars](https://img.shields.io/github/stars/jitsi/docker-jitsi-meet?style=social)

本项目提供基于 Docker 和 Docker Compose 部署 Jitsi Meet 视频会议解决方案的工具。Jitsi Meet 是一款完全加密、100% 开源且免费的视频会议系统，无需账户即可使用。项目发布支持 amd64 和 arm64 架构的 Docker 镜像，包含稳定版与测试版标签，并提供安装手册及 Kubernetes 部署资源链接。
