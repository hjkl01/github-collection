### [basecamp once-campfire](https://github.com/basecamp/once-campfire)  ![GitHub Repo stars](https://img.shields.io/github/stars/basecamp/once-campfire?style=social)

Campfire 是一个基于 Web 的聊天应用，支持多房间访问控制、私聊、文件附件预览、搜索、Web Push 通知、@提及及支持机器人集成的 API。项目提供 Docker 镜像，可实现包含 Web 服务、后台任务、缓存、文件存储及 SSL 的单机器部署，支持通过环境变量配置域名、通知密钥和错误报告。系统采用单租户架构，首次启动引导创建管理员账户，公开房间对所有系统用户开放。
