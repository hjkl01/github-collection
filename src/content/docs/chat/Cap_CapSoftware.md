### [CapSoftware Cap](https://github.com/CapSoftware/Cap)

**核心内容总结：**  
Cap 是一款开源的视频消息工具，功能与 Loom 类似，支持快速录制、编辑和分享视频。用户可通过官网下载 macOS 和 Windows 的桌面应用。  

**主要特性：**  
- 支持自托管，提供 Docker 和 Railway 部署方案；  
- 采用现代化技术栈，包括 Rust（Tauri 桌面端）、React/Next.js（网页端）、MySQL 数据库；  
- 开源协议包含 MIT（部分代码）和 AGPLv3（其他内容）。  

**使用方法：**  
1. 通过官网下载桌面应用或部署自托管服务；  
2. 自托管需参考文档，使用 Docker 或点击 Railway 按钮一键部署。
