### [CapSoftware Cap](https://github.com/CapSoftware/Cap)  ![GitHub Repo stars](https://img.shields.io/github/stars/CapSoftware/Cap?style=social)

Cap 是一款开源的 Loom 替代品，提供视频录制、编辑和分享功能。项目架构包含基于 Tauri（Rust + SolidStart）的桌面端和基于 Next.js 的 Web 端，采用 MySQL 数据库。支持 Docker Compose、Railway 等多种自托管部署方案，核心代码主要采用 AGPLv3 协议。
