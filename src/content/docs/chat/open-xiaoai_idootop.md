### [idootop open-xiaoai](https://github.com/idootop/open-xiaoai)

**核心内容总结：**

Open-XiaoAI 是一个开源项目，通过接管小爱音箱 Pro（LX06/OH2P）的语音输入输出模块，利用多模态大模型和 AI Agent 技术，突破传统智能音箱的“指令-响应”模式，实现情感理解、主动交互等高级功能。主要功能包括：  
1. **自定义唤醒词**：支持更换小爱音箱的唤醒词。  
2. **接入 AI 模型**：可连接小智 AI、MiGPT、Gemini Live API 等外部模型，扩展音箱的对话能力。  
3. **刷机改造**：需先刷入定制固件并安装客户端程序，通过 SSH 连接设备完成配置。  

**使用方法**：  
- 仅支持小爱音箱 Pro（LX06/OH2P）机型。  
- 步骤：刷机 → 安装客户端 → 运行示例程序（如接入 MiGPT、自定义唤醒词等）。  

**主要特性**：  
- 突破传统音箱的固定逻辑，实现更自然的交互体验。  
- 开源且支持用户自定义开发新功能。
