### [chanify chanify](https://github.com/chanify/chanify)  ![GitHub Repo stars](https://img.shields.io/github/stars/chanify/chanify?style=social)

Chanify 是一款安全且简洁的通知工具，允许开发者及系统管理员通过 API 推送通知。其主要功能总结如下：

1. **多格式消息支持**：支持发送文本、链接、图片、音频、文件及自定义动作消息。
2. **自建私有节点**：支持分布式架构，可部署私有节点服务器（无状态或有状态模式），保障用户隐私。
3. **多端客户端**：提供 iOS 和 macOS 应用、Windows 客户端以及 Chrome 浏览器扩展。
4. **灵活部署**：支持二进制文件、Docker、源码编译及 Docker Compose 多种部署方式。
5. **开放接口与扩展**：提供完整的 HTTP API 及 Lua API 插件系统，支持 Webhook 等集成场景。
