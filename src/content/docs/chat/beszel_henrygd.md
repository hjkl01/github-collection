### [henrygd beszel](https://github.com/henrygd/beszel)  ![GitHub Repo stars](https://img.shields.io/github/stars/henrygd/beszel?style=social)

Beszel 是一款轻量级的服务器监控平台，由 Hub 管理端和 Agent 监控端组成。它提供友好的 Web 界面，支持 Docker 容器统计、历史数据追踪及多指标警报（CPU、内存、磁盘、网络、温度等）。项目配置简单，支持多用户管理、OAuth 认证、自动备份及 API 访问。监控范围涵盖主机及容器的 CPU、内存、磁盘、网络、负载、GPU、电池、S.M.A.R.T. 健康状态等。
