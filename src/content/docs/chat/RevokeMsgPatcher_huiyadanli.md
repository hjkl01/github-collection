### [huiyadanli RevokeMsgPatcher](https://github.com/huiyadanli/RevokeMsgPatcher)

**项目功能**  
为Windows平台的微信/QQ/TIM提供防撤回补丁，支持最新版本。微信可安装多开功能，通过修改DLL文件实现消息防撤回。  

**使用方法**  
1. 系统需为Windows 7及以上，安装.NET Framework 4.5.2及以上。  
2. 关闭微信/QQ/TIM后，以管理员身份运行程序，自动获取安装路径（绿色版需手动选择）。  
3. 点击“防撤回”按钮，等待补丁生效（可能触发杀毒软件警告，需放行）。  
4. 软件更新后需重新安装补丁。  

**主要特性**  
- 支持微信/QQ/TIM最新版本，微信支持多开。  
- 补丁基于DLL文件修改，依赖开源项目技术。  
- 提供GitHub及云盘下载渠道。  
- 2.1版本整合NTQQAntiRecall与BetterWX技术。  

**注意事项**  
- 需关闭原程序并以管理员权限运行。  
- 修改DLL可能被杀毒软件拦截，需手动放行。  
- 软件更新后需重新应用补丁。
