### [WhiskeySockets Baileys](https://github.com/WhiskeySockets/Baileys)

Baileys 是一个基于 WebSocket 的 TypeScript 库，用于与 WhatsApp Web API 进行交互。主要功能包括通过 WebSocket 实现 WhatsApp 消息收发、会话管理等操作。  

**使用方法**：需参考官方文档（https://baileys.wiki），旧版指南可查阅 GitHub README。  

**主要特性**：  
1. 支持 TypeScript 开发；  
2. 提供 WebSocket 协议对接 WhatsApp；  
3. 7.0.0 版本有重大变更，需通过指定链接迁移；  
4. 开源 MIT 许可证，可自由使用和修改。  

**注意事项**：  
- 项目与 WhatsApp 无官方关联，使用需遵守其服务条款；  
- 禁止用于垃圾短信、骚扰等违规行为。
