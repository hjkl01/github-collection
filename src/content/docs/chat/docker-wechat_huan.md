### [huan docker-wechat](https://github.com/huan/docker-wechat)  ![GitHub Repo stars](https://img.shields.io/github/stars/huan/docker-wechat?style=social)

DoChat（盒装微信）是一个基于 Docker 和 Wine 的项目，用于在 Linux 桌面环境运行 Windows 版微信客户端。它提供一键启动脚本，支持中文字符显示与输入、复制粘贴图片等功能。项目兼容 Ubuntu、Debian、OpenSUSE、Arch 等多种 Linux 发行版，并允许通过环境变量配置 DPI 缩放、指定微信版本及启用调试模式。需注意，使用该非官方方案存在微信账号被封禁的风险，用户需自行承担。
