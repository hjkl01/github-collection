### [quackduck devzat](https://github.com/quackduck/devzat)  ![GitHub Repo stars](https://img.shields.io/github/stars/quackduck/devzat?style=social)

Devzat 是一个基于 SSH 协议的聊天服务器，用户可通过 SSH 客户端在任何设备（含移动端）接入聊天室，而非传统 Shell 终端。主要功能包括：多房间聊天、私信、Markdown 与代码语法高亮、表情符号替换、内置游戏（如井字棋、猜词）及时间区设置。支持自托管部署，提供 Slack 和 Discord 桥接集成及插件 API 扩展功能。支持端口 443 免密钥登录以适配防火墙环境。
