### [nickrunning wechat-selkies](https://github.com/nickrunning/wechat-selkies)  ![GitHub Repo stars](https://img.shields.io/github/stars/nickrunning/wechat-selkies?style=social)

本项目是基于 Docker 的微信和 QQ Linux 客户端容器化项目，利用 Selkies WebRTC 技术实现在浏览器中直接使用微信/QQ，无需本地安装客户端。核心功能包括配置与聊天记录持久化存储、完整中文支持（字体及输入法）、图片复制、文件传输及窗口切换。项目兼容 AMD64 和 ARM64 架构，支持可选的 GPU 硬件加速，并提供自动启动、桌面快捷方式集成及第三方应用（如 Telegram）安装功能。适用于服务器部署及远程办公场景，支持 Docker 及 Docker Compose 快速部署。
