### [imgcook imove](https://github.com/imgcook/imove)  ![GitHub Repo stars](https://img.shields.io/github/stars/imgcook/imove?style=social)

iMove 是一个逻辑可复用的、面向函数的、流程可视化的 JavaScript 工具库，核心解决复杂逻辑复用问题。项目由画布和 SDK 两部分组成，用户通过画布可视化编排流程并导出 DSL，项目端通过 SDK 调用执行。支持节点复用、参数配置、函数扩展及插件集成，适用于前端事件、网络请求及 Node.js 后端等全场景 JavaScript 流程开发。
