### [rolldown rolldown](https://github.com/rolldown/rolldown)  ![GitHub Repo stars](https://img.shields.io/github/stars/rolldown/rolldown?style=social)

Rolldown 是一个使用 Rust 编写的 JavaScript/TypeScript 打包器，旨在成为 Vite 的未来打包工具。它提供与 Rollup 兼容的 API 和插件接口，功能范围更接近 esbuild。项目目前处于 RC 阶段，内置压缩功能尚在 Alpha 状态。
