### [rolldown rolldown](https://github.com/rolldown/rolldown)

**核心内容总结：**  
Rolldown 是一个用 Rust 编写的 JavaScript/TypeScript 打包工具，旨在成为 Vite 的未来打包方案。它提供与 Rollup 兼容的 API 和插件接口，但功能范围更接近 esbuild。主要特性包括高性能、支持多种平台（如 Windows、macOS、Linux 及 WebAssembly），内置压缩功能（当前为 alpha 阶段）。用户可通过 NPM 安装并参考 [rolldown.rs](https://rolldown.rs/guide/getting-started) 文档使用。项目由 VoidZero Inc. 开发，兼容 Rollup 和 esbuild 的部分代码，采用 MIT 许可证。当前为 Beta 版本，可能存在未完善功能。
