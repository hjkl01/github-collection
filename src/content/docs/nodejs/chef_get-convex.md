### [get-convex chef](https://github.com/get-convex/chef)  ![GitHub Repo stars](https://img.shields.io/github/stars/get-convex/chef?style=social)

Chef 是 Convex 团队开发的 AI 全栈应用构建器，能够生成包含内置数据库、零配置认证、文件上传、实时 UI 及后台工作流的全栈 Web 应用。项目基于 Convex 数据库 API 构建，支持在线托管和本地部署。本地运行需配置 Convex 项目、OAuth 认证及大模型 API 密钥。核心结构包含前端 UI、Agent 逻辑、数据库、CLI 工具及测试框架。
