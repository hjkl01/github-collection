### [is-a-dev register](https://github.com/is-a-dev/register)  ![GitHub Repo stars](https://img.shields.io/github/stars/is-a-dev/register?style=social)

is-a.dev 是一个为开发者提供个人网站 `.is-a.dev` 子域名的服务。用户需 Fork 仓库并遵循文档提交 Pull Request 审核，通过后 DNS 记录将在数分钟内发布。该服务支持举报违反服务条款的行为，并获得 Cloudflare Project Alexandria 赞助。
