### [evanw esbuild](https://github.com/evanw/esbuild)  ![GitHub Repo stars](https://img.shields.io/github/stars/evanw/esbuild?style=social)

esbuild 是一款极快的 JavaScript 打包工具，旨在大幅提升构建性能。其核心功能包括：无需缓存即可实现极速构建；内置支持 JavaScript、CSS、TypeScript 和 JSX；提供便捷的 CLI、JS 及 Go API；支持 ESM 和 CommonJS 模块打包；支持 CSS 模块打包；具备代码压缩、树摇及源码映射功能；同时提供本地服务器、监听模式与插件系统。
