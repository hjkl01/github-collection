### [base node](https://github.com/base/node)  ![GitHub Repo stars](https://img.shields.io/github/stars/base/node?style=social)

本项目提供基于 Optimism OP Stack 的 Base 以太坊 L2 网络节点的 Docker 部署方案。用户可使用 Docker Compose 运行独立节点，支持主网和测试网。项目兼容 Reth、Geth 及 Nethermind 客户端，需配置 L1 RPC 端点并满足特定硬件要求。支持快照加速同步，并提供详细的网络配置与性能优化选项。
