### [vercel turborepo](https://github.com/vercel/turborepo)

Turborepo 是一个高性能的 JavaScript 和 TypeScript 构建系统，由 Vercel 开发并使用 Rust 编写。其核心功能包括加速多项目代码库的构建流程，支持缓存和并行执行任务。用户可通过访问官网（https://turborepo.com）快速上手。项目特性包括：由 Vercel 团队维护、支持主流前端技术栈、提供社区讨论渠道和安全漏洞披露机制。
