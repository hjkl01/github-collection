### [oven-sh bun](https://github.com/oven-sh/bun)  ![GitHub Repo stars](https://img.shields.io/github/stars/oven-sh/bun?style=social)

Bun 是为 JavaScript 和 TypeScript 应用设计的一体化高性能工具集，通过单个可执行文件交付。

1. **核心运行时**：作为 Node.js 的无缝替代品，基于 Zig 和 JavaScriptCore 构建，原生支持 TypeScript 和 JSX，启动速度更快、内存占用更低。
2. **内置工具链**：包含极速的包管理器、脚本运行器、测试运行器及打包器，完全兼容 Node.js 生态。
3. **丰富 API**：提供 HTTP 服务器、WebSockets、数据库连接（SQLite、PostgreSQL、Redis 等）、文件 I/O 及 FFI 支持。
4. **跨平台支持**：兼容 Linux、macOS 和 Windows 系统。

旨在通过单一工具替代开发环境中的多个依赖，显著提升开发效率。
