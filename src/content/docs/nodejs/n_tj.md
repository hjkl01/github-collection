### [tj n](https://github.com/tj/n)  ![GitHub Repo stars](https://img.shields.io/github/stars/tj/n?style=social)

`n` 是一款用于交互式管理 Node.js 版本的命令行工具，适用于 macOS、Linux 及 WSL 等类 Unix 环境，无需配置子壳或环境变量。它支持通过版本号、特定标签（如 lts、latest）或配置文件（如 .nvmrc）来安装和切换 Node.js 版本。工具会缓存已下载的 Node.js 包以支持快速重装和离线使用，并允许直接运行或执行已缓存的版本，同时支持保留当前 npm 版本。此外，`n` 支持自定义下载镜像、系统架构和安装路径，并兼容多种安装方式。
