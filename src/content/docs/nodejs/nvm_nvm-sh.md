### [nvm-sh nvm](https://github.com/nvm-sh/nvm)  ![GitHub Repo stars](https://img.shields.io/github/stars/nvm-sh/nvm?style=social)

nvm 是 Node.js 的版本管理工具，支持 Unix、macOS 和 Windows WSL 平台。它允许用户通过命令行快速安装、切换和管理不同版本的 Node.js。主要功能包括：安装指定版本、设置默认版本、管理长期支持版本 (LTS)、迁移全局 npm 包、根据 .nvmrc 文件自动匹配环境，以及支持 Docker 开发和 CI/CD 集成。nvm 支持 POSIX 兼容的 Shell，按用户安装，无需 sudo 权限。
