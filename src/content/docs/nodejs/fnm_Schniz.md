### [Schniz fnm](https://github.com/Schniz/fnm)  ![GitHub Repo stars](https://img.shields.io/github/stars/Schniz/fnm?style=social)

fnm 是一个基于 Rust 开发的快速 Node.js 版本管理器。它支持 macOS、Windows 和 Linux 跨平台运行，具有单文件、易安装、启动快的特点，兼容 .node-version 和 .nvmrc 文件，用于管理不同版本的 Node.js 环境，并提供 Shell 自动补全及便捷的安装配置方案。
