### [opactorai Claudable](https://github.com/opactorai/Claudable)  ![GitHub Repo stars](https://img.shields.io/github/stars/opactorai/Claudable?style=social)

Claudable 是一款基于 Next.js 的开源 Web 应用构建器，整合了 Claude Code 及 Cursor CLI 等 AI 代理的先进能力。用户只需通过自然语言描述应用需求，即可即时生成生产级代码并获得带有热重载功能的实时预览。项目支持一键部署至 Vercel、集成 Supabase 数据库及 GitHub 版本控制，并提供 Electron 桌面客户端。无需额外订阅费用，支持多种 AI 编码代理（如 Codex CLI、Qwen Code 等），免费构建和部署专业 Web 应用。
