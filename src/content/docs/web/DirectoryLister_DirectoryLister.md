### [DirectoryLister DirectoryLister](https://github.com/DirectoryLister/DirectoryLister)

**项目核心内容总结：**  
Directory Lister 是一个用于快速展示网络文件夹内容的工具，支持浏览、共享和下载文件。其核心功能包括：  
- **快速安装**：无需配置，通过下载、解压并复制文件到服务器即可启动。  
- **核心特性**：支持亮/暗主题切换、自定义文件排序、文件搜索、生成文件哈希校验码、直接显示 README 文件内容、整目录压缩下载、多语言支持。  
- **技术要求**：需 PHP 8.2 及以上版本，依赖 Zip、DOM 和 Fileinfo 扩展。  
- **配置**：通过复制 `.env.example` 为 `.env` 文件并修改配置参数进行设置。  
- **社区支持**：提供赞助、讨论区及文档支持，用户可参与贡献。
