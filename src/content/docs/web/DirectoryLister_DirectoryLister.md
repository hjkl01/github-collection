### [DirectoryLister DirectoryLister](https://github.com/DirectoryLister/DirectoryLister)  ![GitHub Repo stars](https://img.shields.io/github/stars/DirectoryLister/DirectoryLister?style=social)

Directory Lister 是一款用于浏览和共享 Web 可访问文件夹内容的轻量级工具。支持零配置快速安装，提供亮暗主题、自定义排序、文件搜索、哈希校验、README 渲染、整目录 ZIP 下载及多语言支持。基于 PHP 8.2+ 开发。
