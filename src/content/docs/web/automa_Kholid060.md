### [Kholid060 automa](https://github.com/Kholid060/automa)

**核心内容总结**  
Automa 是一款通过可视化连接块实现浏览器自动化的扩展工具，支持自动填写表单、执行重复任务、截图、抓取网站数据，并可设置定时执行。  

**使用方法**  
- 通过 Chrome 或 Firefox 商店下载安装；  
- 或本地手动加载 `build` 目录中的扩展文件。  

**主要特性**  
- 可视化流程编辑（拖拽块构建自动化任务）；  
- 支持跨浏览器（Chrome 和 Firefox）；  
- 提供社区市场共享和下载工作流；  
- 可生成独立扩展包（通过 Automa CEB 工具）；  
- 提供开发文档和构建命令（如 `pnpm dev`、`pnpm build`）。
