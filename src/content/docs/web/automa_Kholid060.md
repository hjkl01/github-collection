### [Kholid060 automa](https://github.com/Kholid060/automa)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kholid060/automa?style=social)

Automa 是一款支持 Chrome 和 Firefox 的浏览器扩展插件，通过连接功能模块实现浏览器自动化操作。其核心功能包括自动填充表单、执行重复任务、网页截图、网站数据抓取以及设置任务执行时间。项目还配备工作流市场供用户分享下载，并支持基于工作流生成独立的扩展程序。
