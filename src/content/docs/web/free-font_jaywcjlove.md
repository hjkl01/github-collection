### [jaywcjlove free-font](https://github.com/jaywcjlove/free-font)  ![GitHub Repo stars](https://img.shields.io/github/stars/jaywcjlove/free-font?style=social)

本项目是一个收录商用免费字体的开源资源库，旨在收集汉字字体及开源英文字体，方便用户下载和使用。主要功能包括自动生成字体预览封面、提供字体分类（黑体、宋体、楷体等）与授权协议说明、支持本地开发与 Docker 部署，并提供多镜像站点以优化访问速度。字体版权归原主所有，使用风险由用户自行承担，大于 50MB 的文件无法提交。
