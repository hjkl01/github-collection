### [adrianhajdin zoom-clone](https://github.com/adrianhajdin/zoom-clone)  ![GitHub Repo stars](https://img.shields.io/github/stars/adrianhajdin/zoom-clone?style=social)

该项目是一个基于 Next.js、TypeScript、Tailwind CSS 和 shadcn 构建的 Zoom 视频会议克隆平台。核心功能如下：

1. **用户认证**：集成 Clerk 实现安全登录，支持社交账号及邮箱密码。
2. **会议管理**：支持创建新会议、预约未来会议、通过链接加入及拥有个人专属房间。
3. **实时互动**：基于 GetStream 提供安全实时音视频，支持录制、屏幕共享、静音控制及表情反应。
4. **参与者管理**：提供会议控制功能，包括参会者列表查看、置顶、静音、禁止视频共享及锁定会议。
5. **历史记录**：可查看过往会议列表详情及会议录制回放。
6. **响应式设计**：适配多种设备屏幕，确保最佳用户体验。
