### [d3george slash-admin](https://github.com/d3george/slash-admin)

**项目核心内容总结：**

Slash Admin 是一个基于 React 19、Vite、shadcn/ui 和 TypeScript 构建的现代管理后台模板，用于快速开发管理类系统。  

**主要功能与特性：**  
- 支持响应式设计，适配不同设备和屏幕尺寸；  
- 提供用户管理、角色管理、权限管理等常见后台功能；  
- 集成 shadcn/ui 组件库，支持主题定制和样式调整；  
- 采用 Zustand 状态管理、React-Query 数据获取，结合 MSW 和 Faker.js 实现数据模拟；  
- 支持国际化语言切换、基于角色的访问控制、嵌套路由配置；  

**使用方法：**  
1. 克隆项目代码：`git clone https://github.com/d3george/slash-admin.git`；  
2. 安装依赖：`pnpm install`；  
3. 启动开发服务器：`pnpm dev`（访问 http://localhost:3001）；  
4. 生产构建：`pnpm build`。
