### [d3george slash-admin](https://github.com/d3george/slash-admin)  ![GitHub Repo stars](https://img.shields.io/github/stars/d3george/slash-admin?style=social)

Slash Admin 是一个基于 React 19、Vite、shadcn/ui 和 TypeScript 构建的现代管理后台模板，旨在帮助开发者快速构建功能强大的管理系统。核心功能包括响应式设计、灵活嵌套路由、基于角色的访问控制，以及内置的用户、角色和权限管理。项目支持国际化与主题定制，采用 Zustand 进行状态管理，React-Query 处理数据获取，并集成 MSW 和 Faker.js 实现 Mock 数据方案。
