### [PGYER codefever](https://github.com/PGYER/codefever)  ![GitHub Repo stars](https://img.shields.io/github/stars/PGYER/codefever?style=social)

CodeFever Community 是一个完全开源、永久免费的 Git 代码托管服务。项目轻量高效，低资源占用即可运行，支持无限仓库数量。功能涵盖代码版本对比、原生分支与标签管理、合并请求、分支保护、多人协作权限控制、Webhook 集成及管理员后台。支持中英双语，提供 Docker 及手动两种安装方式，采用 MIT 协议发布。
