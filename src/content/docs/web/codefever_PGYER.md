### [PGYER codefever](https://github.com/PGYER/codefever)

**项目核心内容总结：**

1. **项目简介**  
CodeFever Community 是一个完全开源的 Git 代码托管服务，提供永久免费、轻量高效、功能完整的代码托管解决方案。

2. **主要特性**  
- 支持无限仓库、代码对比、分支管理、合并请求、分支保护、多人协作；  
- 提供 Webhook 集成、管理后台、中英双语界面；  
- 1 核 1 GB 内存即可运行，对服务器资源要求低。

3. **安装与更新**  
- **安装方式**：推荐使用 Docker 镜像（一行命令完成），或手动从零安装；  
- **更新方式**：支持保留数据的更新操作，详见文档指引。

4. **使用与贡献**  
- 提供仓库管理、管理员设置、Git 操作等使用文档；  
- 支持通过提交 Issue 反馈问题或提出功能需求；  
- 欢迎贡献代码或文档，需遵循 PR 提交规范。

5. **项目背景**  
由蒲公英开发者服务平台团队自主研发，基于 MIT 协议开源，开源版本称为 CodeFever Community，保留公有云版本 CodeFever 的核心功能并优化单机部署能力。
