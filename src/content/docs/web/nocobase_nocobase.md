### [nocobase nocobase](https://github.com/nocobase/nocobase)

**核心内容总结**  

**项目功能**  
NocoBase 是一款基于数据模型驱动的 AI 赋能无代码平台，支持高度扩展和自定义。通过分离数据结构与用户界面，用户可灵活构建复杂业务系统，无需编写代码。  

**主要特性**  
1. **数据模型驱动**：脱离表单/表格限制，支持多场景界面配置，兼容主数据库、外部数据库及第三方 API。  
2. **AI 能力集成**：嵌入 AI 功能（如翻译、分析、助手等），实现 AI 与人工协作，保障安全性与可定制性。  
3. **所见即所得**：一键切换使用模式与配置模式，页面布局类似 Notion，支持普通用户直观操作。  
4. **插件扩展**：所有功能以插件形式存在，支持自定义开发，类似 WordPress 的扩展机制。  

**使用方法**  
- **推荐**：通过 Docker 部署，升级只需替换镜像。  
- **低代码开发**：使用 `create-nocobase-app` CLI 工具。  
- **源码参与**：通过 Git 克隆源码，适合调试或贡献代码。  

**其他**  
官网、文档、社区及案例展示链接已提供，支持快速部署与业务场景适配。
