### [arhamkhnz next-shadcn-admin-dashboard](https://github.com/arhamkhnz/next-shadcn-admin-dashboard)

**项目核心内容总结：**

这是一个基于 Next.js 16、TypeScript 和 Shadcn UI 构建的管理后台模板，提供多仪表盘（默认、CRM、财务等）、认证界面、可定制主题（支持浅色/深色模式及多种配色方案）、灵活布局（可折叠侧边栏、内容宽度可调）等功能，支持响应式设计和移动端适配。  

**主要特性**：  
- 技术栈：Next.js 16（App Router）、Tailwind CSS v4、Shadcn UI、Zod、React Hook Form、TanStack Table 等。  
- 支持预设主题（如 Tangerine、Neo Brutalism）及自定义主题扩展。  
- 采用模块化文件结构，功能组件与逻辑集中管理，便于扩展。  
- 计划新增 RBAC 权限控制、多租户支持等功能。  

**使用方法**：  
1. **一键部署**：通过 [Vercel](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Farhamkhnz%2Fnext-shadcn-admin-dashboard) 直接部署。  
2. **本地运行**：  
   - 克隆仓库：`git clone https://github.com/arhamkhnz/next-shadcn-admin-dashboard.git`  
   - 安装依赖：`npm install`  
   - 启动服务：`npm run dev`  
   - 访问地址：http://localhost:3000  

**注意事项**：  
- 项目持续更新，使用旧版本需注意可能存在的重大变更。  
- 提供历史版本分支（如 Next.js 15、14 + Tailwind CSS v3），但仅维护最新版。
