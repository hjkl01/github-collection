### [filebrowser filebrowser](https://github.com/filebrowser/filebrowser)  ![GitHub Repo stars](https://img.shields.io/github/stars/filebrowser/filebrowser?style=social)

File Browser 是一款单二进制 Web 文件管理工具，属于“自建云”类软件。用户可将其安装于服务器并指定目录，通过 Web 界面实现对文件的上传、删除、预览和编辑。目前项目处于维护模式，优先处理故障、安全问题和 Bug 修复，不再由维护者开发新功能。
