### [tldraw tldraw](https://github.com/tldraw/tldraw)

**项目功能**：tldraw 是一个用于在 React 中创建无限画布体验的库，是数字白板 tldraw.com 的核心软件。  

**使用方法**：通过 `npm i tldraw` 安装，导入 `Tldraw` 组件并嵌入到 React 项目中，即可使用无限画布功能。  

**主要特性**：  
- 支持商业和非商业项目使用，需保留 "Made with tldraw" 水印（商业用途需购买许可证）。  
- 提供详细文档（[tldraw.dev](https://tldraw.dev)）和本地开发示例。  
- 支持社区贡献和开源协作，包含许可证、商标使用规范及贡献指南。
