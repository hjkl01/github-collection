### [zauberzeug nicegui](https://github.com/zauberzeug/nicegui)  ![GitHub Repo stars](https://img.shields.io/github/stars/zauberzeug/nicegui?style=social)

NiceGUI 是一款基于 Python 的 Web UI 框架，界面在浏览器中运行。它支持创建按钮、对话框、图表、3D 场景等组件，适用于微应用、仪表盘、机器人及智能家居项目。核心功能包括代码自动重载、实时数据刷新、双向数据绑定，支持 Web 服务或原生窗口模式。底层基于 FastAPI 后端与 Vue/Quasar 前端，通过 WebSocket 通信，支持 Jupyter Notebook 集成及自定义扩展。
