### [heyui heyui](https://github.com/heyui/heyui)

**项目功能**：HeyUI 是一个基于 Vue 的 UI 组件库，提供丰富的组件支持，适用于前端开发。  

**使用方法**：  
- **基础用法**：通过 CDN 引入 Vue 和 HeyUI 的脚本及 CSS 文件即可使用。  
- **高级用法**：使用 npm 安装后，通过 Vue 的 `import` 引入组件，并加载 Less 主题文件。  

**主要特性**：  
- 支持现代浏览器及 Internet Explorer 11+。  
- 提供在线文档（[heyui.top](http://v2.heyui.top)）和管理模板（[HeyUI Admin Template](http://admin.heyui.top)）。  
- 开源免费，采用 MIT 许可证。  
- 可通过 npm 安装并自定义构建。
