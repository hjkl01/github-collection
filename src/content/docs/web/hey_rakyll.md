### [rakyll hey](https://github.com/rakyll/hey)  ![GitHub Repo stars](https://img.shields.io/github/stars/rakyll/hey?style=social)

hey 是一款轻量级的 Web 应用负载测试工具。它支持发送指定并发数和数量的请求并输出统计信息，兼容 HTTP/2 协议。可自定义 HTTP 方法、请求头、请求体、认证、代理及速率限制，支持基于持续时间或固定数量的测试，并可将结果导出为 CSV。适用于 Linux、macOS 和 Windows 系统。
