### [gphper grm](https://github.com/gphper/grm)  ![GitHub Repo stars](https://img.shields.io/github/stars/gphper/grm?style=social)

该项目是一款基于 Go 和 Vue 的 Web 版 Redis 管理工具。主要功能包括：支持直连和 SSH 连接及数据库切换；支持 String、List、Set、ZSet、Hash、Stream 类型的增删查改；具备用户鉴权、操作日志、服务信息展示及 Redis 监控；支持 Lua 脚本和命令行操作。项目可编译为独立二进制文件，支持 Docker 部署及系统服务化运行。
