### [newpanjing simpleui](https://github.com/newpanjing/simpleui)  ![GitHub Repo stars](https://img.shields.io/github/stars/newpanjing/simpleui?style=social)

Simpleui 是一款基于 Element-UI 和 Vue 开发的 Django Admin 主题插件，旨在优化后台管理界面的美观性与易用性。其核心功能包括：内置 28 款流行主题，支持一键切换；完全兼容原生 Django Admin 界面，无需修改代码，配置简单；支持多标签页功能，使模块管理更清晰；支持 Django 5.3 及 Python 3.12 等最新版本；提供专业版以支持自定义组件、拖拽式首页及图表设计；兼容主流浏览器。
