### [playcanvas engine](https://github.com/playcanvas/engine)  ![GitHub Repo stars](https://img.shields.io/github/stars/playcanvas/engine?style=social)

PlayCanvas Engine 是一款开源游戏引擎，基于 HTML5、WebGL2 和 WebGPU 技术，支持在各类移动及桌面浏览器中运行游戏和交互式 3D 内容。其核心功能包括：高级 2D 与 3D 图形渲染、状态动画系统、基于 ammo.js 的 3D 物理引擎、多种输入接口支持（鼠标、键盘、触摸、手柄、VR 等）、3D 定位音效、基于 glTF 2.0 的资源流式加载，以及 TypeScript/JavaScript 脚本开发能力。该引擎适用于独立开发 HTML5 应用与游戏，也可配合 PlayCanvas 编辑器使用。
