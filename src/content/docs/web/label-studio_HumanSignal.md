### [HumanSignal label-studio](https://github.com/HumanSignal/label-studio)  ![GitHub Repo stars](https://img.shields.io/github/stars/HumanSignal/label-studio?style=social)

Label Studio 是一款开源数据标注工具，支持音频、文本、图像、视频和时间序列等多种数据类型的标注。它提供简洁直观的 UI 界面，帮助用户准备原始数据或优化训练数据以提升机器学习模型准确性。核心功能包括多用户协作标注、多项目管理、可配置标签格式、支持从本地或云存储导入数据。此外，支持集成机器学习模型进行预标注、在线学习和主动学习，并提供 REST API 便于嵌入现有数据流水线。支持多种安装与部署方式。
