### [HumanSignal label-studio](https://github.com/HumanSignal/label-studio)

**核心内容总结：**  
Label Studio 是一个开源的数据标注工具，支持音频、文本、图像、视频和时间序列等多种数据类型的标注，并可导出为多种模型格式。用户可通过 Docker、Docker Compose、pip、poetry、Anaconda 等方式安装使用，支持多用户协作、多项目管理，以及与机器学习模型的集成（如预标注、在线学习、主动学习）。主要特性包括：多用户标注、多项目管理、支持从云存储或文件导入数据、REST API 接口集成、前端库和数据管理库等生态系统。
