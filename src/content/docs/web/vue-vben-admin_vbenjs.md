### [vbenjs vue-vben-admin](https://github.com/vbenjs/vue-vben-admin)  ![GitHub Repo stars](https://img.shields.io/github/stars/vbenjs/vue-vben-admin?style=social)

Vue Vben Admin 是一款基于 Vue 3、Vite 和 TypeScript 开发的免费开源中后台管理前端模板。它提供开箱即用的中后台前端解决方案，核心功能包括多主题定制、完善的国际化支持及内置动态路由权限生成方案。适用于企业级后台管理系统开发及学习参考，兼容现代浏览器环境。
