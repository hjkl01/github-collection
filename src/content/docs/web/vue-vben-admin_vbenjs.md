### [vbenjs vue-vben-admin](https://github.com/vbenjs/vue-vben-admin)

**Vue Vben Admin 核心内容总结：**

**项目功能**  
一个基于 Vue3、Vite 和 TypeScript 的开源中后台管理模板，提供开箱即用的前后端解决方案，适用于学习和项目开发。

**主要特性**  
- 采用 Vue3、Vite 等主流技术栈  
- 支持 TypeScript 和多主题颜色定制  
- 内置国际化和动态路由权限控制  
- 提供中文预览站点及测试账号（vben/123456）  

**使用方法**  
1. 克隆项目：`git clone https://github.com/vbenjs/vue-vben-admin.git`  
2. 安装依赖：进入项目目录后执行 `pnpm install`  
3. 启动开发：`pnpm dev`  
4. 构建项目：`pnpm build`  

**许可证**

**其他**
