### [myide see](https://github.com/myide/see)  ![GitHub Repo stars](https://img.shields.io/github/stars/myide/see?style=social)

See SQL 审核平台是一个基于 Python 和 Vue.js 开发的 SQL 审核与管理工具，集成 Inception、SQLAdvisor 及 SOAR。主要功能包括：
1. **数据库管理**：支持多场地/数据中心数据库配置、集群归纳及库表结构查询。
2. **SQL 操作**：基于 Inception 实现 SQL 语法检测、执行、回滚、定时工单及历史管理。
3. **SQL 查询与优化**：支持表数据查询导出，利用 SQLAdvisor 和 SOAR 提供索引与语句优化建议。
4. **权限与审批**：基于 RBAC 的权限控制，支持灵活的人工审批流程（自动审核 + 经理审批/上线）。
5. **系统运维**：提供用户管理、个性化设置、API 文档、SSO 登录、邮件通知及 Dashboard 数据报表功能。
