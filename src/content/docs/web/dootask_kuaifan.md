### [kuaifan dootask](https://github.com/kuaifan/dootask)  ![GitHub Repo stars](https://img.shields.io/github/stars/kuaifan/dootask?style=social)

DooTask 是一款开源任务管理系统，支持基于 Docker 的一键部署、启停、更新、卸载及密码重置。提供端口修改、HTTPS 配置及数据库备份恢复功能以支持项目迁移与 HTTPS 安全。适用于 Linux/Unix 及 Windows 环境，并支持 NodeJs 开发构建。
