### [wooey wooey](https://github.com/wooey/wooey)

Wooey 是一个通过网页界面运行命令行 Python 脚本的工具，主要用于数据分析、文件处理等场景。其核心功能包括：  
1. **自动记录工作流程**：简化模型保存等操作的文档化；  
2. **降低使用门槛**：无需命令行经验即可运行 Python 脚本；  
3. **灵活封装程序**：用 Python 简化任何程序的包装，避免依赖特定工具（如 Galaxy）。  

**使用方法**：可通过项目[文档](http://wooey.readthedocs.org)安装配置，或直接访问[演示服务器](https://wooey.fly.dev)体验。  
**部署支持**：提供 PyPI 和 Docker 部署方式，便于集成到不同环境。
