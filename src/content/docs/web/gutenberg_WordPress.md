### [WordPress gutenberg](https://github.com/WordPress/gutenberg)  ![GitHub Repo stars](https://img.shields.io/github/stars/WordPress/gutenberg?style=social)

Gutenberg 是 WordPress 的区块编辑器项目，旨在革新网站构建与发布体验。该项目采用模块化设计，将段落、图片等内容封装为独立区块，支持用户通过拖拽方式直观创建富媒体页面，无需使用短代码或自定义 HTML。作为开源项目，Gutenberg 提供插件供用户测试最新特性，并支持开发者扩展定制及社区贡献。
