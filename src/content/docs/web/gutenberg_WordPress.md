### [WordPress gutenberg](https://github.com/WordPress/gutenberg)

**核心内容总结：**  
Gutenberg 是 WordPress 的新一代区块编辑器，采用模块化设计，将页面内容划分为独立的“区块”（如段落、图片、标题等），用户可通过拖拽自由排列，无需依赖短代码或 HTML。项目旨在革新 WordPress 的编辑与协作体验，包含实时协作、异步协作、多语言支持等功能。  

**使用方法：**  
1. **用户使用**：通过 WordPress 后台安装 Gutenberg 插件，或从 WordPress.org 下载；参考官方文档学习操作。  
2. **开发者扩展**：通过官方提供的快速入门指南和 API 文档开发插件或自定义区块。  
3. **贡献代码**：遵循贡献指南，提交代码或设计改进，参与开源社区协作。  

**主要特性：**  
- 模块化编辑：内容以独立区块形式管理，支持灵活布局。  
- 协作功能：支持实时与异步协作、多版本管理及发布流程优化。  
- 开源生态：提供完整开发文档，支持第三方扩展与自定义。
