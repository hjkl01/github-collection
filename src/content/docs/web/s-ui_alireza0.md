### [alireza0 s-ui](https://github.com/alireza0/s-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/alireza0/s-ui?style=social)

S-UI 是一款基于 SagerNet/Sing-Box 的高级 Web 管理面板，支持 Linux、Windows 及 macOS 系统。它具备多协议（VLESS, VMess, Trojan, Hysteria 等）管理能力，支持多客户端与入出站配置、流量统计及系统状态监控。此外，项目提供订阅链接生成（支持 link/json/clash）、高级流量路由配置、客户端限速与到期管理功能，并支持多语言、深色/浅色主题切换及 API 接口访问。
