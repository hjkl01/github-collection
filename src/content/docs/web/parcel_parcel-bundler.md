### [parcel-bundler parcel](https://github.com/parcel-bundler/parcel)  ![GitHub Repo stars](https://img.shields.io/github/stars/parcel-bundler/parcel?style=social)

Parcel 是一款面向 Web 的零配置构建工具。它开箱即用，支持 HTML、CSS、JavaScript 及多种资源文件的自动处理，内置带热重载功能的开发服务器。构建引擎基于 Rust 开发，利用多线程并行处理和缓存机制实现极速构建。项目可自动进行生产优化，包括代码分割、压缩及图片处理，并自动适配不同浏览器环境及 JSX/TypeScript 语法。此外，Parcel 具备强大的可扩展性，可通过配置和插件系统轻松支持任意规模的项目。
