### [parcel-bundler parcel](https://github.com/parcel-bundler/parcel)

**核心内容总结：**  
Parcel 是一个零配置的 Web 构建工具，支持 HTML、CSS、JavaScript 及各类资源文件（图片、字体等），提供开箱即用的开发体验和高性能构建能力。  

**主要特性：**  
1. **零配置**：无需手动配置，自动支持多种语言和文件类型，内置开发服务器、热更新和错误提示。  
2. **极速构建**：基于 Rust 的编译器，利用多核并行处理和缓存技术，构建速度极快。  
3. **自动优化**：生产环境自动执行代码压缩、图片优化、Tree-Shaking、代码分割等操作。  
4. **多环境适配**：支持现代及旧版浏览器，无需配置即可处理 JSX/TypeScript。  
5. **可扩展性**：基础功能无需配置，但可通过插件系统扩展以满足复杂项目需求。  

**使用方法：**  
通过官方文档提供的指南，可快速上手构建 Web 应用、库或迁移旧版本项目。文档地址：https://parceljs.org/docs/。
