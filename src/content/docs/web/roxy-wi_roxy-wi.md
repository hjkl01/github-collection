### [roxy-wi roxy-wi](https://github.com/roxy-wi/roxy-wi)  ![GitHub Repo stars](https://img.shields.io/github/stars/roxy-wi/roxy-wi?style=social)

Roxy-WI 是管理 HAProxy、Nginx、Apache 和 Keepalived 服务器的 Web 图形界面，提供告警、监控及安全功能。
- 服务管理：支持系统服务或 Docker 安装更新 HAProxy、Nginx、Apache、Keepalived 及 GeoIP 管理。
- 配置运维：动态调整连接数、黑白名单及后端节点，一键推送配置、同步主从服务器，支持配置备份与版本回滚。
- 监控分析：统一面板查看前后端状态与日志，收集连接指标，支持无需重启启停服务器。
- 安全权限：支持多用户角色、LDAP、SSH 密钥、SSL 及 WAF 防火墙，可针对不同角色隐藏配置。
- 告警通知：通过 Telegram、Slack、邮件等渠道发送服务状态及配置变更通知。
- 高可用扩展：保障主从高可用，提供 SMON 监控、Web 加速设置，适配移动端，支持 Linux 发行版及 SQLite/MySQL 数据库。
