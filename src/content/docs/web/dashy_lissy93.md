### [lissy93 dashy](https://github.com/lissy93/dashy)

**项目核心内容总结：**  
Dashy 是一款自托管的 Web 仪表板应用，用于整合和管理各类服务与应用，提供统一的界面展示链接、状态监控和操作入口。  

**主要功能：**  
- 支持多语言（含自定义本地化）、主题自定义（可修改样式和图标）、动态小部件（如状态指示器、搜索快捷键）。  
- 提供认证保护、备份恢复、应用状态监控、快捷键操作等功能。  
- 兼容多种部署环境（如 Docker、Nginx 等），支持本地开发和协作贡献。  

**使用方法：**  
1. **快速启动**：通过 Docker 或命令行一键部署。  
2. **配置管理**：通过 YAML 文件自定义仪表板布局、权限、主题等。  
3. **扩展功能**：集成第三方服务（如 Home Assistant、Portainer 等），或通过插件添加新功能。  

**主要特性：**  
- 轻量高效，支持高并发访问；  
- 模块化设计，可灵活扩展；  
- 提供完整文档（含部署、开发、主题设计等指南）及活跃社区支持；  
- 开源 MIT 许可证，允许商业使用与二次开发。  

**适用场景：**  
适合需要集中管理服务器、应用、工具的用户，尤其适合开发者、系统管理员及家庭用户搭建个人技术中心。
