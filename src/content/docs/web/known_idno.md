### [idno known](https://github.com/idno/known)  ![GitHub Repo stars](https://img.shields.io/github/stars/idno/known?style=social)

Idno 是一个开源社交发布平台，基于 PHP 8.2+ 和数据库构建。它支持 Webmention、内容聚合及 ActivityPub 协议，用于社交内容的发布与跨平台同步。系统采用异步事件队列机制处理后台任务（如消息推送、更新传播），确保请求快速响应且任务不丢失。支持通过 Composer、Git 或包管理器安装，并可通过 systemd 或 cron 服务管理后台事件处理 Worker。
