### [pywebio PyWebIO](https://github.com/pywebio/PyWebIO)  ![GitHub Repo stars](https://img.shields.io/github/stars/pywebio/PyWebIO?style=social)

PyWebIO 是一个 Python 库，支持通过脚本方式编写交互式 Web 应用，将浏览器变为“富文本终端”，无需 HTML 和 JS 知识即可构建简单 Web 应用。它采用同步输入输出方式，非声明式布局，侵入性低，可轻松将脚本转换为 Web 应用。支持集成到 Flask、Django、FastAPI 等 Web 服务，支持 asyncio 及数据可视化库，适合快速构建无需复杂 UI 的应用。
