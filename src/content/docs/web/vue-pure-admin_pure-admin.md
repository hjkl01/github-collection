### [pure-admin vue-pure-admin](https://github.com/pure-admin/vue-pure-admin)

**项目核心内容总结：**

1. **项目功能**  
   `vue-pure-admin` 是一个基于 Vue3、Vite、Element-Plus 等技术的开源中后台管理系统模板，提供 PC 端、移动端及暗色风格界面，支持非国际化和国际化两种精简版本，打包体积小于 2.3MB（启用 CDN 后低于 350KB）。

2. **主要特性**  
   - 采用 ESM 规范和最新技术栈（Vue3、TypeScript、Pinia、Tailwindcss 等）  
   - 提供精简版项目结构，适配实际开发需求  
   - 支持多端适配（PC、移动端）及暗色主题  
   - 提供配套文档、视频教程及 Tauri/Electron 版本  
   - 支持 Docker 部署  

3. **使用方法**  
   - 安装方式：通过 `@pureadmin/cli` 脚手架创建项目，或从 GitHub/Gitee 克隆代码  
   - 依赖安装：`pnpm install`  
   - 启动开发：`pnpm dev`  
   - 项目打包：`pnpm build`  
   - Docker 部署：通过 `docker build` 和 `docker run` 命令构建镜像并运行  

4. **其他支持**  
   - 提供中文和英文文档  
   - 支持社区贡献（Issue 提交、PR 提交规范）  
   - 开源免费，采用 MIT 许可证
