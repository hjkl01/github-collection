### [pure-admin vue-pure-admin](https://github.com/pure-admin/vue-pure-admin)  ![GitHub Repo stars](https://img.shields.io/github/stars/pure-admin/vue-pure-admin?style=social)

vue-pure-admin 是一款基于 Vue3、Vite、Element-Plus、TypeScript、Pinia 和 Tailwindcss 开发的开源免费中后台管理系统模板。支持国际化、暗色模式及 PC 与移动端适配，提供精简版、Electron 和 Tauri 桌面端版本。内置 CLI 脚手架，支持 Docker 部署，可用于快速构建企业级后台应用。
