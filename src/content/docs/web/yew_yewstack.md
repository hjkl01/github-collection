### [yewstack yew](https://github.com/yewstack/yew)  ![GitHub Repo stars](https://img.shields.io/github/stars/yewstack/yew?style=social)

Yew 是一个基于 Rust 和 WebAssembly 的现代化前端 Web 应用框架。它支持多线程开发，并提供类似 React JSX 的宏语法用于声明交互式 HTML。框架通过最小化 DOM 调用及支持后台 Web Worker 实现高性能，同时具备 JavaScript 互操作性，允许集成 NPM 包和现有 JavaScript 应用。
