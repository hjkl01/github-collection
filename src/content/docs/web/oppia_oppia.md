### [oppia oppia](https://github.com/oppia/oppia)  ![GitHub Repo stars](https://img.shields.io/github/stars/oppia/oppia?style=social)

Oppia 是一个在线学习工具，允许任何人创建和分享互动学习活动（称为“探索”）。这些活动模拟一对一的导师对话，让学生通过实践学习并获得反馈。项目已开发了基础数学的免费课程，并计划扩展到基础科学和金融知识，以帮助缺乏教育资源的学习者。

Oppia 的网页应用使用 Python、Angular 和 Google App Engine 构建。用户可通过官网 [Oppia.org](https://www.oppia.org) 访问，安装方法详见 [安装指南](https://github.com/oppia/oppia/wiki/Installing-Oppia)。社区欢迎所有贡献者参与开发，包括代码、设计、文档等。项目使用 Apache v2 开源协议，并提供邮件列表和 GitHub 讨论区用于交流与支持。
