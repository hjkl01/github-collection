### [lynx-family lynx](https://github.com/lynx-family/lynx)

**项目核心内容总结：**  
Lynx 是一个开源技术家族，帮助开发者使用现有的 Web 技能（如 CSS 和 React）从单一代码库构建跨平台的原生 UI，支持移动端（Android、iOS）和 Web 端，提供高性能和一致性体验。  

**主要特性：**  
- **跨平台渲染**：支持 Android、iOS 和 Web 的原生渲染，或通过自定义渲染器实现跨平台像素级一致性。  
- **Web 技术兼容**：基于 Web 技术栈设计，降低学习成本。  
- **高性能**：通过多线程引擎实现快速启动和流畅响应。  

**使用方法：**  
- 支持 iOS 10 及以上、Android 5.0（API 21）及以上版本。  
- 推荐使用 macOS 作为开发环境，Windows 和 Linux 可能存在兼容性问题。  
- 入门可通过官方文档的“Hello World”示例或集成到现有应用的指南开始。  

**其他信息：**  
- 项目核心引擎在此仓库，其他相关仓库可访问 [Lynx 家族组织页](https://github.com/lynx-family)。  
- 使用 Apache 2.0 许可证，依赖多个第三方库（如 BoringSSL、V8、React Native 等），并参考了 Chromium、Flutter 等项目的 API 设计。
