### [zhuima awesome-cloudflare](https://github.com/zhuima/awesome-cloudflare)  ![GitHub Repo stars](https://img.shields.io/github/stars/zhuima/awesome-cloudflare?style=social)

本项目是一个基于 Cloudflare 的开源工具收录仓库，旨在为独立开发者早期项目提供省心省时的工具集，帮助提升开发效率并降低成本。收录内容涵盖图床、邮箱、博客、短链、网站分析、隧道、加速、文件分享、监控、开发者工具及教程等，所有工具均基于 Cloudflare 生态系统（如 Workers、Pages、D1、R2 等），强调简单便捷、免费或低成本，支持自部署。
