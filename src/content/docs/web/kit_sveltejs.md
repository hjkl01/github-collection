### [sveltejs kit](https://github.com/sveltejs/kit)  ![GitHub Repo stars](https://img.shields.io/github/stars/sveltejs/kit?style=social)

SvelteKit 是一个简化的 Web 开发框架，核心包为 @sveltejs/kit。它提供多种部署适配器（如 adapter-node、adapter-vercel、adapter-cloudflare、adapter-netlify 等）以适配不同运行环境，并基于 Vite 构建。项目采用 MIT 开源协议。
