### [mui mui-x](https://github.com/mui/mui-x)  ![GitHub Repo stars](https://img.shields.io/github/stars/mui/mui-x?style=social)

MUI X 是一套针对复杂用例的高级 React UI 组件套件，专为数据丰富的应用设计，提供卓越的用户体验与开发体验。核心组件包括数据网格、日期和时间选择器、图表及树视图。该套件扩展了 Material UI 功能，可独立使用并支持完全自定义。项目采用开源核心模式，基础功能免费，高级功能（如多筛选、Excel 导出等）需商业授权。
