### [antiwork gumroad](https://github.com/antiwork/gumroad)  ![GitHub Repo stars](https://img.shields.io/github/stars/antiwork/gumroad?style=social)

Gumroad 是一个电子商务平台，允许创作者直接向消费者销售产品。该仓库包含其 Web 应用程序源代码，功能涵盖商品销售、支付处理、视频与 PDF 生成（含 Logo 及买家信息）、图片处理及搜索。项目基于 Ruby 和 Node.js 技术栈构建，支持通过 Docker 进行本地开发部署。
