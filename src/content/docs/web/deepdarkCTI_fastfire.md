### [fastfire deepdarkCTI](https://github.com/fastfire/deepdarkCTI)

**项目核心内容总结：**  
deepdarkCTI 是一个收集深网和暗网中网络威胁情报（CTI）来源的开源项目，旨在为安全分析提供来自 Telegram、Discord、勒索软件网站、数据泄露论坛、漏洞数据库等平台的威胁情报资源。  

**功能与特性：**  
1. **多平台监控**：覆盖 Telegram 渠道、Discord 群组、勒索软件网站、暗网论坛、数据市场等威胁情报来源。  
2. **社区协作**：提供 Telegram 群组供用户讨论情报分析方法、提交新情报源。  
3. **透明捐赠**：接受用户捐赠，资金用于项目资源建设。  
4. **知识支持**：包含 CTI 定义、情报类型（战略/战术/操作）及分析方法说明。  

**使用方式：**  
- 访问项目官网（[www.deepdarkcti.com](https://www.deepdarkcti.com)）或查看文档获取情报源列表。  
- 通过 Twitter（@fastfire）、Telegram（@fastfire83）或 Bsky（fastfire.bsky.social）申请加入 Telegram 群组参与讨论。  
- 通过 [Buy Me A Coffee](https://www.buymeacoffee.com/fastfire) 页面捐赠支持项目。
