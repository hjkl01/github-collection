### [mui material-ui](https://github.com/mui/material-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/mui/material-ui?style=social)

Material UI 是一个全面的 React 组件库，实现了 Google 的 Material Design 设计系统。它经过十余年开发，功能成熟稳定，被众多全球产品团队信赖。核心功能可通过 MUI X 套件扩展以支持高级用例。仓库内还包含实验性的 Joy UI 组件库，但建议新项目优先使用 Material UI。项目提供完整文档、示例、模板、贡献指南及路线图，采用 MIT 开源协议。
