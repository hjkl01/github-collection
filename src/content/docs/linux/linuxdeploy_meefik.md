### [meefik linuxdeploy](https://github.com/meefik/linuxdeploy)  ![GitHub Repo stars](https://img.shields.io/github/stars/meefik/linuxdeploy?style=social)

Linux Deploy 是一款开源工具，用于在 Android 设备上快速安装 GNU/Linux 操作系统。它支持通过磁盘镜像、目录、分区或内存挂载并安装多种 Linux 发行版（如 Debian、Ubuntu、Arch 等）。系统运行在 chroot 环境中，支持 ext2/3/4 文件系统及 ARM/x86 等架构，所有更改可逆。提供 CLI、SSH、VNC 等控制接口及多种桌面环境，用户可通过界面管理安装进程、服务及配置，推荐使用 root 权限运行。
