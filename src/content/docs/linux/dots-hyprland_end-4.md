### [end-4 dots-hyprland](https://github.com/end-4/dots-hyprland)  ![GitHub Repo stars](https://img.shields.io/github/stars/end-4/dots-hyprland?style=social)

该项目是一套 Hyprland 窗口管理器配置文件，核心采用 Quickshell 构建 illogical-impulse 界面风格。具备应用概览功能，支持显示实时预览、搜索、计算和运行命令；集成 Gemini API 及 Ollama 模型；根据壁纸自动生成 Material 风格配色；安装过程透明，执行前展示所有命令；默认键位适配 Windows 或 GNOME 用户习惯。提供一键安装脚本。另含部分基于 AGS 和 EWW 的旧版本配置，但已停止维护。
