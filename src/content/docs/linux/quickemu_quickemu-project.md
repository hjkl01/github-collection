### [quickemu-project quickemu](https://github.com/quickemu-project/quickemu)  ![GitHub Repo stars](https://img.shields.io/github/stars/quickemu-project/quickemu?style=social)

Quickemu 是一个基于 QEMU 的虚拟机管理工具，旨在简化 Windows、macOS 和 Linux 等操作系统的虚拟机创建与运行。通过 `quickget` 自动下载系统镜像并生成配置，利用 `quickemu` 根据宿主机硬件自动匹配最优配置启动虚拟机。支持 Linux 和 macOS 宿主环境，提供剪贴板共享、文件共享、硬件直通、音频及网络转发等优化功能，无需繁琐配置即可运行近 1000 种操作系统。
