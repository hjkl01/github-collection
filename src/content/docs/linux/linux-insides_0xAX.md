### [0xAX linux-insides](https://github.com/0xAX/linux-insides)  ![GitHub Repo stars](https://img.shields.io/github/stars/0xAX/linux-insides?style=social)

该项目是一部关于 Linux 内核内部机制的开源教程项目，旨在分享内核底层及相关低级主题的知识。内容涵盖启动、初始化、中断、系统调用、内存管理等核心章节，需具备汇编语言和 C 语言基础。项目正持续更新以适配现代内核版本，并提供多语言社区翻译，采用 CC BY-NC-SA 协议开源。
