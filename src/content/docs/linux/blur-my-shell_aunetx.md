### [aunetx blur-my-shell](https://github.com/aunetx/blur-my-shell)  ![GitHub Repo stars](https://img.shields.io/github/stars/aunetx/blur-my-shell?style=social)

Blur my Shell 是一款 GNOME Shell 扩展，旨在为系统界面组件添加模糊视觉效果。

- **支持区域**：覆盖顶部面板、概览界面、Dash to Dock、应用文件夹背景、锁屏、截图窗口选择器及应用窗口。
- **模糊模式**：提供静态模糊（基于壁纸图像，支持自定义效果流水线如高斯模糊、圆角）和动态模糊（直接模糊背景内容，仅支持高斯模糊）。
- **自定义设置**：可调整模糊区域颜色与透明度，设置模糊激活/禁用条件（如全屏或进入概览时），并支持应用窗口模糊的白名单或黑名单模式。
- **兼容性**：兼容 Dash to Dock、Dash to Panel、Window List 等主流扩展，支持 GNOME Shell 3.36 至 46 版本。
