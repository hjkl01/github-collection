### [aisuhua restful-api-design-references](https://github.com/aisuhua/restful-api-design-references)  ![GitHub Repo stars](https://img.shields.io/github/stars/aisuhua/restful-api-design-references?style=social)

本项目是一个 RESTful API 设计参考资料集合，旨在汇聚优质资源辅助开发者进行 API 设计与实现。主要内容如下：

1. **设计原则与规范**：整理 RESTful 设计最佳实践、HTTP 协议指南及安全规范。
2. **基础知识**：包含 HTTP 状态码、幂等性、RPC、JSON 标准等概念解析。
3. **书籍推荐**：收录 RESTful Web APIs、REST CookBook 等系统性教材。
4. **案例参考**：列举 GitHub、Mailgun、Coinbase 等真实 API 设计实例。
5. **工具链推荐**：提供调试工具、文档生成工具及代码高亮插件。
6. **社区与拓展**：链接技术社区及 Markdown 编辑等开发辅助工具。
