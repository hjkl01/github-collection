### [cheahjs free-llm-api-resources](https://github.com/cheahjs/free-llm-api-resources)  ![GitHub Repo stars](https://img.shields.io/github/stars/cheahjs/free-llm-api-resources?style=social)

该项目整理了一个免费大语言模型（LLM）API 资源列表，收录了多种提供免费访问额度或试用积分的服务商。内容划分为“免费提供商”和“试用积分提供商”两类，涵盖 OpenRouter、Google AI Studio、Groq 等主流平台。每个条目详细列出了服务商的 API 使用限制、支持的模型列表及验证要求，旨在帮助用户发现和利用免费或低成本的 LLM 接口。
