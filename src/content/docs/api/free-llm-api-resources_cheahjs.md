**核心内容总结：**  
该文档汇总了多个AI模型和云服务提供商的模型使用限制及试用资源，主要功能包括：  

1. **模型与平台信息**  
   - 详细列出主流平台（如Google Cloud、Fireworks、Baseten等）的模型名称、调用限制（如请求频率、免费额度）及适用场景。  
   - 包括Meta、DeepSeek、Qwen等模型在不同平台的版本和限制条件。  

2. **试用资源与优惠**  
   - 提供多个平台的试用信用额度（如Fireworks $1、Baseten $30、Hyperbolic $1等），部分平台支持免费额度（如Scaleway 100万免费tokens）。  
   - 部分平台需完成验证（如NLP Cloud需手机号验证）或填写问卷获取资源。  

3. **使用方法**  
   - 用户可根据需求选择平台，通过官网申请试用额度或直接调用模型，按平台指引进行模型部署或API调用。  

**主要特性**  
- 覆盖主流模型（Llama、Qwen、DeepSeek等）及云服务（Google Cloud、Modal、Hyperbolic等）。  
- 提供灵活的试用方式（信用额度、免费tokens），部分平台支持按需付费。  
- 明确标注模型限制（如请求频率、免费期），便于用户评估适用性。
