### [fullstorydev grpcurl](https://github.com/fullstorydev/grpcurl)  ![GitHub Repo stars](https://img.shields.io/github/stars/fullstorydev/grpcurl?style=social)

grpcurl 是一款类似于 curl 的命令行工具，用于与 gRPC 服务器交互。它支持通过命令行调用各类 RPC 方法（包括交互式双向流），接受 JSON 格式输入并转换为二进制 Protobuf 发送。工具可通过服务器反射、Proto 源码或 Protoset 文件来浏览、列出及描述服务架构。此外，支持 TLS 及明文通信（含双向认证），并提供 Go 库供二次开发。
