### [swagger-api swagger-ui](https://github.com/swagger-api/swagger-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/swagger-api/swagger-ui?style=social)

Swagger UI 是一个根据 OpenAPI 规范自动生成 API 文档的工具，旨在让开发团队及最终用户无需实现逻辑即可可视化并交互 API 资源，简化后端实现与客户端消费。项目提供三种 npm 模块（标准版、依赖独立版、React 组件版）以适应不同构建环境，兼容 OpenAPI 1.x 至 3.x 多个版本规范，并支持配置、插件定制、OAuth2 授权及深度链接等功能。
