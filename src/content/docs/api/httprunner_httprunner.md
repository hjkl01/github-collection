### [httprunner httprunner](https://github.com/httprunner/httprunner)  ![GitHub Repo stars](https://img.shields.io/github/stars/httprunner/httprunner?style=social)

HttpRunner 是一款基于 Golang 的开源通用智能自动化测试框架。v5 版本融入大模型技术，支持 API 测试（HTTP/HTTP2/WebSocket/RPC）、性能测试及多端 UI 自动化（Android/iOS/Harmony/Browser），其中 UI 自动化采用纯视觉驱动方案（OCR/CV/VLM）。项目提供 `hrp` 命令行工具，支持 YAML/JSON/GoTest 等多种用例格式，具备插件系统、分布式测试支持，并兼容 CI/CD 流程，适用于跨平台测试场景。
