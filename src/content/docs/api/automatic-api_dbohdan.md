### [dbohdan automatic-api](https://github.com/dbohdan/automatic-api)

该README文档列出了多个自动生成REST/GraphQL API的开源项目，核心功能是通过数据库schema自动创建API接口，简化后端开发。所有项目均支持Docker部署，主要特性包括：  
1. **数据库支持**：涵盖PostgreSQL、MySQL、SQLite、MongoDB、Redis等主流数据库；  
2. **API类型**：提供REST或GraphQL接口，部分项目（如Supabase）同时支持两者；  
3. **技术实现**：基于多种编程语言（如Python、TypeScript、Go、Java等）开发，许可证类型多样（MIT、Apache、AGPL等）；  
4. **附加功能**：部分项目具备实时同步（Supabase）、多数据库集成（WunderGraph）、读写分离（subZero）等特性。  
使用方法通常为安装Docker镜像并配置数据库连接，具体实现细节需参考各项目文档。
