### [dbohdan automatic-api](https://github.com/dbohdan/automatic-api)  ![GitHub Repo stars](https://img.shields.io/github/stars/dbohdan/automatic-api?style=social)

本项目整理了一个工具列表，旨在自动为数据库暴露 REST、GraphQL 或其他类型的 API，同时收录了内置 HTTP API 的数据库。列表详细提供了各项目的支持数据库、API 类型、实现语言、许可证及 GitHub 统计等信息。项目欢迎社区贡献，文档及数据遵循 CC BY 4.0 许可协议。
