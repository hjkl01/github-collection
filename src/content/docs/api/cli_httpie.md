### [httpie cli](https://github.com/httpie/cli)

HTTPie 是一个命令行 HTTP 客户端，旨在以人性化的方式实现与 Web 服务的交互。其主要功能包括发送 HTTP 请求、调试 API、支持 JSON 数据、文件上传、自定义请求头、HTTPS 认证等。用户可通过 `http` 和 `https` 命令快速构建请求，支持离线模式生成请求内容。主要特性包括语法简洁、输出格式化与颜色化、支持代理、持久会话及 `wget` 风格下载。使用示例涵盖基本请求、自定义方法/头、JSON 数据传输及 GitHub API 认证等场景。项目提供文档、社区支持（Discord、Twitter）及贡献指南。
