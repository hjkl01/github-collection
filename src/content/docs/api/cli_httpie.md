### [httpie cli](https://github.com/httpie/cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/httpie/cli?style=social)

HTTPie CLI 是一款人类友好的命令行 HTTP 客户端，专为 API 测试、调试及与 HTTP 服务器交互而设计。它允许通过直观简洁的语法创建和发送任意 HTTP 请求，并提供格式化且带颜色的终端输出。主要功能包括内置 JSON 支持、表单与文件上传、HTTPS/代理/认证、自定义请求头、持久会话以及类 wget 的下载功能。
