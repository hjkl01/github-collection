### [usebruno bruno](https://github.com/usebruno/bruno)

**项目核心内容总结：**

**功能**  
Bruno 是一款开源的 API 探索与测试工具，支持本地文件系统存储 API 请求集合，使用自研的 Bru 标记语言记录数据，不依赖云端同步，强调数据隐私。

**主要特性**  
- 跨平台运行（Mac/Windows/Linux）  
- 支持 Git 等版本控制系统协作  
- 本地离线存储，无云同步计划  
- 开源免费（MIT 许可证），部分功能提供付费版本  

**使用方法**  
- 通过官网下载二进制文件或使用包管理器安装（如 Homebrew、Chocolatey、Snap 等）  
- 安装后可直接运行，通过 Git 管理 API 集合  

**其他**  
- 提供多语言文档及社区支持（GitHub、Discord、Twitter 等）  
- 支持用户贡献代码及反馈问题
