### [usebruno bruno](https://github.com/usebruno/bruno)  ![GitHub Repo stars](https://img.shields.io/github/stars/usebruno/bruno?style=social)

Bruno 是一款开源的 API 客户端 IDE，用于探索和测试 API，旨在革新 Postman 等工具现状。它支持将 API 集合直接存储在本地文件系统文件夹中，采用纯文本格式（Bru），便于通过 Git 或其他版本控制系统进行协作。Bruno 坚持完全离线使用，不提供云端同步，重视用户数据隐私。该工具兼容 macOS、Windows 和 Linux 平台，并提供开源版与商业版。
