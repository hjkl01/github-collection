### [hasura graphql-engine](https://github.com/hasura/graphql-engine)  ![GitHub Repo stars](https://img.shields.io/github/stars/hasura/graphql-engine?style=social)

Hasura GraphQL Engine 是一款开源 GraphQL 引擎，通过单一、可组合且安全的 API 端点提供数据访问，助力现代应用程序的构建。它支持 PostgreSQL、MongoDB、ClickHouse 及 MS SQL Server 等多种数据源，并提供完全开源的数据连接器。

项目包含 V2 和 V3 两个版本：
- V2：当前稳定版本。
- V3：支持使用 TypeScript、Python 和 Go SDK 编写自定义业务逻辑。

核心引擎与连接器主要采用 Apache License 2.0 协议开源。
