### [jazzband djangorestframework-simplejwt](https://github.com/jazzband/djangorestframework-simplejwt)

**核心内容总结：**  
Simple JWT 是 Django REST Framework 的 JSON Web Token（JWT）认证插件，用于实现基于令牌的用户认证。其主要特性包括：  
1. **功能**：提供 JWT 生成、验证及刷新机制，支持 Django REST Framework 的认证流程。  
2. **兼容性**：兼容多种 Python 版本（如 3.8+）和 Django 版本（如 3.2+）。  
3. **文档与扩展**：提供详细中文文档（通过 Read the Docs），支持社区翻译贡献（通过 PR 或 inlang 平台）。  

**使用方法**：需通过 pip 安装（`pip install djangorestframework-simplejwt`），并在 Django 项目中配置 JWT 认证类及刷新令牌逻辑（具体实现需参考文档）。
