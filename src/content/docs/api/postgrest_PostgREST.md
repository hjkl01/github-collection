### [PostgREST postgrest](https://github.com/PostgREST/postgrest)  ![GitHub Repo stars](https://img.shields.io/github/stars/PostgREST/postgrest?style=social)

PostgREST 是一个为现有 PostgreSQL 数据库自动生成完整 RESTful API 的工具。它采用 Haskell 编写以确保高性能，支持 JWT 认证并将权限委托给数据库角色。项目通过数据库模式管理 API 版本，自动生成符合 OpenAPI 标准的文档，并强制数据库约束以保障数据完整性，提供比自定义代码更清洁、标准更合规的接口服务。
