### [polarsource polar](https://github.com/polarsource/polar)  ![GitHub Repo stars](https://img.shields.io/github/stars/polarsource/polar?style=social)

Polar 是一个面向开发者的开源支付基础设施与变现平台。核心功能包括快速销售 SaaS 及数字产品（如 GitHub 访问、Discord 支持、许可证密钥等），并支持订阅模式。平台作为记录商户（Merchant of Record），负责处理账单、收据、客户账户及销售税、VAT 等税务合规问题。提供公共 API、Webhook 接口及 JavaScript、Python SDK 供集成，采用按交易收费（4% + 40¢）模式，无月费。
