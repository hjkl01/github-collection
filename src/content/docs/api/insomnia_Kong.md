### [Kong insomnia](https://github.com/Kong/insomnia)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kong/insomnia?style=social)

Insomnia 是一款开源、跨平台的 API 客户端，支持 GraphQL、REST、WebSockets、SSE、gRPC 及 HTTP 兼容协议。主要功能包括 API 调试、设计（原生 OpenAPI 编辑器）、测试（测试套件与集合运行器）、模拟（云或自托管）、CI/CD 集成及团队协作。支持 Mac、Windows 和 Linux 系统，提供本地 Vault、Git 同步和云同步等多种存储选项，并支持第三方插件扩展。
