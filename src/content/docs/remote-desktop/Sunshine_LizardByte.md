### [LizardByte Sunshine](https://github.com/LizardByte/Sunshine)  ![GitHub Repo stars](https://img.shields.io/github/stars/LizardByte/Sunshine?style=social)

Sunshine 是一款自托管的游戏串流服务端，专为 Moonlight 客户端设计。它提供低延迟的云游戏功能，支持 AMD、Intel 和 Nvidia 显卡的硬件编码及软件编码。项目支持通过 Web 界面进行配置和客户端配对，兼容 Windows、Linux、macOS 和 FreeBSD 系统，并提供多种手柄模拟及屏幕捕获方案。
