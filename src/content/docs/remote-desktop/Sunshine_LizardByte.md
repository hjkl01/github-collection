### [LizardByte Sunshine](https://github.com/LizardByte/Sunshine)

**项目核心内容总结：**

**功能**  
Sunshine 是一款支持多平台（Windows/Linux/FreeBSD/macOS）的远程游戏串流软件，通过硬件编码加速（AMD VCE/Intel VAAPI/NVIDIA NVENC）实现低延迟游戏画面传输，支持虚拟游戏手柄映射和Web界面管理。

**使用方法**  
需在主机端安装并配置软件，通过局域网或互联网连接客户端设备（如PC、手机），利用Web界面添加设备并管理串流参数。

**主要特性**  
1. 多平台兼容：支持Windows 11+、Linux主流发行版、FreeBSD 14.3+、macOS 14+  
2. 硬件加速：适配AMD/Intel/NVIDIA显卡的视频编码技术  
3. 虚拟手柄：支持将物理手柄映射为虚拟设备  
4. 网络优化：建议使用5GHz Wi-Fi或千兆以太网连接  
5. 分辨率支持：4K及HDR模式需更高配置（如RTX 2000系列显卡、i5及以上CPU）

**系统要求**  
- 最低配置：Ryzen 3/i3 CPU、4GB内存、支持硬件编码的显卡  
- 4K建议：RTX 2000系列/NVIDIA GTX 1080及以上显卡、i5 CPU  
- HDR建议：Pascal架构显卡（GTX 10系列）或更高、i5 CPU
