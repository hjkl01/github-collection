### [m1k1o neko](https://github.com/m1k1o/neko)  ![GitHub Repo stars](https://img.shields.io/github/stars/m1k1o/neko?style=social)

Neko 是一款基于 Docker 和 WebRTC 技术的自托管虚拟浏览器与桌面工具。它能在隔离的 Linux 容器环境中运行浏览器或应用程序，并通过 WebRTC 实时流传输，支持多用户同时访问与协同控制。核心功能包括安全的隐私浏览、多人实时协作（如观战派对、互动演示、代码调试）、内置音频支持及 API 嵌入能力。相比传统远程桌面方案，Neko 提供多参与者同步控制及更流畅的视频传输体验，支持多种主流浏览器和桌面环境，适用于团队协作、自动化测试及个人隐私工作空间等场景。
