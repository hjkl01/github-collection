### [rustdesk rustdesk](https://github.com/rustdesk/rustdesk)  ![GitHub Repo stars](https://img.shields.io/github/stars/rustdesk/rustdesk?style=social)

RustDesk 是一款使用 Rust 语言编写的开源远程桌面解决方案。它无需配置即可开箱即用，确保用户对数据拥有完全控制权，注重安全性。用户可选择使用官方中继服务器、自建 Rendezvous/Relay 服务器或自定义服务器。功能涵盖屏幕共享、文件传输、剪贴板同步、音频传输、输入控制及 TCP 隧道等。界面基于 Flutter 或 Sciter 构建，支持 Windows、Linux 和 macOS 等操作系统。
