### [screego server](https://github.com/screego/server)  ![GitHub Repo stars](https://img.shields.io/github/stars/screego/server?style=social)

Screego 是一款专为开发者设计的屏幕共享工具，提供高质量、低延迟的屏幕分享功能，旨在解决传统通讯软件中流媒体卡顿或画质差的问题。其核心功能包括多用户屏幕共享、基于 WebRTC 的安全传输、低延迟高分辨率输出，支持通过 Docker 或单二进制文件简便部署，并内置 TURN 服务器以辅助 NAT 穿透。Screego 作为现有协作软件的补充，专注于提供纯粹的屏幕共享服务。
