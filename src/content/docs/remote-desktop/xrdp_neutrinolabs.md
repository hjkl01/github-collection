### [neutrinolabs xrdp](https://github.com/neutrinolabs/xrdp)  ![GitHub Repo stars](https://img.shields.io/github/stars/neutrinolabs/xrdp?style=social)

xrdp 是一款开源的 RDP 服务器，允许用户通过 Microsoft 远程桌面协议（RDP）远程访问 Linux 图形界面。它支持多种 RDP 客户端连接，默认使用 TLS 加密传输。主要功能包括远程桌面访问（支持会话重连、窗口调整、RDP/VNC 代理）、双向剪贴板传输（文本、图片、文件）、音频与麦克风重定向、本地驱动器映射。项目主要面向 GNU/Linux 平台，支持 x86 及 ARM 架构。
