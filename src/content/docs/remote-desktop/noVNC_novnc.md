### [novnc noVNC](https://github.com/novnc/noVNC)  ![GitHub Repo stars](https://img.shields.io/github/stars/novnc/noVNC?style=social)

noVNC 是一个基于 HTML 的 VNC 客户端 JavaScript 库及配套应用程序，支持在现代浏览器（包括 iOS 和 Android 移动端）中运行。它遵循标准 VNC 协议并依赖 WebSocket 连接，支持多种认证方式（如 Classical VNC、RSA-AES、Tight 等）及多种 VNC 编码格式（如 raw、tight、ZRLE 等）。主要功能包括桌面缩放、调整大小、前后鼠标键支持、本地光标渲染、全 Unicode 剪贴板复制/粘贴、触摸手势模拟及多语言翻译。项目可单独部署（支持 snap 包）、通过 websockify 作为代理使用，也可嵌入到其他软件中集成。
