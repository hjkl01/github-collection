### [louislam uptime-kuma](https://github.com/louislam/uptime-kuma)  ![GitHub Repo stars](https://img.shields.io/github/stars/louislam/uptime-kuma?style=social)

Uptime Kuma 是一款易于使用的自托管监控工具。支持 HTTP(s)、TCP、Ping、DNS、Websocket、Steam 游戏服务器及 Docker 容器等多种监控类型，监测间隔为 20 秒。支持 Telegram、Discord、Email 等 90 多种通知服务。界面美观且响应迅速，支持多语言、状态页面、Ping 图表、证书信息、代理及 2FA 双重认证。支持 Docker 和 Node.js 环境部署。
