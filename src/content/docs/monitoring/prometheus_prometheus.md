### [prometheus prometheus](https://github.com/prometheus/prometheus)  ![GitHub Repo stars](https://img.shields.io/github/stars/prometheus/prometheus?style=social)

Prometheus 是云原生计算基金会（CNCF）旗下的系统和监控服务系统。它通过 HTTP 拉取模型定期采集目标指标，基于多维时间序列模型存储数据，并提供强大的 PromQL 查询语言。核心功能包括规则评估、结果展示、告警触发、服务发现及图表仪表盘支持。系统无需依赖分布式存储，单节点即可自治，支持分级和水平联邦。提供二进制、Docker 镜像及源码等多种构建和部署方式。
