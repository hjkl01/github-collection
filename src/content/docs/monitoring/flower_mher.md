### [mher flower](https://github.com/mher/flower)  ![GitHub Repo stars](https://img.shields.io/github/stars/mher/flower?style=social)

Flower 是一款开源 Web 应用，用于监控和管理 Celery 集群。它提供工作进程和任务的实时信息，支持实时监控任务进度与详情、远程控制工作进程（含重启/终止/自动伸缩/队列管理）、Broker 队列统计。此外，项目支持多种身份认证、Prometheus 集成及 REST API，可通过 pip 或 Docker 安装。
