### [ClementTsang bottom](https://github.com/ClementTsang/bottom)  ![GitHub Repo stars](https://img.shields.io/github/stars/ClementTsang/bottom?style=social)

bottom (btm) 是一款跨平台的命令行图形化进程与系统监控工具，支持 Linux、macOS 和 Windows 系统。

核心功能包括：
1. **系统资源可视化**：提供 CPU、内存、网络 I/O、磁盘、温度传感器及电池使用的实时图表展示，支持时间区间缩放。
2. **进程管理**：支持进程的显示、排序、搜索，以及发送终止信号（杀进程）和树状模式展示。
3. **高度可定制**：可通过命令行或配置文件自定义配色主题、组件布局、行为及过滤规则。
4. **便捷使用**：支持多种安装方式，提供类 htop 的基础模式与组件展开模式，并集成多种 Shell 自动补全。
