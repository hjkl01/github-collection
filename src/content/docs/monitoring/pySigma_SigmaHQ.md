### [SigmaHQ pySigma](https://github.com/SigmaHQ/pySigma)  ![GitHub Repo stars](https://img.shields.io/github/stars/SigmaHQ/pySigma?style=social)

pySigma 是一个 Python 库，用于解析和将 Sigma 安全规则转换为查询语句，旨在替代旧版 sigmac 工具链。它将后端（查询语言适配）和流水线（日志数据模型处理）分离为独立项目，以保持核心库简洁且与厂商无关。该项目支持自定义后端开发、规则修饰符处理，并提供配套命令行工具 sigma-cli 用于规则转换。
