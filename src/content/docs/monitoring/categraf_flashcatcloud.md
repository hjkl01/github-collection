### [flashcatcloud categraf](https://github.com/flashcatcloud/categraf)  ![GitHub Repo stars](https://img.shields.io/github/stars/flashcatcloud/categraf?style=social)

Categraf 是一款全栈式数据采集器，支持指标和日志采集。内置近百种插件，支持物理机、虚拟机、容器、K8s 及多种中间件/数据库的数据采集。采用 PUSH 方式和 remote write 协议推送数据，支持对接 Nightingale、Prometheus、VictoriaMetrics 等后端。支持混合云、云原生及多云架构，可无缝代替 Prometheus 抓取数据。项目力求开箱即用，配套推荐仪表盘、告警规则及 OnCall 值班平台。
