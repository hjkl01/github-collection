### [lebinh ngxtop](https://github.com/lebinh/ngxtop)  ![GitHub Repo stars](https://img.shields.io/github/stars/lebinh/ngxtop?style=social)

ngxtop 是一个用于实时解析 Web 服务器（主要是 Nginx）访问日志的命令行工具，输出类似 top 命令的统计指标以进行即时监控。它支持多种日志格式（Nginx、Apache、Caddy），允许自定义日志路径、分组维度（如路径、IP、Host）及输出限制。功能包括状态码过滤、排序、聚合计算（如平均值、总和）等，适用于短期故障排查和实时监控，不适用于长期数据存储系统。
