### [cppla ServerStatus](https://github.com/cppla/ServerStatus)  ![GitHub Repo stars](https://img.shields.io/github/stars/cppla/ServerStatus?style=social)

ServerStatus 中文版是一款多服务器云监控探针工具。支持 Docker 等多种快速部署方式，提供可视化网页监控面板。核心功能包括：实时监控服务器资源状态（CPU、内存、磁盘、网络流量）；HTTP/HTTPS 站点存活监控与 SSL 证书检查；多线路网络 Ping 探测；支持自定义规则的 Watchdog 触发式告警，可对接 Telegram、Server 酱等多种通知渠道。
