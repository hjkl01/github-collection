### [pawelmalak flame](https://github.com/pawelmalak/flame)  ![GitHub Repo stars](https://img.shields.io/github/stars/pawelmalak/flame?style=social)

Flame 是一款自托管的服务端起始页，设计灵感源自 SUI，通过内置 GUI 编辑器允许用户快速搭建应用中心，无需手动编辑文件。核心功能包括：支持图形化管理应用和书签（创建、更新、删除及置顶）；集成带有本地过滤和多搜索引擎的搜索栏；提供认证系统保护隐私；支持自定义 CSS、15 种内置主题、主题构建器及天气组件；具备 Docker 集成能力（根据容器标签自动发现应用）和 Kubernetes 集成能力（根据 Ingress 注解自动发现应用）。
