### [giampaolo psutil](https://github.com/giampaolo/psutil)  ![GitHub Repo stars](https://img.shields.io/github/stars/giampaolo/psutil?style=social)

psutil 是一个跨平台的 Python 库，用于获取正在运行的进程信息及系统资源使用情况（包括 CPU、内存、磁盘、网络和传感器等）。它主要用于系统监控、性能分析、进程资源限制及管理。其功能类似经典 UNIX 命令行工具（如 ps、top、netstat 等），支持 Linux、Windows、macOS 及多种 BSD 和 Unix 变体平台，提供进程详情查询、状态控制及系统硬件指标监控功能。
