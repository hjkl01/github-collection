### [hypertrace hypertrace](https://github.com/hypertrace/hypertrace)  ![GitHub Repo stars](https://img.shields.io/github/stars/hypertrace/hypertrace?style=social)

Hypertrace 是一个云原生的开源分布式追踪与可观测性平台，旨在为分布式系统提供开发和生产环境的可见性。它支持将追踪数据转化为洞察，功能涵盖根因分析、性能瓶颈定位（如慢 API 或数据库查询）、微服务依赖监控、版本发布对比及故障诊断。平台兼容 Zipkin 或 Jaeger 数据源，支持通过 Docker Compose 快速启动，并可利用 Kubernetes Helm 图表在生产环境部署。
