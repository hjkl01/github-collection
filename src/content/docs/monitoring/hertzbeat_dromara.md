### [dromara hertzbeat](https://github.com/dromara/hertzbeat)

**项目核心内容总结：**

**功能**  
HertzBeat 是一个开源监控系统，支持应用性能管理（APM）、主机监控、网络监控、服务监控、日志监控及链路追踪，提供全面的运维观测能力。

**使用方法**  
1. 下载源码并构建 Docker 镜像；  
2. 启动服务后通过 Web 界面访问；  
3. 配置监控项（如主机、服务、日志等）以开始数据采集与分析。

**主要特性**  
- **多维数据采集**：支持多种监控类型及多语言探针；  
- **可视化展示**：提供实时指标、拓扑图、日志分析等可视化能力；  
- **告警通知**：支持多渠道告警（如邮件、企业微信）；  
- **分布式追踪**：集成链路追踪技术，支持微服务调用链分析；  
- **高可用架构**：基于微服务设计，支持水平扩展；  
- **多租户管理**：支持多团队/用户隔离；  
- **数据安全**：采用加密存储与传输，保障数据隐私。  

**其他**  
项目已纳入 CNCF 观测分析领域图谱，采用 Apache 2.0 许可证开源。
