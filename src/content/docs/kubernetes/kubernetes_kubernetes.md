### [kubernetes kubernetes](https://github.com/kubernetes/kubernetes)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubernetes/kubernetes?style=social)

Kubernetes (K8s) 是一个开源系统，用于跨多个主机管理容器化应用程序，提供应用部署、维护和扩展的基本机制。该项目基于 Google 的 Borg 系统经验，由云原生计算基金会 (CNCF) 托管。项目提供官方文档、源码构建指南（支持 Go 或 Docker 环境），并具备完善的社区支持、治理框架及发展路线图。
