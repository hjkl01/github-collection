### [kubernetes minikube](https://github.com/kubernetes/minikube)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubernetes/minikube?style=social)

minikube 是在 macOS、Linux 和 Windows 上实现本地 Kubernetes 集群的工具，旨在支持本地 Kubernetes 应用开发。它具备标准 Kubernetes 功能（如 LoadBalancer、多集群、NodePorts、持久卷、Dashboard、容器运行时配置及 CI 环境支持），同时提供开发者友好特性（如 Addons 市场、NVIDIA/AMD GPU 支持、文件系统挂载）。项目还支持 GitHub Codespace 和 Dev Container 运行环境。
