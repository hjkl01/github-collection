### [helm helm](https://github.com/helm/helm)  ![GitHub Repo stars](https://img.shields.io/github/stars/helm/helm?style=social)

Helm 是用于管理 Charts（预配置 Kubernetes 资源包）的软件包管理工具，相当于 Kubernetes 的 apt/yum。其主要功能包括：
- 查找和使用以 Helm Charts 格式发布的流行软件；
- 将应用程序打包为 Helm Charts 进行分享；
- 创建 Kubernetes 应用的可重复构建；
- 智能管理 Kubernetes 清单文件及 Helm 包发布。

该工具通过渲染模板与 Kubernetes API 通信，支持从本地磁盘或远程仓库获取和管理 Charts 资源。
