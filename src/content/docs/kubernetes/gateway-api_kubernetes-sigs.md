### [kubernetes-sigs gateway-api](https://github.com/kubernetes-sigs/gateway-api)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubernetes-sigs/gateway-api?style=social)

Kubernetes Gateway API 是 SIG Network 下属项目，提供 Kubernetes 网关的 API 规范及自定义资源定义（CRDs）。当前 v1 版本中 Gateway、HTTPRoute 等核心资源已获 GA 支持。项目提供详细的 API 文档、概念介绍及快速入门指南，并包含一致性测试套件，支持实现方验证合规性，方便用户查询各实现的功能支持情况。
