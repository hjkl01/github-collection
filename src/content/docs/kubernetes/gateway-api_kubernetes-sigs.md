### [kubernetes-sigs gateway-api](https://github.com/kubernetes-sigs/gateway-api)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubernetes-sigs/gateway-api?style=social)

**项目功能：**  
Kubernetes Gateway API 是 SIG Network 的一部分，用于定义和管理 Kubernetes 中的网关资源。该项目包含 API 规范和自定义资源定义（CRDs），用于配置如 HTTP、gRPC 等路由策略和后端 TLS 策略。

**使用方法：**  
- 访问项目网站查看 API 规范和详细文档。
- 通过阅读“API 概念”和“安全模型”了解 API 背景和使用场景。
- 参照“入门指南”安装第一个 Gateway 控制器并尝试实践。
- 查阅 API 参考文档和 Go 语言包文档以获取详细信息。

**主要特性：**  
- 支持 GA 级别的资源包括 `GatewayClass`、`Gateway`、`HTTPRoute`、`GRPCRoute` 和 `BackendTLSPolicy`。
- 提供完整的 API 参考和符合性测试文档，支持开发者验证和提交符合性报告。
- 项目维护社区文档、会议安排及贡献指南，并设有 Slack 交流频道。
