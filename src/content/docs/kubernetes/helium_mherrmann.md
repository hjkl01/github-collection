### [mherrmann helium](https://github.com/mherrmann/helium)  ![GitHub Repo stars](https://img.shields.io/github/stars/mherrmann/helium?style=social)

Helium 是一款基于 Python 的 Chrome 和 Firefox 浏览器自动化库，作为 Selenium 的高级封装，它提供更简洁、易读且稳定的 API。其核心功能包括使用可视文本标签定位元素（无需 XPath 或 CSS 选择器）、自动处理 iFrames 切换与窗口管理、内置隐式等待及友好的显式等待机制。Helium 脚本通常比 Selenium 短 30-50%，并支持与原生 Selenium 代码混用。
