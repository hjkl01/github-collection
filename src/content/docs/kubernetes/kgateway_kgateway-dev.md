### [kgateway-dev kgateway](https://github.com/kgateway-dev/kgateway)  ![GitHub Repo stars](https://img.shields.io/github/stars/kgateway-dev/kgateway?style=social)

kgateway 是基于 Envoy 并实现 Kubernetes Gateway API 的 Kubernetes 微服务网关，目前市场上最成熟且广泛部署的 API 网关之一。其核心功能涵盖：作为下一代 API 网关集中暴露服务并管理认证、授权及限流策略；支持混合应用路由，可对接微服务、Serverless 函数及遗留应用；适应复杂多租户环境，提供路由委托和组合策略；具备弹性扩展能力，适用于从轻量级微网关到处理海量请求的中央网关，兼容多云与混合架构。该项目前身为 Gloo，现为 CNCF Sandbox 项目，专注于提供稳定健壮的 Envoy 驱动 API 网关服务。
