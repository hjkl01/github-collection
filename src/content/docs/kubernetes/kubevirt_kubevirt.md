### [kubevirt kubevirt](https://github.com/kubevirt/kubevirt)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubevirt/kubevirt?style=social)

KubeVirt 是一个 Kubernetes 虚拟机管理插件。它通过自定义资源定义（CRD）扩展 API 并新增虚拟机（VM）资源类型，实现通过 Kubernetes API 管理 VM。项目提供控制器和代理，支持在集群上声明式地创建、调度、启动、停止和删除虚拟机，为 Kubernetes 环境下的虚拟化提供基础。
