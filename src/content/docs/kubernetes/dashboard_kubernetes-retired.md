### [kubernetes-retired dashboard](https://github.com/kubernetes-retired/dashboard)  ![GitHub Repo stars](https://img.shields.io/github/stars/kubernetes-retired/dashboard?style=social)

Kubernetes Dashboard 是 Kubernetes 集群的通用 Web 管理界面，支持用户管理集群应用、排查问题及执行集群管理操作。自 7.0.0 版本起，仅支持基于 Helm 的安装方式，不再支持清单（Manifest）安装。该项目目前处于归档状态，不再维护，官方建议替代方案为 Headlamp。项目提供详细的使用文档、访问控制指南及开发者文档，并设有社区支持渠道。
