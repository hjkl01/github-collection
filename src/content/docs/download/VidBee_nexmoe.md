### [nexmoe VidBee](https://github.com/nexmoe/VidBee)  ![GitHub Repo stars](https://img.shields.io/github/stars/nexmoe/VidBee?style=social)

VidBee 是一款基于 yt-dlp 的现代化开源视频下载工具。支持从全球 1000 多个网站（含 YouTube、TikTok 等主流平台）下载视频和音频。核心功能包括：提供直观简洁的界面与下载队列管理（支持暂停、恢复、重试）；支持 RSS 自动订阅并在后台自动下载新视频；包含 Electron 桌面客户端及支持 Docker 部署的 Web + API 服务。
