### [drakkan sftpgo](https://github.com/drakkan/sftpgo)  ![GitHub Repo stars](https://img.shields.io/github/stars/drakkan/sftpgo?style=social)

SFTPGo 是一款全功能、高度可配置的事件驱动型文件传输解决方案。它支持 SFTP、HTTP/S、FTP/S 和 WebDAV 协议，并兼容本地文件系统、加密本地存储、S3 兼容对象存储、Google Cloud Storage、Azure Blob 及其他 SFTP 服务器等存储后端。项目提供开源社区版和企业版：社区版包含核心协议、存储后端及 Web 管理界面；企业版在此基础上扩展了高性能云存储引擎、高级高可用架构、动态自动化逻辑、数据生命周期智能归档、邮件附件自动摄入、数据保护（PGP/杀毒）及文档在线编辑等功能。
