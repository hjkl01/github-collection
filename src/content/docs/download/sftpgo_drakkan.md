### [drakkan sftpgo](https://github.com/drakkan/sftpgo)

**项目核心内容总结：**

SFTPGo 是一个功能全面、高度可配置的事件驱动文件传输解决方案，支持多种协议（SFTP、HTTP/S、FTP/S、WebDAV）和存储后端（本地文件系统、加密本地存储、S3兼容存储、Google Cloud Storage、Azure Blob Storage 等）。  

**主要功能：**  
- 提供 WebAdmin 管理界面，支持用户、文件夹、组等资源的创建与管理。  
- 提供 WebClient 界面，允许用户自助修改凭证、浏览文件、设置双因素认证（兼容 Microsoft Authenticator、Google Authenticator 等）。  
- 支持本地与云存储后端，适用于内部或与合作伙伴的文件交换。  

**使用方法：**  
- 通过 Web 界面进行配置和管理。  
- 部署服务后，用户可通过支持的协议（如 SFTP、HTTP 等）访问文件传输功能。  

**主要特性：**  
- 多协议支持（SFTP、HTTP/S、FTP/S、WebDAV）。  
- 多种存储后端兼容性（本地、云存储）。  
- 企业版提供额外功能与商业支持（开源版仅含基础功能）。  
- 开源版遵循 GNU AGPL-3.0-only 许可证，企业版需商业授权。  
- 提供中文等多语言支持（通过 Crowdin 平台协作翻译）。  

**其他：**  
- 开源版免费使用，企业版提供付费支持与高级功能。  
- 官方文档可通过 [docs.sftpgo.com](https://docs.sftpgo.com/) 查阅。
