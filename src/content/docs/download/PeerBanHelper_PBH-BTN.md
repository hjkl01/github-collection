### [PBH-BTN PeerBanHelper](https://github.com/PBH-BTN/PeerBanHelper)  ![GitHub Repo stars](https://img.shields.io/github/stars/PBH-BTN/PeerBanHelper?style=social)

PeerBanHelper 是一个开源个人网络防火墙安全软件，用于自动封禁不受欢迎、吸血及异常的 BT 客户端。它通过连接支持应用程序（如 BitTorrent 客户端）的 Web API 获取连接信息，识别潜在安全威胁并通知应用断开连接。主要功能涵盖 PeerID 与 Client Name 黑名单、IP/GeoIP/IP 类型黑名单、虚假进度检查、自动连锁封禁、多拨追猎、伪装检查、主动监测、规则订阅及 WebUI 管理。支持基于 GeoIP 信息的归属地查看及按地域、网络类型封禁。兼容 qBittorrent、BiglyBT、Deluge、Transmission、BitComet 等客户端，仅支持 IPv4 或 IPv6 地址。
