### [bitmagnet-io bitmagnet](https://github.com/bitmagnet-io/bitmagnet)

**项目核心内容总结：**  
bitmagnet 是一个自托管的 BitTorrent 索引器、DHT 爬虫、内容分类工具和torrent搜索引擎，提供网页界面、GraphQL API 及 Servarr 堆栈集成。用户需自行部署并配置，可通过其网站（bitmagnet.io）访问服务。主要特性包括：支持自动化爬取和分类内容、提供搜索功能、兼容主流媒体管理工具（如 Servarr），适合搭建私有化 torrent 管理系统。
