### [yt-dlp yt-dlp](https://github.com/yt-dlp/yt-dlp)  ![GitHub Repo stars](https://img.shields.io/github/stars/yt-dlp/yt-dlp?style=social)

yt-dlp 是一款功能丰富的命令行音视频下载工具，支持数千个网站，作为 youtube-dl 的活跃分支项目维护。它提供强大的格式选择、子标题处理、元数据修改、插件扩展及 SponsorBlock 集成等功能。支持跨平台（Windows/Linux/macOS）、自动更新及 Python 嵌入调用。核心依赖为 Python 3.10+，推荐使用 ffmpeg 进行视频合并与后处理。
