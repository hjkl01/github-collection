### [autobrr qui](https://github.com/autobrr/qui)

**核心内容总结：**

该项目名为 **qui**，是一个用于管理和监控 **qBittorrent** 的工具，支持多实例管理，提供丰富的功能和界面，适用于自动化下载和监控需求。

**主要功能：**
- 支持多实例管理，可同时连接多个 qBittorrent 实例。
- 提供 Web 界面，支持通过浏览器操作和监控 qBittorrent。
- 支持 API 访问，可通过 API 实现自动化操作。
- 提供 Prometheus 指标接口，便于监控和集成到监控系统。
- 支持通过 API Key 进行安全访问。
- 提供详细的 Prometheus 指标，包括下载、上传速度、任务状态等。
- 支持通过 Swagger UI 探索和测试 API 接口。
- 支持通过 API 导出 .torrent 文件和备份数据。

**使用方法：**
- 安装项目后，通过 Web 界面访问。
- 配置 qBittorrent 实例连接信息。
- 通过界面或 API 实现对 torrent 的管理、监控、修改等操作。
- 可通过配置文件或环境变量启用 Prometheus 指标服务。

**主要特性：**
- 支持最新的 qBittorrent 功能（如重命名文件、设置临时下载路径等）。
- 提供详细的错误日志和状态信息。
- 支持通过 API Key 进行安全访问和操作。
- 提供丰富的监控指标，便于集成到监控系统中。
- 支持通过 Discord 社区获取帮助和技术支持。
- 提供付费主题和开源贡献支持。

**兼容性：**
- 官方支持 qBittorrent 4.3.9 及以上版本。
- 部分功能需要更高版本的 qBittorrent 才能使用。

**社区与支持：**
- 提供 Discord 社区支持。
- 开源项目，欢迎贡献代码。
- 提供多种支付方式支持开发者的持续维护。
