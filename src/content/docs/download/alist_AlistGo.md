### [AlistGo alist](https://github.com/AlistGo/alist)  ![GitHub Repo stars](https://img.shields.io/github/stars/AlistGo/alist?style=social)

AList 是一个支持多存储的文件管理程序，基于 Gin 和 Solidjs 开发，主要功能包括：  
- **多存储支持**：兼容阿里云盘、OneDrive、Google Drive、FTP、S3 等 30+ 存储类型，支持跨存储文件传输。  
- **文件管理**：提供文件预览（PDF、代码、图片、视频、Office 文档等）、暗黑模式、多语言、密码保护、WebDAV 接入。  
- **部署与扩展**：支持 Docker 部署、Cloudflare Workers 代理，可一键下载文件、打包下载、离线下载、多线程加速。  
- **交互功能**：允许用户上传、删除、重命名、移动文件，支持游客上传。  

**使用方法**：通过 Docker 部署或 Cloudflare Workers 代理快速启动，访问文档（[alistgo.com](https://alistgo.com/)）获取详细配置指南。  

**许可证**：AGPL-3.0 开源协议。  

**注意事项**：  
- 项目仅作文件分享用途，需遵守法律法规，禁止滥用。  
- 不存储用户数据，仅进行流量转发，使用前需自行承担账号封禁、限速等风险。
