### [alexta69 metube](https://github.com/alexta69/metube)  ![GitHub Repo stars](https://img.shields.io/github/stars/alexta69/metube?style=social)

MeTube 是一款基于 yt-dlp 的 Web 图形界面工具，支持从 YouTube 及数十个其他网站下载视频和音频，并支持播放列表。项目主要通过 Docker 部署，可通过环境变量灵活配置下载行为、存储路径、文件命名规则、并发数及网络设置（如 HTTPS 和反向代理）。此外，项目还支持浏览器扩展、书签脚本和 iOS 快捷指令，方便用户将链接直接发送至 MeTube 进行下载。
