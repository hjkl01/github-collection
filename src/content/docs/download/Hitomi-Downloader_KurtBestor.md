### [KurtBestor Hitomi-Downloader](https://github.com/KurtBestor/Hitomi-Downloader)  ![GitHub Repo stars](https://img.shields.io/github/stars/KurtBestor/Hitomi-Downloader?style=social)

**项目核心内容总结：**  
Hitomi-Downloader 是一款功能强大的下载工具，支持多线程加速（单任务最多24线程）、速度限制、BitTorrent/Magnet协议、M3U8/MPD格式视频下载，具备深色模式、剪贴板监控、便携性及任务管理功能。用户可通过 Chrome 扩展、脚本插件或直接下载使用，支持从 4chan、YouTube、Pixiv、Twitter 等 40+ 网站下载内容（包括视频、图片等），并兼容 yt-dlp 扩展的更多站点。  

**使用方法：**  
通过 GitHub 发布页面下载软件，或安装 Chrome 扩展；支持通过用户脚本扩展功能，也可直接复制链接下载。
