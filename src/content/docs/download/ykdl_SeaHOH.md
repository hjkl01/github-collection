### [SeaHOH ykdl](https://github.com/SeaHOH/ykdl)

**YouKuDownLoader** 是一个专注于中国大陆视频网站的视频下载工具，基于 [you-get](https://github.com/soimort/you-get) 分叉开发，主要特性包括：

- **功能**：支持中国大陆主流视频网站的视频下载，提供结构化代码和更优的兼容性。
- **使用方法**：
  1. 安装依赖：需 [FFmpeg](https://ffmpeg.org/)（用于合并媒体文件）和 [mpv](https://mpv.io/)（默认播放器）。
  2. 安装 Python 3 及更新 pip/setuptools。
  3. 通过 `pip3 install ykdl` 或 GitHub 链接安装。
  4. 确保 `ffmpeg`、`mpv`、`python` 及 `Scripts` 目录在系统环境变量中。
- **特性**：不再支持 Python 3.4 及以下版本，专注于国内视频站点，优化代码结构。  

**注意**：需通过 [维基页面](https://github.com/SeaHOH/ykdl/wiki/sites-status) 查看支持的网站状态。
