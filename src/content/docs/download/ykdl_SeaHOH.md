### [SeaHOH ykdl](https://github.com/SeaHOH/ykdl)  ![GitHub Repo stars](https://img.shields.io/github/stars/SeaHOH/ykdl?style=social)

该项目是一个专注于中国大陆视频网站的视频下载工具，基于 you-get 改造而成。它优化了代码结构，不再支持 Python 3.4 及以下版本，依赖 FFmpeg 进行媒体合并，默认使用 mpv 播放器。支持通过 pip 在 macOS、Linux 和 Windows 系统上安装使用。
