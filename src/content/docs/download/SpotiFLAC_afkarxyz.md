### [afkarxyz SpotiFLAC](https://github.com/afkarxyz/SpotiFLAC)  ![GitHub Repo stars](https://img.shields.io/github/stars/afkarxyz/SpotiFLAC?style=social)

SpotiFLAC 是一款支持 Windows、macOS 和 Linux 的免费跨平台工具。其核心功能是从 Tidal、Qobuz、Amazon Music 和 Deezer 等平台获取 Spotify 曲目的高保真 FLAC 音频，用户无需注册或登录 Spotify 账户即可使用。软件通过逆向工程 Spotify Web Player 及调用第三方 API 实现音频获取，仅供教育和私人用途。
