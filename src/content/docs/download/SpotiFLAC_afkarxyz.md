### [afkarxyz SpotiFLAC](https://github.com/afkarxyz/SpotiFLAC)  ![GitHub Repo stars](https://img.shields.io/github/stars/afkarxyz/SpotiFLAC?style=social)

SpotiFLAC 是一款第三方工具，可从 Tidal、Qobuz 和 Amazon Music 获取 Spotify 歌曲的无损 FLAC 格式文件，无需注册账户。支持 Windows 10+、macOS 10.13+ 和 Linux 系统，用户可通过 GitHub 发布页面下载安装包。软件仅用于教育和私人用途，开发者不鼓励或支持版权侵权行为，用户需自行承担法律风险。项目与 Spotify、Tidal 等平台无任何关联，所有使用行为需遵守当地法律及平台服务条款。
