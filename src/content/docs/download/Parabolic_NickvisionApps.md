### [NickvisionApps Parabolic](https://github.com/NickvisionApps/Parabolic)

**项目核心内容总结：**

Parabolic 是一个基于 yt-dlp 的视频下载工具，支持从 YouTube 等网站下载视频及音频，提供以下功能：
- 支持多种格式下载（MP4、WebM、MP3 等）
- 同时运行多个下载任务
- 下载视频元数据及字幕

**使用方法：**
1. **安装方式**：可通过 Flathub、Snap Store、GitHub 发布版安装，支持 Windows、Linux 平台；提供 Firefox 浏览器扩展。
2. **手动构建**：需使用 CMake 和 vcpkg 管理依赖，支持 Windows 和 Linux 平台，需 C++20 编译器。

**其他说明：**
- 项目遵循 GNOME 行为准则
- 法律声明：不鼓励或支持违反 DMCA 法律的视频使用
