### [filecxx FileCentipede](https://github.com/filecxx/FileCentipede)  ![GitHub Repo stars](https://img.shields.io/github/stars/filecxx/FileCentipede?style=social)

File Centipede 是一款免费无广告的全能型互联网文件管理工具，集成了文件上传下载管理器、BitTorrent 客户端、WebDAV、FTP 及 SSH 客户端功能。软件支持多种协议，并提供 Chrome 与 Firefox 浏览器扩展，可下载网页音视频及识别磁力链接。内置 HTTP 请求器、文件合并、编解码、翻译、校验及种子生成等辅助工具。此外，支持代理管理、速度限制、站点规则、目录管理及多语言，基于 C++ 开发，旨在提供快速、定制化和用户友好的体验。
