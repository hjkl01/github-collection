### [filecxx FileCentipede](https://github.com/filecxx/FileCentipede)

**项目功能**  
File Centipede 是一款多功能文件管理工具，支持 HTTP/HTTPS、FTP/FTPS、BitTorrent、WebDAV、SSH/SFTP 等协议，具备文件上传/下载、磁力链接解析、 Torrent 种子创建与管理、网页视频/音频提取等功能。集成浏览器扩展，可直接从网页下载加密视频、音频及抓取磁力链接。

**主要特性**  
- 支持多协议传输（含 BitTorrent、WebDAV、SSH 等）  
- 提供 HTTP 请求器、文件合并、编码工具等辅助功能  
- 浏览器集成：下载网页视频、音频，抓取磁力链接  
- 多语言支持（含中文、英文、日文、韩文等）  
- 支持速度限制、代理管理、站点规则自定义  
- 提供 Torrent 种子创建、磁力链接转换工具  
- 集成文件管理器（支持 WebDAV、FTP、SFTP）  
- 开源依赖库：Boost、Qt、libtorrent  

**使用方法**  
通过官网（[http://filecxx.com](http://filecxx.com)）或仪表盘（[https://w.filecxx.com](https://w.filecxx.com)）下载软件，安装后使用浏览器扩展功能抓取网页资源，或通过协议地址直接添加任务。支持多语言翻译，用户可参与翻译以扩展语言覆盖。
