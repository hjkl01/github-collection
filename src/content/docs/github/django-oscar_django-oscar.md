### [django-oscar django-oscar](https://github.com/django-oscar/django-oscar)  ![GitHub Repo stars](https://img.shields.io/github/stars/django-oscar/django-oscar?style=social)

Oscar 是一个基于 Django 的领域驱动电商框架，核心功能高度可定制，能够灵活适应从大型 B2C 到复杂 B2B 的各种电商需求及特定业务逻辑。项目提供官方及社区扩展，支持 RESTful API、多种支付网关（如 PayPal、Adyen 等）、账户管理、实体店集成、税务计算及发票生成功能。
