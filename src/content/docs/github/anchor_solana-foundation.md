### [solana-foundation anchor](https://github.com/solana-foundation/anchor)  ![GitHub Repo stars](https://img.shields.io/github/stars/solana-foundation/anchor?style=social)

Anchor 是 Solana 上最流行的程序开发框架，旨在简化智能合约的开发流程。其核心功能包括：

- 提供 Rust 领域特定语言（eDSL）用于编写 Solana 程序。
- 定义 IDL（接口描述语言）规范，支持通过 TypeScript 包从 IDL 生成客户端。
- 提供 CLI 和工作区管理工具，用于开发完整的应用程序。

主要组件涵盖 anchor-lang（Rust 原语）、anchor-spl（SPL CPI 客户端）、anchor-client（Rust 客户端）、@anchor-lang/core（TypeScript 客户端）以及 @anchor-lang/cli。其开发工作流类似于以太坊的 Solidity 开发模式。
