### [googlefonts googlesans-code](https://github.com/googlefonts/googlesans-code)

**核心内容总结：**  
Google Sans Code 是一款专为代码设计的等宽字体，旨在提升代码编辑器和终端中的可读性，融合 Google 品牌设计美学，适用于 Gemini 和 Android Studio 等产品。其特性包括：增强可读性、支持多语言（扩展拉丁文）、可变字体（重量轴 300-800）、OpenType 特性（如字形集、本地化形式）。  

**使用方法：**  
可通过 GitHub 下载最新版本的可变字体文件安装，或通过 Git 克隆仓库后使用 `fontc` 工具自行编译。构建需安装 `fontc` 编译器，并执行指定命令生成 Roman 和 Italic 字体文件。  

**主要功能：**  
- 支持多语言及可变字体轴（重量范围 300-800）  
- 优化代码编辑场景的清晰度和字符区分度  
- 提供 OpenType 高级排版功能  
- 自动化 CI/CD 流程确保字体质量
