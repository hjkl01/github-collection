### [Moustachauve cookie-editor](https://github.com/Moustachauve/cookie-editor)

Cookie-Editor 是一款浏览器扩展，用于快速创建、编辑和删除当前页面的 Cookie，适用于开发和测试网页。支持 Chrome、Firefox、Safari、Edge 和 Opera 浏览器，以及部分移动设备（如 Android 和 iOS）。提供批量删除 Cookie 功能，界面简洁易用。用户可通过各浏览器扩展商店安装。构建需使用 npm 和 Grunt，生成文件存放在 `dist` 目录。Safari 需通过 Xcode 15 构建。项目非 Google 官方支持，不保证质量或适用性。
