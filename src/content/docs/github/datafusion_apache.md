### [apache datafusion](https://github.com/apache/datafusion)  ![GitHub Repo stars](https://img.shields.io/github/stars/apache/datafusion?style=social)

Apache DataFusion 是一款用 Rust 编写的可扩展查询引擎，采用 Apache Arrow 作为内存格式。它提供 SQL 和 DataFrame API，具备列式、流式、多线程及向量化的执行引擎，旨在帮助开发者构建高性能的数据库和分析系统。该项目内置支持 CSV、Parquet、JSON 和 Avro 格式，并允许对数据源、查询语言、函数及操作符进行深度定制。此外，它还有配套的 Python 接口和 Spark 加速器，适用于构建领域特定查询引擎、新数据库平台及数据管道。
