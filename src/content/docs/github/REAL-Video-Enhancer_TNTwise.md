### [TNTwise REAL-Video-Enhancer](https://github.com/TNTwise/REAL-Video-Enhancer)  ![GitHub Repo stars](https://img.shields.io/github/stars/TNTwise/REAL-Video-Enhancer?style=social)

REAL Video Enhancer 是一款跨平台（Windows、Linux、macOS）视频增强软件，支持视频帧插值（提升帧率）和超分辨率放大。它提供 TensorRT、PyTorch、NCNN 多种推理后端，兼容不同 GPU 硬件，具备场景切换检测、实时预览及 Discord 状态显示功能，支持加载多种模型进行插值、放大、去噪与解压缩处理。
