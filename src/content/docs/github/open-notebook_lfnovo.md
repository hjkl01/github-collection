### [lfnovo open-notebook](https://github.com/lfnovo/open-notebook)  ![GitHub Repo stars](https://img.shields.io/github/stars/lfnovo/open-notebook?style=social)

Open Notebook 是一款开源、隐私优先的 Google Notebook LM 替代品，主要功能如下：

- **私有化部署**：支持 Docker、云或本地部署，数据完全由用户掌控，无云端依赖。
- **多模型支持**：兼容 16+ 个 AI 提供商（如 OpenAI、Anthropic、Ollama、Google 等），支持灵活选择与成本控制。
- **内容管理**：支持 PDF、视频、音频、网页及 Office 文档等多模态资料的上传与组织。
- **播客生成**：提供专业级多说话人播客生成功能，支持自定义角色与脚本。
- **智能交互**：支持基于研究内容的上下文对话、全文检索及向量搜索。
- **扩展与集成**：开放完整的 REST API，支持自定义内容转换、自动化流程及 MCP 集成。
- **多语言界面**：支持英语、中文、日语、俄语等多种语言。

项目基于 Python、Next.js、React 和 SurrealDB 构建。
