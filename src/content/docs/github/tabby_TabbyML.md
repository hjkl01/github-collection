### [TabbyML tabby](https://github.com/TabbyML/tabby)  ![GitHub Repo stars](https://img.shields.io/github/stars/TabbyML/tabby?style=social)

Tabby 是一个自托管的开源 AI 编程助手，作为 GitHub Copilot 的本地化替代方案。它无需数据库或云服务，支持消费级显卡及 OpenAPI 接口。核心功能包括代码智能补全、聊天对话助手、企业内部知识问答引擎（Answer Engine）及多模型支持。项目提供 VSCode 等 IDE 插件，具备团队管理功能，易于集成现有基础设施并在企业内部部署。
