### [beclab Olares](https://github.com/beclab/Olares)  ![GitHub Repo stars](https://img.shields.io/github/stars/beclab/Olares?style=social)

Olares 是一个开源个人云操作系统，旨在帮助用户在本地硬件托管数据，重获数字资产控制权，替代公有云服务。

**核心功能：**
1. **应用生态**：提供 IaaS、PaaS、SaaS 层的开源替代方案，支持应用沙盒隔离与单点登录（SSO），内置文件管理、同步盘、密码库、阅读器及开发工具。
2. **安全与网络**：集成 Tailscale、HeadScale 等简化网络配置，提供企业级安全认证与数据隐私保护。
3. **AI 与算力**：支持 GPU 管理，可本地部署大模型（LLM）、图像生成模型及私有 AI 知识库。
4. **数据管理**：统一文件系统，具备自动化备份、弹性扩展及高可用性。
5. **多端访问**：提供移动端、桌面端及浏览器客户端，实现随时随地远程访问。

**主要应用场景：** 边缘 AI 推理、个人数据仓库、自托管协作办公、私人媒体服务器、智能家居中枢及去中心化社交平台。

**系统兼容性：** 支持 Ubuntu 24.04 LTS 及 Debian 11 等 Linux 平台。
