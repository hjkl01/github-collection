### [gruntwork-io terragrunt](https://github.com/gruntwork-io/terragrunt)

**核心内容总结：**

Terragrunt 是一个灵活的编排工具，用于扩展基于 OpenTofu/Terraform 的基础设施即代码（IaC）。其主要功能包括支持多环境配置、模块化管理和代码复用，帮助用户高效管理复杂基础设施。  

**使用方法：**  
- 通过 [Terragrunt 官网](https://terragrunt.gruntwork.io) 获取安装说明、快速入门教程和完整文档。  
- 参考 [贡献指南](https://terragrunt.gruntwork.io/docs/community/contributing) 参与开发。  

**主要特性：**  
- 兼容 OpenTofu（≥1.6.0）和 Terraform（≥0.12.0）。  
- 提供 MIT 许可证，支持开源社区使用。  
- 通过 Discord 社区（[链接](https://discord.gg/YENaT9h8jh)）提供讨论、技术支持和协作。  
- 支持商业服务（如专业支持）。
