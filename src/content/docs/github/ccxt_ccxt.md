### [ccxt ccxt](https://github.com/ccxt/ccxt)  ![GitHub Repo stars](https://img.shields.io/github/stars/ccxt/ccxt?style=social)

CCXT 是一个开源的加密货币交易库，支持全球 100 多个加密货币交易所（包括中心化交易所 CEX 和去中心化交易所 DEX）。该库提供统一的 API 接口，支持 JavaScript/TypeScript、Python、C#、PHP 和 Go 等多种编程语言。核心功能包括获取公共市场数据（如行情、订单簿、K 线）以及管理私有账户（如下单、查询余额、资金存取）。通过标准化的数据结构，CCXT 便于进行跨交易所分析、套利及算法交易策略的开发与回测。项目采用 MIT 协议，免费开源，不包含托管资金，用户需自行承担交易风险。
