### [ranuts document](https://github.com/ranuts/document)  ![GitHub Repo stars](https://img.shields.io/github/stars/ranuts/document?style=social)

这是一个基于 OnlyOffice 的纯前端本地 Web 文档编辑器。核心功能是在浏览器中直接编辑文档，所有处理均在本地完成，无需服务器参与，确保隐私安全。支持 DOCX、XLSX、PPTX、CSV 等多种格式，支持离线使用（PWA）、从远程 URL 加载文档及多语言界面切换。采用 WebAssembly 技术实现格式转换，支持 Docker 部署和本地开发。
