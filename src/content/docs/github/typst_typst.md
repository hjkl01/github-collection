### [typst typst](https://github.com/typst/typst)  ![GitHub Repo stars](https://img.shields.io/github/stars/typst/typst?style=social)

Typst 是一个基于标记的排版系统，旨在兼具 LaTeX 的强大功能与易用性。项目包含编译器及命令行工具（CLI），支持本地文档编译。系统内置标记与脚本功能，支持数学公式、文献管理等排版任务。核心优势在于增量编译实现的高速编译、友好的错误提示及一致性设计原则。此外还提供免费的在线协作编辑器。
