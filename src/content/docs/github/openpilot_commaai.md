### [commaai openpilot](https://github.com/commaai/openpilot)  ![GitHub Repo stars](https://img.shields.io/github/stars/commaai/openpilot?style=social)

openpilot 是一款开源的机器人操作系统，用于升级 275 多款支持汽车的驾驶辅助系统。该系统需配合特定硬件设备（如 comma 3X）及汽车线束使用。项目提供发布版、测试版及开发版等软件分支，遵循 ISO26262 安全标准并拥有完善的自动化测试体系。软件为 MIT 许可的 Alpha 研究质量版本，非正式产品，用户需自行遵守当地法律法规。默认上传驾驶数据用于模型改进，用户可自主选择关闭数据收集。
