### [nopSolutions nopCommerce](https://github.com/nopSolutions/nopCommerce)  ![GitHub Repo stars](https://img.shields.io/github/stars/nopSolutions/nopCommerce?style=social)

nopCommerce 是一款免费开源的电子商务解决方案，基于 ASP.NET Core 和 .NET 9 构建。支持跨平台运行（Windows、Linux、Mac）及 Docker 部署，兼容 MS SQL、PostgreSQL 和 MySQL 数据库。平台采用全异步架构，内置多因素认证，支持 Web 农场和 Azure 部署。其架构清晰可定制，提供 Web API 插件支持第三方集成，拥有完善的市场生态，适用于创建和运营各类规模的在线商店。
