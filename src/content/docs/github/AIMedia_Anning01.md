### [Anning01 AIMedia](https://github.com/Anning01/AIMedia)  ![GitHub Repo stars](https://img.shields.io/github/stars/Anning01/AIMedia?style=social)

AIMedia 是一款全自动托管 AI 媒体软件，采用 Django 后端与 PySide6 桌面端架构。主要功能包括：自动抓取抖音、微博、网易新闻等平台的热点新闻；利用 AI 自动生成新闻文本并配图；支持向今日头条、微信公众号、百家号等多个平台自动发布。系统内置任务调度监控、配置管理及微信登录支付等企业级管理功能。
