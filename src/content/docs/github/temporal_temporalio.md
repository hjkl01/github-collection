### [temporalio temporal](https://github.com/temporalio/temporal)  ![GitHub Repo stars](https://img.shields.io/github/stars/temporalio/temporal?style=social)

Temporal 是一个持久化执行平台，支持开发者构建可扩展且可靠的应用程序。其核心功能是以弹性方式执行工作流，自动处理间歇性故障并重试失败操作。该项目源自 Uber Cadence，提供服务器源代码，支持多种编程语言实现工作流、活动和工作器，并可通过 CLI 和 Web UI 进行交互与管理。
