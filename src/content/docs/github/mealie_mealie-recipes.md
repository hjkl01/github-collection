### [mealie-recipes mealie](https://github.com/mealie-recipes/mealie)

**项目核心内容总结：**

**功能**  
Mealie 是一款自托管的食谱管理工具，集成食谱管理、饮食计划和购物清单功能，提供 REST API 后端和 Vue 响应式前端界面。支持通过 URL 自动导入食谱或手动添加家庭食谱，同时提供第三方应用交互的 API 接口。

**使用方法**  
- 通过 URL 导入食谱或手动输入数据创建食谱  
- 使用饮食计划功能规划每周餐食  
- 生成并分类管理购物清单（按超市分区）  
- 通过 Docker 简化部署流程  

**主要特性**  
- 支持 35+ 种语言本地化  
- 提供在线演示（[demo.mealie.io](https://demo.mealie.io/)）和详细文档（[docs.mealie.io](https://docs.mealie.io/)）  
- 支持 Docker 部署及 GitHub 容器镜像（GHCR）  
- 开源社区贡献（含翻译协作平台 Crowdin）  
- AGPL 开源许可证授权
