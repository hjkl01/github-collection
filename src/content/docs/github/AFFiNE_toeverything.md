### [toeverything AFFiNE](https://github.com/toeverything/AFFiNE)  ![GitHub Repo stars](https://img.shields.io/github/stars/toeverything/AFFiNE?style=social)

AFFiNE 是一款支持写作、绘图与规划的开源、隐私优先且本地优先的综合性工作空间，旨在替代 Notion 和 Miro。主要功能如下：

1. **超融合编辑**：文档、画布与表格深度融合，支持在无边画布上编辑富文本、便签、多视图数据库、嵌入网页及幻灯片。
2. **多模态 AI 助手**：集成 AI 功能，支持撰写报告、生成幻灯片、创建思维导图、任务规划及代码原型设计。
3. **本地优先与实时协作**：数据本地存储确保用户所有权，支持跨平台实时同步与多人协作。
4. **自托管与扩展**：支持私有化部署、分支构建及插件系统。
5. **开源生态**：社区版免费开源（MIT 协议），提供知识管理、Wiki 及数字资产管理能力。
