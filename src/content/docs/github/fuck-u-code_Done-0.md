### [Done-0 fuck-u-code](https://github.com/Done-0/fuck-u-code)  ![GitHub Repo stars](https://img.shields.io/github/stars/Done-0/fuck-u-code?style=social)

fuck-u-code 是一款用于检测代码质量并提供幽默反馈的命令行工具。

- 支持 Go、JavaScript、Python、Java 等 14 种编程语言，基于 tree-sitter 进行 AST 语法分析。
- 提供 0~100 综合评分及文件级“垃圾指数”，从复杂度、大小、注释、错误处理、命名、重复、结构七个维度评估代码质量。
- 集成 AI 代码审查功能，兼容 OpenAI、Anthropic、DeepSeek、Gemini 及本地 Ollama 等模型。
- 支持终端彩色输出、Markdown、JSON、HTML 等多种格式，并提供英/中/俄多语言支持。
- 代码分析完全在本地离线运行，保障隐私安全；支持 MCP 协议以便与 Claude Code、Cursor 等 AI 工具集成。
- 提供灵活的配置管理，支持 JSON/YAML 自定义文件、排除规则及评分权重。
