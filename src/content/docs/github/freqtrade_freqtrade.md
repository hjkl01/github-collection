### [freqtrade freqtrade](https://github.com/freqtrade/freqtrade)  ![GitHub Repo stars](https://img.shields.io/github/stars/freqtrade/freqtrade?style=social)

Freqtrade 是一款基于 Python 的免费开源加密货币交易机器人，支持主流交易所的现货与期货交易。项目可通过 Telegram 或 WebUI 进行控制，具备回测、绘图、资金管理、机器学习策略优化（FreqAI）及自适应预测模型等核心功能。支持干运行模式、SQLite 数据持久化、加密货币白名单/黑名单管理以及法币盈亏显示。推荐使用 Docker 部署，需确保系统时钟同步。
