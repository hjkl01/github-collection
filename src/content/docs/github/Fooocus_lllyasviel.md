### [lllyasviel Fooocus](https://github.com/lllyasviel/Fooocus)  ![GitHub Repo stars](https://img.shields.io/github/stars/lllyasviel/Fooocus?style=social)

Fooocus 是一款基于 Stable Diffusion XL 架构的离线、开源、免费图像生成软件。其设计目标是简化操作流程，用户无需复杂参数调整即可生成类似 Midjourney 的高质量图像。核心功能包括文生图、图生图、图像放大、局部重绘（Inpaint/Outpaint）、图像提示（Image Prompt）及换脸（FaceSwap）。软件支持 Windows、Linux、Mac 及 Colab 等多种部署环境，最低仅需 4GB 显存。内置提示词自动优化引擎、多种风格预设（如写实、动漫）及高级控制功能。目前项目处于仅修复漏洞的长期支持（LTS）状态，专注于 SDXL 架构的稳定性与维护。
