### [zephyrproject-rtos zephyr](https://github.com/zephyrproject-rtos/zephyr)

**核心内容总结：**  
Zephyr 是一个可扩展的实时操作系统（RTOS），专为资源受限设备设计，支持多种硬件架构（如 ARM、x86、RISC-V 等）和大量开发板，注重安全性与高效性。主要功能包括轻量级内核、多架构兼容、低功耗优化等。使用方法包括参考官方文档的入门指南、代码示例，并通过 GitHub 获取源码和发布版本。社区支持通过 Discord 和邮件列表提供帮助，问题跟踪和安全相关事宜可通过 GitHub Issues 和安全文档处理。
