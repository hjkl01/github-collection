### [zephyrproject-rtos zephyr](https://github.com/zephyrproject-rtos/zephyr)  ![GitHub Repo stars](https://img.shields.io/github/stars/zephyrproject-rtos/zephyr?style=social)

Zephyr 是一个可扩展的实时操作系统（RTOS），基于小型内核设计，专为资源受限设备优化且注重安全性。它支持多种硬件架构（包括 ARM、Intel x86、RISC-V 等），适用于从嵌入式传感器、可穿戴设备到智能手表及物联网网关的多样化应用场景。
