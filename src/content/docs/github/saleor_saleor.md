### [saleor saleor](https://github.com/saleor/saleor)

**核心内容总结：**  
Saleor 是一个基于 GraphQL 的 API-only 可扩展电商开源平台，支持多语言、多货币、多仓库等特性，适用于企业级需求。主要功能包括：  
- **技术无关性**：无需绑定特定技术栈，支持任意语言和框架扩展。  
- **模块化架构**：通过 Webhook、应用插件、仪表盘 iframe 等实现灵活扩展，核心逻辑与扩展分离。  
- **企业级特性**：支持多渠道管理、促销引擎、支付集成、订单管理、客户偏好、SEO 自由配置等。  
- **开发友好**：提供 Docker Compose 快速部署、CLI 工具（如 `saleor register`）、云服务（Saleor Cloud）和本地开发指南。  

**使用方法**：  
1. **安装**：通过 [Saleor Cloud](https://cloud.saleor.io) 快速部署，或使用 Docker Compose 及 [saleor-platform](https://github.com/saleor/saleor-platform) 本地运行。  
2. **开发**：参考 [文档](https://docs.saleor.io) 和 [贡献指南](./CONTRIBUTING.md)，参与开源社区。  
3. **扩展**：通过 Apps、Webhook、仪表盘 iframe 等方式集成第三方功能。  

**主要特性**：  
- 全球化支持（多语言/货币/仓库）  
- 企业级安全与稳定性  
- 分离的 Dashboard（[仓库](https://github.com/saleor/saleor-dashboard)）和 Storefront 示例（[仓库](https://github.com/saleor/storefront)）  
- 支持云原生部署及本地开发  

**注意事项**：  
- `main` 分支为开发版，生产环境建议使用 3.x 稳定版本。  
- 需通过 [文档](https://docs.saleor.io) 或社区（Discord/Twitter）获取支持。
