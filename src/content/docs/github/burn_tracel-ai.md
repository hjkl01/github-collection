### [tracel-ai burn](https://github.com/tracel-ai/burn)  ![GitHub Repo stars](https://img.shields.io/github/stars/tracel-ai/burn?style=social)

Burn 是基于 Rust 的下一代张量库和深度学习框架，专注于高效、灵活的数值计算、模型训练与推理。它支持广泛的硬件后端（GPU/CPU/WebGPU 等），通过可组合的后端架构实现自动微分、内核融合及多设备路由。项目内置训练监控工具，兼容 ONNX 与 PyTorch 模型导入，支持在浏览器和嵌入式无操作系统环境（no_std）中运行，致力于消除从训练到部署的代码差异，实现跨平台无缝迁移。
