### [github gh-aw](https://github.com/github/gh-aw)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/gh-aw?style=social)

GitHub Agentic Workflows 允许用户使用自然语言 Markdown 编写智能工作流，并在 GitHub Actions 中运行，以实现仓库任务的 AI 自动化。项目内置多层安全防护机制，默认采用只读权限，通过沙箱执行、网络隔离、工具白名单及人工审批等措施，确保 AI 代理在受控边界内安全运行。配套提供 Agent Workflow Firewall 和 MCP Gateway 以增强网络控制与集成能力。
