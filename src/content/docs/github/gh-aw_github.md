### [github gh-aw](https://github.com/github/gh-aw)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/gh-aw?style=social)

GitHub Agentic Workflows 是 GitHub 推出的一个工具，允许用户用自然语言的 Markdown 格式编写“智能代理工作流”，并在 GitHub Actions 中运行。其核心功能是通过 AI 代理自动化仓库中的任务。

**项目功能：**  
- 使用 Markdown 编写 AI 代理工作流，通过 GitHub Actions 自动执行  
- 提供默认只读权限，写操作需通过安全输出（safe-outputs）进行  
- 支持团队成员访问控制和关键操作的人工审批  
- 提供多种安全机制，包括沙箱执行、网络隔离、依赖项 SHA 锁定、工具白名单等  

**使用方法：**  
- 按照快速入门指南安装扩展并添加示例工作流  
- 通过文档和示例学习如何构建和运行工作流  
- 参考安全架构文档了解如何正确配置和使用  

**主要特性：**  
- **安全机制**：多层防护确保 AI 代理在受控环境中运行  
- **易用性**：通过 Markdown 编写工作流，无需复杂编程  
- **可扩展性**：支持与 Agent Workflow Firewall 和 MCP Gateway 等配套项目集成  
- **文档支持**：提供详细的文档、示例和使用指南  

**注意事项：**  
- 使用时需注意安全风险，需人工监督，操作需谨慎  
- 可通过 GitHub 仓库提交反馈和建议
