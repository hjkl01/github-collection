### [saleor storefront](https://github.com/saleor/storefront)  ![GitHub Repo stars](https://img.shields.io/github/stars/saleor/storefront?style=social)

Paper 是一款专为 Saleor 构建的极简生产级无头电商前端模板，基于 Next.js 16、React 19 及 TypeScript 开发。主要功能包括：支持多步骤、移动端优先及组件化的开放结账流程（含国际地址表单）；单一代码库实现多频道、多货币支持；完善的产品展示（多属性变体、动态定价）；集成购物车、客户资料及认证系统。采用 Cache Components 与 Webhook 机制平衡性能与数据实时性，内置无障碍标准并优化了 AI 辅助开发结构，支持主流支付网关，旨在帮助开发者快速构建高性能电商站点。
