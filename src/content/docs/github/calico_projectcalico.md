### [projectcalico calico](https://github.com/projectcalico/calico)  ![GitHub Repo stars](https://img.shields.io/github/stars/projectcalico/calico?style=social)

Calico 是由 Tigera 维护的开源项目，是容器网络和安全领域广泛采用的解决方案。主要功能如下：

1.  **多数据平面支持**：支持 eBPF、标准 Linux、Windows 及 VPP，实现跨发行版、云环境、裸机和虚拟机的互操作。
2.  **高性能与可扩展**：为 Kubernetes 集群优化，提供高速度、低 CPU 消耗且能无缝扩展的网络架构。
3.  **高级安全特性**：提供细粒度访问控制和 WireGuard 加密。
4.  **灵活的网络工具**：完全支持 Kubernetes 网络策略，并提供 BGP、VXLAN 及服务发现等能力。
5.  **组件维护**：包含 Calico API、Operator、VPP dataplane 和 Calico BIRD 等项目。
