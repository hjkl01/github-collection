### [projectcalico calico](https://github.com/projectcalico/calico)

**项目核心内容总结：**

**功能**  
Calico 是由 Tigera 开发的开源容器网络和安全解决方案，支持 Kubernetes 集群的网络通信与策略管理，每日为全球 166 个国家的 800 万+节点提供服务。

**使用方法**  
通过官方提供的 [Quickstart 指南](https://projectcalico.docs.tigera.io/getting-started/kubernetes/quickstart) 快速部署，支持多种 Kubernetes 环境（多云、裸金属、VM 等）。

**主要特性**  
1. **多数据平面支持**：兼容 eBPF、标准 Linux、Windows 和 VPP，适应不同网络需求。  
2. **高性能**：低 CPU 占用，优化集群资源利用率。  
3. **可扩展性**：无缝扩展至大规模 Kubernetes 集群。  
4. **安全能力**：支持细粒度访问控制、WireGuard 加密及 Kubernetes 网络策略标准。  
5. **生态兼容**：跨云平台、多发行版及虚拟化环境。  
6. **社区驱动**：拥有 200+ 全球企业贡献者，提供丰富的文档和培训资源。  

**维护项目**  
包括 Calico API、Operator、VPP 数据平面、BIRD 路由器等核心组件。
