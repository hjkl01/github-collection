### [FFmpeg asm-lessons](https://github.com/FFmpeg/asm-lessons)

**项目核心内容总结：**  
该项目是FFmpeg的汇编语言学习课程，旨在帮助学习者掌握FFmpeg中汇编语言的编写方式，深入理解计算机底层运行原理。  

**功能与使用方法：**  
- 提供系统化的汇编语言课程及配套练习（作业尚未上传）  
- 学习后可具备为FFmpeg项目贡献代码的能力  
- 通过Discord服务器（链接：https://discord.com/invite/Ks5MhUhqfB）获取学习支持  

**主要要求：**  
- 需掌握C语言（尤其指针）及高中数学基础（标量/向量运算等）  
- 课程内容涉及FFmpeg实际开发中汇编语言的应用场景
