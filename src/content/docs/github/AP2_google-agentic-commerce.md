### [google-agentic-commerce AP2](https://github.com/google-agentic-commerce/AP2)  ![GitHub Repo stars](https://img.shields.io/github/stars/google-agentic-commerce/AP2?style=social)

这是一个展示 Agent Payments Protocol (AP2) 关键组件的代码示例与演示项目。项目提供 Python 和 Android 场景代码，支持集成 Google AI 或 Vertex AI，用于构建智能体支付及购物代理应用，并定义了协议的核心对象类型。
