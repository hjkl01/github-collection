### [vosen ZLUDA](https://github.com/vosen/ZLUDA)

**核心内容总结：**  
ZLUDA 是 NVIDIA CUDA 的即插即用替代方案，可在非 NVIDIA GPU 上运行未修改的 CUDA 应用程序，性能接近原生 CUDA。主要特性包括：  
1. **兼容性**：支持非 NVIDIA GPU 运行 CUDA 程序，无需代码修改；  
2. **性能**：提供接近原生 CUDA 的运行效率；  
3. **使用方法**：通过文档（https://zluda.readthedocs.io）或 Discord 社区（https://discord.gg/sg6BNzXuc7）获取安装和配置指导。
