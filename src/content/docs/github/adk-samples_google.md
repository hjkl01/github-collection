### [google adk-samples](https://github.com/google/adk-samples)

**核心内容总结：**  
该项目是基于**Agent Development Kit (ADK)** 的示例代理代码库，提供多种语言（Python/Go/Java）的可运行代理案例，覆盖从简单对话机器人到复杂多代理工作流的常见场景。  

**功能与使用方法：**  
- 包含多种语言的示例代理（如Python中的`academic-research`、`customer-service`等，Go中的`llm-auditor`，Java中的`software-bug-assistant`等）。  
- 需先安装ADK，再根据各语言子目录（`python/`、`go/`、`java/`）中的说明运行示例。  
- 提供ADK文档链接及各语言官方仓库地址，便于进一步开发。  

**主要特性：**  
- 多语言支持（Python/Go/Java）。  
- 涵盖广泛场景（如数据分析、客服、软件调试等）。  
- 提供贡献指南和问题反馈渠道。
