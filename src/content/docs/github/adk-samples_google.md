### [google adk-samples](https://github.com/google/adk-samples)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/adk-samples?style=social)

该项目是 Agent Development Kit (ADK) 的样本仓库，提供基于 ADK 构建的即用型智能体，涵盖从简单对话机器人到复杂多智能体工作流的多种常见用例，旨在加速开发进程。支持 Python、TypeScript、Go 和 Java 语言，运行前需安装 ADK。项目仅用于演示目的，不适用于生产环境。
