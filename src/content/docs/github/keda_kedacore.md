### [kedacore keda](https://github.com/kedacore/keda)  ![GitHub Repo stars](https://img.shields.io/github/stars/kedacore/keda?style=social)

KEDA 是一个基于 Kubernetes 的事件驱动自动伸缩项目，支持对事件驱动工作负载进行细粒度自动伸缩，包括缩容至零。它作为 Kubernetes Metrics Server 运行，用户可通过 Kubernetes 自定义资源定义（CRD）来定义伸缩规则。KEDA 原生集成 Kubernetes 组件（如 HPA），无外部依赖，可在云端和边缘环境运行。
