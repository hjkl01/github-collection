### [steveiliop56 tinyauth](https://github.com/steveiliop56/tinyauth)

**核心内容总结：**  
Tinyauth 是一个轻量级的身份验证中间件，为应用提供登录界面或 OAuth（支持 Google、GitHub 等）保护功能，兼容 Traefik、Nginx、Caddy 等主流代理。  

**功能与特性：**  
- 通过简单配置快速集成到应用中；  
- 提供 Docker Compose 示例（含 Traefik、Whoami 演示）；  
- 支持多语言（通过 Crowdin 翻译）；  
- 开源（GPLv3 许可证），可自由修改和分发；  
- 提供在线 Demo（默认账号 user/password）；  
- 拥有 Discord 社区和贡献指南，支持用户参与开发。  

**使用方法：**  
1. 参考 [文档](https://tinyauth.app/docs) 配置；  
2. 使用 [Docker Compose 示例](./docker-compose.example.yml) 快速部署；  
3. 通过 [Demo](https://demo.tinyauth.app) 测试功能。
