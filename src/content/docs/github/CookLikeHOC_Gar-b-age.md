### [Gar-b-age CookLikeHOC](https://github.com/Gar-b-age/CookLikeHOC)

**项目核心内容总结：**

1. **项目功能**  
   - 提供“像老乡鸡那样做饭”的菜谱/烹饪指南，包含老乡鸡菜品溯源信息及实拍图贡献功能。  
   - 支持网页端访问（[cooklikehoc.soilzhu.su](https://cooklikehoc.soilzhu.su)），并提供AI绘图版网页（[ai.cooklikehoc.soilzhu.su](https://ai.cooklikehoc.soilzhu.su)）。  
   - 通过Docker容器化部署（文档见[docker_support/README.md](https://github.com/Gar-b-age/CookLikeHOC/tree/main/docker_support)）。

2. **使用方法**  
   - 直接访问网页端查看或贡献内容。  
   - 使用Docker部署（需参考docker_support目录的说明）。  

3. **主要特性**  
   - 社区驱动：支持用户贡献实拍图及菜品信息。  
   - AI技术：集成AI绘图功能生成手绘风格图片。  
   - 开源协作：包含贡献者列表及Docker支持（由[@honestAnt](https://github.com/honestAnt)和[@liucongg](https://github.com/liucongg)等贡献）。  

**声明**：项目非老乡鸡官方仓库，与老乡鸡仅存在消费者/商家关系，内容整理自《老乡鸡菜品溯源报告》。
