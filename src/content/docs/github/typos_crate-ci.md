### [crate-ci typos](https://github.com/crate-ci/typos)  ![GitHub Repo stars](https://img.shields.io/github/stars/crate-ci/typos?style=social)

typos 是一款源代码拼写检查工具，用于查找并修正代码中的拼写错误。其主要特点包括：

1.  **高效快速**：适用于大型 monorepo 项目运行。
2.  **低误报率**：适合集成至 Pull Request (PR) 流程中。
3.  **灵活配置**：支持通过配置文件忽略特定标识符、单词或文件，处理人名、本地化内容等误报情况。
4.  **广泛集成**：支持多种安装方式，提供 GitHub Actions、pre-commit、VS Code 及 LSP 等集成，并支持 stdin/stdout 交互及 JSON 格式输出。
