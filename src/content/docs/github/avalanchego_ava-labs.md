### [ava-labs avalanchego](https://github.com/ava-labs/avalanchego)  ![GitHub Repo stars](https://img.shields.io/github/stars/ava-labs/avalanchego?style=social)

AvalancheGo 是 Avalanche 区块链网络的全节点实现，提供高吞吐量和快速交易能力。支持连接主网、Fuji 测试网或创建本地测试网。提供源码编译、APT 安装、二进制包及 Docker 等多种部署方式。包含节点硬件配置要求、状态同步机制、代码生成工具及平台支持说明，遵循 BSD-3 开源协议。
