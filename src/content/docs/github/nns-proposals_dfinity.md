### [dfinity nns-proposals](https://github.com/dfinity/nns-proposals)  ![GitHub Repo stars](https://img.shields.io/github/stars/dfinity/nns-proposals?style=social)

本项目管理互联网计算机（Internet Computer）网络神经系统（NNS）的提案。NNS 以去中心化方式完全控制网络，负责升级协议、管理节点与子网、配置经济参数及冻结恶意软件等治理任务。系统通过接收提案，由网络参与者创建的“神经元”投票决定是否采纳。提案包含摘要、URL、提议者、类型及参数，并根据特定主题（如节点管理、治理、子网管理等）决定处理规则和投票权限。
