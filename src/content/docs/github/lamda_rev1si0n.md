### [rev1si0n lamda](https://github.com/rev1si0n/lamda)  ![GitHub Repo stars](https://img.shields.io/github/stars/rev1si0n/lamda?style=social)

FIRERPA Android 是一款 AI 驱动的下一代 Android 全栈自动化框架。支持 Android 6.0 至 16 系统，提供轻量级无依赖服务。核心功能包括：160+ 接口（系统控制、UI 自动化、OCR、文件管理等）；Python SDK 及 MCP 智能体工具调用；低延迟远程桌面（视频流、触控、UI 诊断）；内置 ADB/SSH/SCP、脚本加密及 Frida/IDA 逆向工具。支持分布式部署与非侵入式长驻服务，适用于生产级自动化工作流。
