### [paperless-ngx paperless-ngx](https://github.com/paperless-ngx/paperless-ngx)  ![GitHub Repo stars](https://img.shields.io/github/stars/paperless-ngx/paperless-ngx?style=social)

Paperless-ngx 是一款文档管理系统，旨在将物理文档转化为可搜索的在线档案以实现无纸化管理。作为 Paperless 和 Paperless-ng 的官方继任者，它支持通过 Docker Compose 或安装脚本快速部署，并提供版本迁移工具。系统核心功能包括文档存储、全文检索、在线管理界面及多语言支持。项目开源并接受社区贡献，但安全警告指出因数据明文存储，建议仅在本地可信服务器上运行，避免在未信任的主机上使用。
