### [OpenMind OM1](https://github.com/OpenMind/OM1)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenMind/OM1?style=social)

OpenMind OM1 是一款模块化 AI 运行时平台，旨在赋能开发者创建和部署多模态 AI 代理，支持数字环境及各类实体机器人（如人形、四足、教育机器人）。

核心功能包括：
1. **多模态交互**：处理网络数据、社交媒体、摄像头及激光雷达等输入，执行运动、导航和对话等物理动作。
2. **硬件扩展**：基于 Python 架构，通过插件支持 ROS2、Zenoh 等接口，便于接入新硬件。
3. **模型集成**：预配置多种大语言模型（LLM）、视觉语言模型（VLM）及语音服务接口。
4. **可视化调试**：提供 WebSim 界面，实时监控系统状态与调试信息。
5. **全自主服务**：结合 BrainPack 组件（SLAM、视频处理、前端界面等）实现闭环自主运行。
6. **自定义配置**：支持通过 JSON5 配置文件灵活调整代理行为、输入输出组合及系统提示。

该平台易于升级和重新配置，以适应不同物理形态的需求。
