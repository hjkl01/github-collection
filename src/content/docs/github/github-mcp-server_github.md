### [github github-mcp-server](https://github.com/github/github-mcp-server)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/github-mcp-server?style=social)

GitHub MCP Server 是将 AI 工具直接接入 GitHub 平台的服务器，使 AI 代理能够通过自然语言与 GitHub 交互。其功能涵盖代码库管理、Issue 与 PR 自动化、CI/CD 工作流监控、代码安全分析及团队协作。项目提供远程托管和本地部署两种模式，支持通过工具集和配置项灵活控制功能范围，兼容 VS Code、Claude Desktop、Cursor 等多种 MCP 宿主环境。
