### [microsoft autogen](https://github.com/microsoft/autogen)  ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/autogen?style=social)

AutoGen 是一个用于创建多智能体 AI 应用程序的框架，支持智能体自主运行或与人类协作。该框架采用分层可扩展设计，提供 Core API、AgentChat API 和 Extensions API，支持 Python 和 .NET 跨语言开发。其生态系统包含 AutoGen Studio（无代码 GUI 工具）和 AutoGen Bench（性能评估套件），并支持 Web 浏览、代码执行、文件处理等多种工具集成。项目旨在简化多智能体工作流的构建，并提供持续维护与社区支持。
