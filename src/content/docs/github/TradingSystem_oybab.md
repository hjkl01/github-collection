### [oybab TradingSystem](https://github.com/oybab/TradingSystem)  ![GitHub Repo stars](https://img.shields.io/github/stars/oybab/TradingSystem?style=social)

这是一个开源、多语言、跨平台的交易系统，适用于个人收支管理及餐饮、零售、酒店等商业场景。系统采用服务器 - 客户端架构，支持 PC、平板及手机多端操作，核心功能包括交易结算、收支分析、会员管理。支持打印机、扫码枪、读卡器等外部硬件。界面语言广泛支持且可扩展，提供源代码供二次开发及定制化集成，最终用户可免费下载二进制文件使用。
