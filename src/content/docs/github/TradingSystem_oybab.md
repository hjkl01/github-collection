### [oybab TradingSystem](https://github.com/oybab/TradingSystem)

**核心内容总结：**  
1. **项目功能**：  
   多语言、跨平台开源交易系统，支持零售、餐饮、酒店等场景的交易管理，支持二次开发及与现有业务系统集成。提供免费二进制文件，支持多语言员工与客户界面（含3种内置语言及可扩展的UI语言）。  

2. **使用方法**：  
   - **普通用户**：下载安装服务器端（需Windows 7及以上）、PC客户端（推荐，含完整功能）、平板客户端或手机客户端（需iOS/Android/UWP系统），设置服务器IP及语言后使用。默认管理员账号：用户名1000，密码123456（需及时修改）。  
   - **开发者**：需安装.NET 4/4.5、Visual C++ 2010 SP1运行库，使用Visual Studio打开源码，配置构建路径（建议使用Ramdisk工具创建Y:盘），通过两个Visual Studio实例分别调试服务端和客户端。  

3. **主要特性**：  
   - 支持热敏打印机、条码扫描器、IC/ID卡读卡器、POS价格屏、收银机等外部设备。  
   - 开源免费，支持多语言（含中文、英文、阿拉伯语等20+种）及扩展。  
   - 提供收入支出分析、个人工具使用、定制化开发（如集成门禁系统、物流服务等）功能。
