### [GoogleChrome chrome-extensions-samples](https://github.com/GoogleChrome/chrome-extensions-samples)  ![GitHub Repo stars](https://img.shields.io/github/stars/GoogleChrome/chrome-extensions-samples?style=social)

本项目是 Chrome 扩展的官方示例代码库，旨在展示扩展 API 的使用。内容涵盖专注于单一 API 的扩展示例及跨多 API 的完整功能扩展示例，并包含已弃用的 Chrome Apps 及 Manifest 版本 2 资源存档。开发者可克隆仓库并在 Chrome 浏览器中加载使用。项目由 Google 开发，遵循 Apache License 2.0 协议。
