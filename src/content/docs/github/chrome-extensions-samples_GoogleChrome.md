### [GoogleChrome chrome-extensions-samples](https://github.com/GoogleChrome/chrome-extensions-samples)

**项目核心内容总结：**  
该项目是Chrome扩展开发的官方示例库，包含针对不同API的扩展样本和功能完整的扩展案例，用于学习Chrome扩展开发。  

**使用方法：**  
克隆仓库后，通过Chrome浏览器的“加载已解压扩展程序”功能安装和测试示例。  

**主要特性：**  
1. 提供两类示例：`api-samples/`（单个API功能演示）和`functional-samples/`（多API整合的完整功能扩展）；  
2. 包含已弃用的Chrome应用平台（`_archive/apps/`）和旧版Manifest V2资源（`_archive/mv2/`）；  
3. 支持通过[Chrome开发者文档](https://developer.chrome.com/docs/extensions/samples/)按类型/权限筛选示例。  

**其他说明：**  
Chrome应用平台已停用，扩展开发需使用Manifest V3。
