### [hashicorp terraform-provider-azurerm](https://github.com/hashicorp/terraform-provider-azurerm)  ![GitHub Repo stars](https://img.shields.io/github/stars/hashicorp/terraform-provider-azurerm?style=social)

该项目是 Terraform 的 Azure 资源管理器（AzureRM）插件，用于管理 Azure 资源管理器中的云资源。它支持通过 Azure CLI、托管身份或服务主体进行身份验证，可用于创建和管理资源组、虚拟网络等资源。此外，项目还提供了开发者构建指南及贡献者指引。
