### [facebook create-react-app](https://github.com/facebook/create-react-app)  ![GitHub Repo stars](https://img.shields.io/github/stars/facebook/create-react-app?style=social)

Create React App 是一个用于快速创建 React 应用的零配置工具。它预集成 Webpack、Babel、ESLint 等工具，提供开箱即用的开发（含热重载）、测试及生产构建功能，支持 React、JSX、ES6、TypeScript 和 PWA。项目已弃用，不建议用于生产环境，但适合学习 React 或构建单页应用。用户可通过 eject 命令自定义配置。
