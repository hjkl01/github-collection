### [kata-containers community](https://github.com/kata-containers/community)  ![GitHub Repo stars](https://img.shields.io/github/stars/kata-containers/community?style=social)

Kata Containers 是一个开源项目，致力于构建兼具容器性能体验与虚拟机（VM）安全隔离优势的轻量级 VM 标准实现。该项目架构无关，支持多种 Hypervisor，兼容 OCI 规范及 Kubernetes。代码融合 Intel Clear Containers 与 Hyper runV 技术，采用 Apache 2 许可证，由 Open Infrastructure Foundation 管理。项目遵循开源、开放设计、开放开发及开放社区原则，设有架构委员会治理技术决策，并提供漏洞管理、贡献指引及相关文档。
