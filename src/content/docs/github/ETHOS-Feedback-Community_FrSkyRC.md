### [FrSkyRC ETHOS-Feedback-Community](https://github.com/FrSkyRC/ETHOS-Feedback-Community)

**项目核心内容总结：**  
FrSky Ethos反馈项目旨在收集用户对系统的使用体验和问题反馈，用于优化和修复软件。用户可通过GitHub提交问题，需确保使用最新固件版本，并检查是否已有相同问题。提交内容需包含固件版本、问题描述、预期行为、复现步骤及截图/视频。项目提供多语言用户手册（中/英/法/意/葡/西/德）及Wiki文档，涵盖固件更新、本地化设置等信息。问题将按优先级处理，用户需在问题解决后确认关闭。
