### [FrSkyRC ETHOS-Feedback-Community](https://github.com/FrSkyRC/ETHOS-Feedback-Community)  ![GitHub Repo stars](https://img.shields.io/github/stars/FrSkyRC/ETHOS-Feedback-Community?style=social)

该项目是 FrSky Ethos 系统的反馈与问题追踪仓库，旨在收集用户操作反馈与 Bug 报告以优化软件更新。项目提供详细的问题报告指南，以及 Ethos Wiki、模拟器资源、多国语言用户手册（含中文）及固件刷新指南等下载链接。
