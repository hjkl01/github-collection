### [coral-xyz anchor](https://github.com/coral-xyz/anchor)  ![GitHub Repo stars](https://img.shields.io/github/stars/coral-xyz/anchor?style=social)

Anchor 是 Solana 智能合约开发框架，也是目前最流行的 Solana 开发工具。它提供 Rust eDSL 用于编写程序，支持 IDL 规范及 TypeScript 客户端生成，并提供 CLI 和工作区管理工具以构建完整应用。包含 anchor-lang、anchor-spl、anchor-client 等核心包。
