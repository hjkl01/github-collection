### [frappe hrms](https://github.com/frappe/hrms)  ![GitHub Repo stars](https://img.shields.io/github/stars/frappe/hrms?style=social)

Frappe HR 是一款开源、现代且易用的 HR 与薪资管理软件，提供完整的 HRMS 解决方案。主要功能包括员工全生命周期管理、考勤与假期配置、费用报销与预支、绩效管理、薪资核算与税务管理，并支持移动端应用。该软件基于 Frappe Framework 构建，可实现与 ERPNext 财务模块的无缝集成。
