### [paritytech polkadot-sdk](https://github.com/paritytech/polkadot-sdk)  ![GitHub Repo stars](https://img.shields.io/github/stars/paritytech/polkadot-sdk?style=social)

Polkadot SDK 是为构建 Polkadot 多链区块链平台提供所需组件的项目，支持不同链之间安全、可扩展地互操作。其核心包含 Substrate、FRAME、Cumulus 和 XCM，用于开发 Polkadot 及其平行链。项目提供构建指南、WASM 和 PolkaVM 编译支持、依赖管理工具（psvm）及开发文档模板。该库整合了原 Substrate、Polkadot 和 Cumulus 仓库，每季度发布稳定版本。
