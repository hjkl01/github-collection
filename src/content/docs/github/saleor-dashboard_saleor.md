### [saleor saleor-dashboard](https://github.com/saleor/saleor-dashboard)  ![GitHub Repo stars](https://img.shields.io/github/stars/saleor/saleor-dashboard?style=social)

Saleor Dashboard 是一款基于 GraphQL 的单页仪表盘应用，专用于管理 Saleor 电商平台。它支持本地开发配置、环境设置、测试运行、Docker 部署及错误追踪，帮助开发者对 Saleor 实例进行可视化管理和操作。
