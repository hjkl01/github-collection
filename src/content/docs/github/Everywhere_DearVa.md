### [DearVa Everywhere](https://github.com/DearVa/Everywhere)  ![GitHub Repo stars](https://img.shields.io/github/stars/DearVa/Everywhere?style=social)

Everywhere 是一款支持上下文感知的交互式 AI 助手。用户可通过快捷键即时调用，直接识别屏幕内容，无需截图或切换应用即可获取帮助。核心功能包括错误排查、网页摘要、即时翻译和文本润色。支持多种主流大模型（OpenAI、Claude 等）及系统工具集成（浏览器、文件、终端），提供现代磨砂玻璃 UI 界面。目前适用于 Windows 和 macOS 平台，支持多语言。
