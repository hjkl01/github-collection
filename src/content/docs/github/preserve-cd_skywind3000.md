### [skywind3000 preserve-cd](https://github.com/skywind3000/preserve-cd)

**项目核心内容总结：**  
"绝版游戏保护工程"旨在保存濒临消失的经典游戏，提供以下内容：  

1. **CD1：DOS经典游戏合集**  
   - 精选64款难以获取的DOS游戏，包含游戏介绍、截图及配置说明，解决DOSBOX兼容性问题。  
   - 提供PDF介绍文档，支持通过谷歌、百度网盘、115、天翼云及BT磁力链接下载（MD5校验：66275d47779d68614232a3e5d524af4a）。  

2. **CD2：绝版经典小游戏**  
   - 收录100款Windows/Flash平台的易上手小游戏，无需充值，适合快速娱乐。  
   - 同样提供PDF介绍文档及多平台下载链接（MD5校验：594091d64d05908fb3e38d2fda1d36e5）。  

3. **增刊：老游戏攻略合集**  
   - 包含DOS、红白机、SFC/MD等主机游戏攻略，提供谷歌、百度网盘、115下载方式。  

**主要特性：**  
- 精选优质内容，避免冗余（如CD1仅650MB精选64款游戏）。  
- 提供多平台下载途径及MD5校验确保文件完整性。  
- 附带游戏介绍文档，提升收藏与怀旧价值。
