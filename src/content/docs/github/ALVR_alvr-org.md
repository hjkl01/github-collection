### [alvr-org ALVR](https://github.com/alvr-org/ALVR)  ![GitHub Repo stars](https://img.shields.io/github/stars/alvr-org/ALVR?style=social)

ALVR 是一个通过 Wi-Fi 将 PC 端的 VR 游戏串流至独立 VR 头显的开源项目。它支持 Windows 10/11 和 Linux 系统，兼容 Quest、Pico、Apple Vision Pro 等多种设备。运行依赖 SteamVR 环境、支持编码的高性能显卡（NVIDIA/AMD/Intel）以及 5GHz Wi-Fi 与有线网络连接。项目采用 MIT 协议，不收集用户个人数据。
