### [codelion openevolve](https://github.com/codelion/openevolve)  ![GitHub Repo stars](https://img.shields.io/github/stars/codelion/openevolve?style=social)

OpenEvolve 是一个先进的开源进化编码代理，旨在将大语言模型（LLM）转化为自主代码优化器，自动发现并优化算法。

核心功能：
1. **自主进化**：结合 MAP-Elites 质量多样性进化与 LLM，无需人工干预即可自动发现突破性算法。
2. **多语言支持**：支持 Python、Rust、Metal 等代码的优化，涵盖 GPU 内核开发、排序算法及数学问题求解。
3. **灵活集成**：兼容任何 OpenAI 标准 API，支持商用模型、本地模型及代理服务器配置。
4. **科研级复现**：内置随机种子机制，确保演化过程完全可复现。
5. **可视化与提示词优化**：提供实时进化跟踪界面，并支持对 LLM 系统提示词进行自动化进化以提升效果。

主要应用于性能优化、算法设计、科学计算、竞赛编程及多目标优化场景，可显著缩短优化时间并超越人类手工优化的性能上限。
