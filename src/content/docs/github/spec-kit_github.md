### [github spec-kit](https://github.com/github/spec-kit)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/spec-kit?style=social)

Spec Kit 是一个开源工具包，旨在通过“规范驱动开发”（Spec-Driven Development）模式加速高质量软件的构建。它将软件规格说明转化为可执行规范，直接生成实现代码，而非仅作为开发指导。项目提供 `specify` CLI 工具，支持与多种 AI 代理（如 Claude、Copilot、Cursor 等）协作，引导开发者遵循结构化工作流：建立项目原则、定义功能规范、制定技术实施方案、分解任务并执行实施。该工具帮助开发者专注于产品场景和预期结果，减少从零开始的随意编码，支持跨技术栈的应用开发与迭代优化。
