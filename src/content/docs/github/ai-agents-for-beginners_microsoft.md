### [microsoft ai-agents-for-beginners](https://github.com/microsoft/ai-agents-for-beginners)  ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/ai-agents-for-beginners?style=social)

这是一个面向初学者的 AI 代理构建课程项目。项目提供 50 多种语言支持，包含 10 多个核心课程，涵盖 AI 代理入门、框架探索、设计模式（工具使用、RAG、规划、多代理等）、可信构建、生产部署及协议等内容。每节课配有视频、文本说明及基于 Microsoft Agent Framework 和 Azure AI Foundry 的 Python 代码示例。此外，项目提供 Discord 社区支持供学员交流和获取帮助。
