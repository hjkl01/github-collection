### [dlvhdr gh-dash](https://github.com/dlvhdr/gh-dash)  ![GitHub Repo stars](https://img.shields.io/github/stars/dlvhdr/gh-dash?style=social)

**项目简介**  
DASH 是一个专为 GitHub 设计的终端用户界面工具，提供丰富的功能且不打断用户工作流程。  

**核心功能**  
- 支持自定义每个仓库的 PR 和 Issues 管理界面  
- 可自定义 Vim 风格键盘快捷键  
- 提供自定义操作以适配特定工作流  
- 完整支持 GitHub 操作（如查看代码差异、评论、推送、更新等）  
- 通过 YAML 配置文件管理所有设置  

**使用方式**  
1. 访问 [gh-dash.dev](https://gh-dash.dev) 获取文档和最新版本  
2. 通过 Discord 社群（链接见 README）提问或交流  
3. 参考贡献指南参与开发  

**特性亮点**  
- 完全在终端中操作 GitHub  
- 高度可定制化界面和快捷键  
- 集成 GitHub 功能与终端交互体验  

**支持方式**  
可通过 GitHub Sponsors 页面捐赠支持项目。
