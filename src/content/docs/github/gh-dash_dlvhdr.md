### [dlvhdr gh-dash](https://github.com/dlvhdr/gh-dash)  ![GitHub Repo stars](https://img.shields.io/github/stars/dlvhdr/gh-dash?style=social)

gh-dash 是一款不打断工作流的 GitHub 终端界面（TUI）工具。它支持自定义仓库级别的 PR 和议题板块，提供可覆盖的 Vim 风格快捷键，并通过 YAML 配置文件管理设置。核心功能涵盖代码对比、评论、检出、推送、更新等 GitHub 操作，集成 Markdown 渲染与 Diff 查看。
