### [trufflesecurity trufflehog](https://github.com/trufflesecurity/trufflehog)  ![GitHub Repo stars](https://img.shields.io/github/stars/trufflesecurity/trufflehog?style=social)

TruffleHog 是一个用于发现、分类、验证和分析泄露凭证（包括 API 密钥、密码、私钥等）的工具。它支持扫描 Git、云存储、CI/CD 平台、文件系统等多种数据源，识别超过 800 种密钥类型。通过 API 测试确认凭证是否有效，输出已验证、未验证或未知结果，并可对泄露凭证进行深度权限分析。支持命令行、Docker、GitHub Action、GitLab CI 及 Pre-commit 钩子等部署方式。
