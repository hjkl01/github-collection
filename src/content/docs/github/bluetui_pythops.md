### [pythops bluetui](https://github.com/pythops/bluetui)  ![GitHub Repo stars](https://img.shields.io/github/stars/pythops/bluetui?style=social)

bluetui 是一款用于在 Linux 系统上管理蓝牙的终端用户界面（TUI）工具。基于 bluez 实现，支持扫描并发现蓝牙设备、控制适配器电源与配对状态，以及对已配对设备进行连接/断开、信任、重命名、收藏等操作。支持多种安装方式，并允许用户通过配置文件自定义按键绑定。
