### [oceanbase miniob](https://github.com/oceanbase/miniob)  ![GitHub Repo stars](https://img.shields.io/github/stars/oceanbase/miniob?style=social)

MiniOB 是由 OceanBase 团队基于华中科技大学数据库课程原型开发的数据库入门学习项目，专为零基础学员设计。该项目旨在通过简化并发、安全等复杂模块，帮助学习者深入理解数据库内核原理并提升工程实战能力。

项目实现了完整的数据库系统架构，涵盖网络服务、SQL 解析与语义分析、查询优化、计划执行、存储引擎、MVCC 事务管理、日志管理、B+ Tree 索引及元数据管理等核心模块。MiniOB 作为全国大学生计算机系统能力大赛的初赛实训平台，配套详尽文档与视频教程，提供在线及本地开发环境，支持初学者系统化学习与实战训练。
