### [inngest inngest](https://github.com/inngest/inngest)  ![GitHub Repo stars](https://img.shields.io/github/stars/inngest/inngest?style=social)

Inngest 是一个持久化函数平台，用于替代队列、状态管理和调度系统，帮助开发者无需接触基础设施即可编写可靠的工作流和后台任务。核心功能包括：

1. **多语言支持**：提供 TypeScript、JavaScript、Python、Go、Kotlin/Java 等 SDK。
2. **本地开发**：支持 Inngest Dev Server，提供与生产环境一致的开发体验。
3. **灵活部署**：支持部署至自有基础设施，或同步至 Inngest 平台及自托管服务器。
4. **函数组件**：包含触发器（事件、Cron、Webhook）、流程控制（并发、限流、优先级）和步骤（自动重试、状态持久化）。
5. **系统架构**：涵盖事件 API、执行器、队列、状态存储、数据库及可视化仪表盘，全面支持自托管。
