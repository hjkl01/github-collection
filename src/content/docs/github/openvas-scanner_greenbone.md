### [greenbone openvas-scanner](https://github.com/greenbone/openvas-scanner)  ![GitHub Repo stars](https://img.shields.io/github/stars/greenbone/openvas-scanner?style=social)

该项目是 Greenbone 社区版的 OpenVAS 扫描器，提供全功能漏洞扫描引擎，用于执行持续更新和扩展的漏洞测试（VTs）。项目包含 C 语言实现及旨在整合扫描栈的 Rust 实现，支持源码构建、Docker 容器化部署，并可与其他 GVM 模块配合使用。
