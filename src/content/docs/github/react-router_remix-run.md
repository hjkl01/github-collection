### [remix-run react-router](https://github.com/remix-run/react-router)  ![GitHub Repo stars](https://img.shields.io/github/stars/remix-run/react-router?style=social)

React Router 是一个用于 React 的多策略路由库，支持从 React 18 到 React 19 的平滑过渡。它既可以作为完整的 React 框架使用，也可以作为库配合自定义架构使用，并提供相关工具包及版本升级指南。
