### [remix-run react-router](https://github.com/remix-run/react-router)

**核心内容总结：**  
React Router 是一个支持多种策略的 React 路由器，兼容 React 18 至 19，可作为完整框架或独立库使用。  

**功能与特性：**  
- 提供两种使用方式：作为 React 框架集成（推荐）或作为独立库嵌入自有架构。  
- 支持从 React 18 到 19 的平滑过渡，适配最新版本特性。  
- 包含多个扩展包（如开发工具、服务端渲染、云flare 集成等），满足不同场景需求。  

**使用方法：**  
- 框架模式：参考 [Framework 安装指南](https://reactrouter.com/start/framework/installation)。  
- 库模式：参考 [Library 安装指南](https://reactrouter.com/start/library/installation)。  

**版本支持：**  
提供 v6 和 v5 的历史版本文档，方便旧项目迁移。
