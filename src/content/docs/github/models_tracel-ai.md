### [tracel-ai models](https://github.com/tracel-ai/models)  ![GitHub Repo stars](https://img.shields.io/github/stars/tracel-ai/models?style=social)

本项目是基于 Burn 深度学习框架的模型集合仓库，汇集了官方及社区提供的深度学习模型与示例。涵盖自然语言处理、计算机视觉、语音识别及图像生成等多种任务，提供了各模型对应的独立实现仓库链接。官方模型采用 MIT 和 Apache License 2.0 双许可协议，社区模型请参照各自仓库的许可信息。
