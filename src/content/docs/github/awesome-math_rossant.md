### [rossant awesome-math](https://github.com/rossant/awesome-math)  ![GitHub Repo stars](https://img.shields.io/github/stars/rossant/awesome-math?style=social)

这是一个精选的数学资源导航项目，旨在为数学学习者提供全面的一站式资源索引。项目汇集了全球范围内的高质量材料，包括在线学习平台、计算软件工具、电子书籍、教学视频、期刊杂志及博客。内容按数学学科分支（如代数、几何、分析、数论、拓扑等）系统分类，并收录了知名院校的学生讲义。除少数付费资源外，绝大多数资料免费开放。
