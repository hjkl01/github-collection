### [rossant awesome-math](https://github.com/rossant/awesome-math)

该文档是一个涵盖数学、计算机科学、工程等领域的教育资源聚合列表，包含全球多所高校的课程讲义和书籍资源。核心内容包括：

1. **资源分类**  
按学科划分20+类别，包含数学分析、数值计算、信号处理、机器学习、生物数学等领域的免费讲义，覆盖基础课程（如微积分、线性代数）到前沿专题（如强化学习、随机过程）。

2. **资源形式**  
提供PDF/网页版教材，部分含视频课程，内容来源包括MIT、哈佛、斯坦福等高校教授及研究人员的公开讲义。

3. **特色内容**  
- 包含学生整理的竞赛/课程笔记（如Evan Chen的数学竞赛讲义）
- 聚合了《统计学习基础》《信号处理导论》等经典教材
- 涵盖数学工具书（如《数学物理方法》）、工程应用（如连续介质力学）等跨学科资源

4. **开放授权**  
所有资源均采用CC0协议开源，可自由用于学习和教学。
