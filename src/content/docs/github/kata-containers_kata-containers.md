### [kata-containers kata-containers](https://github.com/kata-containers/kata-containers)  ![GitHub Repo stars](https://img.shields.io/github/stars/kata-containers/kata-containers?style=social)

Kata Containers 是一个开源项目，旨在构建轻量级虚拟机的标准实现。它让容器具备虚拟机的工作负载隔离性和安全性，同时保持容器的启动速度和性能体验。项目支持多种硬件架构（如 x86_64、aarch64、ppc64le、s390x）及相应的虚拟化技术。核心组件包括容器运行时（Runtime）、代理（Agent）、虚拟机管理程序（如 Dragonball）等，可集成于 containerd、Kubernetes 等容器编排环境中，为容器化应用提供安全隔离的执行环境。
