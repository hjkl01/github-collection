### [browseros-ai BrowserOS](https://github.com/browseros-ai/BrowserOS)

**项目核心内容总结：**  
BrowserOS 是一个开源的 Chromium 分支浏览器，支持本地运行 AI 代理，提供隐私保护功能。用户可通过自定义 API 密钥或本地模型（如 Ollama）实现数据本地化存储，无需依赖云端。  

**使用方法：**  
1. 从官网下载适用于 macOS、Windows、Linux（AppImage/Debian）的安装包；  
2. 可选导入 Chrome 数据；  
3. 连接 OpenAI、Anthropic 或本地模型（Ollama/LMStudio）；  
4. 通过 AI 代理实现自动化操作。  

**主要特性：**  
- 界面类似 Chrome，兼容扩展程序；  
- AI 代理在本地运行，保障隐私；  
- 开源社区驱动，支持 MCP 服务器模式（可被 `claude-code` 或 `gemini-cli` 调用）；  
- 计划推出跨场景 AI 广告拦截功能。  

**差异化优势：**  
相比 Chrome、Brave、Arc/Dia、Perplexity Comet 等竞品，BrowserOS 强调本地化 AI 自动化、隐私保护及开源透明，目标是通过 AI 重构浏览器体验，简化重复性操作（如表单填写、数据抓取）。
