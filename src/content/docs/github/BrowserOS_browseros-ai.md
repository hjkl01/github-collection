### [browseros-ai BrowserOS](https://github.com/browseros-ai/BrowserOS)  ![GitHub Repo stars](https://img.shields.io/github/stars/browseros-ai/BrowserOS?style=social)

BrowserOS 是一款完全开源的 Chromium 浏览器分支，专注于本地化 AI 智能体自动化，提供隐私优先的浏览替代方案。支持自有 API 或本地模型（如 Ollama），确保数据完全保留在本地；兼容 Chrome 界面与扩展；提供可视化工作流构建、定时任务、MCP 服务器控制及结合本地文件操作的功能；内置 LLM 多模型对比聊天与广告拦截。项目采用 AGPL-3.0 协议，支持 macOS、Windows 和 Linux 平台。
