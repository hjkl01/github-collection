### [fyne-io fyne](https://github.com/fyne-io/fyne)  ![GitHub Repo stars](https://img.shields.io/github/stars/fyne-io/fyne?style=social)

Fyne 是一个使用 Go 语言编写的易用 UI 工具箱和应用程序 API，支持通过单一代码库构建跨桌面和移动设备的应用程序。它依赖 Go 1.17+ 及 C 编译器，提供完整的构建、打包和发布工具链，支持 Android 和 iOS 平台部署。应用无需预装库即可运行，具备高便携性，并配套提供系统设置及桌面管理工具。
