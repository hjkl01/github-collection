### [Azure azure-sdk-for-python](https://github.com/Azure/azure-sdk-for-python)  ![GitHub Repo stars](https://img.shields.io/github/stars/Azure/azure-sdk-for-python?style=social)

Azure SDK for Python 是用于 Python 语言连接、管理和使用 Azure 云服务的软件开发工具包。该项目包含针对不同 Azure 服务的独立库集合，主要分为两类：
1. 客户端库：用于与现有 Azure 资源交互及消费（如上传 Blob），包含符合新设计指南的最新发布版本（GA/Preview）和旧稳定版本。
2. 管理库：用于供应和管理 Azure 资源（命名空间以 azure-mgmt- 开头），包含新版本和旧版本。

所有库共享重试、日志、传输协议、认证机制等核心功能，需 Python 3.9 或更高版本支持。项目提供官方文档、技术支持及贡献指南。
