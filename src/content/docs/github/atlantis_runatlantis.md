### [runatlantis atlantis](https://github.com/runatlantis/atlantis)  ![GitHub Repo stars](https://img.shields.io/github/stars/runatlantis/atlantis?style=social)

Atlantis 是一个自托管的 Terraform 拉取请求自动化平台。它通过 Webhooks 监听 Terraform 拉取请求事件，远程执行 `terraform plan`、`import` 和 `apply` 命令，并将输出结果评论在拉取请求中。使用该工具可使 Terraform 变更对团队可见，支持非运维工程师协作，并标准化 Terraform 工作流。
