### [cloudreve cloudreve](https://github.com/cloudreve/cloudreve)  ![GitHub Repo stars](https://img.shields.io/github/stars/cloudreve/cloudreve?style=social)

Cloudreve 是一款自托管的多云文件管理系统，采用 Go 与 React 技术栈。核心功能包括：支持本地及 OneDrive、S3、国内各大云存储等多源存储；支持客户端直传、断点续传及 WebDAV 协议；集成 Aria2/qBittorrent 实现后台下载；提供文件在线预览（媒体/Office/文档）、压缩解压、批量下载及元数据搜索；支持多用户多群组管理、分享链接及界面主题自定义。
