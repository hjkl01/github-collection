### [ironcalc IronCalc](https://github.com/ironcalc/IronCalc)  ![GitHub Repo stars](https://img.shields.io/github/stars/ironcalc/IronCalc?style=social)

IronCalc 是一个基于 Rust 开发的现代化电子表格引擎及工具集，支持 xlsx 文件的读取与写入。它可在 Python、JavaScript、Node.js 等多种编程语言中集成调用，并计划提供终端、桌面及 Web 等多种应用形态。项目允许通过 API 构建模型、设置数据与公式、执行计算并导出文件，采用 MIT 或 Apache 2.0 开源协议，目前处于开发进展中。
