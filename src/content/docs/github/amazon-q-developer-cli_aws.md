### [aws amazon-q-developer-cli](https://github.com/aws/amazon-q-developer-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/aws/amazon-q-developer-cli?style=social)

该项目是 Amazon Q Developer 的命令行界面（CLI）工具，允许用户通过终端与 Amazon Q Developer 进行交互。支持 macOS 和 Linux 系统，基于 Rust 语言开发。目前项目已停止积极维护（仅接收关键安全修复），建议用户使用 Kiro CLI。
