### [ahujasid blender-mcp](https://github.com/ahujasid/blender-mcp)  ![GitHub Repo stars](https://img.shields.io/github/stars/ahujasid/blender-mcp?style=social)

BlenderMCP 是一个基于模型上下文协议 (MCP) 的集成工具，用于连接 Blender 与 Claude AI，实现 AI 对 3D 软件的控制与交互。主要功能包括：通过自然语言提示进行 3D 建模、场景创建与编辑；操作 3D 对象（创建、修改、删除）；管理材质与颜色；检查场景信息；以及执行 Blender Python 代码。支持集成外部资源，如下载 Poly Haven 和 Sketchfab 资产、使用 Hyper3D 生成 3D 模型。该系统由 Blender 插件和 MCP 服务器组成，支持远程连接，并提供可配置的匿名遥测。
