### [saleor saleor-platform](https://github.com/saleor/saleor-platform)  ![GitHub Repo stars](https://img.shields.io/github/stars/saleor/saleor-platform?style=social)

Saleor Platform 是用于 Saleor 项目本地开发的统一仓库，基于 Docker 整合了核心 GraphQL API、Dashboard 管理后台、Mailpit 邮件测试、Jaeger 监控及必要的数据库服务。该平台仅适用于本地开发环境，不支持生产部署。开发者配置 Docker 后，可通过 docker compose 一键启动所有服务，并通过指定端口访问各组件。
