### [Masterminds squirrel](https://github.com/Masterminds/squirrel)  ![GitHub Repo stars](https://img.shields.io/github/stars/Masterminds/squirrel?style=social)

Squirrel 是 Go 语言的链式 SQL 生成器，非 ORM。它支持通过组合组件构建灵活易读的 SQL 查询，涵盖 Select、Insert 等语句及 Where、Join、Limit 等条件操作。功能包括直接执行查询、预编译语句缓存、数据库连接管理，并能适配不同数据库的占位符格式（如 PostgreSQL）。此外，还提供 StatementBuilder 以优化代码结构。
