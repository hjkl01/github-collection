### [gofiber fiber](https://github.com/gofiber/fiber)  ![GitHub Repo stars](https://img.shields.io/github/stars/gofiber/fiber?style=social)

Fiber 是一个基于 Fasthttp 构建的 Go 语言 Web 框架，受 Express 启发，旨在通过零内存分配实现高性能与快速开发。其核心功能包括高效路由、静态文件服务、丰富的中间件生态（涵盖日志、CORS、限流、压缩、安全等）以及模板引擎支持。框架原生支持 WebSocket、Server-Sent Events 和 Socket.io，并能兼容 Go 标准库 net/http 处理器及 Express 风格回调。项目适用于 Go 1.25 及以上版本，采用 MIT 开源协议。
