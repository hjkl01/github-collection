### [tulir whatsmeow](https://github.com/tulir/whatsmeow)  ![GitHub Repo stars](https://img.shields.io/github/stars/tulir/whatsmeow?style=social)

whatsmeow 是一个用于 WhatsApp 网页多设备 API 的 Go 语言库。支持发送和接收私聊及群聊的文本与媒体消息，管理群组及接收变更事件，通过链接加入或创建群组，处理输入状态、送达及已读回执，读写应用状态（联系人列表、聊天设置等），以及处理解密失败重试和发送状态消息（实验性）。目前不支持群广播列表消息和通话功能。
