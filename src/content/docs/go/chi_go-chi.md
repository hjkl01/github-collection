### [go-chi chi](https://github.com/go-chi/chi)  ![GitHub Repo stars](https://img.shields.io/github/stars/go-chi/chi?style=social)

chi 是一个轻量、惯用且可组合的 Go HTTP 路由库，专为构建大型可维护的 REST API 服务设计。它 100% 兼容标准库 net/http，基于 Patricia Radix 树实现，支持中间件栈、路由组及子路由挂载。核心包无外部依赖，配套提供常用中间件包及文档生成功能，利用 Go context 处理请求上下文，性能优异且稳定。
