### [traefik traefik](https://github.com/traefik/traefik)  ![GitHub Repo stars](https://img.shields.io/github/stars/traefik/traefik?style=social)

Traefik 是一个现代 HTTP 反向代理和负载均衡器，旨在简化微服务部署。它能够自动集成 Docker、Kubernetes、Swarm 等基础设施，通过监听编排器或注册中心 API 动态生成路由，实现配置的自动化更新。其主要功能包括：支持 Let's Encrypt 自动 HTTPS、多种负载均衡算法、电路断路器、重试、WebSocket/HTTP/2/gRPC 支持、指标监控和 Web UI。项目以单一二进制文件或 Docker 镜像形式提供。
