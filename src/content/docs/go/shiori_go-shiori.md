### [go-shiori shiori](https://github.com/go-shiori/shiori)  ![GitHub Repo stars](https://img.shields.io/github/stars/go-shiori/shiori?style=social)

Shiori 是一款基于 Go 语言编写的简易书签管理器，支持命令行和 Web 界面，采用单二进制文件分发。核心功能包括：书签的增删改查、支持 Netscape 书签及 Pocket 导入导出、兼容 SQLite3 等多种数据库。默认解析网页内容生成离线存档，并提供浏览器扩展支持。
