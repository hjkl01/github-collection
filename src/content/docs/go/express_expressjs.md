### [expressjs express](https://github.com/expressjs/express)  ![GitHub Repo stars](https://img.shields.io/github/stars/expressjs/express?style=social)

Express 是用于 Node.js 的快速、无偏执、极简 Web 框架。它提供健壮的路由、高性能、HTTP 辅助功能（如重定向、缓存）、支持 14 种以上模板引擎的视图系统以及内容协商。适用于构建单页应用、网站、混合应用或公共 HTTP API，不强制使用特定 ORM 或模板引擎，并提供命令行工具快速生成应用。采用 MIT 许可证。
