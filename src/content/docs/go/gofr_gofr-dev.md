### [gofr-dev gofr](https://github.com/gofr-dev/gofr)  ![GitHub Repo stars](https://img.shields.io/github/stars/gofr-dev/gofr?style=social)

GoFr 是一个基于 Go 的微服务开发框架，旨在简化微服务开发，重点支持 Kubernetes 部署与开箱即用的可观测性。其核心功能包括简单 API 语法与默认 REST 标准、配置管理、日志追踪与指标监控；支持内置及自定义中间件、gRPC、HTTP 服务（含熔断机制）、发布订阅、数据源健康检查、数据库迁移、定时任务、动态日志级别调整、Swagger 文档、抽象文件系统及 WebSocket。项目要求 Go 1.24 或以上版本。
