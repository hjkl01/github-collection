Crocodile 是一个基于 Golang 的分布式任务调度系统，支持通过 Web 界面管理任务，可执行 HTTP 请求、Shell、Python、Go 等脚本，并支持父子任务依赖、多种调度算法（随机、轮询、权重、最小负载）、自定义报警（邮件、微信、钉钉等）、权限分级控制（管理员/普通用户/访客）等功能。系统支持 Linux、Mac、Windows 跨平台运行，提供 Docker 快速部署方式。核心功能包括任务实时日志查看、任务终止、主机分组管理、证书加密通信、任务审计等。部署需配置 Redis 和 MySQL，并通过 `docker-compose up -d` 启动，或手动解压后运行调度中心和 Worker 节点。
