### [pocketbase pocketbase](https://github.com/pocketbase/pocketbase)  ![GitHub Repo stars](https://img.shields.io/github/stars/pocketbase/pocketbase?style=social)

PocketBase 是一款开源的 Go 后端服务，提供单文件可执行方案。核心功能包括嵌入式 SQLite 数据库（支持实时订阅）、内置的文件与用户管理、管理后台 UI 及 REST 风格 API。支持独立运行或作为 Go 框架集成，允许通过 JavaScript 扩展，并提供官方 JS 和 Dart SDK。
