### [sea-team gofound](https://github.com/sea-team/gofound)  ![GitHub Repo stars](https://img.shields.io/github/stars/sea-team/gofound?style=social)

GoFound 是一个基于 Golang 实现的全文检索引擎，支持数据持久化及单机亿级数据毫秒级查找。项目提供 HTTP 接口调用，内置中文分词与可视化管理后台，原生编译无外部依赖，相比 Elasticsearch 内存占用更低、配置更简单。支持 Docker 部署，并提供 Java、Python、Node.js 等多语言 SDK。
