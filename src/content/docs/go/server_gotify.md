### [gotify server](https://github.com/gotify/server)

**项目核心内容总结：**

**功能**  
Gotify 是一个开源的实时消息推送服务器，支持通过 REST API 发送消息、通过 WebSocket 接收消息。提供用户管理、客户端与应用管理功能，并支持插件扩展。  

**主要特性**  
- 提供 Web 界面（UI）和命令行工具（CLI）用于消息发送  
- 支持 Android 客户端（可通过 Google Play 或 F-Droid 下载）  
- 自托管部署，无需依赖第三方推送服务  
- 支持 Docker 镜像部署  
- 提供详细的开发文档、配置指南和 API 文档  

**使用方法**  
1. 通过 [安装文档](https://gotify.net/docs/install) 部署服务器  
2. 使用 REST API 或 CLI 工具发送消息  
3. 通过 Web 界面管理用户和应用  
4. 安装 Android 应用接收推送通知  

**其他**  
- 采用 MIT 许可证  
- 使用语义化版本控制（SemVer）  
- 支持社区贡献（包括插件开发、文档优化等）
