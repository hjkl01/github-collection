### [redis go-redis](https://github.com/redis/go-redis)  ![GitHub Repo stars](https://img.shields.io/github/stars/redis/go-redis?style=social)

go-redis 是 Go 语言的官方 Redis 客户端库，支持 Redis 7.0+ 及 8.x 版本。提供简洁的交互接口，核心功能包括支持 Redis 命令、自动连接池、Pub/Sub 发布订阅、管道与事务、Lua 脚本、Redis Sentinel 及 Redis Cluster 支持。此外还支持多种认证方式、RESP2/3 协议适配、OpenTelemetry 性能监控、自定义缓冲大小配置以及类型化错误处理。
