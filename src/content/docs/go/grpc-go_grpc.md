### [grpc grpc-go](https://github.com/grpc/grpc-go)  ![GitHub Repo stars](https://img.shields.io/github/stars/grpc/grpc-go?style=social)

gRPC-Go 是 gRPC 高性能开源通用 RPC 框架的 Go 语言实现。该框架基于 HTTP/2 协议，优先考虑移动端支持。Go 开发者通过导入 google.golang.org/grpc 库即可构建高性能远程过程调用服务。项目提供完整的 API 文档、示例代码、性能基准测试以及常见安装、日志配置与连接错误处理方案。
