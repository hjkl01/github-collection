### [cosmos cosmos-sdk](https://github.com/cosmos/cosmos-sdk)  ![GitHub Repo stars](https://img.shields.io/github/stars/cosmos/cosmos-sdk?style=social)

Cosmos SDK 是一个基于 Go 的模块化、开源区块链开发框架，用于构建安全、高性能的 Layer 1 应用链。开发者可利用其可组合架构，通过预定义或自定义模块快速搭建主权区块链，覆盖权限、治理、状态管理及代币化等功能。该 SDK 原生集成 IBC 协议实现链间互操作性，兼容 CometBFT 共识引擎。作为 Cosmos 生态核心，它已被 200 多条生产链使用，并提供适用于特定场景的企业级模块。
