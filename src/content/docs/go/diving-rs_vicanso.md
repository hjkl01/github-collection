### [vicanso diving-rs](https://github.com/vicanso/diving-rs)  ![GitHub Repo stars](https://img.shields.io/github/stars/vicanso/diving-rs?style=social)

diving-rs 是基于 Rust 开发的 Docker 镜像层级探索工具，无需依赖 Docker 客户端，支持 Linux、Windows 和 macOS 平台。项目提供终端（默认）和 Web 两种交互模式，支持从 Registry、本地 Docker 或 tar 文件解析镜像数据。支持层级内容缓存配置，终端模式提供文件筛选（如显示修改/删除文件、大于 1MB 文件等）功能，Web 模式可通过 Docker 容器部署并在浏览器中可视化浏览。
