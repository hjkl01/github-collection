### [casdoor casdoor](https://github.com/casdoor/casdoor)  ![GitHub Repo stars](https://img.shields.io/github/stars/casdoor/casdoor?style=social)

Casdoor 是一款开源的 AI 优先身份与访问管理 (IAM) 系统，兼具 AI MCP 网关和认证服务器功能，提供 Web 管理界面。它支持多种身份认证协议和标准，包括 MCP、A2A、OAuth 2.1、OIDC、SAML、CAS、LDAP、SCIM、WebAuthn、TOTP、MFA 及 Face ID，并集成 Google Workspace 和 Azure AD。支持通过源码、Docker 和 Kubernetes Helm 进行部署，采用 Apache-2.0 开源协议。
