### [casdoor casdoor](https://github.com/casdoor/casdoor)

**核心内容总结：**  
Casdoor 是一个开源的 UI 优先身份与访问管理（IAM）及单点登录（SSO）平台，支持 OAuth 2.0、OIDC、SAML、LDAP、MFA 等多种认证协议。  

**主要功能：**  
- 提供 Web 界面管理用户身份和权限；  
- 支持多因素认证（如 WebAuthn、TOTP）、RADIUS 等安全机制；  
- 集成 SCIM、LDAP 等目录服务，适配企业级需求。  

**使用方法：**  
- 通过源码、Docker 或 Kubernetes Helm 安装；  
- 提供在线演示环境（只读和可写两种模式）；  
- 文档齐全，包含 API 接口、集成指南及多语言支持（通过 Crowdin 平台协作翻译）。  

**其他特性：**  
- 开源协议为 Apache-2.0；  
- 提供 Discord 社区支持及问题反馈渠道。
