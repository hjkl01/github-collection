### [evcc-io evcc](https://github.com/evcc-io/evcc)  ![GitHub Repo stars](https://img.shields.io/github/stars/evcc-io/evcc?style=social)

evcc 是一款开源的可扩展电动汽车充电桩控制器及家庭能源管理系统，旨在实现不依赖云服务的本地能源管理。其核心功能包括：支持众多充电桩（含 OCPP、EEBus 协议）、能量计量设备（光伏逆变器、储能电池、智能电表）及车辆集成；提供简洁的用户界面；支持 Modbus、HTTP、MQTT 等插件扩展；提供状态通知、日志记录（InfluxDB、Grafana）及 REST/MQTT API 接口；并兼容 Home Assistant 等主流家庭自动化系统。
