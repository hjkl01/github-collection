### [ethereum go-ethereum](https://github.com/ethereum/go-ethereum)  ![GitHub Repo stars](https://img.shields.io/github/stars/ethereum/go-ethereum?style=social)

Go Ethereum 是以太坊协议的 Go 语言执行层实现。核心客户端 `geth` 支持全节点、归档节点及轻节点模式，可接入主网、测试网或私有网络，提供 JSON-RPC（HTTP、WebSocket、IPC）接口及 JavaScript 控制台用于网络交互。项目包含 `clef` 签名工具、`abigen` 合约代码生成器、`evm` EVM 调试工具及 `devp2p` 网络工具等实用程序，支持 Docker 部署及多种私有网络搭建方案。
