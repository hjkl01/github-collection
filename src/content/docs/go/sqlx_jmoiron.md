### [jmoiron sqlx](https://github.com/jmoiron/sqlx)  ![GitHub Repo stars](https://img.shields.io/github/stars/jmoiron/sqlx?style=social)

sqlx 是 Go 语言标准 database/sql 库的扩展库，其封装类型接口为标准接口的超集，便于无缝集成。主要功能包括：支持将查询结果自动映射到结构体、映射表及切片；支持命名参数查询及预处理语句；提供 Get 和 Select 方法快速获取查询结果；支持使用结构体和映射表进行批量插入；允许运行时自定义驱动程序的绑定变量类型。
