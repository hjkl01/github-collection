### [docker-mailserver docker-mailserver](https://github.com/docker-mailserver/docker-mailserver)  ![GitHub Repo stars](https://img.shields.io/github/stars/docker-mailserver/docker-mailserver?style=social)

Docker Mailserver 是一个生产就绪的容器化全功能邮件服务器。它基于 Postfix 和 Dovecot 构建，提供 SMTP、IMAP、POP3 及 LDAP 支持。项目内置 Rspamd 反垃圾、ClamAV 反病毒、OpenDKIM/OpenDMARC 域名验证及 Fail2ban 安全组件。其配置仅使用文本文件而无需 SQL 数据库，便于部署和升级。此外，它支持 Let's Encrypt 证书、OAuth2 认证、邮件获取（Fetchmail/Getmail6）、邮件过滤（Sieve）以及自动化维护脚本。
