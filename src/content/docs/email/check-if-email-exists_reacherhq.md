### [reacherhq check-if-email-exists](https://github.com/reacherhq/check-if-email-exists)  ![GitHub Repo stars](https://img.shields.io/github/stars/reacherhq/check-if-email-exists?style=social)

check-if-email-exists 是一款开源工具，用于在不发送邮件的情况下验证电子邮件地址是否有效。它提供 HTTP 后端（支持 Docker 部署）、命令行工具（CLI）和 Rust 库三种使用方式。功能涵盖语法校验、DNS/MX 记录验证、SMTP 连接测试、邮件可投递性判断，以及识别临时邮箱、角色账户、邮箱禁用、收件箱满溢、抓全部状态和数据泄露风险，并返回详细的 JSON 格式报告。
