### [stalwartlabs mail-server](https://github.com/stalwartlabs/mail-server)  ![GitHub Repo stars](https://img.shields.io/github/stars/stalwartlabs/mail-server?style=social)

Stalwart 是一款基于 Rust 开发的开源安全邮件与协作服务器，支持 IMAP、JMAP、SMTP、POP3、CalDAV、CardDAV 和 WebDAV 协议。它提供邮件收发、日历调度、联系人管理、文件存储及共享功能，内置反垃圾邮件和反钓鱼过滤器。项目具有高可扩展性和容错性，支持多种存储后端（如 RocksDB、PostgreSQL、S3）及搜索引擎，支持 Kubernetes 集群部署。同时具备完善的身份验证机制（2FA、OIDC 等）、Web 管理后台及监控可观测性工具，适用于从小型部署到大规模生产环境。
