### [cemu-project Cemu](https://github.com/cemu-project/Cemu)

**核心内容总结：**  
Cemu 是一款支持运行大多数 Wii U 游戏和自制软件的模拟器，采用 C/C++ 编写，当前支持 64 位 Windows、Linux 和 macOS 系统。用户可通过 GitHub 发布页面下载最新版本（Windows 提供安装版和便携版，Linux 支持 FlatHub 安装）。macOS 版本尚属实验性，可能存在性能问题。  

项目提供构建指南（BUILD.md），支持开发者自行编译。用户可通过 GitHub Issues 反馈问题，或通过 Discord 和 Matrix 参与讨论。贡献方式包括代码提交、测试游戏、提交兼容性报告等。
