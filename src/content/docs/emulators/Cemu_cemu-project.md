### [cemu-project Cemu](https://github.com/cemu-project/Cemu)  ![GitHub Repo stars](https://img.shields.io/github/stars/cemu-project/Cemu?style=social)

Cemu 是一款基于 C/C++ 开发的开源 Wii U 模拟器，能够在可玩状态下运行大多数 Wii U 游戏及自制软件。项目积极维护更新，当前支持 64 位 Windows、Linux 及 macOS 设备，其中 macOS 版本处于实验阶段。采用 Mozilla Public License 2.0 协议。
