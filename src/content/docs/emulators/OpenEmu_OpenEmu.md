### [OpenEmu OpenEmu](https://github.com/OpenEmu/OpenEmu)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenEmu/OpenEmu?style=social)

**核心内容总结：**  
OpenEmu 是一款基于 macOS 的开源游戏模拟器项目，旨在通过现代技术（如 Cocoa、Metal 等）为经典游戏提供原生体验。其模块化架构支持多种游戏引擎插件（如 Stella、Dolphin、MGBA 等），可运行 Atari、Nintendo、Sega 等多平台经典游戏。项目提供自动更新功能（依赖 Sparkle 库），最低要求 macOS Mojave 10.14.4，构建需 Xcode 14.3 和 macOS Ventura。
