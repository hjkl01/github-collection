### [xenia-project xenia](https://github.com/xenia-project/xenia)  ![GitHub Repo stars](https://img.shields.io/github/stars/xenia-project/xenia?style=social)

Xenia 是一款实验性的 Xbox 360 模拟器，支持 Windows 和 Linux 平台。项目旨在通过合法逆向工程对现代设备及操作系统进行模拟实验、研究与教育，严禁用于非法活动。目前部分游戏可运行，提供游戏兼容性列表、构建指南及快速入门教程。项目正招募具备 C++ 经验的贡献者协助优化与移植。
