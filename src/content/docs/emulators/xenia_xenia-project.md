### [xenia-project xenia](https://github.com/xenia-project/xenia)

**Xenia 项目核心内容总结：**  
Xenia 是一个实验性 Xbox 360 模拟器，旨在通过逆向工程研究和教育现代设备及操作系统的模拟技术。项目提供 Windows 和 Linux 版本，部分游戏可运行，但兼容性不完整（详见游戏兼容性列表）。  

**功能与使用方法：**  
- 提供 [快速入门指南](https://github.com/xenia-project/xenia/wiki/Quickstart) 说明使用方法。  
- 支持通过 [构建文档](docs/building.md) 编译源码，需遵循 C++ 风格指南并使用 `xb` 脚本。  

**主要特性：**  
- 开源社区驱动，支持跨平台（Windows/Linux）构建。  
- 通过 [GitHub Wiki](https://github.com/xenia-project/xenia/wiki) 提供详细技术文档和常见问题解答。  
- 鼓励开发者参与贡献，重点需求包括提升游戏兼容性、优化日志文件体积及 Linux 移植。  

**注意事项：**  
项目明确声明不支持非法用途，所有信息均通过合法途径获取。用户可通过 [Discord](https://discord.gg/Q9mxZf9) 参与讨论，但禁止涉及非法活动。
