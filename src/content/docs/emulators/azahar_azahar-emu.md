### [azahar-emu azahar](https://github.com/azahar-emu/azahar)

**项目核心内容总结：**

Azahar 是一个基于 Citra 的开源 3DS 模拟器，由 PabloMK7 的 Citra 分支和 Lime3DS 项目合并而成，旨在成为未来开发的首选平台。

**使用方法：**  
- **Windows**：从 GitHub 发布页下载最新版本，推荐使用 MSYS2。  
- **MacOS**：选择 `macos-universal`（通用版本）或按芯片架构（`macos-arm64`/`macos-x86_64`）下载。  
- **Android**：推荐通过 Google Play 商店安装，或使用 Obtainium 工具从 GitHub 发布页获取 APK。  
- **Linux**：推荐通过 Flathub 安装 Flatpak 版本，也可从发布页下载 AppImage。  

**主要特性：**  
- 支持多平台（Windows、MacOS、Linux、Android）。  
- 提供构建指南（通过项目 Wiki）。  
- 支持用户贡献（代码提交、翻译、兼容性报告）。  

**最低要求：**  
- **桌面端**：Windows 10（64 位）、macOS 13.4（Ventura）或现代 64 位 Linux；需 OpenGL 4.3/Vulkan 1.1 支持，2GB RAM（推荐 4GB）。  
- **安卓端**：Android 9.0+（64 位）；需 OpenGL ES 3.2/Vulkan 1.1 支持，2GB RAM（推荐 4GB）。  

**其他：**  
- 项目通过 GitHub 管理开发计划（里程碑）。  
- 社区通过 Discord 参与交流。
