### [gopher64 gopher64](https://github.com/gopher64/gopher64)  ![GitHub Repo stars](https://img.shields.io/github/stars/gopher64/gopher64?style=social)

Gopher64 是一款基于 Rust 和 SDL3 开发的开源 Nintendo 64 模拟器，支持 Windows 和 Linux 平台。核心功能包括本地运行 N64 游戏（支持键盘及 Xbox 手柄控制）、通过云端或自建服务器实现网络联机对战（Netplay）、支持便携模式（数据与可执行文件同目录），并提供 Flatpak 安装方式。项目遵循 GPLv3 开源协议。
