### [onsi ginkgo](https://github.com/onsi/ginkgo)  ![GitHub Repo stars](https://img.shields.io/github/stars/onsi/ginkgo?style=social)

Ginkgo 是 Go 语言成熟的测试框架，配合 Gomega 断言库使用，用于编写表达性强的规格说明（Specs）。它提供 BDD 风格 DSL，支持使用 Describe、Context、When、It 等节点组织测试，并通过 BeforeEach/AfterEach 管理前后置逻辑。

运行时支持测试随机化与并行执行，具备上下文管理、超时控制及自动清理能力。提供标签过滤、子集运行、多种格式报告生成及自定义报告功能。附带 CLI 工具支持测试生成、运行、过滤、性能分析及文件监听，适用于单元、集成及性能测试。
