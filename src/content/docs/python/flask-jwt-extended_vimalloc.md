### [vimalloc flask-jwt-extended](https://github.com/vimalloc/flask-jwt-extended)  ![GitHub Repo stars](https://img.shields.io/github/stars/vimalloc/flask-jwt-extended?style=social)

Flask-JWT-Extended 是专为 Flask 框架设计的 JSON Web Token (JWT) 扩展，用于保护应用路由。其主要功能包括支持自定义令牌声明、自动用户加载、令牌声明验证、刷新令牌、新鲜令牌（用于敏感操作）、令牌撤销与黑名单管理，以及支持将令牌存储在 Cookie 中并提供 CSRF 保护。
