### [gruns icecream](https://github.com/gruns/icecream)  ![GitHub Repo stars](https://img.shields.io/github/stars/gruns/icecream?style=social)

IceCream 是一个 Python 调试库，旨在提供更便捷、美观的 print 调试体验。其核心功能 ic() 可自动打印变量名及表达式及其值，支持数据结构的格式化美化与输出高亮，并能可选地包含文件名、行号及父函数等上下文信息。ic() 会返回参数，因此可无缝嵌入现有代码而不影响逻辑。项目支持全局安装（避免重复导入）、临时禁用/启用、自定义输出前缀及格式化规则，并提供未安装时的优雅降级方案。安装方式为 `pip install icecream`。
