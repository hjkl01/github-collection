### [asweigart pyautogui](https://github.com/asweigart/pyautogui)  ![GitHub Repo stars](https://img.shields.io/github/stars/asweigart/pyautogui?style=social)

PyAutoGUI 是一个跨平台的 GUI 自动化 Python 模块，支持 Windows、macOS 和 Linux。其核心功能是通过编程控制鼠标和键盘，包括鼠标移动与点击、键盘按键与文本输入、系统消息框展示以及截图与屏幕图像定位。该库屏蔽了不同操作系统底层 API 的复杂性，提供统一的简单接口。
