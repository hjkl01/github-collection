### [MetaCubeX mihomo](https://github.com/MetaCubeX/mihomo)  ![GitHub Repo stars](https://img.shields.io/github/stars/MetaCubeX/mihomo?style=social)

mihomo 是一个基于 Pydantic 的 Python 库，用于解析崩坏：星穹铁道的 Mihomo API 数据，支持类型提示和自动补全。项目提供 V1 和 V2 两种数据版本接口，可获取玩家基本信息（如等级、签名、成就）、角色列表及资源图标链接。此外，它包含移除重复角色、合并角色数据等实用工具函数，并支持使用 JSON 或 Pickle 格式进行数据持久化存储。
