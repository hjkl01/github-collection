### [cookiecutter-flask cookiecutter-flask](https://github.com/cookiecutter-flask/cookiecutter-flask)  ![GitHub Repo stars](https://img.shields.io/github/stars/cookiecutter-flask/cookiecutter-flask?style=social)

cookiecutter-flask 是一个 Flask Web 项目的 Cookiecutter 模板，支持 Python 3.8 及以上版本。提供 Docker 和标准 pip 两种项目初始化方式。生成的应用程序集成了 Bootstrap 5 和 Font Awesome 前端框架，后端包含 Flask-SQLAlchemy 数据库模型、Flask-Migrate 迁移、Flask-Login 认证、Flask-Bcrypt 密码哈希及 WTForms 表单。项目遵循 Twelve-Factor App 规范进行环境变量配置，采用蓝图和应用工厂设计模式，内置 pytest 测试、Factory-Boy、Click CLI、Webpack 前端构建及 Flask-Cache 缓存，并包含 Procfile 以支持 PaaS 部署。
