### [elliottech lighter-python](https://github.com/elliottech/lighter-python)  ![GitHub Repo stars](https://img.shields.io/github/stars/elliottech/lighter-python?style=social)

Lighter Python 是 Lighter 协议的 Python SDK，基于 Python 3.8+ 开发。

核心功能包括：
1. **账户管理**：查询账户详情、API 密钥、盈亏及资产统计。
2. **交易操作**：支持创建与取消订单，查询订单簿、挂单及成交记录。
3. **实时数据**：通过 WebSocket 实时同步订单簿与账户数据。
4. **资金管理**：查询存款与取款历史，发送交易（含批次），获取交易序号。
5. **市场信息**：获取交易所统计、资金费率、K 线及行情数据。

项目提供完整的 REST API 封装与数据模型，支持异步调用。
