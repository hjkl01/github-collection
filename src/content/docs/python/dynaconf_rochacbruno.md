### [rochacbruno dynaconf](https://github.com/rochacbruno/dynaconf)  ![GitHub Repo stars](https://img.shields.io/github/stars/rochacbruno/dynaconf?style=social)

dynaconf 是一款 Python 配置管理库，支持 TOML、YAML、JSON、INI 及 Python 等多种配置文件格式。该库遵循 12-factor 应用指南，支持通过环境变量覆盖设置，提供分层环境管理（如开发、测试、生产）和敏感信息保护（支持 Hashicorp Vault 和 Redis）。它具备设置验证、解析、模板替换及默认值管理功能，内置 Django 和 Flask 框架扩展，并提供 CLI 工具用于配置的初始化、验证、写入和导出操作。
