### [copier-org copier](https://github.com/copier-org/copier)  ![GitHub Repo stars](https://img.shields.io/github/stars/copier-org/copier?style=social)

Copier 是一个项目模板渲染库及命令行工具。支持从本地路径或 Git 仓库生成项目，可在任意文本文件中动态替换变量值，并防止覆盖现有文件。通过配置文件定义问答问卷，利用模板引擎实现代码脚手架搭建及项目代码的生命周期管理，支持命令行和 Python API 调用。
