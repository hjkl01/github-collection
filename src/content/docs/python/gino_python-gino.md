### [python-gino gino](https://github.com/python-gino/gino)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-gino/gino?style=social)

GINO 是一个基于 SQLAlchemy 核心构建的轻量级 Python asyncio 异步 ORM 库，支持 PostgreSQL（通过 asyncpg）和 MySQL（通过 aiomysql）。其核心功能包括提供稳健的 SQLAlchemy-asyncpg 翻译器、异步引擎与连接、异步友好的 CRUD 模型以及上下文连接与事务管理。它支持复用 SQLAlchemy 核心进行查询，兼容 Alembic 等生态工具，支持多种异步 Web 框架（如 FastAPI、Starlette、aiohttp 等），并提供丰富的 PostgreSQL JSONB 支持。
