### [Mckinsey666 bullet](https://github.com/Mckinsey666/bullet)  ![GitHub Repo stars](https://img.shields.io/github/stars/Mckinsey666/bullet?style=social)

`bullet` 是一个 Python 库，用于在命令行界面（CLI）中创建美观且可定制的提示。它支持丰富的格式设置、颜色、背景色、样式及表情符号。核心功能涵盖项目符号列表、复选框、滚动条、密码输入、是/否问答、数字输入、垂直提示和滑动提示。适用于 Python 3.6 及以上版本。
