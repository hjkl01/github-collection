### [Mckinsey666 bullet](https://github.com/Mckinsey666/bullet)

**项目功能**：`bullet` 是一个用于美化 Python 命令行交互界面（CLI）的工具库，支持自定义提示样式、颜色、背景色、字体样式及表情符号，提供多种输入提示类型（如列表选择、复选框、密码输入、数字输入、是/否确认等）。  

**主要特性**：  
- 支持多种提示类型（Bullet 列表、Check 复选框、Input 输入框、Password 密码输入、YesNo 是/否确认等）。  
- 可高度自定义样式（颜色、背景色、字体等）。  
- 提供滚动条、滑块等交互组件。  

**使用方法**：  
- 安装：通过 `pip install bullet` 或从源码安装。  
- 示例代码见 `examples/` 目录，如 `prompt.py`、`scrollbar.py` 等。  

**支持的提示类型**：`Bullet`、`Check`、`Input`、`Numbers`、`Password`、`YesNo`、`VerticalPrompt`、`SlidePrompt`、`Scrollbar`。
