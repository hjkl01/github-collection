### [joke2k faker](https://github.com/joke2k/faker)  ![GitHub Repo stars](https://img.shields.io/github/stars/joke2k/faker?style=social)

Faker 是一个 Python 库，用于生成虚假数据，适用于数据库初始化、文档创建、压力测试及数据匿名化。它支持通过类实例生成各类数据（如姓名、地址、文本），具备多语言本地化、Pytest 集成、自定义 Provider、生成唯一值及随机种子控制等核心功能，同时支持命令行使用。
