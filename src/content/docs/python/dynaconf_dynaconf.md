### [dynaconf dynaconf](https://github.com/dynaconf/dynaconf)  ![GitHub Repo stars](https://img.shields.io/github/stars/dynaconf/dynaconf?style=social)

Dynaconf 是一款 Python 配置管理库，支持多种文件格式（TOML、YAML、JSON 等）及环境变量覆盖。它具备多环境分层系统、敏感信息保护功能，内置对 Hashicorp Vault 和 Redis 的支持，并提供 Django 和 Flask 框架扩展。此外，还提供用于初始化、验证和导出配置的命令行工具，遵循 12-factor 应用指南进行默认值、验证、解析和模板化管理。
