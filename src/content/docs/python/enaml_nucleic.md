### [nucleic enaml](https://github.com/nucleic/enaml)  ![GitHub Repo stars](https://img.shields.io/github/stars/nucleic/enaml?style=social)

Enaml 是一个基于 Python 的声明式编程语言和框架，旨在以最小努力创建专业级图形用户界面。它基于 Qt 提供大量开箱即用的控件，采用基于约束的布局引擎，支持跨平台自动适配窗口大小。项目实现模型与视图分离，支持数据模型自动刷新界面，便于代码维护。它允许直接集成 Python 代码，支持自定义样式和控件扩展，适用于多平台原生应用开发及快速原型设计。
