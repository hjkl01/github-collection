### [axnsan12 drf-yasg](https://github.com/axnsan12/drf-yasg)  ![GitHub Repo stars](https://img.shields.io/github/stars/axnsan12/drf-yasg?style=social)

drf-yasg 是一个基于 Django Rest Framework 的 Swagger/OpenAPI 2.0 规范生成工具。它能够从 DRF API 自动生成真实的 Swagger 规范（支持 JSON 和 YAML 格式），内置 Swagger UI 和 ReDoc 用于查看文档。主要功能包括支持嵌套序列化器与模式、响应描述、兼容代码生成工具的模型定义、自定义配置钩子、缓存、规范验证及 API 版本化支持。该库兼容 Django 4.0+、Django Rest Framework 3.13+ 和 Python 3.10+，专注于 OpenAPI 2.0 标准。
