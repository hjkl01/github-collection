### [r0x0r pywebview](https://github.com/r0x0r/pywebview)  ![GitHub Repo stars](https://img.shields.io/github/stars/r0x0r/pywebview?style=social)

pywebview 是一个轻量级的原生 Webview 包装器，允许在原生 GUI 窗口中显示 HTML 内容，使桌面应用能够利用 Web 技术。它支持 Windows、macOS、Linux（GTK 或 QT）和 Android 平台，使用原生 GUI（如 WinForms、Cocoa、QT 或 GTK）创建 Web 组件窗口。项目核心功能包括内置 HTTP 服务器、Python DOM 支持、窗口管理、事件系统、原生 GUI 元素（应用菜单和对话框）以及 JavaScript 与 Python 之间的双向通信。在冻结应用时不会捆绑重型 GUI 工具包或 Web 渲染器，从而保持可执行文件体积小。
