### [groveco django-sql-explorer](https://github.com/groveco/django-sql-explorer)  ![GitHub Repo stars](https://img.shields.io/github/stars/groveco/django-sql-explorer?style=social)

SQL Explorer 是一款基于 Django 的数据探索与商业智能工具，旨在简化人与数据之间的交互流程。

1. **多源数据支持**：支持连接 Django 兼容的 SQL 数据库，或直接上传 CSV、JSON 及 SQLite 文件进行查询。
2. **SQL 查询编辑**：提供简易可用的 SQL 编辑器，支持编写、调试、分享查询，并在浏览器中直接查看结果。
3. **AI 辅助功能**：集成 LLM 助手（如 OpenAI），辅助编写和调试 SQL 语句，自动注入相关 Schema 上下文。
4. **数据可视化**：支持浏览器内快速生成统计摘要、透视表、散点图，无需 Excel 即可进行简单分析。
5. **灵活配置与集成**：支持多数据库连接、参数化查询（为无 SQL 经验用户生成友好 UI）、查询历史与日志、定期快照、结果邮件发送及将保存查询暴露为 JSON API。
6. **部署方式**：可嵌入现有 Django 站点或作为独立工具使用。
