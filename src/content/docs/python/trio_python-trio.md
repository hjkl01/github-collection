### [python-trio trio](https://github.com/python-trio/trio)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-trio/trio?style=social)

Trio 是一个面向 Python 的生产级异步 I/O 库，旨在通过原生 async/await 语法帮助开发者编写并行处理多任务 I/O 的程序（如网络爬虫、Web 服务器等）。该项目强调易用性与正确性，基于“结构化并发”设计理念，相比 asyncio 和 Twisted 更为简洁。Trio 支持 Python 3.10 及以上版本，兼容 Linux、macOS、Windows 及 FreeBSD，并采用 MIT 或 Apache 2.0 宽松开源协议。
