### [mingrammer diagrams](https://github.com/mingrammer/diagrams)  ![GitHub Repo stars](https://img.shields.io/github/stars/mingrammer/diagrams?style=social)

Diagrams 是一款通过 Python 代码绘制云系统架构图的工具。支持 AWS、Azure、GCP、Kubernetes、阿里云等多种云厂商及各类节点，适用于架构原型设计与现有系统可视化，支持通过版本控制系统追踪架构变更。工具仅用于绘图，不控制实际云资源或生成基础设施代码。
