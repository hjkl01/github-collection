### [sehmaschine django-grappelli](https://github.com/sehmaschine/django-grappelli)  ![GitHub Repo stars](https://img.shields.io/github/stars/sehmaschine/django-grappelli?style=social)

Django Grappelli 是一个基于网格的 Django 管理界面皮肤扩展，旨在提供美观的替代方案。它兼容 Django 最新稳定版本（如 5.x 和 4.2 LTS）。
