### [arrow-py arrow](https://github.com/arrow-py/arrow)  ![GitHub Repo stars](https://img.shields.io/github/stars/arrow-py/arrow?style=social)

Arrow 是一个 Python 库，旨在提供简洁、人性化的日期和时间处理方式。作为标准 datetime 模块的增强替代方案，它默认支持时区感知和 UTC。核心功能包括：简化日期创建与输入解析；自动格式化与字符串解析；支持 ISO 8601 标准；便捷的时区转换与时间偏移（shift）；生成时间范围、上下界；支持日期人性化表达（多语言）；兼容 dateutil、pytz 及 ZoneInfo 时区对象；并提供完整的类型提示支持。该库旨在减少代码量和模块依赖，简化日期时间操作。
