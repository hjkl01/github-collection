### [agno-agi agno](https://github.com/agno-agi/agno)  ![GitHub Repo stars](https://img.shields.io/github/stars/agno-agi/agno?style=social)

Agno 是一个构建、运行和管理大规模代理软件的运行时平台。它提供框架、运行时和控制平面，支持创建具备记忆、知识及安全护栏的智能体、团队和工作流。核心特性包括：支持快速构建并部署为生产级 API；基于 FastAPI 的无状态后端，支持按用户和会话隔离；通过 AgentOS UI 进行监控和管理；原生支持流式响应、全链路追踪和审计日志。系统运行于用户自有基础设施，确保数据主权，适用于个人代理、多智能体协作及企业级 AI 应用。
