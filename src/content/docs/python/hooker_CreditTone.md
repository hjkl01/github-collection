### [CreditTone hooker](https://github.com/CreditTone/hooker)  ![GitHub Repo stars](https://img.shields.io/github/stars/CreditTone/hooker?style=social)

Hooker 是一个基于 Frida 实现的安卓逆向工作台区工具，旨在为开发者提供舒适的命令行界面与自动化逆向辅助。核心功能包括：

1. **脚本自动化生成**：支持一键生成指定类或方法的 Frida Hook 脚本，自带详细备注与堆栈打印。
2. **SSL Pinning 通杀**：内置 Frida 版 JustTrustMe，支持绕过全网 APP 的 SSL 校验及 BoringSSL 验证。
3. **无感代理设置**：快捷配置 Socks5 代理，通过 iptables 实现应用无感知网络转发。
4. **丰富内置脚本**：包含脱壳、抓包、设备信息获取、绕过 Root/反调试/反 Frida 检测、加密算法 Hook 等实用脚本。
5. **自动化环境管理**：Root 手机连接后自动部署 Frida-server，无需手动配置。
6. **多平台支持**：兼容 Windows（需 WSL）、Mac 及 Linux 环境。

项目内容仅供学习与技术交流使用。
