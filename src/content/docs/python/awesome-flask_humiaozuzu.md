### [humiaozuzu awesome-flask](https://github.com/humiaozuzu/awesome-flask)  ![GitHub Repo stars](https://img.shields.io/github/stars/humiaozuzu/awesome-flask?style=social)

本项目是一个精选的 Flask 资源与插件列表，旨在为开发者提供全面的开发支持。内容涵盖核心扩展（认证、数据库、缓存、邮件、任务队列等）、开发辅助工具（调试、测试、文档）、学习资源（教程、课程、书籍、视频）及实战项目案例与脚手架模板，帮助开发者高效构建、测试和部署基于 Flask 的 Web 应用。
