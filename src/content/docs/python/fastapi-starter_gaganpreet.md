### [gaganpreet fastapi-starter](https://github.com/gaganpreet/fastapi-starter)  ![GitHub Repo stars](https://img.shields.io/github/stars/gaganpreet/fastapi-starter?style=social)

FastAPI-Starter 是一个基于 FastAPI 的项目启动模板，遵循最佳实践与 KISS 原则。主要功能包括：使用 FastAPI Users 处理用户注册、模型与认证；集成 React-Admin 提供现代化管理后台；利用 OpenAPI Generator 自动生成 Typescript 客户端绑定；采用异步优先的 SQLAlchemy 2.0 进行数据库操作与迁移。项目内置 Github Actions 实现 Docker 镜像构建与自动化测试，配备 Dependabot 依赖管理、pytest 单元测试及 Cypress 集成测试，并配置 pre-commit 代码规范检查。提供前后端 Docker 镜像及生产部署方案，支持通过 Cookiecutter 或 Cruft 快速生成项目。
