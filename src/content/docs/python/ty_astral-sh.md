### [astral-sh ty](https://github.com/astral-sh/ty)  ![GitHub Repo stars](https://img.shields.io/github/stars/astral-sh/ty?style=social)

ty 是一款由 Astral 开发、使用 Rust 编写的超快 Python 类型检查器和语言服务器。

主要功能特点：
- **极速性能**：类型检查速度比 mypy 和 Pyright 快 10 到 100 倍。
- **语言服务器**：支持代码导航、自动补全、代码操作、自动导入、内联提示及悬停帮助，具备细粒度增量分析。
- **诊断与配置**：提供丰富上下文诊断，支持规则级别配置、文件级覆盖及抑制注释。
- **类型系统**：支持重新声明、部分类型化代码（渐进式支持），以及交集类型、类型缩小和可达性分析等高级特性。
- **编辑器集成**：兼容 VS Code、PyCharm、Neovim 等主流编辑器。

项目当前处于 Beta 版本。
