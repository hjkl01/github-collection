### [astral-sh ruff](https://github.com/astral-sh/ruff)  ![GitHub Repo stars](https://img.shields.io/github/stars/astral-sh/ruff?style=social)

Ruff 是一个使用 Rust 编写的极快速 Python 代码检查器和格式化工具。其速度比 Flake8 和 Black 等现有工具快 10 至 100 倍，旨在替代包括 Flake8、Black、isort、pydocstyle 在内的多种工具。核心功能包括：支持 800 多个内置规则（兼容 Flake8 插件生态）、自动修复错误、内置缓存加速、支持 `pyproject.toml` 配置、提供编辑器集成及预提交钩子。
