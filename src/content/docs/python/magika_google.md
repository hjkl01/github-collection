### [google magika](https://github.com/google/magika)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/magika?style=social)

Magika 是一款基于深度学习的 AI 文件类型检测工具。它采用轻量级优化模型（仅几 MB），能在单 CPU 上实现毫秒级推理，平均准确率达 99%，且识别速度不受文件大小影响。该工具支持 200 多种文件内容类型（涵盖文本和二进制格式），提供命令行（Rust）、Python API 及 JavaScript/Go 绑定。Magika 主要用于大规模安全扫描和文件分类，支持递归扫描、自定义输出格式及本地 Web 演示。
