### [faisalnazik Django-React-Redux-Boilerplate](https://github.com/faisalnazik/Django-React-Redux-Boilerplate)  ![GitHub Repo stars](https://img.shields.io/github/stars/faisalnazik/Django-React-Redux-Boilerplate?style=social)

这是一个 Django REST Framework 结合 React Redux 的全栈起步模板。

前端部分使用 React 函数组件与 Hooks，集成 Redux Store、Formik 表单验证、JWT 登录示例及服务器错误处理，并支持 Prettier 自动格式化。

后端部分基于 Django REST Framework 构建 API，使用 Django ORM 管理数据库，支持 JWT 认证（注册/登录）、限流设置、Pytest 测试及 Argon2 密码哈希。

旨在提供一个依赖更新完善的快速开发环境。
