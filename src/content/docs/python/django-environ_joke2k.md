### [joke2k django-environ](https://github.com/joke2k/django-environ)  ![GitHub Repo stars](https://img.shields.io/github/stars/joke2k/django-environ?style=social)

django-environ 是一个 Python 包，允许 Django 应用通过环境变量进行配置，遵循十二因子方法论。主要功能包括：从 .env 文件加载环境变量到系统环境，自动进行字符串到指定 Python 类型的转换，解析数据库和缓存等连接 URL 生成相应配置，以及支持 Docker 风格的文件配置。它能帮助开发者简化多环境部署，避免使用未版本控制的设置文件。
