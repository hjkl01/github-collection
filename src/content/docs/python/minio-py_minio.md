### [minio minio-py](https://github.com/minio/minio-py)  ![GitHub Repo stars](https://img.shields.io/github/stars/minio/minio-py?style=social)

MinIO Python 客户端 SDK 提供高层 API，用于访问 MinIO 对象存储或兼容 Amazon S3 的云服务。该库支持 Python 3.10+ 环境，核心功能包括连接存储服务（配置端点及密钥）、管理存储桶（如创建、检查存在性）以及执行对象操作（如上传文件）。项目包含安装指南、客户端配置示例及完整 API 参考。
