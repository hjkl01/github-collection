### [hustcc PyG2Plot](https://github.com/hustcc/PyG2Plot)  ![GitHub Repo stars](https://img.shields.io/github/stars/hustcc/PyG2Plot?style=social)

PyG2Plot 是 AntV G2Plot 的 Python3 绑定库，基于图形语法可轻松创建交互式统计图表。支持通过 `Plot` 类设置选项，渲染 HTML 文件、HTML 字符串，兼容 Jupyter Notebook 和 Jupyter Lab 预览，并支持通过 `JS` API 使用 JavaScript 语法进行回调。
