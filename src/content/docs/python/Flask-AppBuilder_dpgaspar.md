### [dpgaspar Flask-AppBuilder](https://github.com/dpgaspar/Flask-AppBuilder)  ![GitHub Repo stars](https://img.shields.io/github/stars/dpgaspar/Flask-AppBuilder?style=social)

Flask App Builder 是一个基于 Flask 的简易快速应用开发框架。主要功能包括：支持根据数据库模型自动生成 CRUD 操作及 REST API；内置基于角色的安全权限管理系统，支持 OAuth、LDAP 等多种认证方式；兼容 SQLAlchemy 支持的多种数据库及部分 MongoDB，提供自动生成表单（含文件上传）、菜单和列表视图；集成 Google 图表、Bootstrap UI 及多语言国际化功能。
