### [dbader schedule](https://github.com/dbader/schedule)  ![GitHub Repo stars](https://img.shields.io/github/stars/dbader/schedule?style=social)

schedule 是一个轻量级、无外部依赖的 Python 进程内任务调度库，提供人性化且简单的 API。它支持通过直观语法定期执行 Python 函数或其他可调用对象，可配置时间间隔、具体时刻或星期几等多种调度规则。用户只需在主循环中调用 `run_pending()` 即可自动运行待处理任务，适用于 Python 3.7 至 3.12 版本。
