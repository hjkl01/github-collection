### [simonw django-sql-dashboard](https://github.com/simonw/django-sql-dashboard)  ![GitHub Repo stars](https://img.shields.io/github/stars/simonw/django-sql-dashboard?style=social)

django-sql-dashboard 是一个 Django 插件，提供安全认证界面，用于直接对 PostgreSQL 数据库执行只读 SQL 查询。主要功能包括：

- 在浏览器中安全执行只读 SQL 查询并查看结果。
- 支持查询书签、共享链接及创建具有权限控制的仪表盘。
- 支持命名参数生成交互式表单，以及生成图表、进度条等可视化组件。
- 支持在查询结果中渲染 Markdown 和 HTML。
- 支持高效导出 CSV/TSV 文件及将结果复制到电子表格。
- 集成 Django 认证系统管理账户权限。

适用于临时数据分析、调试以及创建可共享的团队报告。
