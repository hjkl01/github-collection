### [plotly dash](https://github.com/plotly/dash)  ![GitHub Repo stars](https://img.shields.io/github/stars/plotly/dash?style=social)

Dash 是一个基于 Plotly.js、React 和 Flask 构建的 Python 框架，用于构建机器学习与数据科学 Web 应用。它将现代 UI 元素直接绑定到 Python 分析代码，支持声明式与响应式编程，内置丰富的交互式图表（含地图）并支持高度自定义样式。项目分为开源版和企业版，企业版提供应用部署管理、Kubernetes 扩展、身份认证、异步作业队列、低代码设计及大数据加速等企业级功能。
