### [RamezIssac django-slick-reporting](https://github.com/RamezIssac/django-slick-reporting)  ![GitHub Repo stars](https://img.shields.io/github/stars/RamezIssac/django-slick-reporting?style=social)

Django Slick Reporting 是一款基于 Django 的一站式报告引擎。它支持通过少量代码快速生成简单、分组、时间序列（日/周/月/年）和交叉表报告，并能通过单行代码创建图表。项目支持自定义复杂计算，具备高性能和易扩展性，提供 ReportView 视图封装与 ReportGenerator 底层生成器两种使用方式。
