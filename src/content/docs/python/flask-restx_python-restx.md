### [python-restx flask-restx](https://github.com/python-restx/flask-restx)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-restx/flask-restx?style=social)

Flask-RESTX 是 Flask 的扩展库，用于快速构建 REST API，作为 Flask-RESTPlus 的社区驱动分支。它提供装饰器和工具来描述 API 结构，并通过 Swagger 自动生成文档，支持最小化配置下的最佳实践。该库兼容 Python 3.9+ 及 Flask 2.0+，提供模型定义、命名空间管理及资源操作功能，简化 API 开发与文档生成流程。
