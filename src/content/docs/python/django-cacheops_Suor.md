### [Suor django-cacheops](https://github.com/Suor/django-cacheops)  ![GitHub Repo stars](https://img.shields.io/github/stars/Suor/django-cacheops?style=social)

django-cacheops 是一款 Django ORM 缓存扩展，支持查询集的自动或手动缓存。它使用 Redis 作为后端，具备自动细粒度的事件驱动缓存失效功能（监听模型变更信号）。该项目还支持通过装饰器缓存函数和视图，提供 Django 及 Jinja2 模板扩展，并包含透明事务支持、防狗群效应机制及性能优化功能，允许灵活配置缓存策略与超时。
