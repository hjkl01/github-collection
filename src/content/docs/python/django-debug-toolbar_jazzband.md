### [jazzband django-debug-toolbar](https://github.com/jazzband/django-debug-toolbar)  ![GitHub Repo stars](https://img.shields.io/github/stars/jazzband/django-debug-toolbar?style=social)

Django Debug Toolbar 是一个 Django 调试工具，通过可配置的面板显示当前请求和响应的调试信息，点击面板可查看详细内容。支持内置及第三方面板，兼容 Django 4.2 及以上版本，实验性支持异步视图。
