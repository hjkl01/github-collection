### [unfoldadmin django-unfold](https://github.com/unfoldadmin/django-unfold)

Unfold 是一个用于 Django 的现代管理界面主题，通过 Tailwind CSS 框架提供美观设计和增强功能。核心特性包括：  
- **界面优化**：支持暗色/亮色模式、可定制主题颜色、响应式布局；  
- **功能增强**：高级筛选（下拉、自动补全、时间范围等）、命令面板快速搜索、分页式内联表单、可排序内联数据；  
- **开发效率**：集成 Crispy Forms 模板、支持 WYSIWYG 编辑器（Trix）、兼容 ArrayField、支持模型字段动态显示/隐藏；  
- **扩展性**：与主流第三方库（如 django-guardian、django-import-export、django-constance 等）深度集成，提供仪表板工具和自定义组件（卡片、图表等）；  
- **部署支持**：可通过 PyPI 安装（`pip install django-unfold`），配置项通过 `settings.py` 调整，支持与原生 Django Admin 并行运行。  

使用方法：安装后通过配置 `settings.py` 启用主题，参考官方文档（[unfoldadmin.com](https://unfoldadmin.com)）进行二次开发和定制。
