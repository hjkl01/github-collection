### [agconti cookiecutter-django-rest](https://github.com/agconti/cookiecutter-django-rest)

**核心内容总结：**  
`cookiecutter-django-rest` 是一个用于快速生成现代、可扩展 REST API 项目的工具，提供认证、用户账户、文档和测试等完整功能支持。  

**使用方法：**  
1. 安装 [cookiecutter](https://github.com/audreyr/cookiecutter)；  
2. 运行命令 `cookiecutter gh:agconti/cookiecutter-django-rest` 生成项目框架。  

**主要特性：**  
- 支持 Python 3.13+、Django 5.0+；  
- 提供 Docker 镜像和 Docker Compose 本地开发环境；  
- 集成 PostgreSQL 16.4+、Django Rest Framework 和完整测试套件；  
- 自动化 CI/CD 流水线及依赖更新管理；  
- 项目结构简洁，包含认证、用户管理等基础模块，可快速扩展。
