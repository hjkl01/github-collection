### [django-cms django-cms](https://github.com/django-cms/django-cms)

**核心内容总结：**  
django CMS 是一个基于 Django 框架的开源企业内容管理系统，支持多语言、多站点、版本控制、SEO 优化等特性。主要功能包括层次化页面管理、草稿/发布工作流、前端内容编辑、可扩展的导航系统以及与其他应用的深度集成。用户可通过官方教程学习使用，查阅稳定版或最新版文档，或使用 Divio 提供的演示平台测试。项目由非营利的 django CMS 协会维护，社区贡献者可参与开发并获得奖励。
