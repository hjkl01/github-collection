### [sehmaschine django-grappelli](https://github.com/sehmaschine/django-grappelli)

**核心内容总结：**

Django Grappelli 是 Django 管理界面的网格布局皮肤，提供更美观、功能增强的替代方案。  
**功能**：基于网格的界面设计，支持自定义扩展，提升 Django Admin 的用户体验。  
**使用方法**：通过 PyPI 安装（`pip install django-grappelli`）或从 GitHub 获取代码。  
**主要特性**：  
- 兼容 Django 5.x（4.0.x 版本）和 Django 4.2 LTS（3.0.x 版本）  
- 提供响应式布局和现代 UI 设计  
- 文档和版本信息完整（支持最新稳定版 Django，不兼容开发版）  
- 开发分支明确区分不同 Django 版本的适配  

**兼容性说明**：仅支持 Django 稳定版本，不兼容 Django 的 master 分支。
