**项目核心内容总结：**  
Django Autocomplete Light 是一个 Django 插件，提供自动补全功能，支持 Django 4.2 及更早版本。主要特性包括：多选/多模型选择支持、通用外键和多对多关系处理、多种前端小部件（如 select2.js）兼容、动态创建小部件、自定义脚本支持、测试 API 以及完整的示例项目。项目提供文档、邮件列表和 Git 仓库，便于开发者使用和扩展。
