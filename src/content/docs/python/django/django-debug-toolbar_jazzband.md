### [jazzband django-debug-toolbar](https://github.com/jazzband/django-debug-toolbar)

**核心内容总结：**  
Django Debug Toolbar 是一个用于 Django 项目的调试工具栏，提供多个可配置面板，显示当前请求/响应的调试信息，点击面板可查看详细内容。  

**功能与特性：**  
- 支持 Django 4.2 及以上版本，实验性支持异步视图（不支持并发请求）。  
- 提供内置面板及社区贡献的第三方面板。  
- 文档包含安装与配置说明（[链接](https://django-debug-toolbar.readthedocs.io/)）。  

**使用方法：**  
通过 PyPI 安装（`pip install django-debug-toolbar`），按文档配置后集成到 Django 项目中。
