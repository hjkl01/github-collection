### [tiangolo fastapi](https://github.com/tiangolo/fastapi)  ![GitHub Repo stars](https://img.shields.io/github/stars/tiangolo/fastapi?style=social)

FastAPI 是一个基于 Python 标准类型提示的现代高性能 Web 框架，专用于构建 API。它具备与 NodeJS 和 Go 相当的高性能，通过类型提示减少代码量与错误，提升开发效率。框架自动处理数据验证、转换与序列化，生成符合 OpenAPI 和 JSON Schema 标准的交互式文档（Swagger UI 和 ReDoc），并提供优秀的编辑器支持。基于 Starlette 和 Pydantic 构建，支持同步和异步编程，设计直观，代码可直接用于生产环境。
