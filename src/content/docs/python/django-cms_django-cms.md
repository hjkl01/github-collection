### [django-cms django-cms](https://github.com/django-cms/django-cms)  ![GitHub Repo stars](https://img.shields.io/github/stars/django-cms/django-cms?style=social)

django CMS 是由 Django 驱动的轻量级、开源企业级内容管理系统。核心功能包括：支持国际化（i18n）和多站点部署；提供前端内联编辑，允许在渲染页面上直接操作；具备灵活的占位符和插件系统以复用内容组件；集成多种富文本编辑器；支持内容版本控制、编辑工作流及无头部署；拥有层级页面树与 SEO 友好 URL；支持可扩展的导航和应用集成（apphooks）。
