### [wtforms flask-wtf](https://github.com/wtforms/flask-wtf)

**核心内容总结：**  
Flask-WTF 是 Flask 框架与 WTForms 的集成库，提供以下功能：  
1. **核心功能**：支持 CSRF 保护、文件上传、reCAPTCHA 验证。  
2. **使用方法**：通过安装库后，在 Flask 应用中使用 WTForms 表单类，自动集成 CSRF 保护等特性。  
3. **主要特性**：简化 Flask 与 WTForms 的结合，提供安全性和便捷的表单处理能力。
