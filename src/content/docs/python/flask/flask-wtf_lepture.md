### [lepture flask-wtf](https://github.com/lepture/flask-wtf)

**Flask-WTF 核心内容总结**  
Flask-WTF 是一个用于简化 Flask 框架与 WTForms 集成的库，主要功能包括：  
1. **CSRF 保护**：自动处理跨站请求伪造攻击，保障表单安全；  
2. **文件上传支持**：提供便捷的文件上传功能；  
3. **reCAPTCHA 集成**：支持验证码功能，防止机器人滥用。  

**使用方法**：通过安装 Flask-WTF 包，结合 WTForms 的表单类，在 Flask 应用中直接调用其提供的扩展功能，如 CSRF 保护和文件上传处理。  

**主要特性**：与 Flask 无缝集成，简化表单处理流程，减少重复代码，提升开发效率。
