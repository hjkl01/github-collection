### [Blaizzy mlx-audio](https://github.com/Blaizzy/mlx-audio)  ![GitHub Repo stars](https://img.shields.io/github/stars/Blaizzy/mlx-audio?style=social)

MLX-Audio 是基于 Apple MLX 框架构建的音频处理库，专为 Apple Silicon（M 系列芯片）优化，提供快速高效的文本转语音（TTS）、语音转文本（STT）及语音转语音（STS）功能。项目支持多种模型架构（如 Kokoro、Whisper、Qwen3 等），具备多语言支持、语音克隆、速度控制、量化优化及语音活动检测能力。提供命令行工具、Python API、交互式 Web 界面及 OpenAI 兼容的 REST API，并包含适用于 iOS/macOS 的 Swift 包集成。
