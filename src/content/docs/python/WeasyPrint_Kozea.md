### [Kozea WeasyPrint](https://github.com/Kozea/WeasyPrint)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kozea/WeasyPrint?style=social)

WeasyPrint 是一个基于 Python 的视觉渲染引擎，旨在将 HTML 和 CSS 转换为 PDF 文档。它支持 Web 打印标准，可生成统计报告、发票、票据等。不同于 WebKit 或 Gecko，其 CSS 布局引擎专为分页设计且易于扩展，采用 BSD 自由软件许可。
