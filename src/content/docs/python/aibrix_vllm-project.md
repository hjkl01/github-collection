### [vllm-project aibrix](https://github.com/vllm-project/aibrix)  ![GitHub Repo stars](https://img.shields.io/github/stars/vllm-project/aibrix?style=social)

AIBrix 是一款基于 Kubernetes 的开源云原生解决方案，旨在构建可扩展的生成式 AI 推理基础设施。它针对企业需求优化了大型语言模型（LLM）的部署、管理与扩展。核心功能包括高密度 LoRA 管理、LLM 网关与路由、动态自动伸缩、统一 AI 运行时、分布式推理、分布式 KV 缓存共享、异构 GPU 推理及硬件故障检测。项目致力于提供高效、低成本的 LLM 推理服务控制平面。
