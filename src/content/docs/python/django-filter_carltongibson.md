### [carltongibson django-filter](https://github.com/carltongibson/django-filter)  ![GitHub Repo stars](https://img.shields.io/github/stars/carltongibson/django-filter?style=social)

Django Filter 是一个可复用的 Django 应用，支持通过 URL 参数声明式地对 QuerySet 进行动态过滤。它提供类似 Django admin list_filter 的界面和类似 ModelForms 的 API。该库既支持标准 Django 视图，也支持 Django REST Framework（DRF），可为 DRF 提供定制的 FilterSet 和过滤后端。
