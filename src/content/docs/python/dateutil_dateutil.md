### [dateutil dateutil](https://github.com/dateutil/dateutil)  ![GitHub Repo stars](https://img.shields.io/github/stars/dateutil/dateutil?style=social)

dateutil 是 Python 标准 datetime 模块的扩展库。主要功能包括：计算相对时间差（如下一月、下周一）；计算两个日期对象间的相对差；基于 iCalendar 规范的灵活日期循环规则；解析多种格式的日期字符串；支持多种时区实现（含 tzfile、环境变量、Windows 注册表等）；提供基于 Olson 数据库的世界时区信息；计算任意年份的复活节日期（支持西方、东正教或儒略历算法）。
