### [yourlabs django-autocomplete-light](https://github.com/yourlabs/django-autocomplete-light)  ![GitHub Repo stars](https://img.shields.io/github/stars/yourlabs/django-autocomplete-light?style=social)

本项目是一个 Django 自动完成库，支持 Django 4.2+ 及更早版本。核心功能涵盖：支持模型和字段的单选/多选；支持通用外键及多对多关系；兼容多种小部件（如 Select2.js）及动态创建；支持自定义选项和依赖选择；提供测试 API 和示例代码。
