### [Nelarius imnodes](https://github.com/Nelarius/imnodes)  ![GitHub Repo stars](https://img.shields.io/github/stars/Nelarius/imnodes?style=social)

imnodes 是一个专为 dear imgui 设计的小型无依赖节点编辑器扩展。它提供即时模式接口，允许用户构建自定义节点编辑器，支持创建节点、链接及输入/输出引脚，并可在节点内嵌套 ImGui 控件。项目提供链接创建、节点选择与悬停等交互事件处理，支持自定义 UI 样式，内置交互式迷你地图以辅助大画布导航。用户可通过复制源码文件快速集成，并支持通过配置文件进行深度定制。
