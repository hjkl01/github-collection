### [lk-geimfari mimesis](https://github.com/lk-geimfari/mimesis)  ![GitHub Repo stars](https://img.shields.io/github/stars/lk-geimfari/mimesis?style=social)

Mimesis 是一款高性能的 Python 虚假数据生成库。它支持 46 种语言区域，能够生成广泛类型的虚假数据。项目具备高可扩展性，支持自定义数据提供者和字段处理器；提供基于模式的数据生成器，支持产生复杂结构及带引用的关系数据。此外，Mimesis 拥有清晰的文档、良好的编辑器支持（全类型提示与自动补全），用户只需导入对应的数据提供者即可快速生成如姓名、邮箱、电话等测试数据。
