### [python-poetry poetry](https://github.com/python-poetry/poetry)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-poetry/poetry?style=social)

Poetry 是一个简化 Python 项目打包和依赖管理的工具。它通过 `pyproject.toml` 统一替代 `setup.py`、`requirements.txt` 等传统配置文件，支持依赖的声明、管理和安装，确保项目环境一致性，并提供脚本定义、可选依赖及依赖组管理功能。
