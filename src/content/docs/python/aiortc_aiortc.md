### [aiortc aiortc](https://github.com/aiortc/aiortc)  ![GitHub Repo stars](https://img.shields.io/github/stars/aiortc/aiortc?style=social)

aiortc 是一个基于 asyncio 的 Python 库，用于实现 Web Real-Time Communication (WebRTC) 和 Object Real-Time Communication (ORTC)。它支持音频、视频和数据信道的收发，涵盖 SDP、ICE、DTLS、SRTP 及纯 Python 实现的 SCTP 等协议功能。该库代码简洁易读，互操作性经过 Chrome 和 Firefox 验证，易于与 Python 生态（如 OpenCV）集成，适合用于理解 WebRTC 原理或构建实时通信产品。项目采用 BSD 许可。
