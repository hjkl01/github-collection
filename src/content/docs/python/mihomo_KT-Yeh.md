### [KT-Yeh mihomo](https://github.com/KT-Yeh/mihomo)  ![GitHub Repo stars](https://img.shields.io/github/stars/KT-Yeh/mihomo?style=social)

该项目是一个用于《崩坏：星穹铁道》的 Python 库，基于 Pydantic 模型解析 Mihomo API 数据。功能包括：异步获取玩家详情与角色信息（支持 V1/V2 模型）、自动获取角色图标 URL、提供角色数据去重与合并工具，以及支持 JSON 和 Pickle 格式的数据持久化。
