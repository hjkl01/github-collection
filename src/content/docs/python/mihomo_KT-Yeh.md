### [KT-Yeh mihomo](https://github.com/KT-Yeh/mihomo)  ![GitHub Repo stars](https://img.shields.io/github/stars/KT-Yeh/mihomo?style=social)

mihomo 是一个用于解析《崩坏：星穹铁道》米哈游 API 数据的 Python 库，基于 Pydantic 模型，支持类型提示和自动补全。功能包括：提供 V1/V2 数据版本接口，获取玩家及角色详情（支持立绘 URL 转换）；内置去除重复角色、合并数据等工具；支持 Pickle 或 JSON 数据持久化。
