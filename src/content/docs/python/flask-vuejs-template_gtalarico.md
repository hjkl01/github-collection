### [gtalarico flask-vuejs-template](https://github.com/gtalarico/flask-vuejs-template)  ![GitHub Repo stars](https://img.shields.io/github/stars/gtalarico/flask-vuejs-template?style=social)

这是一个基于 Flask 和 Vue.js 的 Web 应用开发模板。后端采用 Flask 1.0 和 Flask-RestPlus 构建 REST API，前端基于 Vue-cli 3、Yarn、Vuex 和 Vue Router 构建单页应用。项目内置 PyTest 测试套件，通过 Axios 实现前后端 Ajax 通信。模板预配置了 Heroku 与 Gunicorn 环境，支持一键部署，采用前后端分离架构，涵盖从开发、构建到部署的完整流程。
