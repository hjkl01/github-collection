### [huggingface transformers](https://github.com/huggingface/transformers)  ![GitHub Repo stars](https://img.shields.io/github/stars/huggingface/transformers?style=social)

Transformers 是 Hugging Face 提供的统一模型定义框架，用于文本、视觉、音频、视频及多模态模型的最先进推理与训练。它统一了模型定义，使模型兼容 PyTorch、JAX、TensorFlow 等多种训练框架和推理引擎，并提供超过 100 万个预训练模型检查点。通过统一的 Pipeline API，开发者可轻松实现文本生成、语音识别、图像分类等任务，并支持模型跨框架迁移。该库旨在降低使用门槛，提供高效定制化工具，助力研究人员和开发者快速构建和部署项目。
