### [celery celery](https://github.com/celery/celery)  ![GitHub Repo stars](https://img.shields.io/github/stars/celery/celery?style=social)

Celery 是一个用 Python 编写的分布式任务队列系统，旨在跨线程或机器分配工作负载。它利用消息代理（如 RabbitMQ、Redis）在客户端与 Worker 之间传递任务消息，实现异步任务调度。该系统具有高可用性、高性能、简单灵活的特点，支持多种传输协议、并发模式及结果存储后端，并易于与 Django、Flask 等主流 Web 框架集成。
