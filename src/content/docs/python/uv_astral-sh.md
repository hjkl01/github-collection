### [astral-sh uv](https://github.com/astral-sh/uv)  ![GitHub Repo stars](https://img.shields.io/github/stars/astral-sh/uv?style=social)

`uv` 是一个用 Rust 编写的高性能 Python 包和项目管理器，旨在替代 `pip`、`poetry`、`pyenv`、`virtualenv` 等多个工具，运行速度比 `pip` 快 10 到 100 倍。其核心功能包括统一管理项目依赖与虚拟环境、支持脚本运行及内联依赖声明、执行与安装命令行工具、安装和切换 Python 版本、提供兼容 `pip` 的接口以及通过全局缓存优化磁盘空间。该项目支持 macOS、Linux 和 Windows 平台。
