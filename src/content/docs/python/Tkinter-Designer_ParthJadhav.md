### [ParthJadhav Tkinter-Designer](https://github.com/ParthJadhav/Tkinter-Designer)  ![GitHub Repo stars](https://img.shields.io/github/stars/ParthJadhav/Tkinter-Designer?style=social)

Tkinter Designer 是一款基于 Figma 的 Python Tkinter 图形界面（GUI）自动生成工具。用户可在 Figma 中设计界面，通过输入文件链接和 API 令牌，工具会自动解析设计并生成对应的 Tkinter 代码及资源文件。它支持拖拽操作和多框架设计，旨在比手写代码更高效、便捷地创建美观的 GUI 界面。
