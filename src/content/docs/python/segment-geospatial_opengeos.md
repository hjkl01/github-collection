### [opengeos segment-geospatial](https://github.com/opengeos/segment-geospatial)  ![GitHub Repo stars](https://img.shields.io/github/stars/opengeos/segment-geospatial?style=social)

SamGeo 是一个基于 Segment Anything Model (SAM) 的 Python 包，旨在简化地理空间数据的分割与分析。主要功能包括：从 TMS 服务器下载地图瓦片并生成 GeoTIFF；利用 SAM、HQ-SAM、SAM2/3 及 Fast SAM 等模型对遥感影像进行分割；支持通过文本提示、交互标记或现有矢量文件进行分割；结果可保存为 GeoPackage、Shapefile、GeoJSON 等矢量格式，并支持交互式地图可视化及时间序列影像处理。此外，项目还提供 QGIS 插件及 ArcGIS 集成方案。
