### [jackzhenguo python-small-examples](https://github.com/jackzhenguo/python-small-examples)  ![GitHub Repo stars](https://img.shields.io/github/stars/jackzhenguo/python-small-examples?style=social)

该项目是 Python 经典小案例集合，提供涵盖基础至进阶的实用代码。核心功能涵盖：基础操作（运算、类型转换、数据结构）、函数模块（内置函数、迭代器、生成器、装饰器）、面向对象编程、正则表达式、数据可视化（支持 turtle、matplotlib、pyecharts 等库）、数据分析（NumPy、Pandas）、系统工具（文件操作、时间处理、多线程、数据库）及工作常用算法案例。内容按难度分级，旨在通过简明实例辅助 Python 学习。
