### [Avaiga taipy](https://github.com/Avaiga/taipy)  ![GitHub Repo stars](https://img.shields.io/github/stars/Avaiga/taipy?style=social)

Taipy 是一个基于 Python 的数据与人工智能 Web 应用开发框架，旨在帮助数据科学家和机器学习工程师仅使用 Python 即可快速创建生产级应用，无需学习新语言。其核心功能涵盖用户界面生成、数据集成、流程编排、场景管理、权限控制及定时任务。Taipy 生态系统包含 Designer、Studio 及部署工具，简化了应用的开发、部署、版本管理和运维流程。
