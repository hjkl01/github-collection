### [django channels](https://github.com/django/channels)  ![GitHub Repo stars](https://img.shields.io/github/stars/django/channels?style=social)

Django Channels 是官方 Django 项目，旨在扩展 Django 以支持 WebSocket、长轮询 HTTP、任务卸载及其他异步功能。它采用熟悉的 Django 设计模式和灵活框架，允许自定义行为或开发支持特定协议。项目主要由 Daphne（服务器）、channels_redis（Redis 后端）和 asgiref（ASGI 库）等组件构成。
