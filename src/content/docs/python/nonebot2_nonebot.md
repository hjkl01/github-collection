### [nonebot nonebot2](https://github.com/nonebot/nonebot2)  ![GitHub Repo stars](https://img.shields.io/github/stars/nonebot/nonebot2?style=social)

NoneBot2 是一个现代、跨平台、可扩展的 Python 异步聊天机器人框架。它基于 Python 异步特性和 100% 类型注解，支持通过协议适配器连接多种聊天平台（如 QQ、Telegram、飞书、GitHub 等）及多种 Web 驱动框架。框架提供 NB-CLI 脚手架简化开发流程，拥有丰富的社区资源和插件生态，致力于提供便捷、可靠的机器人开发体验。
