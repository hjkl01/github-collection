### [wbopan moffee](https://github.com/wbopan/moffee)  ![GitHub Repo stars](https://img.shields.io/github/stars/wbopan/moffee?style=social)

moffee 是一款基于 Python 的开源幻灯片制作工具，旨在将 Markdown 文档转换为专业、整洁的演示文稿。其核心功能包括：

1. 自动化排版：自动管理布局、分页和样式，让用户专注于内容创作。
2. 灵活语法：支持标准 Markdown 及 Obsidian 扩展语法，使用 `---`、`<->`、`===` 等符号控制分页与元素水平/垂直布局。
3. 实时预览与导出：提供网页实时预览，支持启动幻灯片放映，并可导出为 PDF 或 HTML 格式。
4. 高度定制：内置多种主题，支持通过 YAML 前端内容配置纵横比、样式及 CSS 属性，也支持局部样式装饰。
