### [faif python-patterns](https://github.com/faif/python-patterns)  ![GitHub Repo stars](https://img.shields.io/github/stars/faif/python-patterns?style=social)

本项目是一个 Python 设计模式和惯用法的代码集合。它包含了创造型、结构型、行为型、可测试性设计、基础模式及其他多种设计模式的具体实现。此外，项目还列出了反模式指南（说明 Python 中不推荐的实践，如单例、上帝对象等），并提供视频资源和贡献规范。旨在帮助开发者根据具体场景权衡利弊，正确选择和应用设计模式。
