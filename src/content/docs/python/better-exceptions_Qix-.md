### [Qix- better-exceptions](https://github.com/Qix-/better-exceptions)  ![GitHub Repo stars](https://img.shields.io/github/stars/Qix-/better-exceptions?style=social)

better-exceptions 是一个 Python 库，用于自动美化并增强异常信息的显示，帮助开发者更高效地排查错误。通过 pip 安装并设置 BETTER_EXCEPTIONS 环境变量即可启用。它支持 Python REPL、unittest 框架及 Django 项目集成，提供异常格式化与输出长度自定义功能。注意在生产环境中应关闭该变量以避免敏感数据泄露。
