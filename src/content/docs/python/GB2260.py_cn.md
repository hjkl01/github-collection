### [cn GB2260.py](https://github.com/cn/GB2260.py)

**项目功能**：提供基于GB/T 2260标准的中国行政区划查询功能，支持通过行政区划代码（如360426）快速获取省、市、县等层级信息。  

**使用方法**：  
1. 安装：`pip install GB2260`  
2. 查询：通过`gb2260.get(代码)`获取对象，例如`gb2260.get(360426)`返回江西省九江市德安县的信息。  
3. 访问属性：可获取行政区划代码（`code`）、名称（`name`）、层级类型（如`is_county`）、上级/下级行政区划对象（如`province`、`prefecture`）。  

**主要特性**：  
- 支持省、市、县三级层级关系查询；  
- 提供`stack()`方法迭代输出完整层级路径（如从省到县的逐级信息）；  
- 数据包含名称、代码、层级类型等关键字段。
