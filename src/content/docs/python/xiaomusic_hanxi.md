### [hanxi xiaomusic](https://github.com/hanxi/xiaomusic)  ![GitHub Repo stars](https://img.shields.io/github/stars/hanxi/xiaomusic?style=social)

XiaoMusic 是一款让小米小爱音箱支持在线听歌与本地播放的音乐服务软件。
1. **语音控制**：支持播放/停止、切歌、循环/随机模式、歌单管理及收藏等语音指令。
2. **音乐源**：支持本地文件、在线歌曲搜索下载及网络歌单播放。
3. **Web 管理**：提供 Web 配置页面，支持小米账号及设备设置，无需环境变量。
4. **设备兼容**：广泛支持多款小爱音箱型号，兼容 mp3、flac 等音频格式。
5. **部署方式**：支持 Docker、Docker Compose 或 Pip 安装。
