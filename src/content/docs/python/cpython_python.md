### [python cpython](https://github.com/python/cpython)  ![GitHub Repo stars](https://img.shields.io/github/stars/python/cpython?style=social)

本项目为 Python 3.15.0 alpha 6 版本的 CPython 参考实现源代码仓库。主要提供 Python 解释器源码，支持在多种操作系统（Unix、Linux、macOS、Windows）上进行构建、安装和测试。文档涵盖构建优化指南（PGO、LTO）、多版本安装方案、测试执行方法、开发者贡献规范、在线文档资源及版权许可证信息。
