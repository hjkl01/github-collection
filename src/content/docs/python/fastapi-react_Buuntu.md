### [Buuntu fastapi-react](https://github.com/Buuntu/fastapi-react)  ![GitHub Repo stars](https://img.shields.io/github/stars/Buuntu/fastapi-react?style=social)

这是一个 FastAPI 与 React 全栈项目的快速启动模板。后端采用 FastAPI、PostgreSQL、SQLAlchemy 和 Celery，集成 JWT 认证、数据库迁移、后台任务及 Pytest 测试。前端基于 React (TypeScript)、MaterialUI 和 react-admin 构建管理后台，包含路由和认证辅助工具。项目通过 Docker Compose 容器化部署，使用 Nginx 反向代理，并集成 Flower 任务监控。旨在提供开箱即用的现代全栈开发环境，简化项目初始化、认证、测试和部署流程。
