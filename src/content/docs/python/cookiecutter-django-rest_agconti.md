### [agconti cookiecutter-django-rest](https://github.com/agconti/cookiecutter-django-rest)  ![GitHub Repo stars](https://img.shields.io/github/stars/agconti/cookiecutter-django-rest?style=social)

cookiecutter-django-rest 是一个用于快速构建可扩展 REST API 的项目脚手架工具。它基于 Python 3.13+、Django 5.0+ 和 PostgreSQL 16.4+ 技术栈，提供 Docker 化的本地开发环境。项目内置认证、用户管理、文档及完整测试覆盖，集成 Django Rest Framework，并通过持续集成与自动化依赖更新确保代码质量与安全，帮助开发者快速启动高质量的 API 开发。
