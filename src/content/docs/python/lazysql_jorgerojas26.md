### [jorgerojas26 lazysql](https://github.com/jorgerojas26/lazysql)  ![GitHub Repo stars](https://img.shields.io/github/stars/jorgerojas26/lazysql?style=social)

Lazysql 是一款基于 Go 语言开发的跨平台终端数据库管理工具（TUI）。支持 MySQL、PostgreSQL、SQLite、MSSQL 等关系型数据库。核心功能包括：多连接与标签管理、Vim 风格快捷键、内置 SQL 编辑器、表格数据浏览与编辑（增删改查）、行过滤与排序、结果导出 CSV、查询历史管理、JSON 查看器，以及支持自定义前置连接命令（如 SSH 隧道）和通过 TOML 配置文件管理连接参数。
