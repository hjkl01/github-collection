### [mrevutskyi flask-restless-ng](https://github.com/mrevutskyi/flask-restless-ng)  ![GitHub Repo stars](https://img.shields.io/github/stars/mrevutskyi/flask-restless-ng?style=social)

Flask-Restless-NG 是一个 Flask 扩展，用于为 SQLAlchemy 或 Flask-SQLAlchemy 定义的数据模型创建符合 JSON API 规范的 URL 端点。它是 Flask-Restless 的改进分支，支持更新的 Flask 和 SQLAlchemy 版本，提供更快的 JSON 序列化速度，能够快速构建 RESTful API 接口。
