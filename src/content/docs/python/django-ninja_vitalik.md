### [vitalik django-ninja](https://github.com/vitalik/django-ninja)  ![GitHub Repo stars](https://img.shields.io/github/stars/vitalik/django-ninja?style=social)

Django Ninja 是一个基于 Django 和 Python 类型提示的 REST API 快速构建框架。它利用 Pydantic 和异步支持实现高性能执行，通过类型提示自动验证请求参数，基于 OpenAPI 和 JSON Schema 标准自动生成交互式文档，并与 Django 核心及 ORM 深度集成，设计直观且适用于生产环境。
