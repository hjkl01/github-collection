### [django-guardian django-guardian](https://github.com/django-guardian/django-guardian)  ![GitHub Repo stars](https://img.shields.io/github/stars/django-guardian/django-guardian?style=social)

django-guardian 是一个基于 Django 授权后端实现对象级权限（per-object permissions）的第三方库。其主要功能包括：允许为特定用户或组分配针对具体模型对象的权限，而非全局权限；通过 `ObjectPermissionBackend` 集成至 Django 认证系统；提供 `GuardedModelAdmin` 类以支持在 Django Admin 面板中管理对象权限；并原生支持 django-unfold 管理后台主题。
