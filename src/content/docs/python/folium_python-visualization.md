### [python-visualization folium](https://github.com/python-visualization/folium)

**核心内容总结：**

folium 是一个 Python 库，结合 Python 的数据处理能力和 Leaflet.js 的地图功能，用于将数据可视化为交互式地图。用户可通过 pip 或 conda 安装（`pip install folium` 或 `conda install -c conda-forge folium`），并参考官方文档（https://python-visualization.github.io/folium/latest/）进行使用。主要特性包括支持多种插件（如 GeoJSON 快速渲染、地图信用信息展示等），以及活跃的社区贡献和扩展生态（如与 GeoPandas、Streamlit 等工具集成）。
