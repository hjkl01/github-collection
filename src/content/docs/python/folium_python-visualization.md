### [python-visualization folium](https://github.com/python-visualization/folium)  ![GitHub Repo stars](https://img.shields.io/github/stars/python-visualization/folium?style=social)

folium 是一个 Python 库，结合了 Python 生态系统的数据处理能力与 Leaflet.js 库的制图能力，允许用户在 Python 中操作数据，并将其可视化为 Leaflet 地图。
