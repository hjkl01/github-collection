### [9001 copyparty](https://github.com/9001/copyparty)  ![GitHub Repo stars](https://img.shields.io/github/stars/9001/copyparty?style=social)

copyparty 是一个基于 Python 的轻量级跨平台文件服务器项目。

主要功能：
1.  **Web 服务**：通过任意 Web 浏览器访问，支持断点续传上传和下载。
2.  **多协议支持**：兼容 HTTP(S)、WebDAV、SFTP、FTP(S)、TFTP 及 SMB/CIFS 协议。
3.  **文件管理**：支持拖拽上传、文件重命名、移动、批量下载（ZIP/TAR）、文件去重与自动压缩。
4.  **多媒体浏览**：内置音频/视频播放器、图片画廊、文本及 Markdown 查看器。
5.  **高级搜索**：支持按文件名、属性、元数据标签（如 MP3 信息）进行搜索。
6.  **权限与安全**：提供细粒度用户权限控制、IP 限制、身份提供商（IdP）集成及文件密钥防猜解。
7.  **部署便捷**：仅需 Python 环境，依赖可选，支持自包含二进制（SFX）、Docker 及主流包管理器。
8.  **辅助工具**：提供 Zeroconf 服务发现、QR 码、Prometheus 监控指标及反向代理支持。
