### [samuelcolvin pydantic](https://github.com/samuelcolvin/pydantic)  ![GitHub Repo stars](https://img.shields.io/github/stars/samuelcolvin/pydantic?style=social)

Pydantic 是一个基于 Python 类型提示的数据验证库。它快速、可扩展，支持使用 Python 类型注解定义数据格式，自动完成数据的验证、解析和转换。项目包含 V1 和 V2 版本（V2 为重写优化版），支持渐进式升级，并能与 linters、IDE 等工具良好集成。
