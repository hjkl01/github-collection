### [keon algorithms](https://github.com/keon/algorithms)  ![GitHub Repo stars](https://img.shields.io/github/stars/keon/algorithms?style=social)

这是一个用 Python 3 实现的轻量级、简洁且文档完善的数据结构与算法库，适合学习与查阅。每个模块文件自包含，附文档字符串、类型提示及复杂度说明。

功能涵盖：
1. **数据结构**：AVL 树、B 树、红黑树、二叉搜索树、堆、哈希表、链表、队列、栈、图、Trie、并查集等。
2. **算法实现**：排序、搜索、图算法（最短路径、最小生成树、流等）、动态规划、回溯、位运算、压缩、字符串匹配、数学运算、矩阵操作等。

项目结构清晰，支持通过 pip 安装并直接导入使用。
