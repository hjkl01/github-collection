### [flasgger flasgger](https://github.com/flasgger/flasgger)  ![GitHub Repo stars](https://img.shields.io/github/stars/flasgger/flasgger?style=social)

Flasgger 是一个 Flask 扩展，用于从 Flask API 视图中自动提取 OpenAPI 规范（Swagger）。它内置了 Swagger UI，允许用户通过浏览器查看和交互 API 资源。该扩展支持通过 docstring、外部 YAML 文件、Python 字典或 Marshmallow Schemas 来定义 API 规范。此外，它提供自动化的请求数据验证功能，确保传入数据符合定义的 Schema，并兼容 Flask-RESTful 及 OpenAPI 3.0。项目还支持自定义 Swagger UI 版本、模板配置及数据验证处理逻辑。
