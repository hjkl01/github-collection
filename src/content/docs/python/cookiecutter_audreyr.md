### [audreyr cookiecutter](https://github.com/audreyr/cookiecutter)  ![GitHub Repo stars](https://img.shields.io/github/stars/audreyr/cookiecutter?style=social)

Cookiecutter 是一款跨平台的命令行工具，用于从项目模板快速生成项目。用户无需 Python 知识即可使用，支持从本地或远程仓库加载模板，通过 cookiecutter.json 自定义变量，并利用钩子脚本处理初始化任务。它兼容多种编程语言和框架，既可作为命令行工具运行，也可作为 Python 库在代码中调用。
