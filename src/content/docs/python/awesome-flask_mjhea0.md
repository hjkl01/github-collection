### [mjhea0 awesome-flask](https://github.com/mjhea0/awesome-flask)  ![GitHub Repo stars](https://img.shields.io/github/stars/mjhea0/awesome-flask?style=social)

该项目是一个精心策划的 Flask Web 框架资源精选列表。内容涵盖第三方扩展（如 Admin、API、认证、数据库、安全等）、学习资源（教程、课程、书籍、视频）、托管方案（PaaS、IaaS、Serverless）以及开源项目模板。旨在为 Flask 开发者提供全面的技术选型参考、开发工具支持和社区学习材料，助力构建基于 Python 的 Web 应用。
