### [pyecharts pyecharts](https://github.com/pyecharts/pyecharts)  ![GitHub Repo stars](https://img.shields.io/github/stars/pyecharts/pyecharts?style=social)

pyecharts 是一个基于 Apache ECharts 的 Python 数据可视化库。它提供简洁的链式 API，支持 30 多种常见图表类型和 400 多种地图文件。项目支持在本地生成 HTML 或图片，兼容 Jupyter Notebook 及 Flask、Django 等主流 Web 框架，具备高度灵活的配置项，适用于数据分析与可视化展示。
