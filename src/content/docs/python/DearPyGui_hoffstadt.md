### [hoffstadt DearPyGui](https://github.com/hoffstadt/DearPyGui)  ![GitHub Repo stars](https://img.shields.io/github/stars/hoffstadt/DearPyGui?style=social)

Dear PyGui 是一个现代、快速且强大的 Python GUI 框架。基于 Dear ImGui 构建，底层使用 C/C++ 并通过 GPU 渲染，支持跨平台（Windows、Linux、macOS、Raspberry Pi）。核心功能包括：完整的主题样式控制、GPU 加速的高性能界面、异步操作支持、百万级数据点快速图表绘制（支持 60fps 及交互）、节点编辑器、内置演示及开发者调试工具。此外还包含绘图 API 与画布功能，可用于自定义绘制、数据分析及 2D 游戏开发，采用 MIT 协议开源。
