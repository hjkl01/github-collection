### [jrief django-admin-sortable2](https://github.com/jrief/django-admin-sortable2)  ![GitHub Repo stars](https://img.shields.io/github/stars/jrief/django-admin-sortable2?style=social)

django-admin-sortable2 是一个 Django 扩展包，为 Django Admin 的列表、Stacked 和 Tabular 内联视图提供通用的拖拽排序功能。它通过提供 mixin 类增强现有的 ModelAdmin 和 Inline 类，无需修改模型基类即可集成。版本 2.0 基于 Sortable.JS 重写，移除了 jQuery 依赖，并支持同时拖拽多个项目。
