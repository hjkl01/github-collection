### [karec cookiecutter-flask-restful](https://github.com/karec/cookiecutter-flask-restful)  ![GitHub Repo stars](https://img.shields.io/github/stars/karec/cookiecutter-flask-restful?style=social)

这是一个基于 Cookiecutter 的 Flask RESTful API 快速启动模板，提供包含应用工厂、蓝图及数据库配置的基础结构。项目集成 Flask、Flask-RESTful、SQLAlchemy、Marshmallow 和 Flask-JWT-Extended 技术栈，支持 JWT 访问与刷新令牌认证及简单分页工具。具备 Flask CLI 集成、数据库迁移、环境变量配置及 OpenAPI/Swagger/ReDoc 文档生成能力。内置基于 pytest、factoryboy 和 tox 的单元测试支持，兼容 Docker 开发环境，可选集成 Celery 异步任务，并支持 WSGI 服务器部署及 Makefile 管理。
