### [gae-init gae-init](https://github.com/gae-init/gae-init)  ![GitHub Repo stars](https://img.shields.io/github/stars/gae-init/gae-init?style=social)

gae-init 是一个用于 Google App Engine 的 Python 快速开发样板。它集成了 Flask、RESTful、Bootstrap 等技术与库，支持项目初始化、本地开发、自动化测试及 GAE 部署，提供包含 OpenID 登录的完整功能，兼容跨平台环境。
