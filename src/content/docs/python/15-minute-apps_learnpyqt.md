### [learnpyqt 15-minute-apps](https://github.com/learnpyqt/15-minute-apps)

**核心内容总结：**  
该项目提供大量使用Python编写的GUI示例，涵盖完整应用（如扫雷、画图工具）和可复用的小部件代码片段，支持PyQt6/PySide6/PySide2/PyQt5库。用户可自由使用、修改代码。  

**使用方法：**  
1. 进入示例文件夹，运行 `pip3 install -r requirements.txt` 安装依赖（通常为GUI库）。  
2. 执行 `python3 main.py` 运行应用。  

**主要特性：**  
- 含多个功能完整应用（如浏览器、媒体播放器、天气工具）及交互小部件（如调色板、密码输入框、动态切换按钮）。  
- 提供教程链接，指导使用PyQt/PySide开发GUI。  
- 代码采用MIT许可证，允许商业或非商业项目使用，需保留许可证声明。  
- 图标由Yusuke Kamiyama设计。
