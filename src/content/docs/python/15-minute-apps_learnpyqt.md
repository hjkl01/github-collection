### [learnpyqt 15-minute-apps](https://github.com/learnpyqt/15-minute-apps)  ![GitHub Repo stars](https://img.shields.io/github/stars/learnpyqt/15-minute-apps?style=social)

该项目包含数百个 Python GUI 示例，支持 PyQt6、PySide6、PySide2 和 PyQt5 框架。内容涵盖完整应用（如扫雷、画图、浏览器、计算器等）及可复用 UI 组件（如图形均衡器、调色板等），并提供教程代码片段。代码采用 MIT 许可证，允许自由复用和修改用于商业或非商业项目。用户安装依赖后可直接运行示例程序。
