### [jfinkels flask-restless](https://github.com/jfinkels/flask-restless)  ![GitHub Repo stars](https://img.shields.io/github/stars/jfinkels/flask-restless?style=social)

Flask-Restless 是一个 Flask 扩展，用于为 SQLAlchemy 或 Flask-SQLAlchemy 模型创建符合 JSON API 规范的 URL 端点，从而构建 RESTful JSON API。它支持数据库搜索、模型序列化与反序列化，并通过视图类实现完整的 JSON API 接口。该项目目前不再维护。
