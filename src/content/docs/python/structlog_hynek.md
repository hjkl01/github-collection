### [hynek structlog](https://github.com/hynek/structlog)  ![GitHub Repo stars](https://img.shields.io/github/stars/hynek/structlog?style=social)

structlog 是 Python 的生产级结构化日志库，秉持“简单、强大、快速”的理念。它基于函数和字典设计，提供熟悉的 API。用户可选择直接输出日志或转发至标准日志系统，支持 JSON、logfmt 及控制台格式化等多种输出格式。该库性能优异，兼容 asyncio 等现代特性，自 2013 年以来广泛应用于各规模生产环境。
