### [graphql-python flask-graphql](https://github.com/graphql-python/flask-graphql)  ![GitHub Repo stars](https://img.shields.io/github/stars/graphql-python/flask-graphql?style=social)

Flask-GraphQL 是一个为 Flask 应用提供 GraphQL 支持的库。它通过 GraphQLView 类，允许开发者在 Flask 应用中快速注册 GraphQL 端点。核心功能包括：支持 GraphiQL IDE 调试与探索、启用批量查询与订阅、配置中间件、Schema、上下文及根值。该库兼容 Graphene v3 和 graphql-core，便于将 GraphQL 服务部署至 Flask 框架。
