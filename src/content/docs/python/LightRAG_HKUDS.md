### [HKUDS LightRAG](https://github.com/HKUDS/LightRAG)  ![GitHub Repo stars](https://img.shields.io/github/stars/HKUDS/LightRAG?style=social)

LightRAG 是一个简单快速的检索增强生成（RAG）系统，核心通过结合知识图谱技术提升检索质量。它支持多种大语言模型、嵌入模型及重排序模型，提供灵活的存储方案（如文件系统、Neo4J、PostgreSQL、MongoDB 等）。系统集成 RAG-Anything 支持多模态文档处理，具备实体与关系的增删改查、合并及引用溯源功能。此外，LightRAG 提供 Web 界面与 API，支持 Token 追踪、Langfuse 观察及 RAGAS 评估，适用于从小型到大规模数据集的知识管理与问答场景。
