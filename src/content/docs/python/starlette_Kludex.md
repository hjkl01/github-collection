### [Kludex starlette](https://github.com/Kludex/starlette)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kludex/starlette?style=social)

Starlette 是一个轻量级的 ASGI 框架和工具包，旨在构建高性能 Python 异步 Web 服务。它提供低复杂度的 HTTP 支持，并涵盖 WebSocket、后台任务、启动/关闭事件、测试客户端、CORS、GZip、静态文件、流式响应、会话及 Cookie 等功能。框架依赖极少（仅需 anyio），兼容 asyncio 和 trio 后端，代码 100% 类型注解且测试覆盖率完整。它既可作为完整框架使用，也可作为 ASGI 工具包灵活组合其独立组件。
