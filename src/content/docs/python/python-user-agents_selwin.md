### [selwin python-user-agents](https://github.com/selwin/python-user-agents)  ![GitHub Repo stars](https://img.shields.io/github/stars/selwin/python-user-agents?style=social)

`user_agents` 是一个 Python 库，用于解析 HTTP 用户代理字符串以识别设备类型（手机、平板、PC）及其能力。基于 `ua-parser` 实现，可获取浏览器、操作系统及设备详情（品牌、型号、版本）。核心功能包括判断是否为移动设备、平板或 PC，检测触控能力，识别爬虫/机器人，并提供设备信息字符串化输出及属性查询方法。
