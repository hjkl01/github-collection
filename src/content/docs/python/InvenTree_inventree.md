### [inventree InvenTree](https://github.com/inventree/InvenTree)  ![GitHub Repo stars](https://img.shields.io/github/stars/inventree/InvenTree?style=social)

InvenTree 是一款开源库存管理系统，提供强大的底层库存控制和零件跟踪功能。系统基于 Python/Django 构建，提供 Web 管理界面和 REST API 以支持外部交互。内置强大的插件系统允许自定义扩展，并配套有移动端应用。支持 Docker 等多种部署方式。
