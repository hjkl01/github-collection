### [plotly plotly](https://github.com/plotly/plotly)  ![GitHub Repo stars](https://img.shields.io/github/stars/plotly/plotly?style=social)

plotly 是一个 R 语言包，通过开源 JavaScript 图形库 plotly.js 创建交互式网络图形。主要功能包括：使用 ggplotly() 将静态 ggplot2 图形转换为交互式版本，支持动态刻度、样式修改、布局调整及动画；通过 plot_ly() 函数直接调用 plotly.js，支持平行坐标、地图及 3D 曲面等特殊图表；此外还支持 Shiny 集成、视图链接、静态图像生成及自定义事件处理等高级功能。
