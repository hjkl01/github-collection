### [streamlit streamlit](https://github.com/streamlit/streamlit)  ![GitHub Repo stars](https://img.shields.io/github/stars/streamlit/streamlit?style=social)

Streamlit 是一个开源 Python 库，旨在快速将 Python 脚本转换为交互式 Web 应用。它支持构建仪表盘、报告或聊天应用，具有代码简洁、实时编辑更新、丰富的组件（如输入控件、数据表格、图表、布局）及多页面应用支持等特点。此外，它还提供免费社区云平台供用户部署、管理和分享应用，项目采用 Apache 2.0 开源协议。
