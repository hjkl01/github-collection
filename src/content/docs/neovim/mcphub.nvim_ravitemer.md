### [ravitemer mcphub.nvim](https://github.com/ravitemer/mcphub.nvim)  ![GitHub Repo stars](https://img.shields.io/github/stars/ravitemer/mcphub.nvim?style=social)

**项目核心内容总结：**  
MCP HUB 是一个为 Neovim 设计的 MCP（Model Context Protocol）客户端，用于无缝集成 MCP 服务器到编辑流程中，支持工具、资源、提示的管理与测试，并兼容多种聊天插件（如 CodeCompanion.nvim、Avante.nvim）。  

**主要功能：**  
- 支持 MCP 协议的工具、资源、提示管理，实时更新列表；  
- 兼容多种传输协议（Streamable-HTTP、SSE、STDIO）及 OAuth、Header 认证；  
- 集成主流 Neovim 插件，支持图像响应、函数调用等高级功能；  
- 提供市场发现 MCP 服务器功能，支持手动或自动安装；  
- 配置支持环境变量、JSON5 语法、VS Code 兼容格式；  
- 支持多实例同步、热重载、延迟关闭等高级特性。  

**使用方法：**  
访问项目文档网站获取详细指南，通过 Discord 社区获取帮助，开发者可阅读 CONTRIBUTING.md 参与贡献。
