### [sudo-tee opencode.nvim](https://github.com/sudo-tee/opencode.nvim)

**核心内容总结：**

**项目功能：**  
Opencode 是一款基于 Neovim 的 AI 编码助手插件，提供代码修改、分析、会话管理、快照与恢复点、用户自定义命令等功能。支持内置代理（Build/Plan 模式）和自定义代理配置，可切换不同 LLM 模型及 MCP 服务器。

**使用方法：**  
1. 安装 Opencode 并配置 LLM 提供商（如 `opencode auth login`）。  
2. 通过输入窗口使用 `/` 触发内置命令（如 `/share` 共享会话、`/undo` 撤销操作）。  
3. 使用 `@文件名` 引用上下文内容，或通过快照功能（实验性）查看、还原代码变更。  
4. 自定义代理规则、提示保护逻辑及事件钩子（如文件修改后触发操作）。

**主要特性：**  
- **代理模式**：Build 模式支持代码修改，Plan 模式仅用于分析。  
- **快照与恢复点**：自动记录代码变更，支持对比和还原（实验性）。  
- **上下文操作**：通过 `@` 引用文件、使用 `/` 命令扩展功能。  
- **高亮与主题**：自定义 UI 颜色（如提示、差异、代理状态）。  
- **安全控制**：通过 `prompt_guard` 防止误操作，支持自定义事件钩子（如文件编辑后处理）。
