### [NeogitOrg neogit](https://github.com/NeogitOrg/neogit)  ![GitHub Repo stars](https://img.shields.io/github/stars/NeogitOrg/neogit?style=social)

Neogit 是一款专为 Neovim 设计的 Git 管理插件，灵感源自 Magit。它提供统一的界面支持在编辑器内完成状态查看、文件暂存、提交、分支管理、合并、变基、回退及 Stash 等完整 Git 操作。插件支持分屏、悬浮窗及 Tab 页等多种窗口模式，可集成 Telescope 等工具，并提供丰富的 Lua 配置选项以自定义键位、高亮样式及显示逻辑。
