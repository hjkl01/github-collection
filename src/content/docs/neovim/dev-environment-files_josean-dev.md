### [josean-dev dev-environment-files](https://github.com/josean-dev/dev-environment-files)  ![GitHub Repo stars](https://img.shields.io/github/stars/josean-dev/dev-environment-files?style=social)

该项目提供了一套针对 macOS 的完整开发环境配置文件，核心内容包括：
- **终端与 Shell**：WezTerm 终端及 Zsh Shell 配置，集成 fzf、fd、bat 等常用命令行工具。
- **窗口管理**：Tmux 配置，以及 Yabai 和 Aerospace 平铺窗口管理器设置。
- **界面定制**：基于 Sketchybar 的自定义菜单栏配置。
- **编辑器配置**：基于 lazy.nvim 的 Neovim 配置，整合 LSP、自动补全、Git 集成、Treesitter 高亮及丰富插件。
- **环境依赖**：提供 iTerm2、Nerd Font、Ripgrep 等必要工具的安装指令。
