### [jstkdng ueberzugpp](https://github.com/jstkdng/ueberzugpp)  ![GitHub Repo stars](https://img.shields.io/github/stars/jstkdng/ueberzugpp?style=social)

Überzug++ 是用 C++ 编写的命令行工具，作为已废弃的 ueberzug 的替代方案，用于在终端中显示图像。它支持通过 X11/Wayland 子窗口、sixel、kitty 和 iterm2 等协议渲染图片。核心功能包括支持 Wayland 环境、MacOS、tmux、多种图像及动画格式、缓存加速及内存优化。项目提供 `layer` 命令和 JSON 接口供应用调用，兼容原 ueberzug 生态，支持配置文件和环境变量定制。
