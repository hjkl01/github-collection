### [ibhagwan fzf-lua](https://github.com/ibhagwan/fzf-lua)  ![GitHub Repo stars](https://img.shields.io/github/stars/ibhagwan/fzf-lua?style=social)

fzf-lua 是一个基于 Neovim 的模糊搜索插件，利用 fzf 或 skim 实现文件、缓冲区、Grep、Git、LSP 及诊断等多种命令选择器。它支持高度自定义的 UI 布局、键位绑定、预览模式及颜色配置，并提供多种预设配置文件以适配不同需求。此外，插件支持 Neovim API 注册及插入模式完成扩展，需 Neovim 0.9+ 及对应依赖环境。
