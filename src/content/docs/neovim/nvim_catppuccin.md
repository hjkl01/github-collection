### [catppuccin nvim](https://github.com/catppuccin/nvim)

Catppuccin 是一款高度可定制的 Neovim/Vim 颜色主题插件，支持 LSP、Treesitter、Git 等多种插件的配置。核心功能包括：  
1. **多终端兼容**：需终端支持 True Color（如 iTerm2、Alacritty）以正确显示颜色；  
2. **配置灵活**：通过 Lua 配置文件自定义颜色、编译路径等参数；  
3. **性能优化**：预编译配置文件提升执行效率；  
4. **插件适配**：与 Telescope、LSP、Git 等主流插件兼容，提供统一视觉风格；  
5. **安装使用**：通过 Lua 脚本加载，支持 Vim 8.2+ 和 Neovim 0.5+。  

**注意事项**：需在终端中启用 True Color 和 Italic 字体支持（如使用 tmux），否则可能出现颜色显示异常。
