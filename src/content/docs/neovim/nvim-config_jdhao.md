### [jdhao nvim-config](https://github.com/jdhao/nvim-config)  ![GitHub Repo stars](https://img.shields.io/github/stars/jdhao/nvim-config?style=social)

这是一个为 Neovim 设计的跨平台（Linux、macOS、Windows）个人配置文件项目，提供了高度文档化的参考方案。项目基于 Neovim 最新稳定版维护，不建议直接克隆使用，而是建议作为参考借鉴，鼓励用户根据需求裁剪配置。

核心功能涵盖：
1. **插件管理**：通过 Lazy.nvim 管理所有插件。
2. **编程辅助**：集成 nvim-cmp 实现智能代码补全，支持 LSP 协议、语法高亮（Treesitter）、代码片段（Ultisnips）及 Markdown/LaTeX 预览。
3. **效率提升**：提供 fzf-lua 项目级模糊搜索、Fugitive Git 集成、异步代码执行、快速跳转（Hop）及自动配对等功能。
4. **界面体验**：配置了美观的状态栏（Lualine）、文件树（nvim-tree）、GUI 风格通知、丰富配色方案及自定义快捷键与命令。

项目附带详细的安装指南、快捷键速查表及自定义命令说明。
