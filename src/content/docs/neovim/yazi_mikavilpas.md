### [mikavilpas yazi](https://github.com/mikavilpas/yazi)  ![GitHub Repo stars](https://img.shields.io/github/stars/mikavilpas/yazi?style=social)

Yazi 是一款基于 Rust 编写的高性能异步终端文件管理器。它采用全异步 I/O 和多线程任务调度，内置多种终端协议的图像预览及代码高亮加速功能。支持基于 Lua 的并发插件系统，可自定义 UI、预览器及预加载器。集成 fd、rg、fzf 等工具，提供 Vim 风格交互界面，支持多标签页、批量重命名、跨目录选择及主题定制。同时兼容主流终端，并可通过 Überzug++ 扩展对 X11/Wayland 的支持。
