### [NvChad NvChad](https://github.com/NvChad/NvChad)  ![GitHub Repo stars](https://img.shields.io/github/stars/NvChad/NvChad?style=social)

NvChad 是一个基于 Lua 的 Neovim 配置框架，以美观界面和极速启动（约 0.02-0.07 秒）为特点。它通过 93% 的延迟加载优化性能，集成文件导航、模糊搜索、LSP、代码补全、语法高亮等核心插件，并提供 56 种主题。此外，它还包含自研的状态栏、标签栏及快捷键提示等 UI 组件，支持通过 Starter 配置进行自定义扩展，适用于追求高性能与美学的 Neovim 用户。
