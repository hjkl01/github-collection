### [jwasham coding-interview-university](https://github.com/jwasham/coding-interview-university)

该README是一个计算机科学学习资源目录，核心功能是提供系统化的学习资料索引。主要包含以下内容：

1. **分类资源**：按主题划分算法、数据结构、操作系统、分布式系统等领域的课程视频（如MIT、Stanford等名校公开课），并附详细章节链接。

2. **论文合集**：收录计算机科学经典论文（如Google File System、MapReduce、Spanner等），包含论文原文及部分配套视频。

3. **实践资源**：提供算法实现代码库（如Princeton大学算法课程代码），以及在线课程目录（如Open Source Society计算机科学课程库）。

4. **学习路径**：涵盖从基础课程到前沿技术的完整学习链，包括操作系统、人工智能、密码学、大数据处理等方向。

使用方法：根据学习需求选择对应分类，通过链接访问具体课程、论文或代码实现。
