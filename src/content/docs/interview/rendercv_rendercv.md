### [rendercv rendercv](https://github.com/rendercv/rendercv)  ![GitHub Repo stars](https://img.shields.io/github/stars/rendercv/rendercv?style=social)

RenderCV 是一款面向学者和工程师的简历生成工具。用户通过 YAML 文件编写简历内容，运行命令即可生成排版专业的 PDF 文档。主要功能包括：基于文本的版本控制，专注于内容撰写而非排版；自动处理专业排版，确保对齐和间距一致；支持丰富的设计主题定制（颜色、字体、边距等）；提供 JSON Schema 交互式编辑和严格的内容验证；以及支持多语言配置。
