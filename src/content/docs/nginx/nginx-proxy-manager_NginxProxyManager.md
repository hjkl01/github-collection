### [NginxProxyManager nginx-proxy-manager](https://github.com/NginxProxyManager/nginx-proxy-manager)  ![GitHub Repo stars](https://img.shields.io/github/stars/NginxProxyManager/nginx-proxy-manager?style=social)

Nginx Proxy Manager 是一款基于 Docker 的图形化反向代理管理工具。它允许用户无需深入配置 Nginx 即可轻松实现网站转发、重定向及 SSL 证书管理（支持 Let's Encrypt 自动签发）。主要功能包括：灵活的域名转发与流转发、免费 SSL 证书集成、访问控制与 HTTP 认证、用户权限管理及审计日志。项目旨在降低反向代理的门槛，适用于家庭网络或自建服务的托管场景。
