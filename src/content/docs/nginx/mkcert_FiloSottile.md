### [FiloSottile mkcert](https://github.com/FiloSottile/mkcert)  ![GitHub Repo stars](https://img.shields.io/github/stars/FiloSottile/mkcert?style=social)

mkcert 是一个用于生成本地可信开发证书的简单工具，可自动在系统信任库中创建并安装本地 CA，无需配置。它能生成支持内部域名（如 localhost）且受信任的证书，解决真实 CA 无法签发或自签名证书不信任的问题。支持 macOS、Linux、Windows 及主流浏览器和 Java 信任库，并提供客户端认证、ECDSA、S/MIME 等高级选项。仅限开发环境使用，严禁泄露私钥或用于生产场景。
