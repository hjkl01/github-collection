### [evertramos nginx-proxy-automation](https://github.com/evertramos/nginx-proxy-automation)  ![GitHub Repo stars](https://img.shields.io/github/stars/evertramos/nginx-proxy-automation?style=social)

该项目是一个基于 Docker 和 Nginx 的反向代理自动化部署工具。核心功能是通过 Shell 脚本自动配置反向代理服务及 Let's Encrypt SSL 证书。用户运行初始化脚本即可快速搭建环境，支持通过 Docker 环境变量动态管理多个 Web 服务的域名映射。项目还包含针对主流云服务器平台（如 AWS、Digital Ocean、OVH）的部署教程视频及相关服务器自动化脚本链接。
