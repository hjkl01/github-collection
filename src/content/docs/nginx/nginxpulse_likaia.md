### [likaia nginxpulse](https://github.com/likaia/nginxpulse)  ![GitHub Repo stars](https://img.shields.io/github/stars/likaia/nginxpulse?style=social)

NginxPulse 是一款轻量级 Nginx 访问日志分析与可视化面板。主要功能包括实时流量统计、PV 过滤、IP 归属地解析及客户端识别。系统后端采用 Go 语言与 PostgreSQL 数据库，前端基于 Vue 3，支持 Docker、Docker Compose 及单体部署。IP 归属地查询采用本地缓存与远程 API 异步结合策略，支持移动端访问，并提供日志保留与时区配置功能。
