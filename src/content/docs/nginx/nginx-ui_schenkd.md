### [schenkd nginx-ui](https://github.com/schenkd/nginx-ui)  ![GitHub Repo stars](https://img.shields.io/github/stars/schenkd/nginx-ui?style=social)

Nginx UI 是一个基于 Docker 的 Nginx 配置 Web 管理工具。它提供图形化界面动态读取和更新 Nginx 配置文件，支持域名的添加、启用、禁用及删除，方便无服务器权限的用户修改生产环境配置。应用本身不带认证，但可配合 Nginx 反向代理配置 Basic Auth 以限制访问。
