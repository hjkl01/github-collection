### [trimstray nginx-admins-handbook](https://github.com/trimstray/nginx-admins-handbook)  ![GitHub Repo stars](https://img.shields.io/github/stars/trimstray/nginx-admins-handbook?style=social)

本项目《Nginx Admin's Handbook》是一份关于 NGINX 服务器管理的基础知识、技巧、注意事项和潜在陷阱的综合指南。主要功能与内容如下：

1. **核心知识**：涵盖 HTTP 基础、SSL/TLS 原理与配置、NGINX 基础架构、配置语法、连接与请求处理机制。
2. **最佳实践与规则**：提供了详细的配置规则，包括基础规则、调试方法、性能优化、安全加固（Hardening）、反向代理及负载均衡的最佳实践，并标注了优先级。
3. **辅助工具与脚本**：包含自动安装脚本、静态错误页生成器、服务器名称解析器、监控测试工具、日志分析命令、调试命令、Shell 别名及常用配置片段（如证书生成、访问控制、重定向等）。
4. **资源汇总**：整理了官方文档、第三方发行版、相关书籍、对比评测、各类配置生成/分析/管理工具、性能基准测试工具及安全测试工具的外部资源链接。
5. **安全报告与清单**：提供配置报告（如 SSL Labs 评分）、可打印的安全加固速查表以及配置检查清单，帮助管理员确保 NGINX 的安全性与性能。

该项目旨在为系统管理员和 Web 应用专家提供一套全面、一致且实用的 NGINX 配置与管理参考手册。
