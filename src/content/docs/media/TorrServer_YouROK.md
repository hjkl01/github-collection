### [YouROK TorrServer](https://github.com/YouROK/TorrServer)  ![GitHub Repo stars](https://img.shields.io/github/stars/YouROK/TorrServer?style=social)

TorrServer 是一款在线流式传输 BT 种子的工具，支持无需预先下载即可观看。核心功能包含种子缓存及 HTTP 协议数据传输，缓存大小可动态调整。支持 Windows、Linux、macOS 及 Docker 多平台部署，提供跨浏览器 Web 界面。功能涵盖 API 集成、DLNA 服务器、WebDAV、HTTP 认证、IP 黑白名单及代理设置，适用于多设备远程访问与观看。
