### [seerr-team seerr](https://github.com/seerr-team/seerr)  ![GitHub Repo stars](https://img.shields.io/github/stars/seerr-team/seerr?style=social)

**Seerr** 是一款开源的媒体请求管理软件，支持与 Jellyfin、Plex 和 Emby 等媒体服务器集成，并与 Sonarr 和 Radarr 等现有服务无缝对接。用户可以通过简单友好的界面请求电影、剧集或特定季集，管理员可方便地进行审批和管理。

### 主要功能包括：
- 支持 Jellyfin、Plex 和 Emby 的用户认证与管理；
- 支持 PostgreSQL 和 SQLite 数据库；
- 支持电影、剧集和混合类库；
- 支持通过 SMTP 修改邮箱；
- 自动扫描媒体库，跟踪已有的内容；
- 提供灵活的请求系统和直观的管理界面；
- 支持通知代理、权限管理、观看清单和屏蔽清单；
- 移动端友好设计，支持随时审批请求；
- 提供 API 接口和多语言支持。
