### [FreeTubeApp FreeTube](https://github.com/FreeTubeApp/FreeTube)  ![GitHub Repo stars](https://img.shields.io/github/stars/FreeTubeApp/FreeTube?style=social)

FreeTube 是一款开源的、注重隐私的桌面 YouTube 播放器，支持 Windows、macOS 和 Linux 系统。其核心功能包括去除广告、防止 Google 通过 Cookie 和 JavaScript 追踪用户，以及在不使用官方 API 的情况下播放视频。用户可无需账号订阅频道，所有订阅、播放列表和历史记录均本地存储。此外，它还支持使用内置提取器或 Invidious API、集成 SponsorBlock 和 DeArrow、外部播放器、浏览器扩展跳转以及自定义主题等功能。目前项目处于 Beta 阶段。
