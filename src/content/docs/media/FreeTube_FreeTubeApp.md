### [FreeTubeApp FreeTube](https://github.com/FreeTubeApp/FreeTube)

**核心内容总结：**  
FreeTube 是一款开源的桌面 YouTube 播放器，注重用户隐私，可无广告观看视频并防止 Google 通过 Cookie 和 JavaScript 跟踪用户。支持 Windows 10 及以上、macOS 12 及以上和 Linux 系统，基于 Electron 框架开发。  

**主要功能：**  
- 无广告观看 YouTube 视频  
- 本地存储订阅、播放列表和历史记录（不上传云端）  
- 支持 Invidious API 或内置提取器获取数据  
- 可连接 Tor 等代理隐藏 IP  
- 多主题、截图、多窗口、迷你播放器、键盘快捷键等  
- 支持订阅管理、导出/导入订阅、观看 YouTube 趋势视频、章节功能等  

**使用方法：**  
通过 GitHub 发布版、官网或 Flatpak 等渠道下载安装。可配合 LibRedirect 或 RedirectTube 浏览器扩展，实现 YouTube 链接直接跳转至 FreeTube。  

**其他信息：**  
- 项目处于 Beta 阶段，建议通过 GitHub 提交问题反馈  
- 提供中文翻译支持，可通过 Weblate 参与本地化  
- 可通过比特币地址捐款支持项目维护  
- 使用 GNU AGPLv3 开源协议授权
