### [Lightricks LTX-Video](https://github.com/Lightricks/LTX-Video)  ![GitHub Repo stars](https://img.shields.io/github/stars/Lightricks/LTX-Video?style=social)

LTX-Video 是一款基于 DiT 架构的开源视频生成模型，核心功能包括文本/图像生成视频、视频延伸与转换及音画同步生成。模型支持最高 4K 分辨率和 60 FPS 的高保真输出，具备多关键帧控制、深度/姿态/边缘检测等精细控制，并提供 LoRA 微调工具。项目提供多种模型版本（2B/13B 及蒸馏、量化版），兼容 ComfyUI、Diffusers 及 API 部署，支持本地与云端高效推理。LTX-2 作为新一代模型，进一步实现了音画同步生成及更强大的视频创作功能。
