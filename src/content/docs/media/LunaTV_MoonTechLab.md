### [MoonTechLab LunaTV](https://github.com/MoonTechLab/LunaTV)

LunaTV 是一个基于 Next.js、TypeScript 和 Tailwind CSS 构建的影视资源聚合平台，支持多资源站接口整合、自定义分类搜索及播放记录同步。用户可通过 Docker 部署，配置环境变量（如存储方式选择 Redis/Kvrocks/Upstash、设置管理员密码、关闭公网注册等）并上传 JSON 格式配置文件实现资源站接入。项目提供搜索接口流式输出、豆瓣数据代理加速（支持腾讯云/阿里云 CDN）、移动端适配（Selene 客户端）及 Android TV 支持（OrionTV 后端）。核心特性包括：自定义分类导航、接口缓存优化、数据本地化存储、安全防护（强制密码保护）及 MIT 开源协议授权。项目明确声明仅限个人非商业使用，用户需自行承担法律风险。
