### [MoonTechLab LunaTV](https://github.com/MoonTechLab/LunaTV)  ![GitHub Repo stars](https://img.shields.io/github/stars/MoonTechLab/LunaTV?style=social)

MoonTV 是一个基于 Next.js 14、Tailwind CSS 和 TypeScript 构建的跨平台影视聚合播放器。核心功能包括多源聚合搜索、在线播放（集成 HLS.js 与 ArtPlayer）、收藏与播放记录多端同步（支持 Kvrocks/Redis/Upstash）、PWA 离线缓存及响应式布局。支持 Android TV 使用及智能去广告（实验性）。部署后为空壳，需自行配置影视资源源，支持 Docker 及 Zeabur 部署方案。
