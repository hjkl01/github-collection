### [AzuraCast AzuraCast](https://github.com/AzuraCast/AzuraCast)

AzuraCast 是一个自托管的网络电台管理套件，提供一键安装和直观的网页管理界面，可快速部署电台站。支持各类规模的电台，兼容低价 VPS 服务器，当前为测试版。主要功能包括：  
- 通过 Web 界面管理电台所有设置  
- 内置软件包含流媒体服务器、自动播放列表、实时监控等  
- 提供演示站点（需账号密码登录）  
- 支持多语言（依赖 CrowdIn 翻译平台）  
- 使用 Docker 镜像构建（由 Depot 支持）  

**使用方法**：需通过安装指南在服务器上部署，要求具备基础 Linux 终端操作知识。  

**特性**：  
- 采用 AGPL-3.0 开源协议，免费开放源代码  
- 社区支持（Discord、Mastodon、Bluesky 等）  
- 依赖 JetBrains、Netlify 等企业赞助开发  
- 提供中文等多语言文档及故障排查指南
