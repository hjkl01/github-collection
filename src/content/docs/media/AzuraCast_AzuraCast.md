### [AzuraCast AzuraCast](https://github.com/AzuraCast/AzuraCast)  ![GitHub Repo stars](https://img.shields.io/github/stars/AzuraCast/AzuraCast?style=social)

AzuraCast 是一款自托管的一体化网络广播管理套件。它提供简便的安装流程和直观的网络界面，使用户能迅速搭建完整工作的网络电台，并通过 Web 界面管理电台的所有运营方面。该软件适用于各种类型的电台，可运行于低配置 VPS 上，集成了广播流媒体所需的核心组件。
