### [remotion-dev remotion](https://github.com/remotion-dev/remotion)  ![GitHub Repo stars](https://img.shields.io/github/stars/remotion-dev/remotion?style=social)

**核心内容总结：**  
Remotion 是一个基于 React 的视频生成框架，支持通过编程方式利用 CSS、Canvas、SVG 等 Web 技术及算法实现复杂视频效果。主要特性包括：  
- **技术整合**：兼容 Web 技术栈（CSS、SVG、WebGL 等）与编程能力（变量、函数、API）。  
- **React 优势**：支持组件复用、快速刷新及生态包。  
- **使用方法**：安装 Node.js 后通过 `npx create-video@latest` 快速启动，或参考官方文档。  
- **案例**：支持制作个性化年鉴（如 GitHub Unwrapped）、动态教程等，项目展示见官网。  
- **资源**：提供详细文档（[remotion.dev/docs](https://www.remotion.dev/docs)）及 API 参考。  
- **许可**：部分场景需申请公司许可证，详情见 [LICENSE](LICENSE.md)。
