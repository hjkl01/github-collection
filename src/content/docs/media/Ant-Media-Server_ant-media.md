### [ant-media Ant-Media-Server](https://github.com/ant-media/Ant-Media-Server)

**Ant Media Server 核心内容总结：**

**项目功能：**  
开源媒体服务器，支持超低延迟（WebRTC ~0.5秒）直播、实时通信（RTC）、录制及云扩展，兼容多种协议（WebRTC、SRT、RTMP、HLS、RTSP、H.265/HEVC等），适用于实时视频应用、直播、自适应码率（ABR）等场景。

**主要特性：**  
- **超低延迟**：WebRTC延迟低于0.5秒，支持LL-HLS等低延迟协议  
- **多协议支持**：覆盖WebRTC、SRT、RTMP、HLS、RTSP、CMAF等  
- **集成与安全**：提供多平台SDK，支持SSL/TLS、令牌认证、IP过滤等  
- **版本选择**：社区版（基础功能）与企业版（高级特性及支持）  

**使用方法：**  
- 快速体验：通过WebRTC演示、AWS云服务试用  
- 部署方式：提供Linux脚本安装、SSL配置、云平台（AWS/Azure/GCP）一键部署及Docker/Kubernetes方案  
- 获取资源：文档、社区讨论、开源代码下载及企业版试用  

**典型应用：**  
用于医疗远程会诊、在线教育、电竞直播、拍卖、媒体娱乐、视频监控等实时视频场景。  

**支持方式：**  
社区版通过GitHub讨论获取支持，企业版提供邮件支持及定制化服务包。
