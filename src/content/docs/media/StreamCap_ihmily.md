### [ihmily StreamCap](https://github.com/ihmily/StreamCap)  ![GitHub Repo stars](https://img.shields.io/github/stars/ihmily/StreamCap?style=social)

StreamCap 是一款基于 FFmpeg 和 StreamGet 的多平台直播流录制客户端，覆盖 40+ 国内外主流直播平台。支持 Windows、MacOS 及 Web 端运行，具备循环监控、定时监控和批量录制功能。支持多种视频及音频输出格式，录制完成后可自动转码为 MP4，并提供直播状态消息推送。支持预构建程序、源码运行及 Docker 部署。
