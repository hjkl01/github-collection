### [jellyfin jellyfin-web](https://github.com/jellyfin/jellyfin-web)  ![GitHub Repo stars](https://img.shields.io/github/stars/jellyfin/jellyfin-web?style=social)

Jellyfin Web 是 Jellyfin 媒体服务器的 Web 前端，为桌面浏览器、Android 和 iOS 等客户端提供用户界面。项目包含管理后台、启动向导、稳定版与实验性应用，支持插件动态加载。集成了本地化、主题、React 组件及工具库，采用模块化目录结构，支持多语言翻译，代码架构正按 Bulletproof React 标准重构。
