### [ProudMuBai GoFilm](https://github.com/ProudMuBai/GoFilm)  ![GitHub Repo stars](https://img.shields.io/github/stars/ProudMuBai/GoFilm?style=social)

GoFilm 是一款基于 Vue 和 Gin 实现的在线观影网站。前端采用 Vite + Vue + ElementPlus 技术栈，后端使用 Gin + Gorm + Go-Redis 配合 gocolly 资源采集与 cron 定时更新服务。主要功能涵盖影视资源搜索、播放、分类导航及详情展示，支持移动端观看历史与采集失败重试。项目提供管理后台，用于配置采集站点、定时任务、网站信息及影片数据。支持 Docker 及 1Panel 可视化部署方式。
