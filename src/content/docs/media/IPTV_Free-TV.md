### [Free-TV IPTV](https://github.com/Free-TV/IPTV)  ![GitHub Repo stars](https://img.shields.io/github/stars/Free-TV/IPTV?style=social)

该项目提供全球免费电视频道的 M3U 播放列表，涵盖各地免费地面波及网络电视服务（如 Pluto TV、Plex TV 等）。用户可通过 IPTV 播放器访问指定链接使用。项目仅收录官方免费、高清、主流频道，排除付费、成人、宗教及政治敏感内容。频道列表基于 Markdown 文件维护，用户可通过 Pull Request 提交频道信息的修改。
