### [reisxd TizenTube](https://github.com/reisxd/TizenTube)

**TizenTube 核心内容总结：**  

**项目功能**  
TizenTube 是一个 TizenBrew 模块，通过移除广告、支持 SponsorBlock 和 DeArrow 等功能，优化流媒体网站的观看体验。  

**使用方法**  
1. 安装 [TizenBrew](https://github.com/reisxd/TizenBrew) 并按指引完成设置。  
2. TizenTube 默认集成于 TizenBrew，若未显示，可通过 TizenBrew 模块管理器添加 NPM 模块 `@foxreis/tizentube`。  

**主要特性**  
- 广告拦截  
- SponsorBlock 和 DeArrow 支持  
- 画中画模式  
- 自定义主题颜色  
- 支持用户功能请求（通过 GitHub 提交 issue）  

**其他**  
Android TV 用户可使用 [TizenTube Cobalt](https://github.com/reisxd/TizenTubeCobalt) 项目。
