### [streamaserver streama](https://github.com/streamaserver/streama)

Streama 是一款自托管的媒体管理工具，功能类似 Netflix，用于集中管理本地电影、电视剧并提供播放服务。核心功能包括：  
- **媒体管理**：通过 The Movie Database API 自动匹配影视信息（如封面、简介），支持手动上传或通过本地文件目录批量识别内容。  
- **播放体验**：HTML5 播放器支持分集浏览（类似 Netflix）、进度记录、多季剧集管理，未来将支持字幕和多画质切换。  
- **用户与权限**：管理员可创建用户并分配权限（仅限观看或管理内容），支持多用户协作。  
- **本地文件整合**：通过设置本地目录，可直接读取文件并自动匹配到对应影视条目，支持正则表达式自定义文件名规则。  

**使用方法**：  
1. 安装后配置上传目录、The Movie Database API 密钥等基础设置。  
2. 通过网页界面上传视频文件，或使用本地文件浏览器/批量匹配功能导入已有内容。  
3. 用户可访问仪表板查看观看记录、继续未完成的剧集，管理员通过后台管理影视库和用户权限。  

**主要特性**：  
- 自动填充影视元数据（标题、简介、海报等）。  
- 支持本地文件夹直接读取与批量创建影视条目。  
- 管理员面板提供便捷的影视管理、用户控制功能。  
- 播放器支持分集跳转、进度保存，兼容主流浏览器。
