### [kaltura platform-install-packages](https://github.com/kaltura/platform-install-packages)

**项目核心内容总结：**  
Kaltura安装包项目提供官方二进制包和包管理器，用于在服务器或集群环境中部署Kaltura开源视频平台。支持多种操作系统（如Red Hat、CentOS、Debian、Ubuntu及Docker），涵盖单服务器安装、集群部署、Chef/Vagrant自动化工具集成等方案，并提供升级、监控、端口配置等文档指导。项目采用AGPLv3开源许可证，社区贡献者可参与测试、文档编写及CI/CD流程优化，用户可通过官方渠道获取商业支持服务。
