### [awolfly9 IPProxyTool](https://github.com/awolfly9/IPProxyTool)  ![GitHub Repo stars](https://img.shields.io/github/stars/awolfly9/IPProxyTool?style=social)

IPProxyTool 是一个基于 Scrapy 的免费代理 IP 采集与管理工具。核心功能包括：利用爬虫抓取多个代理网站数据并支持扩展；验证 IP 有效性、HTTPS 支持及匿名度；将有效 IP 存储至 MySQL 数据库；提供 HTTP 接口支持按条件筛选获取、删除或插入代理；支持 Python 环境直接运行及 Docker 容器化部署。
