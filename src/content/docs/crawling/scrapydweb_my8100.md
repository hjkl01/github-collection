### [my8100 scrapydweb](https://github.com/my8100/scrapydweb)  ![GitHub Repo stars](https://img.shields.io/github/stars/my8100/scrapydweb?style=social)

ScrapydWeb 是一个用于管理 Scrapyd 集群的 Web 应用程序，支持 Scrapy 日志分析与可视化。主要功能包括：

1. **集群管理**：支持所有 Scrapyd JSON API，可分组、过滤节点，并实现多节点一键执行命令。
2. **日志分析**：提供统计收集、进度可视化和日志分类功能。
3. **增强特性**：支持项目自动打包、集成 LogParser、定时任务、监控告警、移动端 UI 及 Web 界面基础认证。
