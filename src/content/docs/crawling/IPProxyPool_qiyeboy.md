### [qiyeboy IPProxyPool](https://github.com/qiyeboy/IPProxyPool)  ![GitHub Repo stars](https://img.shields.io/github/stars/qiyeboy/IPProxyPool?style=social)

IPProxyPool 是一个支持 Python 2 和 Python 3 的代理 IP 池项目。主要功能包括：自动从多个代理网站爬取 IP 并检测有效性，采用评分机制维护 IP 质量；支持 SQLite、MySQL、MongoDB 等多种数据库存储；提供 Web API 接口，支持按匿名类型、协议、地区等条件查询和删除代理 IP；支持通过配置文件自定义爬取规则、检测逻辑及数据库连接方式。
