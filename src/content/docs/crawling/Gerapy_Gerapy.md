### [Gerapy Gerapy](https://github.com/Gerapy/Gerapy)  ![GitHub Repo stars](https://img.shields.io/github/stars/Gerapy/Gerapy?style=social)

Gerapy 是一个基于 Scrapy、Scrapyd、Django 和 Vue.js 的分布式爬虫管理框架。它通过 Web 界面提供 Scrapy 项目的可视化配置、代码自动生成及拖拽导入功能。主要功能包括自动化部署、作业监控、定时任务调度与权限管理。支持 Docker 容器化部署，可快速初始化并运行。
