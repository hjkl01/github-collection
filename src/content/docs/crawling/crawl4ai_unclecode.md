### [unclecode crawl4ai](https://github.com/unclecode/crawl4ai)  ![GitHub Repo stars](https://img.shields.io/github/stars/unclecode/crawl4ai?style=social)

Crawl4AI 是一款开源的、大语言模型（LLM）友好的网页爬虫与抓取工具，旨在将网页内容转换为适合 RAG、智能体及数据管道使用的干净 Markdown 和结构化数据。

核心功能包括：
1. **内容转换**：生成清洁 Markdown，支持智能过滤、引用标记及自定义策略。
2. **数据提取**：支持 LLM 驱动、CSS/XPath 选择器或自定义 Schema 的结构化 JSON 提取。
3. **浏览器自动化**：基于 Playwright，支持无头模式、会话管理、代理及反检测隐身功能。
4. **动态处理**：可执行 JavaScript、处理懒加载与滚动、支持截图及 IFrame 内容提取。
5. **部署与运维**：提供 Docker 镜像、REST API、命令行工具，支持深度爬取、缓存、状态恢复及实时监控仪表盘。

项目采用异步架构，具备高性能、可控制且低成本的特性，适用于构建自动化 Web 数据采集系统。
