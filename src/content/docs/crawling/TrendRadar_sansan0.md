### [sansan0 TrendRadar](https://github.com/sansan0/TrendRadar)  ![GitHub Repo stars](https://img.shields.io/github/stars/sansan0/TrendRadar?style=social)

TrendRadar 是一款轻量级热点资讯监控与推送助手，旨在帮助用户告别无效刷屏，只获取真正关心的新闻资讯。

核心功能总结：
1. **全网热点聚合**：默认监控知乎、微博、抖音、百度等 11+ 主流平台热搜，支持自定义平台，同时兼容 RSS/Atom 订阅源抓取。
2. **精准内容筛选**：支持关键词、正则表达式、必须词/过滤词等配置，只推送匹配内容；提供当日汇总、当前榜单、增量监控三种推送模式。
3. **AI 智能分析**：集成大模型（支持 DeepSeek、OpenAI 等），自动生成热点趋势概述、情感倾向分析及跨平台关联报告，支持多语言翻译。
4. **多渠道推送通知**：支持企业微信、个人微信、飞书、钉钉、Telegram、邮件、Bark、Slack 及通用 Webhook 等多种通知渠道。
5. **灵活部署与存储**：支持 GitHub Actions、Docker 及本地部署；数据存储可选本地 SQLite 或远程云存储（S3 兼容）。
6. **可视化配置**：提供 Web 端可视化配置编辑器，支持自定义推送时间窗口、调度策略及 AI 分析频率。
