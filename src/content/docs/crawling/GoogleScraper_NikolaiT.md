### [NikolaiT GoogleScraper](https://github.com/NikolaiT/GoogleScraper)  ![GitHub Repo stars](https://img.shields.io/github/stars/NikolaiT/GoogleScraper?style=social)

GoogleScraper 是基于 Python 3 的开源搜索引擎抓取工具，支持 Google、Bing、Yahoo、Yandex、Baidu、Duckduckgo 等引擎。核心功能包括：

1. 双模式抓取：提供高性能 HTTP 异步多线程模式与 Selenium 浏览器模拟模式，支持大规模并发。
2. 多样化支持：支持代理配置（Socks/Http）、多关键词批量处理及网页、图片、视频等多种搜索类型。
3. 数据导出：支持将结果保存为 JSON 文件或 SQLite 数据库。

注意：该项目已于 2019 年停止维护，官方推荐使用基于 JavaScript/Puppeteer 的继任项目（如 se-scraper 或 Crawling-Infrastructure）。
