### [seleniumbase SeleniumBase](https://github.com/seleniumbase/SeleniumBase)  ![GitHub Repo stars](https://img.shields.io/github/stars/seleniumbase/SeleniumBase?style=social)

SeleniumBase 是一个基于 Python 的全功能浏览器自动化框架，支持端到端测试、网页爬虫及浏览器控制。它提供 UC 模式和 CDP 模式以绕过反爬虫检测、Cloudflare 挑战及验证码。框架封装 Selenium 以自动处理元素等待、驱动管理及超时设置，并内置测试录制器。此外，它提供可视化仪表盘、多格式报告（如 HTML/Allure），支持 CI/CD 及移动端测试，兼容 Chrome、Firefox 等浏览器及 Windows、macOS、Linux 系统，并与 pytest 等测试框架集成。
