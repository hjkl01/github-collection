### [g1879 DrissionPage](https://github.com/g1879/DrissionPage)  ![GitHub Repo stars](https://img.shields.io/github/stars/g1879/DrissionPage?style=social)

DrissionPage 是一个基于 Python 的网页自动化工具，集浏览器控制与数据包收发功能于一体。它采用自研内核，不依赖 webdriver，无需配置驱动，运行速度快。支持跨 iframe 查找元素、多标签页操作、全页截图及直接读取浏览器缓存。内置极简定位语法、自动等待重试、强大下载工具及 POM 模式，代码简洁优雅。兼容 Chromium 内核浏览器，支持 Windows、Linux、Mac 系统。
