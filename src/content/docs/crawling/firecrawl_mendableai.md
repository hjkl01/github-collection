### [mendableai firecrawl](https://github.com/mendableai/firecrawl)  ![GitHub Repo stars](https://img.shields.io/github/stars/mendableai/firecrawl?style=social)

Firecrawl 是一个将网站数据转换为大语言模型（LLM）就绪格式的 API 服务。核心功能涵盖：
- **数据抓取**：将 URL 转换为 Markdown、HTML、结构化 JSON 或截图。
- **网络搜索**：搜索网页并获取结果页面的完整内容。
- **AI 代理**：根据自然语言描述，自动搜索、导航并提取数据。
- **全站爬取**：一次性抓取整个网站的所有页面。
- **URL 发现**：即时发现网站上的所有链接。
- **交互操作**：支持抓取前点击、滚动、输入、等待等行为。
- **媒体解析**：自动提取 PDF、DOCX 和图片中的文本。
- **批量处理**：异步批量抓取大量 URL。
项目提供 Python 和 Node.js SDK，能处理 JavaScript 渲染、代理及鉴权页面。
