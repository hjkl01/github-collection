### [gocolly colly](https://github.com/gocolly/colly)  ![GitHub Repo stars](https://img.shields.io/github/stars/gocolly/colly?style=social)

Colly 是一个轻量、优雅的 Go 语言网络爬虫框架。它提供简洁的 API 接口，用于编写爬虫程序，可高效地从网站提取结构化数据，适用于数据挖掘、处理或归档。主要功能包括：高并发请求（单核超 1000 请求/秒）、自动管理请求延迟与并发限制、Cookie 与会话自动处理、支持同步/异步/并行抓取、缓存机制、非 Unicode 响应自动编码、Robots.txt 支持、分布式抓取及环境变量配置。
