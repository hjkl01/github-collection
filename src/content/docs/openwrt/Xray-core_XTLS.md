### [XTLS Xray-core](https://github.com/XTLS/Xray-core)

Project X 是基于 XTLS 协议开发的网络工具集，核心组件包括 Xray-core 和 REALITY，主要用于实现加密传输、代理服务及网络通信。其主要功能包括支持 VLESS、XTLS、XUDP 等协议，提供多平台客户端（Windows、Android、iOS、Linux 等），并兼容 WireGuard 等扩展功能。使用方法包括通过 Linux 脚本、Docker、Homebrew 等方式安装，或使用预编译的 GUI 客户端（如 v2rayN、ShadowSocksR 等）。项目特性包括跨平台支持、可自定义编译（提供 Windows/Linux/macOS 的编译命令）、丰富的社区资源（文档、教程、贡献指南）及活跃的开源生态。
