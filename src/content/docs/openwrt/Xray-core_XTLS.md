### [XTLS Xray-core](https://github.com/XTLS/Xray-core)  ![GitHub Repo stars](https://img.shields.io/github/stars/XTLS/Xray-core?style=social)

Project X 是基于 XTLS 协议的网络工具集，核心组件为 Xray-core 和 REALITY。它支持 VLESS、XTLS、REALITY、XUDP、PLUX 等多种协议，提供高性能网络连接能力。项目兼容 Windows、Linux、macOS、Android、iOS、OpenWrt 等多平台，支持 Docker、脚本安装、Web 面板及各类客户端等多种部署方式。
