### [juewuy ShellCrash](https://github.com/juewuy/ShellCrash)  ![GitHub Repo stars](https://img.shields.io/github/stars/juewuy/ShellCrash?style=social)

ShellCrash 是一款用于在 Shell 环境中便捷部署和管理 mihomo/sing-box 内核的脚本工具。核心功能包括多内核切换、订阅配置在线导入、定时任务自动更新、本地 Web 仪表盘管理、路由与本地模式切换以及一键在线维护。项目兼容各类基于 Linux 的设备，如 OpenWrt 路由器、Linux 服务器、第三方固件、Docker 环境及虚拟机。安装后通过命令行 crash 进入管理界面进行配置。
