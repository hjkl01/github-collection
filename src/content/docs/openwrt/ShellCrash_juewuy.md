### [juewuy ShellCrash](https://github.com/juewuy/ShellCrash)  ![GitHub Repo stars](https://img.shields.io/github/stars/juewuy/ShellCrash?style=social)

**项目核心内容总结：**  
ShellCrash 是一个基于 Shell 环境的管理脚本，用于便捷管理 [Crash](https://github.com/Dreamacro/Crash) 功能。支持在线导入 Crash 共享的规则、配置链接，配置定时任务及自动更新配置文件，提供本地网页面板管理内置规则。支持路由模式、原生模式等切换，支持在线更新。  

**设备兼容性：**  
支持基于 OpenWrt 或二次开发的路由器设备、标准 Linux 系统（如 Debian/CentOS/Armbian 等）、Padavan 固件（保守模式）、Pandora 固件及各类 Linux 内核定制设备。如需更多设备支持，需提交设备信息（通过 `uname -a` 获取核心信息）至 TG 群或提交 Issue。
