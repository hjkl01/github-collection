### [ThisSeanZhang landscape](https://github.com/ThisSeanZhang/landscape)  ![GitHub Repo stars](https://img.shields.io/github/stars/ThisSeanZhang/landscape?style=social)

Landscape 是一款基于 Web 的 Linux 路由器配置工具，采用 Rust 开发并结合 eBPF 技术。其核心功能包括：流量控制（支持基于 IP、MAC、域名及地理位置的分流、eBPF 路由及 NAT）、DNS 管理（每流独立配置与缓存、支持 DoH/DoT/DoQ 及防污染）、网络配置（静态 IP、DHCP、PPPoE、IPv6 及 Wi-Fi 管理）、以及 Docker 流量路由和 GeoIP 自动更新。系统需 Linux 内核 6.9 及以上版本。
