### [netbirdio netbird](https://github.com/netbirdio/netbird)  ![GitHub Repo stars](https://img.shields.io/github/stars/netbirdio/netbird?style=social)

NetBird 是一个集配置自由的点对点（P2P）私有网络和集中式访问控制系统于一体的安全平台，旨在简化组织或家庭的私有网络创建。

- **网络连接**：基于 WireGuard 自动建立加密隧道，无需手动配置端口、防火墙规则或网关，支持 NAT 穿透、路由至外部网络及连接中继备用。
- **安全访问**：提供细粒度访问控制策略、SSO 与 MFA 支持、用户组与规则管理、设备状态检查、活动日志记录及量子抗性加密。
- **系统管理**：拥有 Web 管理后台、公共 API、自动对等发现、私有 DNS 解析及批量配置密钥功能。
- **平台支持**：兼容 Linux、macOS、Windows、Android、iOS、OpenWRT 及 Docker 等多种环境。
- **部署方式**：支持 NetBird Cloud 云服务及自托管模式（需满足特定服务器与 Docker 要求）。
