### [ehang-io nps](https://github.com/ehang-io/nps)  ![GitHub Repo stars](https://img.shields.io/github/stars/ehang-io/nps?style=social)

NPS 是一款轻量级、高性能的内网穿透代理服务器，配备强大的 Web 管理终端。它支持 TCP、UDP、HTTP(S)、SOCKS5、P2P 等多种协议，兼容 Linux、Windows、macOS 及 Synology 等跨平台系统，并支持安装为系统服务。功能涵盖 HTTPS 集成（支持多证书）、缓存、压缩、加密、流量与带宽限制、端口复用及高级域名解析（如 URL 路由、泛解析、自定义请求头配置）。项目支持多用户管理，提供实时的流量、带宽及系统信息监控，大部分配置可通过 Web 界面完成。
