### [judge0 judge0](https://github.com/judge0/judge0)  ![GitHub Repo stars](https://img.shields.io/github/stars/judge0/judge0?style=social)

Judge0 是一个健壮、快速、可扩展且沙箱化的开源在线代码执行系统。它支持 90 多种编程语言的代码安全编译与执行，适用于 AI 代理、编程竞赛平台、在线教育、在线编辑器及招聘评估等应用场景。项目提供简单的 HTTP JSON API 和 Python SDK 便于集成，支持自建部署或云服务（SaaS），具备高并发扩展能力，并支持多文件项目、自定义资源限制及详细执行结果反馈。
