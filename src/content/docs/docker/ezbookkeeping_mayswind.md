### [mayswind ezbookkeeping](https://github.com/mayswind/ezbookkeeping)  ![GitHub Repo stars](https://img.shields.io/github/stars/mayswind/ezbookkeeping?style=social)

ezBookkeeping 是一款轻量级、开源的自托管个人财务管理应用。它提供面向移动端和桌面端的友好界面（支持 PWA），帮助用户记录日常交易、导入导出数据、搜索过滤账单，并利用内置图表分析财务趋势。核心功能包括双层账户分类、交易图片附件、位置跟踪、定时交易、AI 收据识别及多语言多货币支持。应用注重隐私与安全，支持双因素认证，资源占用极低，可通过 Docker 或二进制方式轻松部署在树莓派、NAS 等低配置设备上。
