### [lirantal dockly](https://github.com/lirantal/dockly)  ![GitHub Repo stars](https://img.shields.io/github/stars/lirantal/dockly?style=social)

Dockly 是一个沉浸式终端界面工具，用于管理 Docker 容器、服务和镜像。它支持通过 npm 全局安装，运行后默认自动连接本地 Docker 守护进程，并允许通过命令行选项自定义连接参数（如 Socket 路径、远程主机、端口和协议）。用户也可以将 Dockly 作为 Docker 容器运行。
