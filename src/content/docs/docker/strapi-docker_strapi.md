### [strapi strapi-docker](https://github.com/strapi/strapi-docker)  ![GitHub Repo stars](https://img.shields.io/github/stars/strapi/strapi-docker?style=social)

本项目是 Strapi v3 的 Docker 容器化项目，提供两个核心镜像：`strapi/strapi` 用于创建新项目或运行现有项目（支持卷挂载及数据库环境变量），默认启动开发模式；`strapi/base` 用于构建生产环境镜像。注：仅支持 Strapi v3，v4 需使用社区工具。
