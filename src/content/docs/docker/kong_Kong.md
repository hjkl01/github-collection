### [Kong kong](https://github.com/Kong/kong)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kong/kong?style=social)

Kong Gateway 是一款云原生、平台无关且可扩展的高性能 API 网关，支持 API、LLM 及 MCP 流量治理。它作为微服务与 AI 流量的编排核心层，提供代理、路由、负载均衡、健康检查及多种认证功能。项目具备高级 AI 流量能力，包括多 LLM 支持、语义安全与 MCP 流量监控。支持通过 Lua、Go 或 JavaScript 插件扩展功能（如限流、日志、转换），并原生支持 Kubernetes Ingress 及多种无厂商锁定的部署模式。
