### [Lissy93 dashy](https://github.com/Lissy93/dashy)  ![GitHub Repo stars](https://img.shields.io/github/stars/Lissy93/dashy?style=social)

Dashy 是一款开源的自托管服务仪表盘，旨在通过单一入口帮助用户组织和管理各类自托管应用。其主要功能包括：

1. **服务聚合**：支持多页面布局，可自定义链接、图标（支持多种来源）及描述。
2. **实时监控**：提供服务的在线状态检测、响应时间分析及动态数据组件（Widgets）。
3. **高度定制**：内置多主题及 UI 编辑器，支持自定义 CSS、布局、文本及多语言界面（10+ 种）。
4. **安全认证**：支持基本认证、Keycloak SSO 及多用户权限管理，注重隐私保护。
5. **交互体验**：具备即时搜索、键盘快捷键及多种应用打开方式（新标签页、弹窗、工作空间等）。
6. **灵活部署**：支持 Docker、源码及云平台一键部署，提供 UI 配置编辑器及加密云端备份功能。
