### [portainer portainer](https://github.com/portainer/portainer)  ![GitHub Repo stars](https://img.shields.io/github/stars/portainer/portainer?style=social)

Portainer 是一个轻量级的容器化应用服务交付平台，支持管理 Docker、Swarm、Kubernetes 和 ACI 环境。它通过智能图形界面（GUI）和 API 管理容器、镜像、卷及网络等资源，可部署为单个容器运行于 Linux 或 Windows 环境。项目分为开源的社区版（CE）和商业版（BE），后者包含 RBAC 等企业级高级功能。
