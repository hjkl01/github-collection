### [mrjackwills oxker](https://github.com/mrjackwills/oxker)  ![GitHub Repo stars](https://img.shields.io/github/stars/mrjackwills/oxker?style=social)

oxker 是一款基于 Rust 开发的终端用户界面（TUI）工具，专为查看和控制 Docker 容器设计。其主要功能包括：

1. **容器管理**：以终端界面可视化展示容器列表，支持排序、筛选及查看详情。
2. **容器控制**：支持执行启动/停止、进入容器内部（exec）、检查配置（inspect）等操作。
3. **日志处理**：提供实时日志查看，支持关键字搜索、日志保存、时间戳控制及面板布局调整。
4. **灵活配置**：允许自定义按键映射、颜色主题、日志显示样式及 Docker 主机连接地址。
