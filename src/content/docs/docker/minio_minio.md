### [minio minio](https://github.com/minio/minio)  ![GitHub Repo stars](https://img.shields.io/github/stars/minio/minio?style=social)

MinIO 是一款高性能、兼容 S3 API 的对象存储解决方案，专为 AI/ML、数据分析和数据密集型工作负载设计。采用 GNU AGPL v3.0 开源协议，支持源码编译、Docker 及 Kubernetes 部署，提供 Web 控制台和 mc 命令行工具进行对象管理。当前社区版仓库已停止维护，仅提供源码分发，建议源码构建或使用 AIStor 替代方案。
