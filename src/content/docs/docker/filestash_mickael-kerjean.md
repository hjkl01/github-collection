### [mickael-kerjean filestash](https://github.com/mickael-kerjean/filestash)

Filestash 是一款支持多种协议和云存储的文件管理平台，提供类似 Dropbox 的图形化操作界面。核心功能包括：  
1. **多协议支持**：可通过 FTP/SFTP/WebDAV/S3/NFS/SMB 等协议管理文件，支持 Dropbox、Google Drive 等云存储。  
2. **插件扩展**：通过插件系统实现功能扩展，包括自定义主题（如 Dropbox、Google Drive 风格）、文件预览（支持 TIFF、FITS、GeoJSON 等 200+ 格式）、API 集成（MCP 接口）等。  
3. **跨平台使用**：支持浏览器、SFTP 客户端、第三方工具访问，提供本地网络驱动挂载功能。  
4. **开发友好**：提供丰富的接口（如存储、认证、搜索等），允许开发者通过插件实现定制化功能。  

**使用方法**：可通过官方演示链接体验，或参考文档进行安装部署。支持 Docker 部署，社区提供 IRC 和 Open Collective 支持。项目目标是成为比 Dropbox 等工具更全面的文件管理平台。
