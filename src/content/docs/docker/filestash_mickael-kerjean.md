### [mickael-kerjean filestash](https://github.com/mickael-kerjean/filestash)  ![GitHub Repo stars](https://img.shields.io/github/stars/mickael-kerjean/filestash?style=social)

Filestash 是一个基于插件架构的存储无关型类 Dropbox 文件管理及通用数据访问平台，主要功能如下：

1.  **多协议支持**：兼容 FTP、SFTP、S3、SMB、WebDAV、IPFS 等约 20 种存储协议及各类认证提供商。
2.  **高度模块化**：核心简洁，通过插件生态系统扩展存储、认证、API、网关及前端功能，支持虚拟文件系统。
3.  **Web 客户端**：原生 JS 构建，支持主题定制、动态插件补丁及工作流自动化。
4.  **丰富文件预览**：支持摄影、科学、工程、3D、医学等海量专业格式（如 raw、fits、dwg、parquet 等）的直接打开。
5.  **企业级特性**：提供 RBAC 权限、API 接口、SFTP/MCP 网关（用于 LLM 数据访问）、版本控制、杀毒及配额管理等。
6.  **AI 集成**：具备语义搜索和智能文件夹功能。
