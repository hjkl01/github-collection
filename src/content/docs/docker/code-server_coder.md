### [coder code-server](https://github.com/coder/code-server)

**项目核心内容总结：**  
code-server 是一个可在任意设备上运行 VS Code 的工具，通过浏览器访问，支持远程开发。  

**主要功能：**  
- 跨设备使用统一开发环境，支持云服务器加速编译、测试等任务。  
- 通过云端运行高强度任务，节省本地设备电量。  

**使用方法：**  
1. 通过安装脚本（提供 `--dry-run` 预览和正式安装命令）。  
2. 手动安装或部署到团队（使用 [coder/coder](https://cdr.co/coder-github)）。  
3. 一键部署到云服务商（如 [deploy-code-server](https://github.com/coder/deploy-code-server)）。  
4. 与 devcontainers 集成（适用于已有 devcontainers 项目）。  

**要求：**  
- 最低配置：Linux 系统、启用 WebSocket、1GB RAM、2vCPU。  

**其他：**  
- 提供详细配置指南、常见问题解答及贡献方式。  
- 支持社区交流（GitHub Discussions、Slack、Discord）。  
- 团队可使用 [coder/coder](https://cdr.co/coder-github) 管理远程开发。
