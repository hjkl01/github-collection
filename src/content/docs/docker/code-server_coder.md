### [coder code-server](https://github.com/coder/code-server)  ![GitHub Repo stars](https://img.shields.io/github/stars/coder/code-server?style=social)

code-server 允许用户在任意机器上运行 Visual Studio Code 并通过浏览器远程访问。它确保跨设备开发环境一致，利用云服务器加速编译与测试，并节省本地设备电量。项目适用于 Linux 环境，提供多种安装部署方式，服务于个人及团队的远程开发需求。
