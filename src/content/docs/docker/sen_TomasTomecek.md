### [TomasTomecek sen](https://github.com/TomasTomecek/sen)  ![GitHub Repo stars](https://img.shields.io/github/stars/TomasTomecek/sen?style=social)

sen 是一个用于 Docker 的终端用户界面（TUI）工具。主要功能包括交互式管理容器和镜像（启动、停止、重启、删除等），提供仪表盘视图及检查功能，支持日志获取与实时流式传输。界面具备搜索过滤能力，采用 Vim 风格键位绑定，并接收 Docker 实时状态更新通知。此外支持镜像树状视图及磁盘空间分析。项目目前处于维护模式，兼容 Docker 和 Podman。
