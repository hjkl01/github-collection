### [strapi-community strapi-tool-dockerize](https://github.com/strapi-community/strapi-tool-dockerize)  ![GitHub Repo stars](https://img.shields.io/github/stars/strapi-community/strapi-tool-dockerize?style=social)

`@strapi-community/dockerize` 是一个用于快速为 Strapi 项目添加 Docker 支持的命令行工具。

核心功能：
1. **自动化部署**：自动检测 yarn 或 npm 包管理器，并生成 Docker Compose 文件。
2. **数据库支持**：内置支持 PostgreSQL 16、MySQL 8 及 MariaDB。
3. **灵活配置**：提供 CLI 参数，可自定义数据库连接信息、项目类型及环境变量。
4. **项目管理**：支持重置项目配置。

该工具主要针对 Strapi v5 版本。
