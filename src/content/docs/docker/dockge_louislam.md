### [louislam dockge](https://github.com/louislam/dockge)  ![GitHub Repo stars](https://img.shields.io/github/stars/louislam/dockge?style=social)

Dockge 是一款美观、易用且响应式的自托管 Docker Compose 堆栈管理工具。其主要功能包括：管理 compose.yaml 文件（创建、编辑、启停、重启、删除及更新镜像）；提供交互式编辑器与 Web 终端；支持多代理模式，可在单一界面管理多个 Docker 宿主机的堆栈；支持将 docker run 命令转换为 compose.yaml。采用基于文件的本地存储结构，堆栈文件保留在本地，兼容原生 docker compose 命令，支持 Docker 20+ 或 Podman，界面实时反映操作进度与终端输出。
