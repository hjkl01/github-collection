### [moby moby](https://github.com/moby/moby)  ![GitHub Repo stars](https://img.shields.io/github/stars/moby/moby?style=social)

Moby 是由 Docker 创建的开源项目，旨在加速软件容器化。它提供了一套模块化、可互换的组件（如构建工具、注册表、编排工具和运行时），作为“乐高套装”供开发者组装自定义容器系统。该项目面向工程师和开发者，提供功能性 API 用于构建工具，而非作为最终用户产品。它是 Docker 产品的开源上游，允许组件复用，但仅提供社区支持。
