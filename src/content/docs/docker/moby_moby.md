### [moby moby](https://github.com/moby/moby)

**Moby 项目核心内容总结：**  

1. **项目功能**  
   Moby 是 Docker 开源的容器化工具集，提供容器构建、镜像仓库、编排、运行时等组件，可作为模块化构建块与其他工具结合，用于开发定制的容器系统。  

2. **主要特性**  
   - **模块化设计**：组件功能明确，通过标准化 API 协作。  
   - **灵活可替换**：提供完整功能组件，但支持替换为其他实现。  
   - **安全默认**：保障安全性的同时兼顾易用性。  
   - **开发者导向**：API 侧重功能实现，面向开发者而非终端用户。  

3. **目标用户**  
   面向工程师、集成者及容器技术爱好者，用于修改、实验、构建容器系统，非商业支持场景。  

4. **与 Docker 的关系**  
   Moby 是 Docker 项目的开源上游，Docker 产品基于 Moby 开发，其他项目也可复用其组件。  

5. **许可证**
