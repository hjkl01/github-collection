### [JanDeDobbeleer oh-my-posh](https://github.com/JanDeDobbeleer/oh-my-posh)

Oh My Posh 是一个跨平台的终端提示主题引擎，支持 PowerShell、Zsh、Fish 等主流 Shell，提供高度可定制的提示符样式。主要特性包括：  
- **高度可配置**：支持自定义主题、颜色、格式，满足个性化需求  
- **多平台兼容**：适用于 Windows、Linux、macOS  
- **功能丰富**：支持主提示符、副提示符、右侧提示、临时提示等多类型提示  
- **性能优化**：运行快速，资源占用低  
- **社区支持**：提供官方文档（[ohmyposh.dev](https://ohmyposh.dev)）、Mastodon 和 Discord 社群  

项目基于 MIT 许可证开源，可通过 GitHub 获取，支持多种 Shell 的安装方式（如 PowerShell 的 `Install-Module` 或 Linux 的包管理器）。
