### [wavetermdev waveterm](https://github.com/wavetermdev/waveterm)  ![GitHub Repo stars](https://img.shields.io/github/stars/wavetermdev/waveterm?style=social)

Wave Terminal 是一款开源的、集成 AI 功能的终端模拟器，支持 macOS、Linux 和 Windows 平台。

核心功能包括：

1.  **AI 集成**：支持多种 AI 模型（如 OpenAI、Claude、Ollama 等），具备上下文感知能力，可读取终端输出、执行文件操作，支持自带密钥（BYOK）及本地模型部署。
2.  **SSH 增强**：支持持久化 SSH 会话，断网或重启后自动重连；内置远程文件图形编辑器及多格式文件内联预览。
3.  **灵活布局**：支持拖拽界面组织终端块、编辑器、浏览器和 AI 助手，提供快速全屏切换功能。
4.  **工作流管理**：通过 `wsh` 命令行系统管理工作区，实现本地与远程主机间的文件无缝同步与复制。
5.  **安全与自定义**：利用系统原生后端安全存储密钥，支持主题、样式及背景自定义。
