### [karol-broda snitch](https://github.com/karol-broda/snitch)  ![GitHub Repo stars](https://img.shields.io/github/stars/karol-broda/snitch?style=social)

snitch 是一款面向人类友好的网络连接检查工具，旨在替代 `ss` / `netstat`。支持 Linux 和 macOS 系统，提供整洁的 TUI 交互式界面和样式化表格。核心功能包括实时监控网络状态、多维度过滤（协议、状态、IP 版本）、多格式输出（样式表、纯文本、JSON、CSV）、DNS 及端口解析、进程信息查看。支持主题定制、配置记忆、流式输出及自动升级。
