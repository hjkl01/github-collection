### [starship starship](https://github.com/starship/starship)

**Starship** 是一个跨平台的命令行提示符工具，支持多种 Shell（如 Bash、Zsh、Fish 等），提供高度可定制的提示符界面。其核心功能包括：

1. **功能**  
   - 显示当前 Shell 环境信息（如 Git 分支、时间、路径等）  
   - 支持图标和彩色显示，提升可读性  
   - 通过预设主题（Presets）快速应用不同风格  

2. **使用方法**  
   - 安装：通过包管理器（如 Homebrew、Chocolatey）或从发布页下载安装包  
   - 配置：根据使用的 Shell 类型，在配置文件中添加初始化脚本（如 `eval "$(starship init zsh)"`）  
   - 自定义：通过配置文件调整提示符样式、模块显示等  

3. **主要特性**  
   - 跨平台支持（Windows、macOS、Linux）  
   - 轻量级且加载速度快  
   - 支持多语言文档和社区贡献的翻译  
   - 提供官方文档和社区预设主题供参考  

项目采用 ISC 许可证，可通过 GitHub 赞助支持开发。
