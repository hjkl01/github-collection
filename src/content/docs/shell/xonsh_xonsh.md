### [xonsh xonsh](https://github.com/xonsh/xonsh)  ![GitHub Repo stars](https://img.shields.io/github/stars/xonsh/xonsh?style=social)

Xonsh 是一款由 Python 驱动的跨平台 Shell，语言为 Python 3 超集，实现了 Shell 功能与 Python 代码的无缝集成。它具备全功能及 AI 友好特性，支持在 Shell 环境中混合运行 Shell 命令与 Python 脚本。项目提供扩展系统（xontribs），构建丰富的 Shell 生态，兼容 Jupyter、conda、Starship 等多种工具，并支持多种安装部署方式。
