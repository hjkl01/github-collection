### [ghostty-org ghostty](https://github.com/ghostty-org/ghostty)  ![GitHub Repo stars](https://img.shields.io/github/stars/ghostty-org/ghostty?style=social)

Ghostty 是一款高性能、原生且功能丰富的跨平台终端模拟器。它符合终端标准，可无缝替换现有终端，支持多窗口、标签页及分屏功能。渲染引擎针对 macOS (Metal) 和 Linux (OpenGL) 优化，IO 处理性能优异。核心采用 Zig 编写，提供 `libghostty` C 库供第三方嵌入，并内置本地崩溃报告机制。
