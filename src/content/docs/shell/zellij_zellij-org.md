### [zellij-org zellij](https://github.com/zellij-org/zellij)  ![GitHub Repo stars](https://img.shields.io/github/stars/zellij-org/zellij?style=social)

Zellij 是一个面向开发者、运维人员及终端爱好者的终端工作区（终端复用器）。它强调在简洁与强大功能之间取得平衡，提供优秀的开箱即用体验。主要功能包括：支持深度定制与布局自动化、真正的多人协作、独特的浮动与堆叠分屏界面、基于 WebAssembly 的插件系统（支持任意编译型语言编写），以及内置 Web 客户端（可无需本地终端）。
