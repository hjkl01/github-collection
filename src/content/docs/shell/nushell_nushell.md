### [nushell nushell](https://github.com/nushell/nushell)  ![GitHub Repo stars](https://img.shields.io/github/stars/nushell/nushell?style=social)

Nushell 是一款现代化的跨平台命令行 Shell，旨在将数据视为结构化内容（如表格、记录）而非纯文本流。主要功能包括：
1. 结构化管道：命令通过管道连接，支持数据的生成、过滤和消费。
2. 结构化文件处理：支持将文件及 URL 内容解析为结构化数据，便于字段查询。
3. 插件系统：支持基于 JSON-RPC 的插件扩展，遵循统一数据模型。
4. 跨平台兼容：原生支持 Windows、macOS 和 Linux，兼容现有可执行文件。
5. 函数式流程：通过管道实现数据加载、变换和保存，避免可变状态。
