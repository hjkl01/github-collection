### [sxyazi yazi](https://github.com/sxyazi/yazi)  ![GitHub Repo stars](https://img.shields.io/github/stars/sxyazi/yazi?style=social)

Yazi 是一款基于 Rust 和异步 I/O 开发的极速终端文件管理器。它提供高效、用户友好且高度可定制的文件管理体验。核心特性包括：全异步 IO 支持、多线程任务调度、内置多种终端图像协议预览、代码高亮与图像预加载、基于 Lua 的并发插件系统、虚拟文件系统、数据分发服务及包管理器。此外，它集成 ripgrep、fd 等工具，支持 Vim 风格操作、多标签管理、批量重命名、归档解压、Git 集成、主题系统及鼠标支持。项目目前处于公共测试版阶段，可作为日常主力使用。
