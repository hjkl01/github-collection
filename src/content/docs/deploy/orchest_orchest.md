### [orchest orchest](https://github.com/orchest/orchest)  ![GitHub Repo stars](https://img.shields.io/github/stars/orchest/orchest?style=social)

Orchest 是一个数据管道构建工具，支持直接使用 Python、R 或 Julia 编写代码，无需框架或 YAML 配置。功能包括通过 UI 可视化构建管道、在 Notebook 和脚本中编码、运行作业子集或周期任务、定义跨机器依赖环境、启动持久服务以及使用 Git 管理项目版本。
注：项目已停止活跃开发，官方建议改用 Apache Airflow。
