### [keephq keep](https://github.com/keephq/keep)  ![GitHub Repo stars](https://img.shields.io/github/stars/keephq/keep?style=social)

Keep 是一个开源 AIOps 和告警管理平台，旨在提供统一的全景视图来管理所有告警和事件。其核心功能包括告警去重、丰富、过滤、关联以及与监控工具的双向集成。平台支持基于声明式 YAML 的自动化工作流（类似 GitHub Actions），允许自定义触发器、步骤和动作以实现流程自动化。具备 AI 驱动的告警关联和摘要能力（AIOps 2.0），广泛集成观测工具、AI 后端、数据库、通讯平台、事件管理及工单系统。同时提供企业级安全认证、灵活部署方案及生产规模的高可用性扩展能力。
