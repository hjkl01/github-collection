### [hashicorp terraform-provider-aws](https://github.com/hashicorp/terraform-provider-aws)

**核心内容总结：**  
Terraform AWS Provider 是 Terraform 的插件，用于管理 AWS 资源。用户可通过 Terraform 配置文件定义和自动化 AWS 服务（如 EC2、S3 等），实现基础设施即代码（IaC）。项目提供贡献指南、开发路线图、FAQ 和教程，支持社区讨论（discuss.hashicorp.com）。主要特性包括与 Terraform 深度集成、覆盖 AWS 主要服务、完善的文档支持。使用方法需通过 Terraform 配置文件调用 Provider 接口，具体实现参考官方文档。
