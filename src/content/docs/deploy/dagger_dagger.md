### [dagger dagger](https://github.com/dagger/dagger)  ![GitHub Repo stars](https://img.shields.io/github/stars/dagger/dagger?style=social)

Dagger 是一个自动化软件交付平台，用于构建、测试和部署代码。它具备可编程、本地优先、可重复和可观测的特性。主要功能包括：提供跨语言系统 API 编排容器与资源；支持 8 种语言的 SDK；实现内容寻址的增量执行与缓存；仅需容器运行时即可在本地、CI 或云端运行；内置 OpenTelemetry 追踪，确保交付过程的可靠性与一致性。
