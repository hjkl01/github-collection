### [infracost infracost](https://github.com/infracost/infracost)  ![GitHub Repo stars](https://img.shields.io/github/stars/infracost/infracost?style=social)

Infracost 是一款专为 Terraform 设计的云端成本估算与 FinOps 最佳实践工具。它使工程师能够在终端、VS Code 或拉取请求中，在实施更改前查看成本明细和月度成本差异。项目支持 AWS、Azure 和 Google 云超过 1,100 种资源及基于使用量的资源，并提供 Infracost Cloud SaaS 服务，支持团队策略管理、标签策略及合规性检查。
