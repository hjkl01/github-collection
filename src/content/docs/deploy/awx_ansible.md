### [ansible awx](https://github.com/ansible/awx)

**核心内容总结：**  
AWX 是基于 Ansible 构建的自动化平台，提供 Web 界面、REST API 和任务引擎，用于自动化运维流程。用户可通过 [INSTALL.md](./INSTALL.md) 安装，使用文档详见 [AWX 官方文档](https://ansible.readthedocs.io/projects/awx/en/latest/)。项目当前因大规模重构暂停新版本发布，详情需关注 [Ansible 论坛](https://forum.ansible.com/)。贡献代码需提交至 `devel` 分支，使用 `git commit --signoff` 签署提交，避免合并提交。问题反馈请参考 [ISSUES.md](./ISSUES.md)。项目遵循 [Ansible 行为准则](http://docs.ansible.com/ansible/latest/community/code_of_conduct.html)。
