### [external-secrets external-secrets](https://github.com/external-secrets/external-secrets)  ![GitHub Repo stars](https://img.shields.io/github/stars/external-secrets/external-secrets?style=social)

外部密钥操作符（External Secrets Operator）是一款 Kubernetes 操作符，用于集成外部密钥管理系统（如 AWS Secrets Manager、HashiCorp Vault、Google Secrets Manager、Azure Key Vault 等）。该操作符能够读取外部 API 中的信息，并自动将这些值注入到 Kubernetes Secret 中，实现密钥的自动化同步。
