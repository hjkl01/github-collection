### [kimdre doco-cd](https://github.com/kimdre/doco-cd)  ![GitHub Repo stars](https://img.shields.io/github/stars/kimdre/doco-cd?style=social)

doco-cd 是一个轻量级 GitOps 工具，用于自动部署和更新 Docker Compose 项目、服务及 Swarm 堆栈。它支持通过轮询或 Webhooks 触发，兼容多种 Git 提供者。项目基于 Go 语言开发，资源消耗极低，支持外部密钥管理、数据加密、通知推送及 Prometheus 监控，可作为 Docker 环境下的 Portainer 或 ArgoCD 替代品。
