### [harness drone](https://github.com/harness/drone)  ![GitHub Repo stars](https://img.shields.io/github/stars/harness/drone?style=social)

Harness Open Source 是一个开源开发平台，提供代码托管、自动化 DevOps 流水线、托管开发环境（Gitspaces）及制品注册表功能。该项目作为 Drone 的继任者，旨在提供端到端的开源 DevOps 解决方案。支持 Docker 或本地二进制部署，包含 Web 界面、REST API 和 CLI 工具，支持通过 Docker 容器执行流水线任务。
