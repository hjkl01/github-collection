### [aws aws-sdk-go-v2](https://github.com/aws/aws-sdk-go-v2)  ![GitHub Repo stars](https://img.shields.io/github/stars/aws/aws-sdk-go-v2?style=social)

AWS SDK for Go v2 是为 Go 语言设计的 AWS 服务开发工具包，支持通过代码调用 AWS 服务 API（如 DynamoDB）。它可从环境变量、共享凭证及配置文件中加载配置，要求 Go 1.24 及以上版本。项目提供 API 参考文档、迁移指南，并支持通过 GitHub 反馈问题、提交功能请求及代码贡献。
