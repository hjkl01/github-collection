### [zhenorzz goploy](https://github.com/zhenorzz/goploy)  ![GitHub Repo stars](https://img.shields.io/github/stars/zhenorzz/goploy?style=social)

Goploy 是一款基于 Web 的自动化部署系统工具，支持通过网页一键完成代码发布与回滚。它支持自动化构建项目，兼容 Git、SVN、FTP、SFTP 等代码源，可实现跨操作系统部署。此外，还提供 RBAC 权限控制、多维度监控（HTTP、TCP、Ping、进程、脚本及服务器）、定时任务管理、Xterm 终端访问、SFTP 服务及 LDAP 集成等功能。
