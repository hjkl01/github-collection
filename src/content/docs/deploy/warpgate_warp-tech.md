### [warp-tech warpgate](https://github.com/warp-tech/warpgate)  ![GitHub Repo stars](https://img.shields.io/github/stars/warp-tech/warpgate?style=social)

Warpgate 是一个智能且完全透明的 SSH、HTTPS、MySQL 和 PostgreSQL 堡垒机。它无需客户端应用或 SSH 包装器，部署于 DMZ 区域后即可实现用户与特定主机及 URL 的精确绑定。内置 Web 管理界面支持原生双因素认证（2FA）和单点登录（SSO），提供命令级审计及完整的会话记录（含实时查看与回放）。连接对客户端透明，非传统跳板机，采用自托管模式，由单二进制 Rust 文件构成且无额外依赖。
