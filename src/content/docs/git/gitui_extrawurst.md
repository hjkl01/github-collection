### [extrawurst gitui](https://github.com/extrawurst/gitui)  ![GitHub Repo stars](https://img.shields.io/github/stars/extrawurst/gitui?style=social)

GitUI 是一款基于 Rust 开发的终端 Git 图形界面工具，旨在提供类似 GUI 的便捷操作体验与终端的高效性。其核心功能包括：
- **基础操作**：支持全键盘控制、上下文帮助、代码检查、提交与修正（含 Hooks 支持）、文件及行块暂存与还原。
- **仓库管理**：涵盖分支管理（创建/删除/切换）、远程推送与拉取、暂存管理、提交历史浏览与搜索、子模块支持。
- **高级特性**：支持 GPG 提交签名、异步 Git API、响应式 UI。
- **性能优势**：相比同类工具（如 lazygit, tig），在处理大型仓库时速度更快、内存占用更低、无卡顿或崩溃。
- **可配置性**：支持跨平台部署、自定义主题与按键绑定（如 Vim 风格）。
