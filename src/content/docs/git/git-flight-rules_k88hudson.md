### [k88hudson git-flight-rules](https://github.com/k88hudson/git-flight-rules)  ![GitHub Repo stars](https://img.shields.io/github/stars/k88hudson/git-flight-rules?style=social)

这是一个名为"Git 飞行规则”的 Git 故障排除与操作指南。项目借鉴 NASA 宇航员手册理念，记录了当 Git 使用出现问题时的具体解决方案和标准操作流程。内容覆盖仓库管理、提交编辑与修正、暂存与丢弃操作、分支管理、合并与变基、历史查找、子模块、敏感数据清理、调试及配置优化等场景。通过提供具体的命令行示例，帮助用户快速解决误删提交、推送冲突等常见问题，规范 Git 开发流程。
