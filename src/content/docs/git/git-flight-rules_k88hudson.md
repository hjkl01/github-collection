### [k88hudson git-flight-rules](https://github.com/k88hudson/git-flight-rules)

**核心内容总结：**  
该项目是一份全面的Git使用指南，涵盖版本控制的基础操作与进阶技巧。主要功能包括：  
1. **常用命令**：提交代码（`commit`）、分支管理（`checkout`/`merge`）、冲突解决（`rebase`）、历史回退（`reset`/`reflog`）等。  
2. **配置管理**：设置全局用户信息（`user.name`/`user.email`）、忽略文件权限变更（`core.fileMode`）、凭证缓存（`osxkeychain`/`gnome-keyring`）等。  
3. **问题恢复**：通过`reflog`追踪分支变动历史，恢复误操作（如硬重置、错误合并）。  
4. **效率工具**：提供Git Bash/PowerShell快捷命令（如`alias`自定义函数），推荐GUI客户端（如GitKraken、Sourcetree）。  
5. **学习资源**：附带书籍推荐（《Pro Git》）、教程链接（Atlassian Git教程）、脚本工具（git-extras）等辅助学习材料。  

**主要特性**：  
- 结构化命令示例，适合不同场景（如合并分支、撤销提交）。  
- 强调`reflog`作为“安全网”的作用，解决误操作问题。  
- 覆盖从基础语法到高级工作流（如变基、分支策略）的完整知识链。
