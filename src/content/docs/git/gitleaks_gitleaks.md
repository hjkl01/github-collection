### [gitleaks gitleaks](https://github.com/gitleaks/gitleaks)  ![GitHub Repo stars](https://img.shields.io/github/stars/gitleaks/gitleaks?style=social)

Gitleaks 是一款检测 Git 仓库、文件及标准输入中密码、API 密钥和令牌等敏感信息的工具。它通过正则表达式和熵值分析识别密钥泄露，支持多种安装与集成方式（如 Docker、Homebrew、GitHub Action、Pre-commit）。核心功能包括自定义检测规则、允许列表过滤、基线管理以忽略历史问题、自动解码编码文本及扫描归档文件内容。支持多种报告格式输出，并提供指纹忽略、代码注释忽略等机制以减少误报。
