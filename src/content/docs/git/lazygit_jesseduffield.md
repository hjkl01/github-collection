### [jesseduffield lazygit](https://github.com/jesseduffield/lazygit)  ![GitHub Repo stars](https://img.shields.io/github/stars/jesseduffield/lazygit?style=social)

Lazygit 是一个用于 Git 命令的简洁终端用户界面，旨在简化复杂的 Git 操作流程。主要功能涵盖交互式变基、拣选提交、二分查找、单行代码暂存、修改历史提交、工作树管理、提交图可视化、提交比对及撤销重做操作。同时支持自定义命令、Git Flow 支持，并兼容 Windows、MacOS 和 Linux 系统。
