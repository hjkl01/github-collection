### [isacikgoz gitbatch](https://github.com/isacikgoz/gitbatch)  ![GitHub Repo stars](https://img.shields.io/github/stars/isacikgoz/gitbatch?style=social)

GitBatch 是一个基于 Go 的命令行工具，专注于批量管理多个 Git 仓库。它支持批量操作（如拉取更新），同时也提供对单个仓库的微管理功能（如 add、reset、stash、commit 等）。用户可在仓库父目录下直接运行该工具。
