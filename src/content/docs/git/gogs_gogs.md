### [gogs gogs](https://github.com/gogs/gogs)  ![GitHub Repo stars](https://img.shields.io/github/stars/gogs/gogs?style=social)

Gogs 是一个使用 Go 语言编写的简单、稳定且可扩展的自托管 Git 服务，支持跨平台部署（Linux、macOS、Windows、ARM 等）。它提供用户与组织管理、仓库访问（SSH/HTTP/HTTPS）、代码协作（Issue、Pull Request、Wiki）、Webhooks 集成、多种认证方式（SMTP、LDAP、GitHub 等）、数据库支持（MySQL、PostgreSQL、SQLite 等）及多语言本地化功能。项目对硬件要求低，易于安装，采用 MIT 开源协议，适合搭建私有 Git 平台供个人及团队使用。
