### [FredrikNoren ungit](https://github.com/FredrikNoren/ungit)  ![GitHub Repo stars](https://img.shields.io/github/stars/FredrikNoren/ungit?style=social)

Ungit 是一个基于 Web 的跨平台 Git 图形界面工具，旨在简化 Git 操作并提升易用性。它通过浏览器提供直观的操作界面，支持在本地或远程服务器（如云主机）上运行。核心功能包括：分支与提交可视化、GitHub 及 Gerrit 集成、PGP/GPG 签名支持、自定义配置、自动刷新、外部合并工具支持以及主流代码编辑器的插件集成。
