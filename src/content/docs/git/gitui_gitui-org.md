### [gitui-org gitui](https://github.com/gitui-org/gitui)  ![GitHub Repo stars](https://img.shields.io/github/stars/gitui-org/gitui?style=social)

GitUI 是一款基于 Rust 开发的终端 Git 图形界面工具，旨在提供终端内的 Git GUI 操作体验。它支持快速直观的键盘控制与上下文帮助，无需记忆大量快捷键。核心功能包括：文件的暂存、提交、回退、重置及代码暂存操作；分支列表管理（创建、切换、删除等）；远程推送与拉取；提交日志浏览、搜索与差异对比；以及子模块和 GPG 提交签名支持。该工具针对大型仓库优化，具有高性能、低内存占用及运行稳定的特点，支持自定义主题和快捷键绑定。
