### [MichaelMure git-bug](https://github.com/MichaelMure/git-bug)  ![GitHub Repo stars](https://img.shields.io/github/stars/MichaelMure/git-bug?style=social)

git-bug 是一个独立的去中心化、离线优先的问题追踪工具。它将问题、评论等数据作为 Git 对象嵌入仓库（而非普通文件），利用 Git 架构实现版本控制和分布式同步。核心功能包括：原生 Git 存储、毫秒级搜索性能、支持 GitHub/GitLab 等第三方平台桥接同步，以及提供 CLI、TUI 和 Web 多种交互界面，集成简单。
