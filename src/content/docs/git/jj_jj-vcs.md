### [jj-vcs jj](https://github.com/jj-vcs/jj)  ![GitHub Repo stars](https://img.shields.io/github/stars/jj-vcs/jj?style=social)

Jujutsu 是一款强大的版本控制系统，旨在简化代码版本管理。它采用抽象存储架构，当前以 Git 为存储后端，完全兼容主流 Git 工具及远程仓库。核心特性包括：工作区变更自动记录为提交，无需暂存区或 stash；记录所有操作日志，支持便捷撤销与状态恢复；修改提交后自动变基并传播冲突解决状态；提供丰富的历史记录重写命令。此外，它具备实验性的并发安全存储能力。项目目前处于实验阶段但功能较完善，采用 Apache 2.0 开源协议。
