### [mergestat mergestat](https://github.com/mergestat/mergestat)  ![GitHub Repo stars](https://img.shields.io/github/stars/mergestat/mergestat?style=social)

MergeStat 是一个基于 SQL 的 Git 仓库数据查询与分析工具，支持 GitHub API 等数据源。它提供 Web 管理界面和 Docker 部署方式，用于仓库添加及数据同步。使用 GitHub API 功能需提供个人访问令牌（PAT）。另有 CLI 工具 MergeStat-Lite 用于本地查询。
