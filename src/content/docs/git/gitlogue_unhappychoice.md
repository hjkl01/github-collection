### [unhappychoice gitlogue](https://github.com/unhappychoice/gitlogue)  ![GitHub Repo stars](https://img.shields.io/github/stars/unhappychoice/gitlogue?style=social)

gitlogue 是一款基于 Rust 的终端 Git 提交回放工具，将 Git 历史转化为带有打字动画、语法高亮和文件树过渡的可视化动画体验。支持屏幕保护模式、指定提交范围回放、实时 Diff 预览及多语言主题自定义。适用于代码演示、教育学习、终端装饰或展示项目演变历史。
