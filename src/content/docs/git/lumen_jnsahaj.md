### [jnsahaj lumen](https://github.com/jnsahaj/lumen)  ![GitHub Repo stars](https://img.shields.io/github/stars/jnsahaj/lumen?style=social)

Lumen 是一款基于 Rust 开发的命令行工具，专为 AI 编程时代提供代码审查与辅助功能。

**核心功能：**
1. **交互式 Diff 查看器**：终端内提供语法高亮的代码对比、注释管理、主题定制，支持查看本地变更、提交历史、分支差异及 GitHub 拉取请求。
2. **智能提交信息生成**：根据暂存更改自动生成符合规范的 Commit Message。
3. **变更解释**：分析并解释代码变更的具体内容、原因及潜在影响。
4. **Git 命令生成**：通过自然语言描述生成相应的 Git 命令，并提供安全提示与确认机制。
5. **多 AI 模型集成**：支持 OpenAI、Claude、Ollama 等多种 AI 提供商，用于生成提交信息、解释变更及辅助命令生成。

**配置：**
支持通过配置文件、环境变量或命令行参数灵活配置 AI 提供商、模型及 Diff 查看器主题。
