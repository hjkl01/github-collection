### [seanmonstar reqwest](https://github.com/seanmonstar/reqwest)  ![GitHub Repo stars](https://img.shields.io/github/stars/seanmonstar/reqwest?style=social)

Reqwest 是一个为 Rust 编写的易用且功能齐全的 HTTP 客户端库。
核心功能包括：
- 支持异步和阻塞模式
- 支持 JSON、urlencoded、multipart 及纯文本请求体
- 支持自定义重定向策略和 HTTP 代理
- 支持 rustls 或系统原生 TLS 的 HTTPS 连接
- 内置 Cookie 存储及 WASM 支持
