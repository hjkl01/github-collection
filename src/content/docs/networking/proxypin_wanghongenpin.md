### [wanghongenpin proxypin](https://github.com/wanghongenpin/proxypin)  ![GitHub Repo stars](https://img.shields.io/github/stars/wanghongenpin/proxypin?style=social)

ProxyPin 是一款基于 Flutter 开发的开源免费 HTTP(S) 流量捕获代理工具，支持 Windows、Mac、Android、iOS、Linux 全平台，可捕获 Flutter 应用流量。主要功能包括：

1. **扫码连接**：支持移动端扫码连接，无需手动配置 WiFi 代理，支持配置同步。
2. **流量管理**：支持域名过滤、请求拦截、关键字及响应类型搜索。
3. **请求处理**：支持 JavaScript 脚本处理、请求重写、重定向、本地映射及 AES 解密。
4. **历史与工具**：自动保存流量记录，支持 HAR 格式导入导出，内置工具箱和编码工具。
