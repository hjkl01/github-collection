### [wanghongenpin proxypin](https://github.com/wanghongenpin/proxypin)

ProxyPin 是一款开源免费的 HTTP(S) 流量抓包工具，支持 Windows、Mac、Android、iOS 和 Linux 全平台。基于 Flutter 开发，界面美观易用，可捕获 Flutter 应用流量。

**核心功能**  
- 移动端扫码连接：无需手动配置 WiFi 代理，支持跨终端扫码同步配置  
- 域名过滤：精准拦截目标流量，避免干扰其他应用  
- 请求处理：支持重写/拦截/映射请求，可自定义响应内容  
- 脚本扩展：通过 JavaScript 脚本处理请求或响应  
- 数据管理：自动保存流量记录，支持 HAR 格式导出/导入  
- 辅助工具：含编码工具、正则表达式、二维码生成等功能  

**使用方法**  
通过扫码实现设备间流量转发，支持多终端协同；首次在 Mac 上使用需前往系统设置允许任意来源应用。  
可通过 GitHub 发布页、应用商店或 Telegram 获取下载链接。
