### [quic-go quic-go](https://github.com/quic-go/quic-go)  ![GitHub Repo stars](https://img.shields.io/github/stars/quic-go/quic-go?style=social)

quic-go 是一个用纯 Go 语言实现的 QUIC 协议库，支持 RFC 9000 至 9002 标准，以及 HTTP/3、QPACK 和 HTTP Datagrams。它额外实现了 QUIC Version 2、不可靠数据报扩展、DPLPMTUD、qlog 日志记录及流重置等功能。该项目采用 MIT 协议开源，为 Go 应用提供高性能的网络通信支持。
