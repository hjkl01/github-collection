### [EasyTier EasyTier](https://github.com/EasyTier/EasyTier)  ![GitHub Repo stars](https://img.shields.io/github/stars/EasyTier/EasyTier?style=social)

EasyTier 是一个基于 Rust 和 Tokio 开发的简单、安全、去中心化的虚拟私有网络（VPN）解决方案。

1. **核心架构**：节点对等独立，无需中心化服务，支持跨平台（Win/Mac/Linux/Android 等）和多架构（X86/ARM/MIPS）。
2. **安全与传输**：采用 AES-GCM 或 WireGuard 加密，支持 TCP/UDP/WSS/WG 协议，具备零拷贝高性能传输。
3. **网络能力**：支持高效 NAT 穿透（UDP/IPv6），KCP/QUIC 抗丢包优化，智能路由选择，支持子网代理共享。
4. **部署与管理**：提供 Web 管理界面及命令行操作，支持零配置部署、共享节点快速组网及自建私有网络。
5. **扩展集成**：支持 WireGuard 客户端接入及自建公共共享节点。
