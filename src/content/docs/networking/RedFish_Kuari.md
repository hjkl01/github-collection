### [Kuari RedFish](https://github.com/Kuari/RedFish)

RedisFish 是一款跨平台、以数据操作为核心的 Redis 图形化客户端，支持处理 Redis 基础数据类型（如哈希、列表等），提供可视化数据编辑、状态颜色区分（如蓝色表示新数据、黄色表示编辑数据）、多选操作、JSON 自动格式化与编辑、键列表分页查询及前缀过滤功能。支持两种键列表显示模式（直接列表或按前缀分文件夹分类）和控制台多模式操作（单条/多条命令执行），具备数据库监控（CPU、内存、连接数等）、发布/订阅图形操作、TLS 加密、暗黑模式、中英文系统自动切换及 Mac 平台自动更新功能。用户可通过右键快捷键缩放数据编辑，或使用 Command/Ctrl + 左键复制内容。项目提供多语言支持，可通过 GitHub 发布页面下载。
