### [mitmproxy mitmproxy](https://github.com/mitmproxy/mitmproxy)

mitmproxy 是一个支持 SSL/TLS 的交互式抓包工具，可拦截和调试 HTTP/1、HTTP/2 及 WebSocket 流量。项目包含三个核心工具：mitmproxy（带控制台界面）、mitmdump（命令行工具，类似 tcpdump）和 mitmweb（基于 Web 的界面）。主要特性包括：支持 HTTPS 流量解密、协议版本全面覆盖、提供多种操作界面。用户可通过官方文档链接安装（提供预编译版本和源码安装说明），使用 GitHub 讨论区寻求帮助，开发者可通过 CONTRIBUTING.md 参与贡献。
