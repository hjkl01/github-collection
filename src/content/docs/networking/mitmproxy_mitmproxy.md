### [mitmproxy mitmproxy](https://github.com/mitmproxy/mitmproxy)  ![GitHub Repo stars](https://img.shields.io/github/stars/mitmproxy/mitmproxy?style=social)

mitmproxy 是一款支持 SSL/TLS 的交互式拦截代理工具，提供控制台界面，兼容 HTTP/1、HTTP/2 及 WebSockets 协议。它包含三个主要工具：mitmproxy（核心交互式版本）、mitmdump（命令行版本，类似 HTTP 的 tcpdump）以及 mitmweb（基于 Web 的管理界面）。
