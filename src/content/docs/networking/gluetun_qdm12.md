### [qdm12 gluetun](https://github.com/qdm12/gluetun)  ![GitHub Repo stars](https://img.shields.io/github/stars/qdm12/gluetun?style=social)

Gluetun 是一款基于 Docker 的轻量级多功能 VPN 客户端，基于 Alpine 系统构建，镜像体积小。支持多种主流 VPN 服务商（如 ExpressVPN、NordVPN 等），兼容 OpenVPN 及 Wireguard 协议。内置 DNS over TLS、DNS 广告与恶意域名过滤、防火墙防泄漏（Kill Switch）功能。提供 Shadowsocks 和 HTTP 代理服务，支持连接其他容器或局域网设备。兼容多 CPU 架构（amd64, ARM, ppc64le 等）及 Kubernetes 旁路容器模式，支持特定服务商的端口转发和 Split Horizon DNS 配置。
