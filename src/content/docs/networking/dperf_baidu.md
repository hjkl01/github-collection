### [baidu dperf](https://github.com/baidu/dperf)  ![GitHub Repo stars](https://img.shields.io/github/stars/baidu/dperf?style=social)

dperf 是一个基于 DPDK 的高性能网络流量生成和负载测试工具。它能够在单台 x86 服务器上生成海量流量，支持千万级 HTTP 连接每秒、数百 Gbps 吞吐量及数十亿并发连接。主要功能包括：对四层负载均衡器和网关进行负载及稳定性测试；进行云服务器网络性能基准测试；评估网卡和 CPU 的包处理能力；以及作为高性能 HTTP 服务器或客户端。工具提供实时统计数据，可监控连接速率、吞吐量、丢包和错误情况。
