### [UwUDev ygege](https://github.com/UwUDev/ygege)  ![GitHub Repo stars](https://img.shields.io/github/stars/UwUDev/ygege?style=social)

Ygégé 是一个基于 Rust 的高性能 YGG Torrent 索引器。具备自动解析域名、自动化绕过 Cloudflare 挑战、会话缓存及重连、绕过 DNS 欺骗等功能。支持低内存消耗运行，提供模块化搜索（按名称、种子数、日期等）及 torrent 详情获取（描述、大小、种子/催种数等）。无需浏览器驱动或外部依赖，支持集成至 Prowlarr 和 Jackett 作为自定义索引器，可选配置 IMDB/TMDB 元数据。提供 Docker 镜像及源码编译支持，利用 HTTP 客户端模拟 TLS 指纹以规避检测。
