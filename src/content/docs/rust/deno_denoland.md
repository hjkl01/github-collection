### [denoland deno](https://github.com/denoland/deno)  ![GitHub Repo stars](https://img.shields.io/github/stars/denoland/deno?style=social)

Deno 是一个基于 V8、Rust 和 Tokio 构建的 JavaScript、TypeScript 和 WebAssembly 运行时。它具有安全默认设置和优秀的开发者体验，主要用于构建 Web 服务器及其他应用程序。项目提供多种安装方式、官方文档、标准库及 JSR 包注册表。
