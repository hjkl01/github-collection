### [sunface rust-by-practice](https://github.com/sunface/rust-by-practice)  ![GitHub Repo stars](https://img.shields.io/github/stars/sunface/rust-by-practice?style=social)

该项目是一个 Rust 学习与实践平台，通过挑战性示例、练习和项目帮助开发者深入掌握 Rust，填补学习理论与实际项目的差距。

核心功能：
1. 每章包含示例、练习和实践，覆盖异步、线程、数据结构等 Rust 核心内容。
2. 提供大量可在线阅读、编辑和运行的练习代码。
3. 难度从易到超难，每道练习均提供解决方案。
4. 支持在线阅读及使用 mdbook 本地运行。
