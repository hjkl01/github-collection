### [sunface rust-course](https://github.com/sunface/rust-course)  ![GitHub Repo stars](https://img.shields.io/github/stars/sunface/rust-course?style=social)

《Rust 语言圣经》是一个全面、深入且持续更新的 Rust 在线教程。项目涵盖从入门到精通的完整知识体系，包含基础教学、高级专题（如性能优化、异步编程、WASM 等）、实用工具索引、避坑指南、实战代码片段（Cookbook）及配套练习题。旨在帮助中国用户高效学习 Rust，适用于新手入门、开发者进阶及提升生产力。
