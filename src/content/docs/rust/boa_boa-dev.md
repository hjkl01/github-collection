### [boa-dev boa](https://github.com/boa-dev/boa)  ![GitHub Repo stars](https://img.shields.io/github/stars/boa-dev/boa?style=social)

Boa 是一个使用 Rust 编写的实验性 JavaScript 引擎，具备词法分析、解析和解释功能，支持超过 90% 的最新 ECMAScript 规范。项目包含多个核心模块（如引擎、AST、CLI、REPL、垃圾回收器等），支持库集成、命令行运行及 WebAssembly 部署。此外提供 API 文档、Test262 合规性报告及性能基准测试，并持续跟进 ECMAScript 标准的更新。
