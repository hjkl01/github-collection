### [emilk egui](https://github.com/emilk/egui)  ![GitHub Repo stars](https://img.shields.io/github/stars/emilk/egui?style=social)

egui 是一个基于 Rust 的简单、快速且高度可移植的即时模式图形用户界面（GUI）库。它采用纯 Rust 实现（无 unsafe 代码），支持在 Web（Wasm）、原生平台（Windows、macOS、Linux、Android）及游戏引擎中运行。核心功能包括丰富的控件（按钮、滑块、文本编辑、图像、窗口等）、自动布局、自定义绘制及样式定制。通过官方框架 eframe，可实现跨平台应用开发，代码在 Web 和原生环境通用。项目注重易用性、可集成性和安全性，支持 60Hz 响应速度，适用于构建 Rust 交互式应用或游戏界面，但不提供原生外观。
