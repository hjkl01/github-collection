### [esp-rs esp-hal](https://github.com/esp-rs/esp-hal)  ![GitHub Repo stars](https://img.shields.io/github/stars/esp-rs/esp-hal?style=social)

esp-hal 是专为乐鑫（Espressif）设备设计的基于 Rust 的裸机（no_std）硬件抽象层（HAL）。支持 ESP32、ESP32-C（C2/C3/C6）、ESP32-H（H2）及 ESP32-S（S2/S3）系列芯片，并通过 esp-lp-hal 提供低功耗 RISC-V 核心编程支持。项目包含完整文档、示例及开发指南，遵循 MIT 和 Apache-2.0 双重许可。
