### [leptos-rs leptos](https://github.com/leptos-rs/leptos)  ![GitHub Repo stars](https://img.shields.io/github/stars/leptos-rs/leptos?style=social)

Leptos 是一个基于 Rust 的全栈同构 Web 框架，利用细粒度响应式系统构建声明式用户界面，无虚拟 DOM 开销。它支持客户端渲染、服务器端渲染（SSR）及水合渲染，并提供数据与 HTML 的流式传输。通过服务器函数实现前后端逻辑无缝集成，无需维护独立 API。框架基于 Web 标准构建，提供 `view!` 宏或构建器语法定义 UI，并支持 `cargo-leptos` 简化构建。
