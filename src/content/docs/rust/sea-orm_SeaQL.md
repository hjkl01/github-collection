### [SeaQL sea-orm](https://github.com/SeaQL/sea-orm)  ![GitHub Repo stars](https://img.shields.io/github/stars/SeaQL/sea-orm?style=social)

SeaORM 是一个面向 Rust 的 Web 服务 ORM 库。它支持 1-1、1-N、M-N 等复杂关系建模，并提供智能加载器自动解决 N+1 查询问题。项目具备强大的迁移系统，支持 Entity First 与 Schema First 工作流及自动模式同步。通过 ActiveModel API 可实现嵌套对象的原子化持久操作。此外，SeaORM 提供原生 SQL 支持、同步模式兼容、Seaography GraphQL 框架集成及 SeaORM Pro 管理后台生成能力，并支持 SQL Server 等主流数据库，满足生产级应用需求。
