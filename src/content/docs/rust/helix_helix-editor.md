### [helix-editor helix](https://github.com/helix-editor/helix)  ![GitHub Repo stars](https://img.shields.io/github/stars/helix-editor/helix?style=social)

Helix 是一款使用 Rust 编写的终端文本编辑器，设计灵感源自 Kakoune 和 Neovim。其核心功能包括：Vim 风格模态编辑、多选择编辑、内置语言服务器支持（LSP），以及基于 tree-sitter 的智能增量语法高亮和代码编辑。
