### [gfx-rs wgpu-native](https://github.com/gfx-rs/wgpu-native)  ![GitHub Repo stars](https://img.shields.io/github/stars/gfx-rs/wgpu-native?style=social)

wgpu-native 是一个基于 wgpu-core 的 Rust 原生 WebGPU 实现。它提供 C FFI 接口，支持 Rust、Python、.NET、Go、C++ 等多种语言绑定。项目提供 macOS、Windows 和 Linux 的预构建二进制文件，并包含用于基本渲染和计算的 C 语言示例。
