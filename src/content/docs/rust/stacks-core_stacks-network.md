### [stacks-network stacks-core](https://github.com/stacks-network/stacks-core)  ![GitHub Repo stars](https://img.shields.io/github/stars/stacks-network/stacks-core?style=social)

该项目是使用 Rust 编写的 Stacks 区块链参考实现。它作为一个 Layer-2 区块链，利用比特币作为基础层安全性，无需修改比特币即可支持去中心化应用和可预测的智能合约（使用 Clarity 语言）。项目采用证明传输（PoX）挖矿机制，在比特币链上进行领导者选举，并在 Stacks 链上生成新区块。
