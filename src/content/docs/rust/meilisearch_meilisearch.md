### [meilisearch meilisearch](https://github.com/meilisearch/meilisearch)

**核心内容总结：**  
Meilisearch 是一个开源的搜索引擎，提供快速、相关性高的全文搜索、语义搜索及混合搜索功能，支持多语言、拼写容错、过滤、排序、地理搜索等特性。用户可通过 RESTful API 或 SDK（支持多种编程语言）集成到应用中，文档和示例（如电影搜索、电商网站）辅助快速上手。  

主要特性包括：  
- **混合搜索**：结合全文与语义搜索；  
- **实时响应**：毫秒级搜索结果；  
- **高级功能**：过滤、分面搜索、多租户管理、安全策略；  
- **多语言支持**：覆盖全球语言；  
- **扩展性**：企业版支持分片等高级功能。  

提供 **社区版（MIT 许可，免费开源）** 和 **企业版（需商业授权）** 两种版本。用户可选择本地部署或使用 Meilisearch Cloud 服务。  

**注意事项：**  
- 默认收集匿名使用数据，用户可自行关闭；  
- 企业版需联系销售获取授权。
