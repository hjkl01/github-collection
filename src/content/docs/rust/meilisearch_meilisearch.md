### [meilisearch meilisearch](https://github.com/meilisearch/meilisearch)  ![GitHub Repo stars](https://img.shields.io/github/stars/meilisearch/meilisearch?style=social)

Meilisearch 是一款极速、易部署的开源搜索引擎，可无缝集成至应用与网站。支持混合搜索（语义与全文）、即时搜索（<50ms 响应）、容错拼写、筛选与分面搜索、排序、同义词、地理搜索及多语言优化。提供 RESTful API 与多语言 SDK，具备 AI 就绪特性及细粒度安全管理（API 密钥、多租户）。分为社区版（MIT 开源）和企业版（含分片等功能），安装维护简便。
