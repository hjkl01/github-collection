### [FyroxEngine Fyrox](https://github.com/FyroxEngine/Fyrox)

Fyrox 是一个用 Rust 编写的现代 2D/3D 游戏引擎，提供场景编辑器，曾用名 rg3d。其核心功能包括：  
- 支持 2D/3D 游戏开发，具备完整生产级功能  
- 提供官方教程书籍（[fyrox-book.github.io](https://fyrox-book.github.io/)）和在线示例（[fyrox.rs/examples.html](https://fyrox.rs/examples.html)）  
- 可通过 Discord 服务器（[discord.gg/xENF5Uh](https://discord.gg/xENF5Uh)）或 GitHub 讨论区获取社区支持  
- 支持通过 crates.io 安装（[crates.io/crates/fyrox](https://crates.io/crates/fyrox)）  
- 提供文档（[docs.rs/Fyrox](https://docs.rs/Fyrox/)）和源码示例（[Fyrox-demo-projects](https://github.com/FyroxEngine/Fyrox-demo-projects)）  
- 开源协议为 MIT，持续集成状态可通过 GitHub 查看
