### [charlotte-os Catten](https://github.com/charlotte-os/Catten)  ![GitHub Repo stars](https://img.shields.io/github/stars/charlotte-os/Catten?style=social)

Catten 是 CharlotteOS 的通用单体内核，强调清晰度、安全性与灵活性，可作为其他系统的基础。项目主要使用 Rust 开发，支持 x86-64、Aarch64 及 RISC-V64 架构。内核采用最小化系统调用接口，提供类型化、能力保护的系统命名空间，兼容 UEFI、ACPI 及设备树固件。目前处于早期开发阶段，支持标准内存、存储、显示、输入及网络硬件，采用 AGPL-3.0 协议开源。
