### [rust-lang rust-clippy](https://github.com/rust-lang/rust-clippy)  ![GitHub Repo stars](https://img.shields.io/github/stars/rust-lang/rust-clippy?style=social)

Clippy 是 Rust 代码检查工具，包含 800 多个 lint 规则，用于捕获常见错误并改进代码质量。规则分为正确性、风格、性能等类别，并设有不同默认级别。支持通过 cargo clippy 命令运行，可自动修复部分建议。允许通过代码属性、命令行参数或 clippy.toml 配置文件自定义 lint 级别与行为，支持指定最小 Rust 版本（MSRV）及集成至 CI 流程。
