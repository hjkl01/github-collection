### [matrix-construct tuwunel](https://github.com/matrix-construct/tuwunel)  ![GitHub Repo stars](https://img.shields.io/github/stars/matrix-construct/tuwunel?style=social)

Tuwunel 是一款用 Rust 编写的高性能 Matrix 家庭服务器，可作为 Synapse 的替代方案。它完全遵循 Matrix 规范，支持各类客户端、网桥和机器人，具备可扩展、低成本及企业级特性。作为 conduwuit 的官方继任者，该项目支持多种部署方式（Docker、静态二进制、系统包等），并允许从 conduwuit 无缝迁移。获瑞士政府支持，致力于提供稳定、安全且易于维护的通信服务。
