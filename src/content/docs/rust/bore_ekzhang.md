### [ekzhang bore](https://github.com/ekzhang/bore)  ![GitHub Repo stars](https://img.shields.io/github/stars/ekzhang/bore?style=social)

bore 是一款基于 Rust 编写的轻量级 TCP 隧道工具，核心功能是将本地端口暴露到远程服务器，从而绕过 NAT 防火墙实现公网访问。它类似 ngrok 或 localtunnel，但设计更简单、高效，易于安装和自托管。项目专注于 TCP 流量转发，支持本地端口映射、自托管服务器部署、端口指定、本地主机绑定及可选的密钥认证，仅需客户端和服务端各一个二进制文件即可运行。
