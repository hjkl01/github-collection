### [rust-lang cargo](https://github.com/rust-lang/cargo)  ![GitHub Repo stars](https://img.shields.io/github/stars/rust-lang/cargo?style=social)

Cargo 是 Rust 项目的官方包管理器与构建工具，主要用于下载依赖并编译项目。它支持通过第三方扩展新增子命令，发布节奏与 Rust 保持一致。源码编译需 Rust 工具链、C 编译器及 Git 等环境。项目采用 MIT 和 Apache 2.0 双重许可证。
