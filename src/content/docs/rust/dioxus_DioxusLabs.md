### [DioxusLabs dioxus](https://github.com/DioxusLabs/dioxus)  ![GitHub Repo stars](https://img.shields.io/github/stars/DioxusLabs/dioxus?style=social)

Dioxus 是一款基于 Rust 的跨平台应用开发框架，支持通过单一代码库构建 Web、桌面（macOS/Linux/Windows）、移动端（iOS/Android）及服务器端应用。其核心功能包括零配置启动与集成热重载、基于信号的高效状态管理、内置 CLI 工具链及全栈 Web 框架能力。支持多种渲染后端（Webview、WGPU、SSR 等），具备原生系统 API 调用能力，并提供丰富的基础组件库与完善的文档体系。
