### [davidlattimore wild](https://github.com/davidlattimore/wild)  ![GitHub Repo stars](https://img.shields.io/github/stars/davidlattimore/wild?style=social)

Wild 是一个旨在为迭代开发提供极速链接性能的 Rust 链接器。虽然增量链接尚未实现，但当前非增量链接速度已非常快，未来计划实现该功能。目前主要支持 Linux 平台（x86-64、ARM64、RISC-V、LoongArch64），支持静态链接、动态链接、共享库、调试信息及基础链接脚本。Wild 可用作 Rust 和 C/C++ 项目的默认链接器。
