### [gfx-rs wgpu](https://github.com/gfx-rs/wgpu)  ![GitHub Repo stars](https://img.shields.io/github/stars/gfx-rs/wgpu?style=social)

wgpu 是一个跨平台的、安全的纯 Rust 图形 API。它原生支持 Vulkan、Metal、D3D12 和 OpenGL，并支持在 WebAssembly 上运行 WebGL2 和 WebGPU。该 API 基于 WebGPU 标准，是 Firefox、Servo 和 Deno 中 WebGPU 集成的核心。此外，通过 wgpu-native 可在 C 等其他语言中使用。项目提供示例、文档及环境配置说明，致力于遵循 WebGPU 和 WGSL 规范。
