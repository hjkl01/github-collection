### [mozilla uniffi-rs](https://github.com/mozilla/uniffi-rs)  ![GitHub Repo stars](https://img.shields.io/github/stars/mozilla/uniffi-rs?style=social)

UniFFI 是一个 Rust 多语言绑定生成器，用于构建跨平台组件。它允许开发者用 Rust 编写核心逻辑，通过接口定义文件或宏定义对象模型，编译为共享库后，可生成 Kotlin、Swift、Python、Ruby 等语言的绑定。这使得不同平台的语言能直接调用 Rust 代码，实现跨语言互操作。
