### [rust-lang mdBook](https://github.com/rust-lang/mdBook)

**项目功能**：mdBook 是一个从 Markdown 文件生成现代在线书籍的工具。  
**使用方法**：通过 [用户指南](https://rust-lang.github.io/mdBook/) 获取安装和使用说明，并可参考该指南作为书籍样例。  
**主要特性**：支持 Markdown 格式、提供插件扩展功能、可生成交互式文档。
