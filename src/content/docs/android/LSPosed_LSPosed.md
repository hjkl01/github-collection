### [LSPosed LSPosed](https://github.com/LSPosed/LSPosed)  ![GitHub Repo stars](https://img.shields.io/github/stars/LSPosed/LSPosed?style=social)

LSPosed Framework 是一个基于 Riru 或 Zygisk 的 Android 模块，提供与原版 Xposed API 一致的 ART 挂钩框架。它利用 LSPlant 技术在内存中修改系统及应用行为，无需修改 APK。支持 Android 8.1 至 14 版本，完全兼容原版 Xposed 模块，支持开发者编写和部署功能模块。
