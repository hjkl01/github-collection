### [tiann KernelSU](https://github.com/tiann/KernelSU)  ![GitHub Repo stars](https://img.shields.io/github/stars/tiann/KernelSU?style=social)

KernelSU 是一款基于内核的 Android 设备 Root 解决方案。核心功能包括：提供基于内核的 `su` 权限管理、基于 metamodules 的模块系统（支持无系统修改）以及 App Profile 权限控制功能。项目官方支持 Android GKI 2.0 设备（内核 5.10+）及旧内核（需手动编译），兼容 WSA、ChromeOS 等环境，架构仅支持 arm64-v8a 和 x86_64。
