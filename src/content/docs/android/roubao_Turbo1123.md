### [Turbo1123 roubao](https://github.com/Turbo1123/roubao)  ![GitHub Repo stars](https://img.shields.io/github/stars/Turbo1123/roubao?style=social)

肉包 Roubao 是一款基于视觉语言模型（VLM）和 Multi-Agent 协作架构的开源 AI 手机自动化助手，核心功能总结如下：

1.  **无需电脑**：原生 Kotlin 实现，利用 Shizuku 获取系统级权限，在手机本地闭环完成截图、分析及操作指令执行，无需 ADB 或电脑中转。
2.  **智能自动化**：支持自然语言指令控制手机，具备 Tools（原子能力）+ Skills（用户意图）双层架构，可智能决策执行“委托模式”（DeepLink 跳转 AI 应用）或"GUI 自动化模式”。
3.  **高度定制与安防**：支持接入多种 VLM 模型（如通义千问、GPT-4V），采用 Material 3 UI 设计，支持 API Key 加密存储及敏感页面自动停止执行。
