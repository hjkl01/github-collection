### [mzlogin awesome-adb](https://github.com/mzlogin/awesome-adb)  ![GitHub Repo stars](https://img.shields.io/github/stars/mzlogin/awesome-adb?style=social)

本项目是一个 Android Debug Bridge (ADB) 命令集合与使用指南。主要功能包括：

1.  **基本操作**：命令语法、目标设备指定、服务启动/停止及 Root 权限管理。
2.  **连接管理**：支持 USB 连接及多种场景下的无线连接配置（含 Android 11+）。
3.  **应用管理**：涵盖 APK 安装、卸载、启用/禁用、数据清理及详细信息查询。
4.  **交互控制**：支持启动/停止 Activity 和 Service、发送广播、模拟按键、滑动及文本输入。
5.  **信息查看**：提供日志（Logcat/Kernel）、设备硬件/系统信息、网络参数及文件传输。
6.  **系统设置**：可修改分辨率、屏幕密度、显示区域，控制 USB 调试及系统属性。
7.  **实用工具**：包含屏幕截图与录制、重新挂载系统分区、压力测试、重启及刷机模式切换。
8.  **安全与 Shell**：提供 SELinux/dm_verity 控制、常用 Linux Shell 命令参考及常见问题排查。
