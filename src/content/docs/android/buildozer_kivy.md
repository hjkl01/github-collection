### [kivy buildozer](https://github.com/kivy/buildozer)  ![GitHub Repo stars](https://img.shields.io/github/stars/kivy/buildozer?style=social)

Buildozer 是一个将 Python 应用程序打包为多平台（Android、iOS、Windows、macOS、Linux）二进制软件的开发工具。通过提供 `buildozer.spec` 配置文件，它可自动处理构建依赖并生成安装包。由 Kivy 团队维护，基于 `python-for-android` 和 `kivy-ios` 等项目，不仅支持 Kivy 框架，也可用于其他 GUI 框架。支持命令行操作，并提供 Docker 镜像及 GitHub Actions 集成部署。
