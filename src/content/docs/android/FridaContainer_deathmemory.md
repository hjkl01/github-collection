### [deathmemory FridaContainer](https://github.com/deathmemory/FridaContainer)  ![GitHub Repo stars](https://img.shields.io/github/stars/deathmemory/FridaContainer?style=social)

FridaContainer 是一个整合常用 Frida 脚本的逆向效率提升工具，支持 Android、iOS 平台及跨平台通用功能。

主要功能：
- Android：反调试绕过、堆栈打印、Dump Dex、SSL Pinning 绕过、JNI Hook 及方法追踪。
- iOS：函数地址获取、堆栈打印、UI 结构 Dump、数据类型转换。
- 通用：模块信息处理、地址查模块、寄存器获取、模块 Dump。

提供 PyCharm API 代码补全支持，支持源码实时编译开发。
