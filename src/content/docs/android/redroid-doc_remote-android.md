### [remote-android redroid-doc](https://github.com/remote-android/redroid-doc)  ![GitHub Repo stars](https://img.shields.io/github/stars/remote-android/redroid-doc?style=social)

redroid 是一款 GPU 加速的云端 Android 解决方案，支持在 Linux 宿主机（如 Docker、Podman、Kubernetes）上部署多个 Android 实例，兼容 arm64 和 amd64 架构，适用于云游戏、虚拟化手机及自动化测试。项目支持 Android 8.1 至 16 多个版本，提供显示、网络、GPU 模式等灵活配置。此外，支持通过原生桥接在 x86 实例运行 ARM 应用，可集成 GMS 服务，并计划引入 WebRTC 流媒体功能。构建流程类似 AOSP，建议使用 Docker。
