### [AutoHotkey AutoHotkey](https://github.com/AutoHotkey/AutoHotkey)

**项目功能**  
AutoHotkey 是一款开源自动化工具，通过自定义脚本语言实现键盘热键绑定、重复任务自动化，支持创建宏和控制软件行为。

**使用方法**  
- **开发环境**：需使用 Microsoft Visual Studio 2022 或 VS Code（需安装 C++ 插件和构建工具）打开项目源码进行编译。  
- **编译配置**：支持 Debug/Release/Self-contained 等模式，可选择 Win32 或 x64 平台，Unicode 或 ANSI 字符集。  
- **版本支持**：v2 为主推版本，v1 已停止维护但社区仍提供支持。

**主要特性**  
- 提供多版本构建选项（如 DLL 实验性模块）。  
- 支持 Windows XP 及以上系统（部分旧系统需额外补丁）。  
- 社区论坛提供技术支持，可提交问题或报告漏洞。  
- 编译后可生成 AutoHotkey.exe 或 AutoHotkeySC.bin 用于脚本执行。
