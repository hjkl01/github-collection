### [Anuken Mindustry](https://github.com/Anuken/Mindustry)  ![GitHub Repo stars](https://img.shields.io/github/stars/Anuken/Mindustry?style=social)

Mindustry 是一款基于 Java 开发的自动化塔防即时战略（RTS）游戏。项目支持桌面端（Windows/Linux/Mac）、服务器端及 Android 移动端，开源并提供构建指南与文档。用户可通过 itch.io、Google Play、F-Droid 等平台下载体验。
