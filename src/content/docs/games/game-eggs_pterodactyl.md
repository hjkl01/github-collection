### [pterodactyl game-eggs](https://github.com/pterodactyl/game-eggs)  ![GitHub Repo stars](https://img.shields.io/github/stars/pterodactyl/game-eggs?style=social)

本项目是一个分类整理的游戏服务器配置与资源列表仓库。它提供了多款流行游戏及其衍生服务器版本的搭建指南、配置文件及 MOD 支持，涵盖生存、射击、驾驶、模拟等多种类型。每个条目均链接至具体配置文件夹，用户可查看详细设置与部署方法，并支持提交 Issue 或 PR 进行内容改进。
