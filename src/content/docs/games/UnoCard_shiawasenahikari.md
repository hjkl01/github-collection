### [shiawasenahikari UnoCard](https://github.com/shiawasenahikari/UnoCard)

**项目核心内容总结：**

该项目是一个基于 UNO 卡牌游戏规则开发的桌面和 Android 平台应用，支持多人对战。玩家可从牌库中抓牌，按照规则出牌，使用特殊功能卡（如 +2、反转、跳过、万能牌、+4 王牌等）干扰对手或改变游戏节奏。游戏支持多种特殊规则，如“7-0”交换手牌、“叠加”功能等，增加了游戏的策略性和趣味性。

**使用方法：**

- **桌面版**：用户可通过 GitHub 下载编译好的安装包（.apk）进行安装，或使用 Android Studio 编译源码后运行。
- **Android 版**：用户可从 GitHub 下载 APK 安装包，安装后即可在手机上运行游戏。
- **开发环境**：支持 Windows、Linux、macOS 系统，需安装 Qt、Android Studio 等开发工具。

**主要特性：**

- 支持标准 UNO 卡牌规则和多种自定义特殊规则。
- 支持多人对战，包括电脑 AI 和真人玩家。
- 界面友好，操作简单，适合各年龄段玩家。
- 自动识别并处理游戏中的违规操作（如非法使用 +4 王牌）。
- 提供“UNO”自动提示功能，避免玩家因忘记喊“UNO”而受罚。
