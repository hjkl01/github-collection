### [1995chen dnf](https://github.com/1995chen/dnf)  ![GitHub Repo stars](https://img.shields.io/github/stars/1995chen/dnf?style=social)

该项目是一个将《地下城与勇士》（DNF）整合为 Docker 镜像的私服部署方案，基于 CentOS 系统，通过环境变量和初始化脚本实现游戏服务器的快速部署。核心功能包括支持多区/大区架构、插件登录器支持、MySQL 数据库自动初始化、进程管理 Web 页面及端口映射配置。项目提供完整的服务器初始化、启动验证、常见问题排查及客户端登录器配置指南。项目声明仅用于学习研究，严禁用于商业对外开服。
