### [Stephan-S FS25_AutoDrive](https://github.com/Stephan-S/FS25_AutoDrive)

**项目核心内容总结：**  
FS25_AutoDrive 是一款为《Farming Simulator 25》设计的自动导航模组，提供路径规划、自动行驶、交叉路口处理及倒车功能。用户可通过 GitHub 下载最新版本（3.0.0.8），并加入 Discord 服务器获取支持。模组包含独立扩展功能，如可放置的交叉路口、T型路口及倒车至标记点的模块，另有预设 AD 路线的建筑模组可供下载。课程编辑器由社区成员维护，支持自定义路径设计。
