### [AvarianKnight pma-voice](https://github.com/AvarianKnight/pma-voice)  ![GitHub Repo stars](https://img.shields.io/github/stars/AvarianKnight/pma-voice?style=social)

pma-voice 是一款专为 FiveM/RedM 平台设计的基于 Mumble 服务器的语音通讯系统。其核心功能包括近距离语音交流（支持低语、正常、喊叫三种模式）、无线电台通讯、私人呼叫系统以及可自定义的 UI 界面。系统支持多种音频模式配置（原生 3D、2D 等），允许管理员管理玩家语音状态（如静音），并提供丰富的客户端与服务端 API 导出及状态获取器以便脚本集成。该系统不支持与其他语音系统共存。
