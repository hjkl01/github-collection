### [game1024 OpenSpeedy](https://github.com/game1024/OpenSpeedy)  ![GitHub Repo stars](https://img.shields.io/github/stars/game1024/OpenSpeedy?style=social)

OpenSpeedy 是一款开源免费的 Windows 游戏加速工具，通过 Hook Windows 系统时间函数（如 Sleep、GetTickCount 等）在 Ring3 级别突破游戏帧率限制，提供流畅的加速体验。项目支持自定义加速倍率，兼容 x86 和 x64 进程，资源占用低且无需修改内核。界面简洁易用，适用于多种游戏引擎。注意：仅供学习研究，在线游戏使用可能导致封号或物理引擎异常。
