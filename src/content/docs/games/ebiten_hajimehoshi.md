### [hajimehoshi ebiten](https://github.com/hajimehoshi/ebiten)  ![GitHub Repo stars](https://img.shields.io/github/stars/hajimehoshi/ebiten?style=social)

Ebitengine 是一款专为 Go 语言设计的开源简单 2D 游戏引擎，支持快速开发并部署跨平台的 2D 游戏。它支持 Windows、macOS、Linux、Android、iOS、WebAssembly 及 Nintendo Switch 等平台。核心功能包括 2D 图形处理（几何与颜色变换、离屏渲染、文本渲染、自定义着色器等）、多类型输入支持（鼠标、键盘、手柄、触摸）以及音频播放（支持多种格式）。
