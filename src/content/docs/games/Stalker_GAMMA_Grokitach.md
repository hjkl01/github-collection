### [Grokitach Stalker_GAMMA](https://github.com/Grokitach/Stalker_GAMMA)  ![GitHub Repo stars](https://img.shields.io/github/stars/Grokitach/Stalker_GAMMA?style=social)

G.A.M.M.A. 是一款专为《S.T.A.L.K.E.R. Anomaly》打造的自动模块化模改整合包，旨在提供易于安装、自动更新且平衡的生存游戏体验。主要功能包括：

1. **玩法重塑**：移除装备交易与随机战利品，强调工具包获取与修复，北方区域越危险奖励越好。
2. **系统优化**：重构枪械逻辑、物品合成、治疗系统及区域访问权限。
3. **视听升级**：全面翻新图形、模型、动画及战斗 AI。
4. **部署方式**：整合包免费，组件自动从 ModDB 或 GitHub 下载，支持 GOG 一键安装。
