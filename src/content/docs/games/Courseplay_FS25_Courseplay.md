### [Courseplay Courseplay_FS25](https://github.com/Courseplay/Courseplay_FS25)

**项目核心内容总结：**  
Courseplay 是 Farming Simulator 25 的自动化驾驶模组，提供车辆路径规划、任务自动化等功能，支持多语言教程和详细文档。  

**使用方法：**  
通过 ModHub 或 GitHub 下载最新版本（FS25_Courseplay.zip），仅官方渠道（ModHub 和 GitHub）提供正版，其他来源可能为盗版。  

**主要特性：**  
- 自动化农田作业（如播种、收割）  
- 车辆协同与路径优化  
- 支持自定义车辆配置  
- 提供中文/英文教程及游戏内菜单指引  

**注意事项：**  
- 开发者版本可能存在 bug，需用户反馈问题（需提供日志、版本号、复现步骤）  
- 非官方渠道下载可能导致风险  
- 捐款支持（PayPal/Buy Me A Coffee）用于项目开发
