### [beyond-all-reason spring](https://github.com/beyond-all-reason/spring)  ![GitHub Repo stars](https://img.shields.io/github/stars/beyond-all-reason/spring?style=social)

Recoil 是一款开源的实时战略游戏引擎，作为 Spring 引擎 105.0 的分支延续项目。支持通过 GitHub 克隆源码进行编译（提供 Docker 及原生环境方案），也可下载安装预编译包。项目提供社区 Discord 支持，遵循开源许可协议及 AI 使用政策。
