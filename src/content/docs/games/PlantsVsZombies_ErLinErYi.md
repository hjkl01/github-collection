### [ErLinErYi PlantsVsZombies](https://github.com/ErLinErYi/PlantsVsZombies)  ![GitHub Repo stars](https://img.shields.io/github/stars/ErLinErYi/PlantsVsZombies?style=social)

本项目是基于 Cocos2d-X v3.16 引擎使用 C++ 全栈开发的《植物大战僵尸》复刻游戏。项目包含 104 个可玩关卡、14 种僵尸和 13 种植物，动画采用 Dragonbones/Spine 制作。相比原版，游戏在分辨率、画质、刷新率及游戏模式上有所提升，难度稍高。该项目适合作为 Cocos2d-X 和 C++ 初学者的入门学习项目，或为游戏开发者提供借鉴。因版权原因资源文件需通过教程单独下载，仅供学习交流，严禁商业用途。
