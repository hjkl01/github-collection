### [multiprocessio datastation](https://github.com/multiprocessio/datastation)  ![GitHub Repo stars](https://img.shields.io/github/stars/multiprocessio/datastation?style=social)

DataStation 是一款面向开发者的开源跨平台数据 IDE（支持 Windows、macOS、Linux）。它支持连接 20 多种数据库（如 PostgreSQL、MySQL）和 13 多种文件格式，能够从日志、指标及 HTTP 服务器等数据源获取数据并构建图表、表格。用户可使用 Python、JavaScript、R、SQL 等语言编写嵌入式脚本对数据进行合并与处理，并支持数据导出及 HTTP 面板功能。
