### [gunnarmorling 1brc](https://github.com/gunnarmorling/1brc)  ![GitHub Repo stars](https://img.shields.io/github/stars/gunnarmorling/1brc?style=social)

该项目是一个名为“十亿行挑战”（1BRC）的编程竞赛平台，旨在探索现代 Java 处理十亿行数据时的性能极限。主要功能包括提供数据生成工具与基准测试程序，要求参与者编写 Java 程序读取包含十亿行温度测量值的文件，计算每个气象站的温度最小值、平均值和最大值。挑战鼓励利用虚拟线程、SIMD、内存映射及原生编译等优化技术，参赛者需在指定硬件环境下提交实现以比拼速度并进入排行榜，规则限制仅限 Java 语言、无外部依赖库及单个源文件。
