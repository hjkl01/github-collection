### [pudo dataset](https://github.com/pudo/dataset)

**项目核心内容总结：**

**功能**：dataset 是一个简化数据库操作的工具，使数据库读写操作如同处理 JSON 文件般便捷。  
**使用方法**：通过 `pip install dataset` 命令安装，支持 Python 代码中直接操作数据库。  
**主要特性**：  
- 提供简单 API 实现数据库数据的快速存取；  
- 支持多种数据库（需依赖对应数据库驱动）。  

**注意事项**：  
自 1.0 版本起，数据导出功能已独立为 `datafreeze` 包，原项目仅保留核心数据库操作功能。
