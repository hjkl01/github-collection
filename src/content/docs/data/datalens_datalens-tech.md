### [datalens-tech datalens](https://github.com/datalens-tech/datalens)  ![GitHub Repo stars](https://img.shields.io/github/stars/datalens-tech/datalens?style=social)

DataLens 是一款现代化的开源商业智能与数据可视化系统，由 Yandex 开发。它提供数据处理引擎、用户界面及认证授权功能，支持连接 ClickHouse、PostgreSQL 等多种数据源，并具备图表可视化（支持 Highcharts 和 D3.js）及地图集成能力。系统架构包含前端 UI、后端服务、元数据存储、认证及工作流管理模块，支持 Docker Compose 和 Kubernetes Helm 部署，也可作为云服务使用。此外，系统支持灵活的配置选项，如自定义外部数据库、调整服务组件及基于角色的用户权限管理。
