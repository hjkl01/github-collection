### [PostHog posthog](https://github.com/PostHog/posthog)  ![GitHub Repo stars](https://img.shields.io/github/stars/PostHog/posthog?style=social)

PostHog 是一款面向产品构建的一站式开源平台，集成产品分析、Web 分析、会话回放、功能标志、A/B 实验、错误追踪、用户调查、数据仓库、数据管道、LLM 分析及工作流自动化等功能。支持 PostHog Cloud 快速部署或 Docker 自托管，提供多种语言 SDK 与 API，帮助团队监控用户行为、分析数据并优化产品体验。
