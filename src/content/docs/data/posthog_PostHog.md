### [PostHog posthog](https://github.com/PostHog/posthog)  ![GitHub Repo stars](https://img.shields.io/github/stars/PostHog/posthog?style=social)

PostHog 是一个开源的一站式平台，帮助用户构建成功的产品。它提供多种工具，包括产品分析、网页分析、会话回放、功能标志、实验、错误追踪、调查问卷、数据仓库、数据管道、LLM 分析和工作流程。用户可以通过 [PostHog Cloud](https://posthog.com/) 快速开始，也可以通过 Docker 自行部署开源版本。

主要功能包括：

- **产品分析**：自动捕获或手动记录事件，使用可视化或 SQL 分析数据。
- **网页分析**：监控网页流量和用户会话，跟踪转化率、网页性能和收入。
- **会话回放**：观看用户在网站或移动应用上的真实操作，诊断问题和理解行为。
- **功能标志**：安全地向特定用户或群体发布功能。
- **实验**：测试变更并测量其对目标指标的影响，支持无代码设置实验。
- **错误追踪**：追踪错误，获取警报，并解决问题以改进产品。
- **调查问卷**：使用无代码模板或自定义调查工具收集反馈。
- **数据仓库**：从 Stripe、Hubspot 等外部工具同步数据，与产品数据一起查询。
- **数据管道**：对传入数据运行自定义过滤和转换，实时或批量导出到 25+ 工具或仓库。
- **LLM 分析**：追踪 LLM 应用的追踪、生成、延迟和成本。
- **工作流程**：自动化操作或向用户发送消息。

用户可以通过安装 JavaScript 片段、SDK 或使用 API 来集成 PostHog。平台支持多种前端、移动端和后端语言和框架，包括 JavaScript、React、Vue、Android、iOS、Python、Node、PHP、Ruby 等。

PostHog 开源代码在 MIT expat 许可下发布，部分高级功能在付费计划中提供。用户可通过 [PostHog Cloud](https://posthog.com/) 使用免费 tier，每月有事件、记录、标志请求、异常和调查响应的免费额度。项目也欢迎贡献代码和功能请求。
