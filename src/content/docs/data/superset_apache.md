### [apache superset](https://github.com/apache/superset)  ![GitHub Repo stars](https://img.shields.io/github/stars/apache/superset?style=social)

Apache Superset 是一款现代化的企业级商业智能（BI）Web 应用程序。它提供无代码图表构建器、强大的 Web 端 SQL 编辑器及轻量级语义层，支持几乎所有 SQL 数据库或数据引擎。项目具备丰富的可视化图表、可配置缓存层、可扩展的安全认证、API 接口和云原生架构，旨在替代或增强专有商业智能工具。
