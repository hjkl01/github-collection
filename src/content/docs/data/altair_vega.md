### [vega altair](https://github.com/vega/altair)  ![GitHub Repo stars](https://img.shields.io/github/stars/vega/altair?style=social)

Vega-Altair 是基于 Vega-Lite 规范的 Python 声明式统计可视化库。其 API 简洁一致，能以少量代码生成美观图表，支持可视化和交互的声明式定义。支持在 JupyterLab、VS Code 等环境显示，可导出 PNG/SVG、HTML、JSON 及在线编辑器格式，内部 API 确保与 Vega-Lite 规范完全一致。
