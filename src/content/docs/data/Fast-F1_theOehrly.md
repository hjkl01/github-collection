### [theOehrly Fast-F1](https://github.com/theOehrly/Fast-F1)  ![GitHub Repo stars](https://img.shields.io/github/stars/theOehrly/Fast-F1?style=social)

FastF1 是一款用于访问和分析一级方程式（F1）数据的 Python 包。主要功能包括：
- 获取 F1 计时、遥测、赛程、会话结果及历史数据；
- 兼容 Ergast API 接口；
- 数据以扩展的 Pandas DataFrame 形式提供，内置针对 F1 的专用处理函数；
- 集成 Matplotlib 支持数据可视化；
- 实现 API 请求缓存以加速脚本运行。
