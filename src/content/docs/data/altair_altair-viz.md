### [altair-viz altair](https://github.com/altair-viz/altair)  ![GitHub Repo stars](https://img.shields.io/github/stars/altair-viz/altair?style=social)

Vega-Altair 是一个基于 Vega-Lite 规范的 Python 声明式统计可视化库。它提供简洁、一致且类型安全的 API，旨在用最少代码生成美观有效的数据可视化图表。核心功能包括：在 JupyterLab、VS Code 等环境中展示；支持导出为 PNG、SVG、HTML 等格式；支持序列化 JSON；以及提供声明式的交互功能（如刷选过滤）。
