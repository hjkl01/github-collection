### [getredash redash](https://github.com/getredash/redash)  ![GitHub Repo stars](https://img.shields.io/github/stars/getredash/redash?style=social)

Redash 是一个基于浏览器的数据探索、查询、可视化与协作平台。它支持 SQL 和 NoSQL 查询编辑，提供拖拽式可视化图表生成及仪表盘组合功能，允许共享查询与可视化结果。核心特性包括易于上手、定时刷新、数据变更警报、REST API 支持，以及兼容超过 35 种主流数据源（如 AWS、Google、各类数据库等），旨在帮助组织成员进行数据驱动的决策。
