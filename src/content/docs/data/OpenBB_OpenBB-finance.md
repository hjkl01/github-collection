### [OpenBB-finance OpenBB](https://github.com/OpenBB-finance/OpenBB)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenBB-finance/OpenBB?style=social)

OpenBB Open Data Platform (ODP) 是一个开源金融数据集成工具集，旨在将专有、许可及公开数据源统一整合并暴露给下游应用（如 AI 助手、研究仪表板）。其核心架构遵循“一次连接，处处消费”理念，支持数据在 Python 环境、OpenBB Workspace、Excel、MCP 服务器及 REST API 等多平台间无缝调用。项目提供 Python 包与 CLI 工具，支持直接获取金融数据（如股价历史），并可作为后端服务与 OpenBB Workspace 无缝集成，为量化研究、分析可视化及 AI 应用提供数据基础设施。
