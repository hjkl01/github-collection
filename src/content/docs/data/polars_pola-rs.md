### [pola-rs polars](https://github.com/pola-rs/polars)  ![GitHub Repo stars](https://img.shields.io/github/stars/pola-rs/polars?style=social)

Polars 是一款用 Rust 编写的高性能 DataFrame 查询引擎，支持 Python、Rust、Node.js、R 和 SQL 多语言接口。其特点为高性能、轻量级（零依赖）且启动迅速。核心功能包括：支持懒执行与立即执行模式、多线程及 SIMD 加速、查询优化、强大的表达式 API，以及通过流式处理支持超过内存容量的大数据集处理。底层基于 Apache Arrow 列式格式。
