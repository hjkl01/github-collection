### [ibis-project ibis](https://github.com/ibis-project/ibis)  ![GitHub Repo stars](https://img.shields.io/github/stars/ibis-project/ibis?style=social)

Ibis 是一个便携式 Python DataFrame 库，通过统一 API 支持 20 多种后端（含数据库及计算引擎）。它可将 Python 表达式编译为后端原生语言（主要是 SQL），支持混合使用 Python 与 SQL 代码，并提供惰性求值及交互式探索功能，便于在本地迭代并通过简单配置实现远程部署。
