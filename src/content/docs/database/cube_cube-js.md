### [cube-js cube](https://github.com/cube-js/cube)  ![GitHub Repo stars](https://img.shields.io/github/stars/cube-js/cube?style=social)

Cube Core 是一个开源语义层，可用于构建嵌入式分析、商业智能（BI）工具或为 AI 代理提供数据上下文。它采用无头架构，支持 REST、GraphQL 和 SQL 多种 API。内置关系缓存引擎，支持秒级低延迟和高并发请求，兼容各类 SQL 数据源（如 Snowflake、BigQuery、Postgres 等）。通过解耦语义层与特定工具，允许组织一次定义指标即可在不同场景复用。支持通过 Docker 进行本地部署或自托管。
