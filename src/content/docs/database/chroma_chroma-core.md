### [chroma-core chroma](https://github.com/chroma-core/chroma)

**核心内容总结：**

Chroma 是一个开源的嵌入式数据库，用于快速构建 Python 或 JavaScript 的 LLM 应用。主要功能包括添加文档、查询相似结果等，支持通过 `pip install chromadb`（Python）或 `npm install chromadb`（JavaScript）安装，提供简单 API。主要特性包括：  
- **易用性**：支持开发、测试、生产环境，API 一致；  
- **功能丰富**：支持查询、过滤、正则表达式等；  
- **集成工具**：兼容 LangChain、LlamaIndex 等；  
- **开源免费**：Apache 2.0 许可证。  

提供托管服务 Chroma Cloud，支持快速创建数据库，适用于“Chat your data”等场景（如通过自然语言查询文档并生成 LLM 上下文）。支持多种嵌入模型（如 Sentence Transformers、OpenAI、Cohere），用户可通过 Discord 参与贡献或提交 PR。
