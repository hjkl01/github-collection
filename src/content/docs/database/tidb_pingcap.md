### [pingcap tidb](https://github.com/pingcap/tidb)  ![GitHub Repo stars](https://img.shields.io/github/stars/pingcap/tidb?style=social)

TiDB 是一个开源、云原生的分布式 SQL 数据库，旨在提供高可用性、弹性扩展、强一致性和高性能。核心功能包括：支持基于两阶段提交的分布式事务（ACID 合规）；计算与存储分离架构，支持在线横向和纵向扩展；内置 Raft 共识协议保障数据多副本及自动故障转移；融合 TiKV 行存储和 TiFlash 列存储支持混合事务/分析处理（HTAP）；完全兼容 MySQL 8.0 协议；支持公有云、私有云及 Kubernetes 部署；遵循 Apache 2.0 开源协议。
