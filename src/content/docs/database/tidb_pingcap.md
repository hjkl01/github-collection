### [pingcap tidb](https://github.com/pingcap/tidb)  ![GitHub Repo stars](https://img.shields.io/github/stars/pingcap/tidb?style=social)

### 核心内容总结

**项目功能**  
TiDB 是一个开源、云原生的分布式 SQL 数据库，支持高可用性、水平和垂直扩展、强一致性及高性能。其核心功能包括分布式事务、混合事务/分析处理（HTAP）、MySQL 兼容性、云原生部署（支持 Kubernetes 和 TiDB Cloud）。

**使用方法**  
- **本地部署**：通过 TiDB 快速入门指南搭建本地测试集群。  
- **Kubernetes**：支持在 Kubernetes 上部署。  
- **TiDB Cloud**：提供一键创建集群服务，无需自行管理基础设施。  

**主要特性**  
- **分布式事务**：确保跨节点数据一致性。  
- **HTAP**：同时支持事务处理（OLTP）和分析处理（OLAP）。  
- **MySQL 兼容性**：语法和协议兼容 MySQL，便于迁移。  
- **云原生**：支持 Kubernetes 部署及 TiDB Cloud 服务。  
- **开源**：采用 Apache 2.0 许可证，社区活跃，提供贡献指南和问题列表。  

**其他**
