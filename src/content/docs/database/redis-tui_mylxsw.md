### [mylxsw redis-tui](https://github.com/mylxsw/redis-tui)

**核心内容总结：**  
redis-Tui 是一个基于 CLI 的 Redis 文本界面客户端，提供以下主要功能：  
1. **解决官方环境禁用 `KEYS` 命令的问题**，支持获取键列表；  
2. **命令自动补全功能**，提升操作效率；  
3. **命令执行历史记录**，支持通过上下箭头快速切换历史命令；  
4. **待完善功能**：`SCAN` 命令的返回值格式化尚未正确处理。
