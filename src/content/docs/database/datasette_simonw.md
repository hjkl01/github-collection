### [simonw datasette](https://github.com/simonw/datasette)  ![GitHub Repo stars](https://img.shields.io/github/stars/simonw/datasette?style=social)

Datasette 是一款开源多用途工具，用于探索和数据发布。它支持将任意形状或规模的数据转化为交互式网站及配套的 API。

主要功能：
1. **数据服务**：通过命令行启动 Web 服务器，提供数据浏览界面和 API 接口。
2. **元数据管理**：支持通过 JSON 文件添加许可、来源等元数据信息。
3. **一键部署**：支持将 SQLite 数据库直接发布至 Heroku 或 Google Cloud Run。
4. **无服务器版本**：提供基于 WebAssembly 的 Lite 版本，完全在浏览器中运行，无需后端。
