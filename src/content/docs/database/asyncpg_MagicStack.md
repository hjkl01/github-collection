### [MagicStack asyncpg](https://github.com/MagicStack/asyncpg)  ![GitHub Repo stars](https://img.shields.io/github/stars/MagicStack/asyncpg?style=social)

asyncpg 是一款专为 PostgreSQL 和 Python asyncio 设计的高效数据库客户端库。它原生实现 PostgreSQL 服务器二进制协议，性能比 psycopg3 快 5 倍。核心功能包括支持预编译语句、可滚动游标、查询结果部分迭代，以及自动处理复合类型、数组和自定义数据类型的编解码。安装简单且基本无额外依赖。
