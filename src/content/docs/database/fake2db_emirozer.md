### [emirozer fake2db](https://github.com/emirozer/fake2db)  ![GitHub Repo stars](https://img.shields.io/github/stars/emirozer/fake2db?style=social)

fake2db 是一个用于生成测试用虚假有效数据数据库的工具。支持 SQLite、MySQL、PostgreSQL、MongoDB、Redis、CouchDB 及 FoundationDB SQL Layer。提供命令行接口，支持配置行数、数据库类型、名称、连接参数、数据本地化语言及随机种子，允许通过 `--custom` 参数自定义数据库列项内容。
