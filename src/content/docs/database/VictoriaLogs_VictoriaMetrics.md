### [VictoriaMetrics VictoriaLogs](https://github.com/VictoriaMetrics/VictoriaLogs)  ![GitHub Repo stars](https://img.shields.io/github/stars/VictoriaMetrics/VictoriaLogs?style=social)

VictoriaLogs 是一款快速、易用的开源免费日志数据库。它支持单节点和集群部署，提供内置 Web UI 及 Grafana 插件支持，并提供二进制发布版、Docker 镜像和源代码。
