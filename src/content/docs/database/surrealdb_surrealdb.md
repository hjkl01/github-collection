### [surrealdb surrealdb](https://github.com/surrealdb/surrealdb)  ![GitHub Repo stars](https://img.shields.io/github/stars/surrealdb/surrealdb?style=social)

SurrealDB 是一款基于 Rust 构建的多模型数据库，将文档、图、关系、时序、地理空间和键值数据模型统一于单一引擎中。它支持全文、向量和混合搜索，具备实时事件驱动能力及内置用户认证，可用作后端即服务。支持嵌入式（应用内、WebAssembly、边缘）、自托管单节点或云分布式集群部署。核心特性包括使用 SurrealQL 查询语言，支持 ACID 事务、细粒度行级权限控制、Schema 定义、动态计算字段及嵌入式 JavaScript 扩展。兼容多种编程语言 SDK，旨在简化数据库与 API 架构，加速应用开发。
