### [luin medis](https://github.com/luin/medis)  ![GitHub Repo stars](https://img.shields.io/github/stars/luin/medis?style=social)

Medis 是一款基于 Electron、React 和 Redux 构建的 Redis 图形化管理应用。支持键值查看与编辑、SSH 隧道连接远程服务器、自定义命令终端及配置管理。具备 JSON 和 MessagePack 格式的高亮验证功能，利用 SCAN 命令实现海量键数据的高效处理且不阻塞服务器。兼容 Redis 2.8 及以上版本，提供 Windows 和 macOS 版本。
