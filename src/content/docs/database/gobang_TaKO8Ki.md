### [TaKO8Ki gobang](https://github.com/TaKO8Ki/gobang)  ![GitHub Repo stars](https://img.shields.io/github/stars/TaKO8Ki/gobang?style=social)

gobang 是一款使用 Rust 编写的跨平台 TUI（文本用户界面）数据库管理工具。支持 macOS、Windows 和 Linux 系统，可管理 MySQL、PostgreSQL 及 SQLite 数据库。项目采用纯键盘操作模式，通过配置文件管理数据库连接，并提供记录、列、约束等视图的导航与管理功能。
