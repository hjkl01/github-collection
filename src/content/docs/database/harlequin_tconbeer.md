### [tconbeer harlequin](https://github.com/tconbeer/harlequin)  ![GitHub Repo stars](https://img.shields.io/github/stars/tconbeer/harlequin?style=social)

Harlequin 是一款运行于终端的 SQL 集成开发环境（IDE）。它基于 Python 开发，兼容 Linux、macOS 和 Windows 系统。项目默认内置 DuckDB 数据库适配器，并支持通过安装插件连接 PostgreSQL、MySQL、SQLite 等多种数据库。Harlequin 提供命令行交互界面，支持自定义主题、快捷键绑定、文件管理、数据查询与导出等功能，同时也支持与 Django 集成使用。
