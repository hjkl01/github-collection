### [timescale pg-aiguide](https://github.com/timescale/pg-aiguide)  ![GitHub Repo stars](https://img.shields.io/github/stars/timescale/pg-aiguide?style=social)

pg-aiguide 是一款面向 AI 编码助手的 PostgreSQL 专家工具，旨在提升 AI 生成的数据库代码质量。其主要功能包括：

1.  **文档语义搜索**：支持跨官方 PostgreSQL 手册（版本感知）、TimescaleDB 及 PostGIS 进行语义和关键词搜索。
2.  **AI 优化技能库**：提供涵盖模式设计、索引策略、数据完整性、性能调优及命名规范的最佳实践技能。
3.  **灵活集成**：支持作为公共 MCP 服务器适配任意 AI 编码代理，或作为 Claude Code 插件使用。

该工具能有效解决 AI 生成代码过时、缺少约束索引及不符合最佳实践等问题，帮助生成更健壮、高性能且可维护的数据库架构。
