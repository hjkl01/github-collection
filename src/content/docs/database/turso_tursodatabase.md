### [tursodatabase turso](https://github.com/tursodatabase/turso)  ![GitHub Repo stars](https://img.shields.io/github/stars/tursodatabase/turso?style=social)

Turso Database 是一个基于 Rust 编写的进程内 SQL 数据库，与 SQLite 兼容。支持多语言绑定（Go、JavaScript、Python、Java 等）、异步 I/O 及跨平台部署（Linux、macOS、Windows、浏览器）。具备变更数据捕获（CDC）、向量搜索、全文搜索及 MCP Server 模式。当前处于 BETA 阶段，不建议用于生产环境。
