### [tursodatabase turso](https://github.com/tursodatabase/turso)

**核心内容总结：**

Turso Database 是一个基于 Rust 重写的 SQLite 进化版，支持异步操作、向量搜索等特性。主要功能包括：  
1. **项目功能**：提供命令行工具、多语言库，支持与 IDE（如 Claude、Cursor）通过 MCP 模式集成，实现自然语言查询数据库。  
2. **使用方法**：通过命令行安装和运行，支持 JSON-RPC 协议直接交互；集成到开发环境后，可通过自然语言调用数据库工具（如创建表、查询数据）。  
3. **主要特性**：  
   - 原生异步支持，提升性能；  
   - 向量搜索功能，适用于 AI 和机器学习场景；  
   - 数据可靠性测试奖励计划，鼓励发现并修复数据损坏问题；  
   - 与 Antithesis、Blacksmith 等合作伙伴协作，推动技术发展。  
4. **开发状态**：目前处于积极开发阶段，尚未适合生产环境使用，但计划逐步完善功能。  

**注意事项**：  
- 项目核心内容已涵盖功能、使用方法及关键特性，未涉及额外信息。
