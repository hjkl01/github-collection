### [PostgresApp PostgresApp](https://github.com/PostgresApp/PostgresApp)  ![GitHub Repo stars](https://img.shields.io/github/stars/PostgresApp/PostgresApp?style=social)

Postgres.app 是 macOS 上运行 PostgreSQL 的最简便方式。它提供图形界面用于启动和停止服务器，支持同时运行多个版本的 PostgreSQL。应用集成了 PostgreSQL 核心及多种扩展（如 PostGIS、pgvector、pgrouting 等）和完整的命令行工具。首次启动时自动初始化数据库集群，并以当前用户身份运行服务，同时内置了 pl/pgsql 调试器支持。
