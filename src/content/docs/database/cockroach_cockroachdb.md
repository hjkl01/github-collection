### [cockroachdb cockroach](https://github.com/cockroachdb/cockroach)  ![GitHub Repo stars](https://img.shields.io/github/stars/cockroachdb/cockroach?style=social)

CockroachDB 是一款云原生分布式 SQL 数据库，专为构建、扩展和管理现代数据密集型应用而设计。它基于强一致性键值存储，支持水平扩展，提供标准 SQL 接口和强一致性 ACID 事务。系统具备高容错性，可自动容忍磁盘、机器、机架及数据中心故障且无需人工干预。兼容 PostgreSQL 协议，支持多种客户端驱动，并提供 CockroachCloud 托管服务及灵活的部署方案。
