### [drawdb-io drawdb](https://github.com/drawdb-io/drawdb)

**核心内容总结：**  
drawDB 是一款免费、简单直观的数据库结构编辑器和 SQL 生成器，支持在浏览器中快速构建实体关系图（ER 图），无需注册即可导出 SQL 脚本、自定义编辑器界面。  

**使用方法：**  
- 本地开发：通过 `git clone` 获取代码，运行 `npm install` 和 `npm run dev` 启动开发环境。  
- 构建：执行 `npm run build` 生成项目文件。  
- Docker 部署：使用 `docker build` 和 `docker run` 命令进行容器化部署。  
- 如需共享功能，需额外配置服务器和环境变量（非必需）。  

**主要特性：**  
- 浏览器端直接操作，无需账户；  
- 支持一键生成 SQL 脚本；  
- 提供自定义编辑器功能；  
- 支持 Docker 部署及本地开发。
