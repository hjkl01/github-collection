### [Kuari RedisFish](https://github.com/Kuari/RedisFish)  ![GitHub Repo stars](https://img.shields.io/github/stars/Kuari/RedisFish?style=social)

RedisFish 是一款跨平台、数据导向的 Redis GUI 客户端。核心功能如下：

- 支持 Redis 基本数据类型处理，提供状态颜色区分、数据多选、表格直接编辑及右键放大查看。
- 支持 JSON 格式化与编辑，Key 列表支持分页查询、前缀筛选及多种展示模式切换。
- 控制台支持单/多命令执行，并提供命令执行日志查看功能。
- 具备数据库监控能力（CPU、内存、连接客户端数等），支持发布/订阅图形化操作。
- 支持 TLS 加密、暗色模式、中英文多语言自动切换及自动更新。
