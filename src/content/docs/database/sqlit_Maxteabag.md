### [Maxteabag sqlit](https://github.com/Maxteabag/sqlit)  ![GitHub Repo stars](https://img.shields.io/github/stars/Maxteabag/sqlit?style=social)

sqlit 是一款轻量级终端数据库管理工具，旨在通过命令行快速连接和查询数据库。

**主要功能：**

- **多数据库兼容**：支持 SQL Server、PostgreSQL、MySQL、SQLite 等 20 余种主流及云数据库。
- **终端交互**：具备 Vim 风格快捷键、SQL 语法高亮、查询历史及智能上下文帮助。
- **数据操作**：支持加载百万行数据、结果过滤、模糊搜索及结构浏览（表/视图等）。
- **连接集成**：自动发现 Docker 容器数据库，支持 SSH 隧道、云服务商 CLI 集成。
- **安全与配置**：凭据存储于系统密钥环，无需配置文件，自动管理驱动依赖。
- **命令行模式**：支持直接执行 SQL 脚本、多格式输出及连接管理命令。
