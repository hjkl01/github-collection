### [dragonflydb dragonfly](https://github.com/dragonflydb/dragonfly)  ![GitHub Repo stars](https://img.shields.io/github/stars/dragonflydb/dragonfly?style=social)

Dragonfly 是一款专为现代云工作负载设计的高性能内存数据存储。它完全兼容 Redis 和 Memcached API，用户无需修改代码即可接入。相比传统内存数据库，Dragonfly 吞吐量最高可达 25 倍，资源占用减少 80%，内存效率提升 30%。项目支持约 185 个 Redis 命令及大多数 Memcached 命令，采用无共享架构实现多核扩展，具备自适应缓存算法、无 fork 快照技术及原子多键操作功能。此外，它提供原生 HTTP 控制台、Prometheus 兼容指标及模拟集群模式支持。
