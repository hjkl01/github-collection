### [cmushroom redis-pro](https://github.com/cmushroom/redis-pro)  ![GitHub Repo stars](https://img.shields.io/github/stars/cmushroom/redis-pro?style=social)

Redis Pro 是一款基于 SwiftUI 开发的 Redis 管理工具，适用于 macOS 平台（支持 Intel 和 Apple Silicon），兼容 Redis 3.x 至 6.x 版本。核心功能包括客户端列表管理（查看与终止）、慢日志查询、配置更新、SSH 登录、批量删除数据等，界面支持暗色模式。
