### [cmushroom redis-pro](https://github.com/cmushroom/redis-pro)

Redis Pro 是一个使用 SwiftUI 编写的简单易用的 Redis 管理工具，支持 macOS（Intel/Apple Silicon）平台，未来将支持 iPadOS。主要功能包括：客户端连接管理、慢日志查看、配置更新、SSH 登录、批量删除、终端操作（待实现）等。提供两种安装方式：通过 GitHub Release 下载安装包，或使用 Homebrew 安装（`brew install redis-pro`）。依赖 SwiftNIO 的 RediStack 客户端、日志框架 Puppy 及 JSON 处理库 SwiftyJSON。当前支持 Redis 3.x 至 6.x 版本，需 macOS 11.0 及以上系统。
