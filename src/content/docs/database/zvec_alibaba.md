### [alibaba zvec](https://github.com/alibaba/zvec)  ![GitHub Repo stars](https://img.shields.io/github/stars/alibaba/zvec?style=social)

Zvec 是基于阿里巴巴 Proxima 引擎的开源轻量级进程内向量数据库。支持直接嵌入 Python 或 Node.js 应用，无需服务器配置，兼容笔记本、服务器及边缘设备。支持稠密与稀疏向量混合查询及混合搜索，可毫秒级处理数十亿向量，提供生产级低延迟、可扩展的相似度搜索服务。
