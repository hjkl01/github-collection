### [sqlectron sqlectron-gui](https://github.com/sqlectron/sqlectron-gui)

**项目核心内容总结**  
Sqlectron 是一个轻量、跨平台的 SQL 客户端，支持多种数据库（如 PostgreSQL、MySQL 等）。主要功能包括：  
1. **跨平台支持**：支持多操作系统（Windows、macOS、Linux）。  
2. **多数据库兼容**：可通过链接查看当前支持的数据库列表。  
3. **配置管理**：支持保存服务器连接信息及自定义配置（详见配置文档）。  
4. **终端界面**：提供命令行版本（Sqlectron-term）作为补充工具。  

**使用方法**：  
- 通过 [下载页面](https://github.com/sqlectron/sqlectron-gui/releases) 获取安装包或源码。  
- 参考 [应用文档](docs/app) 了解操作说明。  
- 通过 [贡献指南](CONTRIBUTING.md) 参与开发。  

**特性亮点**：  
- 简单直观的图形界面。  
- 提供终端模式，满足不同用户需求。  
- 文档详细，便于配置和扩展。
