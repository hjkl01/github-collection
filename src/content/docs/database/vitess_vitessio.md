### [vitessio vitess](https://github.com/vitessio/vitess)  ![GitHub Repo stars](https://img.shields.io/github/stars/vitessio/vitess?style=social)

Vitess 是一个基于 MySQL 的云原生水平可扩展分布式数据库系统。它通过通用分片机制实现无限扩展，使应用程序代码和查询对底层数据分布透明。系统支持分片的拆分与合并，切换过程原子化且仅需数秒。自 2011 年起，该项目作为核心组件被 YouTube、Slack 等大型企业广泛用于数据库基础设施。
