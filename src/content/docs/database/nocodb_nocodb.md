### [nocodb nocodb](https://github.com/nocodb/nocodb)  ![GitHub Repo stars](https://img.shields.io/github/stars/nocodb/nocodb?style=social)

NocoDB 是一款快速易用的在线数据库构建工具，提供类电子表格的数据库操作界面。核心功能包括：支持表格、字段和行的增删改查及排序过滤；提供网格、画廊、表单、看板、日历等多种视图类型；具备细粒度访问控制、角色权限管理和视图共享功能；内置工作流自动化应用商店，支持集成聊天、邮件、存储等服务；提供 REST APIs 和 SDK 以实现程序化访问；支持 Docker、自动安装脚本及多平台二进制文件部署。
