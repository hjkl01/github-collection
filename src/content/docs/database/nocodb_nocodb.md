### [nocodb nocodb](https://github.com/nocodb/nocodb)  ![GitHub Repo stars](https://img.shields.io/github/stars/nocodb/nocodb?style=social)

NocoDB 是一个开源的数据库工具，提供类似 Airtable 的在线数据库构建功能。其核心功能包括：  
1. **功能特性**  
   - 支持创建/管理表格、列、行，提供网格、画廊、表单、看板、日历等多视图类型  
   - 支持字段排序、过滤、分组、隐藏，以及 ID、链接、公式、附件等多样化的单元格类型  
   - 提供基于角色的细粒度访问控制和共享权限（公开/私密/密码保护）  
   - 集成应用商店，支持 Slack、邮件、云存储等自动化工作流  
   - 提供 REST API 和 SDK 实现程序化访问  

2. **使用方法**  
   - 通过 Docker 安装（支持 SQLite/PostgreSQL）  
   - 使用 Auto-upstall 脚本一键部署  
   - 提供 Windows/macOS/Linux 的二进制文件安装  
   - 本地运行后通过 http://localhost:8080/dashboard 访问  

3. **其他信息**  
   - 采用 AGPLv3 开源协议  
   - 支持多语言协作（中文等语言可通过链接切换）  
   - 社区贡献者可参考 GitHub 的贡献指南参与开发
