### [sqlalchemy sqlalchemy](https://github.com/sqlalchemy/sqlalchemy)  ![GitHub Repo stars](https://img.shields.io/github/stars/sqlalchemy/sqlalchemy?style=social)

SQLAlchemy 是一个 Python SQL 工具集和对象关系映射器（ORM），提供企业级持久化模式与高效的数据库访问。主要功能包括：工业级 ORM 支持对象透明持久化和自动事务同步；关系型查询系统通过对象模型暴露完整 SQL 能力（如连接、子查询）；灵活的预加载系统；独立的 Core SQL 构建层及 DBAPI 交互层，包含 SQL 表达式语言、模式元数据、连接池及自定义类型支持；支持数据库结构自省与 DDL 生成。项目分为 Core 和 ORM 组件，开发者可完全控制对象模型与数据库模式设计，强调标准事务管理、查询控制及使用绑定参数保障安全。
