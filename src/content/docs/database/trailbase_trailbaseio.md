### [trailbaseio trailbase](https://github.com/trailbaseio/trailbase)  ![GitHub Repo stars](https://img.shields.io/github/stars/trailbaseio/trailbase?style=social)

TrailBase 是一个基于 Rust、SQLite 和 WebAssembly 构建的开源单可执行文件后端服务，旨在作为 Firebase 的替代方案。它提供亚毫秒级延迟，支持类型安全的 REST 和实时 API，具备多数据库、地理空间、SSR、认证及管理界面等功能。项目易于自托管（支持 Docker 及多平台二进制），消除了对专用缓存的需求并确保数据一致性，同时提供多种主流编程语言（如 JS/TS、Flutter、Rust 等）的客户端库。
