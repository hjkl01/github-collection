### [arangodb arangodb](https://github.com/arangodb/arangodb)  ![GitHub Repo stars](https://img.shields.io/github/stars/arangodb/arangodb?style=social)

ArangoDB 是一款支持原生图、文档和 JSON 存储的可扩展多模型数据库，通过单一查询语言（AQL）管理数据。核心功能包括水平扩展、高可用性、集成搜索引擎（ArangoSearch）、事务处理及灵活建模。支持本地、云端及容器部署，提供社区版与企业版，具备智能图处理、热备份、加密审计等企业级特性。
