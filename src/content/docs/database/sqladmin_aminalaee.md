### [aminalaee sqladmin](https://github.com/aminalaee/sqladmin)  ![GitHub Repo stars](https://img.shields.io/github/stars/aminalaee/sqladmin?style=social)

SQLAdmin 是一个专为 FastAPI 和 Starlette 框架设计的 SQLAlchemy 模型管理后台。它支持 SQLAlchemy 同步/异步引擎，利用 WTForms 构建表单，兼容 SQLModel，并提供基于 Tabler 的 UI 界面，用于快速搭建数据管理系统。
