### [centerofci mathesar](https://github.com/centerofci/mathesar)  ![GitHub Repo stars](https://img.shields.io/github/stars/centerofci/mathesar?style=social)

Mathesar 是一款开源自托管的 PostgreSQL 管理 Web 应用，提供类电子表格界面，让用户无需编写代码即可查看、编辑、查询和协作处理数据。它支持直接操作原生 Postgres 架构与表，利用原生角色权限控制访问，提供无 SQL 查询构建器、表单创建、数据导入导出及架构迁移等功能。所有数据完全保留在用户服务器中，确保数据安全，可无缝集成到现有基础设施中。
