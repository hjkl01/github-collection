### [dbeaver dbeaver](https://github.com/dbeaver/dbeaver)  ![GitHub Repo stars](https://img.shields.io/github/stars/dbeaver/dbeaver?style=social)

DBeaver 是一款免费的跨平台数据库管理工具，面向开发人员、SQL 程序员及数据库管理员。
主要特性：
1. **数据库支持**：开箱即用支持 100 多种数据库驱动，兼容任何 JDBC 或 ODBC 数据库（如 MySQL、PostgreSQL、Oracle 等），支持 SSH 隧道及自定义驱动。
2. **功能丰富**：包含 SQL 编辑器、数据编辑器、模式编辑器、ER 图、数据导入导出与迁移、SQL 执行计划分析、空间数据查看及数据库管理仪表盘。
3. **AI 集成**：支持 OpenAI 或 Copilot 的智能代码补全与生成。
4. **版本架构**：基于 Java 和 Eclipse RCP 平台。社区版免费；专业版扩展支持 NoSQL（如 MongoDB、Redis）及更多数据源；衍生出 Web 版工具 CloudBeaver。
