### [ydb-platform ydb](https://github.com/ydb-platform/ydb)

YDB 是一个开源的分布式 SQL 数据库，具备高可用性、可扩展性、严格一致性和 ACID 事务特性。其核心功能包括行/列存储表、持久化队列（topics）、跨节点事务、自动灾难恢复、水平扩展的存储与计算层，以及兼容 PostgreSQL 和 Kafka 的模式。支持通过 Ansible、Kubernetes 或手动部署集群，最低系统要求为 8GB 内存的 x86 64 位平台，主要运行于 Ubuntu，开发环境支持 MacOS 和 Windows。用户可通过快速入门指南部署单节点集群，或使用 Ansible/Kubernetes 部署多节点集群进行测试和生产环境使用。
