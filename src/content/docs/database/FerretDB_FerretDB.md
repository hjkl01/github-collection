### [FerretDB FerretDB](https://github.com/FerretDB/FerretDB)  ![GitHub Repo stars](https://img.shields.io/github/stars/FerretDB/FerretDB?style=social)

FerretDB 是一个开源的 MongoDB 替代方案。它作为代理将 MongoDB 5.0+ 的 Wire 协议查询转换为 SQL，并使用带有 DocumentDB 扩展的 PostgreSQL 作为数据库引擎。FerretDB 兼容 MongoDB 驱动和常用工具，可作为 MongoDB 的无缝替代品，为用户提供易于使用的开源文档数据库解决方案。
