### [pgmodeler pgmodeler](https://github.com/pgmodeler/pgmodeler)  ![GitHub Repo stars](https://img.shields.io/github/stars/pgmodeler/pgmodeler?style=social)

pgModeler 是一款专为 PostgreSQL 生态系统设计的开源跨平台数据库建模工具，支持 Linux、Windows 和 macOS 系统。其核心功能包括：

1. **可视化建模**：提供直观的拖放界面，用于创建和编辑复杂的数据库架构（如表、视图、函数等）。
2. **逆向工程**：连接现有数据库实例，自动生成全面的可视化模型。
3. **智能同步（Diff）**：对比模型与实时数据库或模型文件，自动生成 SQL 脚本以同步两者。
4. **集成管理**：内置数据库管理模块，可执行自定义 SQL、浏览数据及管理服务器对象。

项目采用 GNU GPL v3 许可证开源。
