### [BluePointLilac ContextMenuManager](https://github.com/BluePointLilac/ContextMenuManager)

**ContextMenuManager** 是一款用于管理 Windows 右键菜单的工具，支持启用/禁用文件、文件夹、新建、发送到等右键菜单项，可修改名称、图标、导航路径，或自定义添加新菜单项。  

**核心功能**  
- 管理右键菜单项（启用/禁用/删除/修改属性）  
- 支持自定义添加菜单命令  
- 适配 Windows 7/8/8.1/10/Vista 系统及 32/64 位架构  
- 支持高分屏（最佳缩放 150%）  

**使用方法**  
1. 下载最新版本（提供 ZIP 或 EXE 文件）  
2. 根据系统安装对应 .NET 3.5/4.0 运行库（部分系统需手动安装）  
3. 通过程序界面操作注册表项和文件路径  

**注意事项**  
- 可能被安全软件误报为病毒，需添加白名单  
- 不建议与其他右键菜单管理工具同时使用  
- 部分菜单项因系统限制可能无法完全显示或操作
