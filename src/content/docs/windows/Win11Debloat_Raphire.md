### [Raphire Win11Debloat](https://github.com/Raphire/Win11Debloat)  ![GitHub Repo stars](https://img.shields.io/github/stars/Raphire/Win11Debloat?style=social)

Win11Debloat 是一款轻量级 PowerShell 脚本，用于快速优化和清理 Windows 系统体验。核心功能包括：移除预装臃肿软件，禁用遥测、诊断数据、跟踪及系统广告，自定义开始菜单、任务栏、资源管理器及右键菜单界面。此外支持禁用 Copilot 等 AI 功能，控制 Windows 更新行为，优化系统设置（如鼠标加速、快速启动），调整外观（如深色模式），并支持启用 Sandbox、WSL 等可选功能。提供 Sysprep 模式和多用户配置等高级特性，支持命令行调用，所有更改大多可逆。
