### [Atlas-OS Atlas](https://github.com/Atlas-OS/Atlas)  ![GitHub Repo stars](https://img.shields.io/github/stars/Atlas-OS/Atlas?style=social)

AtlasOS 是一个开源的 Windows 轻量级修改项目，致力于优化性能、隐私和可用性。它移除 Windows 内置遥测以增强隐私，在确保兼容性的前提下提升系统性能，清除无用应用和广告以提高易用性，并提供可配置的安全选项。项目基于透明的 Playbooks 机制实现修改，不重新分发修改后的 Windows ISO，符合微软使用条款。
