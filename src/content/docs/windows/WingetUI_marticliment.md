### [marticliment WingetUI](https://github.com/marticliment/WingetUI)  ![GitHub Repo stars](https://img.shields.io/github/stars/marticliment/WingetUI?style=social)

UniGetUI 是一款为 Windows 10/11 设计的统一图形界面工具，整合了 WinGet、Scoop、Chocolatey、Pip、Npm 等多种命令行包管理器。用户可通过它便捷地搜索、安装、更新和卸载软件，支持批量操作、查看包详情、自定义安装选项、自动更新通知，以及软件列表的导出备份与恢复。
