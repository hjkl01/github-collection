### [marticliment WingetUI](https://github.com/marticliment/WingetUI)

**项目核心内容总结：**  

**功能：**  
UniGetUI 是一款基于 Winget/Scoop 的图形化软件包管理工具，支持安装、升级、搜索软件包，提供多语言界面（含中文）。  

**使用方法：**  
通过图形界面操作，或使用命令行参数（如 `--install`、`--upgrade` 等）管理软件包，支持 Winget/Scoop 包的搜索与安装。  

**主要特性：**  
- 支持 Winget/Scoop 多种包管理器；  
- 界面简洁，支持多语言（含中文）；  
- 开源，可在 GitHub 获取源码及贡献者信息；  
- 提供常见问题解答（如包名截断、安全提示等）。  

**注意事项：**  
- 部分功能依赖 Winget/Scoop 的稳定性；  
- 从非官方渠道下载时需注意安全风险。
