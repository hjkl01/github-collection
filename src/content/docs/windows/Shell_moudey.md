### [moudey Shell](https://github.com/moudey/Shell)  ![GitHub Repo stars](https://img.shields.io/github/stars/moudey/Shell?style=social)

Shell 是一款 Windows 文件资源管理器右键菜单管理工具。它允许用户自定义右键菜单项，创建命令以访问网页、文件、文件夹和应用，同时支持修改或移除系统及第三方软件的菜单项。功能涵盖外观自定义、子菜单与嵌套菜单支持、表达式语法、多种图标图片格式、搜索过滤及多列布局，支持纯文本配置，具有轻量便携、资源占用低的特点，兼容 Windows 7 至 11 系统。
