### [moudey Shell](https://github.com/moudey/Shell)

**项目核心内容总结：**  
Shell 是一款用于管理 Windows 资源管理器右键菜单的工具，支持自定义添加、修改或删除菜单项，可创建快捷命令访问网页、文件、文件夹或启动应用程序。  

**主要功能与特性：**  
- 轻量便携，操作简单，资源占用低。  
- 支持高度自定义外观（颜色、图标、多列布局等）。  
- 可添加子菜单、分隔符等复杂结构，支持嵌套菜单。  
- 兼容所有文件系统对象（文件、文件夹、桌面、任务栏等）。  
- 内置表达式语法及变量，支持搜索过滤功能。  
- 支持多种图标格式（.ico/.png/.bmp/SVG）及嵌入式图标。  
- 配置文件为纯文本格式，便于快速编辑。  

**系统要求：**  
Microsoft Windows 7/8/10/11  

**使用方法：**  
通过官网下载安装，通过文本配置文件自定义菜单项，支持实时修改和删除系统或第三方软件的右键菜单项。
