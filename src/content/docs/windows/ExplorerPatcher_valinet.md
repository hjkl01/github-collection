### [valinet ExplorerPatcher](https://github.com/valinet/ExplorerPatcher)

**核心内容总结：**

**项目功能**  
ExplorerPatcher 是一款用于增强 Windows 操作体验的工具，可自定义任务栏样式、开始菜单（支持 Windows 10 风格）及 Alt+Tab 窗口切换器，优化桌面交互体验。

**使用方法**  
1. 根据设备处理器类型（Intel/AMD 或 Snapdragon）下载对应安装文件（`ep_setup.exe` 或 `ep_setup_arm64.exe`）。  
2. 运行安装程序，自动提升权限后关闭并重装 `explorer.exe`，完成后桌面与任务栏将恢复。  
3. 右键任务栏进入“属性”界面，分别调整“任务栏样式”“开始菜单”及“窗口切换器（Alt+Tab）”为 Windows 10 风格。

**主要特性**  
- 支持多平台（x64/ARM64）。  
- 提供多种卸载方式（通过控制面板、安装程序命令行参数或重命名文件运行）。  
- 内置更新功能，支持手动下载最新版本安装文件升级。  
- 社区支持（Discord 服务器）及捐赠渠道。
