### [rayenghanmi RyTuneX](https://github.com/rayenghanmi/RyTuneX)

**项目核心内容总结：**

**功能**  
RyTuneX 是一款专为 Windows 10（20H1 及以上）和 Windows 11 设计的系统优化工具，提供以下核心功能：  
- **卸载预装应用**：一键移除系统自带的冗余软件（“去广告”功能）。  
- **系统优化**：调整资源占用、性能参数，支持基础与高级选项。  
- **隐私保护**：关闭 Windows 的数据收集与跟踪功能。  
- **跨版本兼容**：适配 Windows 10 和 11，界面统一。  

**使用方法**  
- **安装**：  
  - 自动安装：通过终端运行 `winget install rytunex`。  
  - 手动安装：从 GitHub 下载压缩包并运行安装程序。  
- **操作**：  
  - 优化系统：在 **Optimize** 标签页调整设置，重启生效。  
  - 卸载应用：在 **Debloat** 标签页选择应用并卸载。  
  - 隐私设置：在 **Privacy** 标签页开关数据收集功能。  

**主要特性**  
- 基于 WinUI 3 和 .NET 8 开发，界面现代简洁。  
- 提供多语言支持（含中、英、法、德、日等 13 种语言）。  
- 开源项目，支持社区贡献（翻译、代码提交）。  
- 需在优化前创建系统还原点，避免风险。
