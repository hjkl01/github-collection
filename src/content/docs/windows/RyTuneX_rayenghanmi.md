### [rayenghanmi RyTuneX](https://github.com/rayenghanmi/RyTuneX)  ![GitHub Repo stars](https://img.shields.io/github/stars/rayenghanmi/RyTuneX?style=social)

RyTuneX 是一款专为 Windows 10 和 11 设计的系统优化与精简工具。核心功能包括：卸载预装应用（去臃肿）、系统性能优化（降低资源占用）、隐私保护（阻止遥测和数据收集）、Windows 功能管理以及系统信息查看。基于 WinUI 3 和 .NET 8 构建，提供现代化用户界面，旨在提升系统的性能、隐私性和可用性。
