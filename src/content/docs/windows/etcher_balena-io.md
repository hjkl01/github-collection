### [balena-io etcher](https://github.com/balena-io/etcher)  ![GitHub Repo stars](https://img.shields.io/github/stars/balena-io/etcher?style=social)

Etcher 是一款安全便捷的操作系统镜像烧录工具，可将镜像写入 SD 卡及 USB 驱动器。它基于 Web 技术构建，具备防止误写入硬盘、验证数据完整性的功能，并支持直接刷写支持 USB 设备启动模式的 Raspberry Pi 设备。该软件兼容 Linux、Windows 和 macOS 操作系统。
