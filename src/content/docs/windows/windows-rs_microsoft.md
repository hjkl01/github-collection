### [microsoft windows-rs](https://github.com/microsoft/windows-rs)  ![GitHub Repo stars](https://img.shields.io/github/stars/microsoft/windows-rs?style=social)

该项目是 Rust 语言调用 Windows API 的库集合（windows-rs）。它基于 Windows API 元数据自动生成代码，为 Rust 开发者提供自然且符合习惯的接口，支持调用 C 风格 API、COM 及 WinRT。核心包含 `windows`（安全绑定）、`windows-sys`（原始绑定）及 `windows-bindgen`（代码生成）等库，涵盖注册表、服务、异步等 Windows 系统功能支持。
