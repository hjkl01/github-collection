### [DavidXanatos TaskExplorer](https://github.com/DavidXanatos/TaskExplorer)  ![GitHub Repo stars](https://img.shields.io/github/stars/DavidXanatos/TaskExplorer?style=social)

TaskExplorer 是一款专为 Windows 设计的高效任务管理工具，旨在实时监控运行中的应用并提供深度系统洞察。界面注重速度与效率，支持键盘导航与动态刷新。核心功能涵盖多类详情面板：线程面板提供堆栈跟踪以诊断瓶颈；内存面板支持内存查看与编辑；句柄面板展示文件及系统句柄信息；套接字面板监控网络连接与流量；模块面板管理 DLL 加载与注入；此外还包括 Token、环境、Windows、GDI 及 .NET 面板。工具支持多进程并行检查，提供 CPU、内存、磁盘、网络及 GPU 资源的实时性能图表，并允许控制系统服务。基于 Qt 框架与 Process Hacker 库开发，兼容 Windows 7 及以上 32/64 位系统。
