### [memstechtips Winhance](https://github.com/memstechtips/Winhance)  ![GitHub Repo stars](https://img.shields.io/github/stars/memstechtips/Winhance?style=social)

Winhance 是一款适用于 Windows 10 和 11 的 C# 应用程序，旨在无需重装系统的情况下精简、优化和自定义 Windows 体验。核心功能包括：
- 软件管理：清理或安装 Windows 内置应用及功能，通过 WinGet 一键安装外部软件。
- 系统优化：调节隐私、性能、更新、电源、声音及通知等系统设置。
- 界面定制：自定义主题、任务栏、开始菜单和资源管理器外观。
- 高级工具：创建自定义 Windows ISO 镜像及生成自动应答文件。
- 配置管理：保存和导入系统配置以便快速还原。
