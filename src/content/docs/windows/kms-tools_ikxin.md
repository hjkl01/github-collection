### [ikxin kms-tools](https://github.com/ikxin/kms-tools)  ![GitHub Repo stars](https://img.shields.io/github/stars/ikxin/kms-tools?style=social)

本项目是一个一站式 KMS 工具箱，集成了 KMS 激活脚本生成、KMS 服务器检测及服务器状态监控功能。基于 Nuxt 和 Vue.js 技术栈，分为全栈版本和静态版本。全栈版本依赖 Node.js 环境运行，用于调用 vlmcs 执行检测服务；静态版本通过 API 接口交互。支持 Docker、Zeabur 及各类静态托管平台部署。
