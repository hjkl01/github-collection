### [ChrisTitusTech winutil](https://github.com/ChrisTitusTech/winutil)  ![GitHub Repo stars](https://img.shields.io/github/stars/ChrisTitusTech/winutil?style=social)

该项目是一个名为 WinUtil 的 Windows 系统维护工具，通过 PowerShell 脚本集成多项任务。核心功能包括简化软件安装、系统去臃肿与优化调整、配置故障排查以及修复 Windows 更新。运行需具备管理员权限。
