### [xerrors Yuxi-Know](https://github.com/xerrors/Yuxi-Know)  ![GitHub Repo stars](https://img.shields.io/github/stars/xerrors/Yuxi-Know?style=social)

语析是一个基于大模型的知识库与知识图谱智能体开发平台。核心功能包括：基于 LangGraph v1 的多智能体架构开发，支持子智能体、工具调用与中间件机制；支持 PDF、Word、Markdown 等多格式文档上传与解析的 RAG 知识库，提供 Embedding/Rerank 配置及知识库评估功能；基于 LightRAG 的知识图谱构建、可视化与属性管理，可参与智能体推理并自动生成思维导图；采用 Vue + FastAPI 架构，支持 Docker 部署、暗黑模式及生产环境优化。该平台旨在帮助用户构建 RAG+ 知识图谱智能体，实现业务文档向可推理知识库的转化。
