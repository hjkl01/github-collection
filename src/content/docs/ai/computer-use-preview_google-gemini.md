### [google-gemini computer-use-preview](https://github.com/google-gemini/computer-use-preview)  ![GitHub Repo stars](https://img.shields.io/github/stars/google-gemini/computer-use-preview?style=social)

Computer Use Preview 是一个 AI 浏览器自动化代理工具，允许用户通过自然语言指令控制浏览器执行网页任务。项目支持 Gemini Developer API 和 Vertex AI 两种模型接入方式，提供本地 Playwright 和云端 Browserbase 两种运行环境。用户可通过命令行脚本指定任务查询、模型及环境来运行代理，默认使用 gemini-2.5-computer-use-preview 系列模型。
