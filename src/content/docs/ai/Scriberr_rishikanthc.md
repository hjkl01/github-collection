### [rishikanthc Scriberr](https://github.com/rishikanthc/Scriberr)  ![GitHub Repo stars](https://img.shields.io/github/stars/rishikanthc/Scriberr?style=social)

Scriberr 是一款开源、完全离线的音频转录应用程序，专为重视隐私和性能的用户设计。它支持在本地机器上转录音频和视频，确保数据不上传至第三方云服务，并使用 Whisper、NVIDIA Parakeet 或 Canary 等模型提供高精度转录及词级时间标记。核心功能包括智能说话人检测、支持连接 Ollama 或 OpenAI API 进行对话与摘要生成、内置音频录制与笔记功能，以及通过 API 和文件夹监控实现工作流自动化。项目支持 PWA 安装提供原生体验，并可通过 Homebrew 安装或 Docker 部署（支持 CPU 及 NVIDIA GPU 加速）。
