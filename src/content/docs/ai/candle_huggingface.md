### [huggingface candle](https://github.com/huggingface/candle)  ![GitHub Repo stars](https://img.shields.io/github/stars/huggingface/candle?style=social)

Candle 是一个基于 Rust 的极简机器学习框架，专注于性能（支持 GPU）和易用性。它提供类似 PyTorch 的简洁语法，支持 CPU、CUDA 和 WebAssembly 后端。框架内置丰富的模型示例，涵盖大语言模型（如 LLaMA、Phi）、文生图模型（如 Stable Diffusion）、语音识别（如 Whisper）及计算机视觉模型。支持多种模型文件格式加载与量化，旨在实现无服务器推理并消除 Python 依赖，适合轻量级部署。
