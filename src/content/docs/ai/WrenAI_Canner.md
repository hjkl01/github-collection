### [Canner WrenAI](https://github.com/Canner/WrenAI)  ![GitHub Repo stars](https://img.shields.io/github/stars/Canner/WrenAI?style=social)

Wren AI 是一款开源生成式商业智能（GenBI）智能体。它支持通过自然语言查询数据库，自动生成准确的 SQL 语句和数据图表。主要功能包括：使用自然语言提问获取数据答案与摘要，自动生成图表与报告，通过语义层（MDL）确保查询准确性与规范性，以及提供 API 以便将查询与图表功能嵌入应用程序。项目支持多种主流数据源及各类大语言模型，可本地部署或使用云端服务。
