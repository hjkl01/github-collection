### [MoonshotAI kimi-cli](https://github.com/MoonshotAI/kimi-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/MoonshotAI/kimi-cli?style=social)

Kimi Code CLI 是一款运行在终端中的 AI 智能代理，旨在辅助完成软件开发任务及终端操作。它能够读取和编辑代码、执行 Shell 命令、搜索及获取网页内容，并支持在执行过程中自主规划与调整动作。

核心功能特性包括：
1.  **Shell 命令模式**：支持通过快捷键（Ctrl-X）切换模式，在代理内直接运行 Shell 命令。
2.  **IDE 集成**：提供 VS Code 扩展，并原生支持 Agent Client Protocol (ACP)，可接入 Zed、JetBrains 等兼容编辑器。
3.  **Zsh 增强**：支持通过 Zsh 插件将 AI 代理能力集成到 Shell 环境中。
4.  **MCP 支持**：支持 Model Context Protocol，可通过命令行子命令或配置文件灵活管理 MCP 服务器。
