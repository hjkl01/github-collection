### [cjpais Handy](https://github.com/cjpais/Handy)  ![GitHub Repo stars](https://img.shields.io/github/stars/cjpais/Handy?style=social)

Handy 是一款免费、开源且可扩展的跨平台桌面语音转文本应用，支持完全离线运行以保障用户隐私。用户可通过配置快捷键触发录音，转写后的文字会自动粘贴到当前激活的文本框中。应用基于 Tauri 框架（Rust 后端 + React 前端）构建，兼容 Windows、macOS 和 Linux 系统，支持 Whisper 系列及 Parakeet V3 模型（含 GPU 加速与自动语言检测），并提供命令行控制及自定义模型安装功能。
