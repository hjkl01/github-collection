### [open-webui open-webui](https://github.com/open-webui/open-webui)  ![GitHub Repo stars](https://img.shields.io/github/stars/open-webui/open-webui?style=social)

Open WebUI 是一款功能丰富、用户友好的自托管 AI 平台，支持完全离线运行，内置 RAG 推理引擎。核心功能包括：

1. **灵活部署**：支持 Docker、Kubernetes 及 Python pip 安装。
2. **模型集成**：兼容 Ollama 和 OpenAI 接口，支持多模型并行对话。
3. **交互增强**：提供 Markdown/LaTeX 支持，集成语音/视频通话及图像生成编辑。
4. **知识增强**：支持本地 RAG（兼容多种向量数据库）、网络搜索及网页浏览。
5. **安全管控**：具备细粒度权限、RBAC、企业级认证及数据持久化。
6. **扩展能力**：支持 Python 函数调用、插件框架及生产级监控。
7. **多端体验**：支持响应式设计、PWA 及多语言国际化。
