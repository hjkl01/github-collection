### [AsyncFuncAI deepwiki-open](https://github.com/AsyncFuncAI/deepwiki-open)  ![GitHub Repo stars](https://img.shields.io/github/stars/AsyncFuncAI/deepwiki-open?style=social)

DeepWiki-Open 是一款自动化代码仓库 Wiki 生成工具，支持 GitHub、GitLab 和 BitBucket。核心功能包括：

1. **自动生成文档与图表**：快速分析代码结构，生成全面文档及 Mermaid 可视化架构与数据流图。
2. **智能问答与研究**：提供基于 RAG 技术的智能问答（Ask 功能），支持多轮对话与 DeepResearch 深入探究。
3. **多模型支持**：兼容 Google Gemini、OpenAI、Ollama、Azure OpenAI 等多种大模型及嵌入服务。
4. **私有仓库访问**：支持通过访问令牌安全访问私有代码仓库。
5. **灵活部署**：支持 Docker 及手动部署，允许自定义 API 密钥与配置参数。
