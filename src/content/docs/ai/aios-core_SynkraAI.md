### [SynkraAI aios-core](https://github.com/SynkraAI/aios-core)  ![GitHub Repo stars](https://img.shields.io/github/stars/SynkraAI/aios-core?style=social)

Synkra AIOX 是一个由 AI 驱动的全栈开发框架，采用智能体驱动的敏捷开发模式。其核心功能总结如下：

1. **CLI 优先架构**：遵循 CLI 第一、可观测性第二、UI 第三的设计原则，确保系统可通过命令行完全控制，无需依赖界面即可运行。
2. **专用智能体协作**：内置 11 个专用智能体角色（如分析师、产品经理、架构师、开发、QA 等），分为元智能体、规划智能体和开发智能体，通过协作完成从需求规划到代码实现的全过程。
3. **自主开发引擎 (ADE)**：包含需求规范管道、执行引擎、QA 审核及自动恢复系统，实现从用户请求到可运行代码的自动化闭环。
4. **领域扩展能力**：通过 Squads 机制，支持用户创建自定义智能体团队，不仅适用于软件开发，还可扩展至创意写作、商业策略及教育等领域。
5. **多 IDE 集成**：兼容 Cursor、Claude Code、Codex CLI、Gemini CLI 等多种主流编辑器与工具链，支持跨平台部署。
6. **企业级支持**：提供 AIOX Pro 版本，包含持久化记忆层、高级数据分析及企业系统集成（如 Jira、Linear 等）。
