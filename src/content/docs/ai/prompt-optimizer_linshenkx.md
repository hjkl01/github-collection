### [linshenkx prompt-optimizer](https://github.com/linshenkx/prompt-optimizer)  ![GitHub Repo stars](https://img.shields.io/github/stars/linshenkx/prompt-optimizer?style=social)

Prompt Optimizer（提示词优化器）是一个 AI 提示词优化工具，旨在帮助用户编写更高质量的提示词以提升 AI 输出效果。支持 Web 应用、桌面应用、Chrome 插件和 Docker 四种使用方式。

核心功能：
1. **智能优化**：支持一键优化、多轮迭代及系统/用户提示词双模式优化。
2. **多模型集成**：支持 OpenAI、Gemini、DeepSeek、智谱等主流大模型，以及文生图（T2I）和图生图（I2I）功能。
3. **高级测试**：提供上下文变量管理、多轮会话测试及工具调用（Function Calling）支持。
4. **安全与集成**：纯客户端处理，数据直接与 AI 服务商交互，支持密码访问控制及 MCP 协议集成。

用户可选择在线体验或自行部署（支持 Vercel、Docker 等），并可灵活配置各类 API 密钥及自定义模型参数。
