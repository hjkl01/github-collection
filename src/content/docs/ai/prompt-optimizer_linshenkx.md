### 项目简介

Prompt Optimizer 是一个强大的 AI 提示词优化工具，帮助用户编写更高质量的提示词，从而提升 AI 输出效果。支持多种使用方式，包括 Web 应用、桌面应用、Chrome 插件和 Docker 部署。

---

### 核心功能

- **智能优化**：一键优化提示词，支持多轮迭代改进。
- **双模式优化**：支持系统提示词和用户提示词优化。
- **对比测试**：支持原始与优化后提示词的实时对比。
- **多模型集成**：支持 OpenAI、Gemini、DeepSeek、智谱 AI、SiliconFlow 等主流模型。
- **图像生成**：支持文生图（T2I）和图生图（I2I），集成 Gemini、Seedream 等图像模型。
- **高级测试模式**：上下文变量管理、多轮会话测试、工具调用支持。
- **安全架构**：纯客户端处理，数据不经过中间服务器。
- **多端支持**：支持 Web、桌面、Chrome 插件和 Docker。
- **MCP 协议支持**：可与 Claude Desktop 等 MCP 兼容应用集成。

---

### 使用方式

1. **在线版本**：访问 [https://prompt.always200.com](https://prompt.always200.com)
2. **Vercel 部署**：一键部署或 Fork 后手动部署。
3. **桌面应用**：从 [GitHub Releases](https://github.com/linshenkx/prompt-optimizer/releases) 下载。
4. **Chrome 插件**：在 Chrome 应用商店安装。
5. **Docker 部署**：提供简单部署命令和 Compose 配置。
6. **MCP Server**：支持与 Claude Desktop 等工具集成。

---

### 主要特性

- **角色扮演对话**：提升小模型在角色扮演场景中的表现。
- **知识图谱提取**：保障生产环境中模型输出的稳定性和格式一致性。
- **诗歌创作**：辅助用户细化创作需求，与 AI 共创作品。
- **API 密钥配置**：支持界面和环境变量两种配置方式。
- **本地开发**：提供完整的开发文档和命令。
