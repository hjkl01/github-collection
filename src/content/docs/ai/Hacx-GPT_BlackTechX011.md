### [BlackTechX011 Hacx-GPT](https://github.com/BlackTechX011/Hacx-GPT)

**项目核心内容总结：**  
HacxGPT 是一个开源 AI 框架，通过集成第三方 API（如 OpenRouter 或 DeepSeek）并结合定制系统提示，模拟其私有模型的行为。  

**主要功能：**  
- 提供基于第三方 API 的 AI 对话能力，支持无限制交互；  
- 通过定制系统提示增强模型表现，模拟私有模型功能；  
- 支持跨平台使用（Kali Linux、Ubuntu、Termux）。  

**使用方法：**  
1. 获取 OpenRouter 或 DeepSeek 的免费 API 密钥；  
2. 通过安装脚本（Windows 使用 `install.bat`，Linux/Termux 运行 `bash install.sh`）部署项目；  
3. 修改 `HacxGPT.py` 中的 `API_PROVIDER` 变量选择 API 服务商；  
4. 运行 `python3 HacxGPT.py`，首次启动时输入 API 密钥。  

**主要特性：**  
- 强大的 AI 对话能力，支持复杂任务处理；  
- 无限制交互框架，突破常规模型限制；  
- 提供简洁命令行界面（CLI）；  
- 支持多平台部署。  

**注意事项：**  
- 项目代码为第三方 API 的封装，非核心私有模型；  
- 私有模型需通过 Telegram 频道申请（[https://t.me/HacxGPT](https://t.me/HacxGPT)）。
