### [BlackTechX011 Hacx-GPT](https://github.com/BlackTechX011/Hacx-GPT)  ![GitHub Repo stars](https://img.shields.io/github/stars/BlackTechX011/Hacx-GPT?style=social)

HacxGPT-CLI 是一个开源跨平台命令行工具，旨在提供无审查限制的 AI 对话体验。

**核心功能：**
1.  **多模型接入**：支持 OpenRouter、Groq 及 HacxGPT 自有 API，兼容多种开源和闭源模型。
2.  **自研本地引擎**：内置独立高性能 API 引擎，零第三方 SDK 依赖，确保速度与自主控制。
3.  **会话持久化**：支持会话保存、加载及列表管理，内置自动更新系统。
4.  **安全隐私**：API 密钥采用机器绑定加密存储，配置仅保存在本地，不上传至服务器。
5.  **无限制访问**：通过高级提示词减少模型审查，优化思维链（CoT）渲染。
6.  **跨平台兼容**：支持 Linux、Windows、macOS 及 Termux 等系统。

工具本身免费（需用户自备 API Key），另有付费生产级模型服务可选。
