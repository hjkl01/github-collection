### [github copilot-cli](https://github.com/github/copilot-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/github/copilot-cli?style=social)

GitHub Copilot CLI 是一款将 AI 编程助手直接集成到命令行终端的工具。它支持 Linux、macOS 和 Windows 平台，允许用户通过自然语言对话与 AI 协作进行代码构建、调试、编辑和重构。核心功能涵盖原生终端开发体验、深度 GitHub 集成（访问仓库、Issue 及 PR）、智能代理协作能力、MCP 扩展支持以及通过 LSP 服务器增强代码智能。项目要求所有操作需经用户确认，支持多模型切换，并需拥有活跃的 Copilot 订阅。
