### [virattt ai-hedge-fund](https://github.com/virattt/ai-hedge-fund)  ![GitHub Repo stars](https://img.shields.io/github/stars/virattt/ai-hedge-fund?style=social)

这是一个教育用途的 AI 驱动对冲基金概念验证项目，旨在通过多 Agent 协作模拟投资交易决策。系统集成了模仿知名投资者（如巴菲特、芒格等）的策略角色，以及负责估值、市场情绪、基本面、技术分析和风险管理的职能代理。项目仅用于学习和研究，不涉及真实交易。支持通过命令行或 Web 界面运行，提供回测功能，可自定义股票代码、时间及本地大模型模式，需配置 LLM 和金融数据 API 密钥。
