### [ChromeDevTools chrome-devtools-mcp](https://github.com/ChromeDevTools/chrome-devtools-mcp)  ![GitHub Repo stars](https://img.shields.io/github/stars/ChromeDevTools/chrome-devtools-mcp?style=social)

`chrome-devtools-mcp` 是一个 MCP（Model-Context-Protocol）服务器，允许 AI 编程代理（如 Gemini、Claude、Cursor、Copilot）控制和检查实时 Chrome 浏览器。它结合 Chrome DevTools 和 Puppeteer 实现核心功能，包括性能分析、高级浏览器调试（网络请求、截图、控制台消息）及可靠的自动化操作（点击、导航、脚本执行等）。项目支持配置于多种 MCP 客户端，运行需 Node.js v20.19+ 和 Chrome 环境。
