### [OpenBMB VoxCPM](https://github.com/OpenBMB/VoxCPM)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenBMB/VoxCPM?style=social)

VoxCPM 是一款基于 MiniCPM 骨干网络的无分词器文本转语音（TTS）系统，采用端到端扩散自回归架构直接在连续空间生成语音。核心功能包括上下文感知的富有表现力语音生成和真实零样本语音克隆，仅需短参考音频即可还原音色、情感及节奏。系统支持流式合成，推理效率高（RTF 低至 0.15），提供 Python 库、CLI 接口及 Web 演示。包含 VoxCPM1.5 和 VoxCPM-0.5B 两个版本，支持全参数及 LoRA 微调，主要支持中英文，开源协议为 Apache-2.0。
