### [badlogic pi-mono](https://github.com/badlogic/pi-mono)  ![GitHub Repo stars](https://img.shields.io/github/stars/badlogic/pi-mono?style=social)

该项目是用于构建 AI 代理和管理 LLM 部署的工具集（Pi Monorepo）。核心功能如下：

1. **统一 LLM API**：支持 OpenAI、Anthropic、Google 等多提供商接口。
2. **代理运行时**：提供包含工具调用和状态管理功能的 Agent 核心。
3. **编码助手**：提供交互式 CLI 编码代理及 Slack 机器人集成。
4. **交互界面**：包含支持差异渲染的终端 UI 库及 AI 聊天 Web 组件。
5. **部署管理**：提供管理 GPU 节点上 vLLM 部署的命令行工具。
