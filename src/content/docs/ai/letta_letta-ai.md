### [letta-ai letta](https://github.com/letta-ai/letta)  ![GitHub Repo stars](https://img.shields.io/github/stars/letta-ai/letta?style=social)

Letta（原 MemGPT）是构建具备高级记忆能力、可随时间学习和自我改进的有状态 AI 智能体的平台。

核心功能包括：
1. **本地运行**：提供 CLI 工具（Letta Code），支持在终端运行智能体，具备技能（Skills）、子智能体（Subagents）及模型无关性。
2. **API 集成**：提供 Python 和 TypeScript SDK，方便将智能体集成至应用程序。
3. **记忆与工具**：支持记忆块配置、工具调用（如网络搜索）及持续对话管理。
4. **开源生态**：开放源代码，支持社区贡献与开发者协作。
