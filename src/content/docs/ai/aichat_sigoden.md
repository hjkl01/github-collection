### [sigoden aichat](https://github.com/sigoden/aichat)  ![GitHub Repo stars](https://img.shields.io/github/stars/sigoden/aichat?style=social)

AIChat 是一款全功能的 LLM 命令行工具，主要功能总结如下：

1.  **多模型支持**：通过统一接口集成 20 多种主流大模型提供商（如 OpenAI、Claude、Ollama、Gemini 等）。
2.  **交互模式**：提供命令行 (CMD) 和交互式 REPL 模式，内置 Shell 助手可将自然语言任务转换为精确的 Shell 命令。
3.  **多源输入**：支持 stdin、本地文件/目录、远程 URL 及外部命令等多种数据输入与组合形式。
4.  **增强功能**：支持角色定制、会话保持上下文、宏命令录制、RAG（检索增强生成）、函数调用及 AI 智能体。
5.  **本地服务**：内置轻量级 HTTP 服务器，提供 Chat Completions、Embeddings、Rerank 等 API 接口及 LLM Playground 网页交互界面。
6.  **模型对比**：集成 LLM Arena 功能，支持网页端多模型侧边对比评测。
7.  **自定义**：支持用户自定义深色和浅色主题。
