### [droidrun droidrun](https://github.com/droidrun/droidrun)  ![GitHub Repo stars](https://img.shields.io/github/stars/droidrun/droidrun?style=social)

DroidRun 是一个通过大语言模型（LLM）代理控制 Android 和 iOS 设备的强大框架。它允许用户使用自然语言命令自动化设备交互。主要功能包括支持多种 LLM 提供商（如 OpenAI、Gemini 等）、多步骤任务规划能力、截图视觉分析、可扩展的 Python API 以及易于调试的命令行工具。适用于移动端 UI 自动化测试、重复任务自动化、非技术用户引导工作流及远程协助等场景。
