### [OpenBMB UltraRAG](https://github.com/OpenBMB/UltraRAG)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenBMB/UltraRAG?style=social)

UltraRAG 是一款基于模型上下文协议 (MCP) 架构的轻量级 RAG 开发框架，旨在实现低代码、低门槛、快部署。该框架将检索与生成等核心组件标准化为独立 MCP 服务器，开发者仅需通过 YAML 配置即可编排包含循环和条件分支的复杂工作流。项目配备 UltraRAG UI 可视化 IDE，支持管道构建、调试、知识库管理及一键生成交互式 Web 应用。其核心优势包括模块化高复用扩展、内置统一评估基准以及快速原型生成，适用于学术研究实验与工业级应用开发。
