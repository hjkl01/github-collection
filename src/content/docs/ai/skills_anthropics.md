### [anthropics skills](https://github.com/anthropics/skills)  ![GitHub Repo stars](https://img.shields.io/github/stars/anthropics/skills?style=social)

这是 Anthropic 为 Claude 提供的技能系统仓库。其核心功能是通过动态加载指令和资源，使 Claude 能够以可重复的方式完成创意设计、技术开发、企业工作及文档处理等专项任务。仓库提供示例、规范及模板，支持通过 Claude Code、Claude.ai 及 API 调用或创建自定义技能。主要用于演示教育，包含开源示例及源码可见的文档技能。
