### [cexll myclaude](https://github.com/cexll/myclaude)  ![GitHub Repo stars](https://img.shields.io/github/stars/cexll/myclaude?style=social)

该项目是一个基于 Claude Code 的多智能体开发自动化系统（myclaude），支持 Codex、Claude、Gemini 和 OpenCode 多后端执行。核心功能包括多场景工作流模块（如功能开发 do、多智能体编排 omo、敏捷 bmad、需求转代码 requirements、核心命令 essentials 及 SPARV 流程 sparv），以及浏览器自动化、代码代理、需求与测试用例生成等独立技能。系统采用双核架构（Claude Code 编排与 codeagent-wrapper 执行），支持通过 npx 灵活安装、配置模块，并根据不同开发场景推荐相应工作流。
