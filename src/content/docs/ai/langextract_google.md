### [google langextract](https://github.com/google/langextract)  ![GitHub Repo stars](https://img.shields.io/github/stars/google/langextract?style=social)

LangExtract 是一款基于大语言模型的 Python 库，用于根据用户指令从非结构化文本中抽取结构化信息。其核心功能包括：支持提取结果与原文位置的精确映射与溯源验证；通过分块、并行处理及多次抽取策略优化长文档提取效率；提供交互式 HTML 可视化文件以便审查提取实体；兼容云端（Gemini、OpenAI）及本地（Ollama）多种模型，支持自定义模型提供者；无需模型微调，仅需定义提示词和示例即可适配医疗、文学等任意领域任务。
