### [simular-ai Agent-S](https://github.com/simular-ai/Agent-S)  ![GitHub Repo stars](https://img.shields.io/github/stars/simular-ai/Agent-S?style=social)

Agent S 是一个开源框架，旨在构建能像人类一样操作计算机的智能图形用户界面（GUI）智能体。它支持 Windows、macOS 和 Linux 系统，通过结合大语言模型与定位模型，将自然语言指令转化为屏幕操作或代码执行。该项目最新版本的智能体在 OSWorld 基准测试中超越了人类表现（72.60%）。它提供命令行和 SDK 接口，支持本地代码执行环境，可应用于复杂任务自动化、数据处理、文件管理及系统配置等场景。
