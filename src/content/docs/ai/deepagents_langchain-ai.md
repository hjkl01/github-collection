### [langchain-ai deepagents](https://github.com/langchain-ai/deepagents)  ![GitHub Repo stars](https://img.shields.io/github/stars/langchain-ai/deepagents?style=social)

Deep Agents 是一个开箱即用的 AI Agent 框架（Harness），基于 LangGraph 构建，旨在提供无需复杂配置即可运行的智能体。其核心功能包括任务规划、文件系统读写、沙箱 Shell 执行、子 Agent 协作及自动上下文管理。项目支持立即运行，并可灵活自定义模型、工具和提示词。此外，它提供 CLI 终端交互界面，支持会话恢复、网络搜索及远程沙箱集成，兼容多种大模型提供商，完全开源且生产就绪。
