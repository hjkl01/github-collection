### [ollama ollama](https://github.com/ollama/ollama)  ![GitHub Repo stars](https://img.shields.io/github/stars/ollama/ollama?style=social)

Ollama 是一个跨平台的开源工具，用于在本地运行和管理大语言模型。它支持 macOS、Windows、Linux 及 Docker 部署，提供命令行、REST API 及多语言 SDK 接口。基于 llama.cpp 后端，Ollama 拥有丰富的社区生态，集成支持聊天界面、代码辅助、AI 助手、RAG 知识库、终端工具及多种云部署方案，便于开发者快速构建 AI 应用。
