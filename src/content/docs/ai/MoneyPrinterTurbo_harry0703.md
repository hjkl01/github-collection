### [harry0703 MoneyPrinterTurbo](https://github.com/harry0703/MoneyPrinterTurbo)  ![GitHub Repo stars](https://img.shields.io/github/stars/harry0703/MoneyPrinterTurbo?style=social)

MoneyPrinterTurbo 是一个全自动短视频生成工具。输入视频主题或关键词，即可自动完成视频文案、素材匹配、字幕制作及背景音乐合成。支持 Web 界面和 API 调用，兼容竖屏（9:16）和横屏（16:9）高清尺寸，支持批量生成。支持中文和英文，提供多种语音合成选项。支持接入 OpenAI、DeepSeek 等多种大模型。素材支持高清无版权资源或本地文件。无需显卡，建议 4 核 CPU 和 4G 内存，支持 Windows、MacOS 及 Docker 部署。
