### [vllm-project semantic-router](https://github.com/vllm-project/semantic-router)  ![GitHub Repo stars](https://img.shields.io/github/stars/vllm-project/semantic-router?style=social)

vLLM Semantic Router 是为 vLLM 构建的语义路由系统，旨在为混合模型（MoM）提供系统级智能。它位于模型与真实世界之间，负责捕获请求、响应和上下文信号，并基于信号驱动进行决策。核心功能包括异构 LLM 工作负载的语义缓存、多模型间的高效协作、安全防护（实时检测幻觉、防止越狱和隐私泄露）以及构建自学习系统。项目提供命令行工具（CLI）和服务端支持，包含配置管理、服务启停、日志查看及监控仪表盘。
