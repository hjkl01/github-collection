### [databricks-solutions ai-dev-kit](https://github.com/databricks-solutions/ai-dev-kit)  ![GitHub Repo stars](https://img.shields.io/github/stars/databricks-solutions/ai-dev-kit?style=social)

Databricks AI Dev Kit 是专为 Databricks 平台设计的 AI 驱动开发工具包，旨在为 AI 编码助手（如 Claude Code、Cursor 等）提供可信源和工具，加速在 Databricks 上的构建过程。

核心功能与组件如下：

1.  **支持构建**：涵盖 Spark 声明式管道、Databricks Jobs、AI/BI 仪表盘、Unity Catalog、Genie 空间、RAG 知识助手、MLflow 实验、模型服务及全栈应用。
2.  **四大组件**：
    *   **Python 核心库**：提供高级 Databricks 函数，兼容 LangChain 等框架。
    *   **MCP 服务器**：向 AI 助手暴露 50+ 可执行工具。
    *   **技能库**：19 个 Markdown 文档，传授 Databricks 模式与最佳实践。
    *   **Web 构建器**：全栈 Web 应用，提供聊天 UI 支持可视化开发。
3.  **部署集成**：支持通过脚本快速安装至项目或全局环境，兼容多种 AI 编程环境。
