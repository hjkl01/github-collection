### [upstash context7](https://github.com/upstash/context7)  ![GitHub Repo stars](https://img.shields.io/github/stars/upstash/context7?style=social)

Context7 MCP 是一个模型上下文协议 (MCP) 服务器，旨在为大语言模型提供实时、最新且特定版本的代码库文档与示例。它解决了 LLM 依赖过时训练数据导致 API 幻觉或代码陈旧的问题，通过直接从官方来源抓取文档并注入提示上下文，确保代码生成的准确性。用户只需在提示中添加 `use context7`，即可自动检索相关技术库资料，无需切换标签页。项目支持 Cursor、Claude Code 等多种开发环境，提供库 ID 解析与文档查询工具，支持 API 密钥及 OAuth 认证，显著提升 AI 编程助手的技术文档获取能力。
