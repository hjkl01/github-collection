### [ComposioHQ awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills)  ![GitHub Repo stars](https://img.shields.io/github/stars/ComposioHQ/awesome-claude-skills?style=social)

这是一个精选的 Claude 技能（Skills）集合，旨在提升 Claude.ai、Claude Code 及 Claude API 的生产力。项目允许 Claude 执行文本生成外的实际操作，通过 Composio 集成 500+ 应用（如发送邮件、创建工单等）。内容涵盖文档处理、开发工具、数据分析、商务营销及自动化等多个领域的预置技能，支持跨平台通用。此外，项目提供技能创建模板、最佳实践指南及社区贡献规范，鼓励用户构建自定义技能以复用工作流。
