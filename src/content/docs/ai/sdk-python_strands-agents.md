### [strands-agents sdk-python](https://github.com/strands-agents/sdk-python)

**项目核心内容总结：**  
Strands Agents 是一个基于模型驱动的AI代理开发工具包，支持快速构建和部署AI代理系统。其功能包括：  
1. **核心功能**：提供轻量级、可定制的代理框架，支持多种模型（如Amazon Bedrock、Gemini、OpenAI等）和工具集成，内置MCP协议以接入预置工具库。  
2. **使用方法**：通过pip安装后，使用Python代码创建Agent实例，配置模型和工具（如示例中使用`calculator`工具）。  
3. **主要特性**：  
   - 支持多模型提供商（Amazon、Google、OpenAI等）及自定义模型。  
   - 提供双向音频流媒体功能，适用于实时语音交互场景。  
   - 集成工具扩展性，如数学计算、对话终止等功能。  
   - 文档齐全，涵盖部署、开发及生产环境配置指导。
