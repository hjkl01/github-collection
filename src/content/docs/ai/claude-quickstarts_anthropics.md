### [anthropics claude-quickstarts](https://github.com/anthropics/claude-quickstarts)

**核心内容总结：**  
Claude Quickstarts 是一组帮助开发者快速使用 Claude API 的项目模板，包含以下功能：  

1. **项目功能**  
   - **客户支持代理**：基于 Claude 的自然语言处理能力，构建 AI 辅助的客户支持系统。  
   - **财务数据分析**：通过交互式数据可视化，实现聊天式财务数据分析。  
   - **计算机操作演示**：支持 Claude 控制桌面计算机，包含最新 `computer_use_20251124` 工具的缩放功能。  
   - **自主编码代理**：利用 Claude Agent SDK 实现两代理协作（初始化器 + 编码代理），通过 Git 持续化开发进度。  

2. **使用方法**  
   - 克隆仓库，进入对应项目目录。  
   - 安装依赖，配置 Claude API 密钥环境变量。  
   - 运行项目（每个项目附带独立的 README 指南）。  

3. **主要特性**  
   - 提供完整代码示例与详细文档。  
   - 支持 Claude API 的高级功能（如计算机操作、自主编码）。  
   - 开源社区可贡献新项目或优化现有内容。  

**注意事项**：  
- 需注册 [Anthropic 控制台](https://console.anthropic.com) 获取 API 密钥。  
- 项目依赖 Claude API 及相关 SDK 工具。
