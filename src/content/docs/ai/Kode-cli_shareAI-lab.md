### [shareAI-lab Kode-cli](https://github.com/shareAI-lab/Kode-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/shareAI-lab/Kode-cli?style=social)

Kode 是一款运行在终端的 AI 编程助手，核心功能包括理解代码库、编辑文件、运行命令及自动化开发工作流。

主要特性：
1. **多模型协作**：支持灵活切换不同 AI 模型，提供专家咨询、子代理任务分配及并行处理能力。
2. **智能交互**：集成 AGENTS.md 标准支持文档生成，提供 @ 提及系统用于调用模型、代理和文件，支持命令自动补全。
3. **扩展性强**：兼容 MCP 服务器、Agent Skills 插件及技能市场，支持自定义工具和工作流。
4. **跨平台部署**：支持 Windows 原生二进制及 Node.js 回退，提供安全模式与 YOLO 模式切换，兼容 Docker 及 ACP 协议。
