### [livekit agents](https://github.com/livekit/agents)  ![GitHub Repo stars](https://img.shields.io/github/stars/livekit/agents?style=social)

LiveKit Agents 是一个开源 Python 框架，用于构建运行于服务器端的实时可编程 AI 智能体。它支持创建具备视听理解能力的多模态语音助手。核心特性包括：灵活集成 STT、LLM、TTS 及实时 API；内置任务调度与分发；支持 WebRTC、电话集成及客户端数据交互；提供语义转折检测、MCP 原生支持及内置测试框架。框架完全开源，支持私有化部署或 LiveKit Cloud，适用于语音对话、视频头像、多智能体协作等多种场景。
