### [vercel-labs json-render](https://github.com/vercel-labs/json-render)  ![GitHub Repo stars](https://img.shields.io/github/stars/vercel-labs/json-render?style=social)

json-render 是一个生成式 UI 框架，允许通过自然语言提示生成动态、个性化的用户界面，同时基于预定义组件和操作确保输出的安全性与可预测性。它支持跨平台渲染，涵盖 React、Vue、React Native、Remotion 视频、React PDF 文档、React Email 邮件及图片生成。核心流程为用户定义组件目录与行为约束，AI 生成符合 Schema 的 JSON 规范，由框架安全渲染。此外，项目支持流式输出、动态属性绑定、状态驱动逻辑及条件可见性，并提供 36 个预构建的 shadcn/ui 组件。
