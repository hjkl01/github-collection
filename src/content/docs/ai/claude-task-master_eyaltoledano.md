### [eyaltoledano claude-task-master](https://github.com/eyaltoledano/claude-task-master)

**项目功能：**  
Task Master 是一个用于任务管理的 AI 工具，支持通过自然语言与 AI 助手交互，帮助用户解析需求、生成任务、管理项目进度、研究信息等。它可与多个编辑器（如 Cursor、Windsurf、VS Code）集成，并支持通过命令行方式进行操作。

**使用方法：**  
1. **集成到编辑器（如 Cursor）：**  
   - 安装并启用 Task Master 的 MCP 服务。  
   - 在编辑器中通过自然语言指令初始化项目、生成任务、查看任务列表、研究信息等。  
2. **命令行方式：**  
   - 全局或本地安装 `task-master-ai`。  
   - 使用命令如 `task-master init` 初始化项目、`task-master parse-prd` 解析需求文档、`task-master list` 查看任务等。

**主要特性：**  
- 支持与多个编辑器集成，提供无缝的 AI 助手体验。  
- 提供丰富的任务管理功能，如任务生成、查看、修改、研究等。  
- 支持 Claude Code CLI，无需 API 密钥即可使用 Claude 模型。  
- 提供多种工具加载模式（如 `all`、`standard`、`core`），以优化性能。  
- 提供详细的项目需求文档（PRD）支持，提升任务生成的准确性。  
- 支持任务之间的依赖关系、标签管理、任务迁移等高级功能。
