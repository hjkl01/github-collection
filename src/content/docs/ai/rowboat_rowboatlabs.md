### [rowboatlabs rowboat](https://github.com/rowboatlabs/rowboat)  ![GitHub Repo stars](https://img.shields.io/github/stars/rowboatlabs/rowboat?style=social)

Rowboat 是一款开源的本地优先 AI 协作工具，旨在将工作转化为知识图谱并辅助执行任务。

**核心功能：**
1. **本地知识图谱**：连接 Gmail、会议笔记等数据源，在本地构建持久化的知识图谱，数据以纯 Markdown 格式存储，支持手动编辑与 Obsidian 兼容。
2. **智能辅助生成**：基于知识图谱上下文，自动生成会议简报、邮件草稿、文档及 PDF 演示文稿，并能自动提取语音备忘录的关键信息。
3. **后台自动化**：支持运行后台 Agent 自动处理邮件回复、日报生成、项目更新等重复性任务。
4. **灵活扩展**：支持本地模型（如 Ollama）或自有 API 模型，通过 MCP 协议集成搜索、数据库、CRM 等外部工具。
5. **隐私与掌控**：所有数据均在本地设备处理，无云端锁定或专有格式，用户可随时查看、备份或删除。
