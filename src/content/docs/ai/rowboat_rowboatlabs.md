### [rowboatlabs rowboat](https://github.com/rowboatlabs/rowboat)  ![GitHub Repo stars](https://img.shields.io/github/stars/rowboatlabs/rowboat?style=social)

**项目名称：Rowboat（开源AI协作者）**

**核心功能：**  
Rowboat 是一个本地优先的开源 AI 协作者，能够将你的工作转化为知识图谱并基于此进行操作。它连接你的邮件和会议记录，构建长期知识图谱，帮助你更高效地完成工作，所有处理都在你的设备上私密运行。

**主要功能：**  
- 生成报告、PPT 等文件  
- 会议准备（自动整理历史决策、问题和相关邮件）  
- 邮件草稿撰写（基于历史和承诺）  
- 创建和更新 Markdown 格式的知识图谱  
- 录制语音备忘录并自动提取关键点  
- 支持多种模型（本地或云端）和第三方工具集成  

**使用方法：**  
1. 下载适用于 Mac/Windows/Linux 的版本。  
2. 连接 Google 服务（Gmail、日历、Drive），按照文档设置。  
3. 可选配置语音识别（Deepgram API）、网络搜索（Brave/Exa API）。  
4. Rowboat 会自动建立知识图谱，支持 Markdown 编辑和可视化。  

**主要特性：**  
- **本地优先**：所有数据以 Markdown 格式存储在本地，无隐私风险。  
- **长期知识积累**：构建可编辑、可查看的显式知识图谱。  
- **AI 协助**：基于上下文生成文档、邮件、会议准备等。  
- **背景代理任务**：自动执行重复任务（如每日语音简报、邮件草稿生成）。  
- **可扩展性**：支持 MCP 协议，可集成搜索、数据库、CRM、Slack、GitHub 等工具。  
- **多模型支持**：兼容 Ollama、LM Studio 或自定义 API 模型。  

**适用场景：**  
- 会议准备  
- 邮件撰写  
- 文档与 PPT 生成  
- 自动化任务处理  
- 项目跟进与跟进管理  

**扩展支持：**  
- 可连接 Exa、Twitter、ElevenLabs、Slack、Linear/Jira 等工具。  
- 提供 Discord 社区和 Twitter 官方账号。
