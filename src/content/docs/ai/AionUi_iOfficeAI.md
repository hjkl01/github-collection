### [iOfficeAI AionUi](https://github.com/iOfficeAI/AionUi)  ![GitHub Repo stars](https://img.shields.io/github/stars/iOfficeAI/AionUi?style=social)

AionUi 是一款免费开源的跨平台 AI 协作桌面应用，旨在让 AI 智能体与用户共同在电脑上工作。其核心功能总结如下：

*   **零配置内置智能体**：安装即可使用，无需额外配置，支持文件读写、网页搜索、图片生成及 MCP 工具调用。
*   **多智能体集成**：自动检测并整合本地已安装的 CLI 智能体（如 Claude Code、Codex、Qwen Code 等），提供统一的操作界面。
*   **全模型兼容**：支持 20+ 主流 AI 平台及模型（包括 OpenAI、Gemini、Anthropic、Ollama 及国内大模型），允许用户使用任意 API Key。
*   **远程控制与自动化**：支持 WebUI 及第三方平台（Telegram、飞书、钉钉）远程接入；提供 Cron 定时任务功能，实现 24/7 无人值守自动化运行。
*   **生产力增强工具**：内置 12+ 专业助手（如 PPT 生成、文档转换），支持文件批量管理、Excel 智能分析、10+ 种格式文档即时预览与编辑。
*   **隐私与安全**：所有数据本地 SQLite 存储，不上传云端；支持 macOS、Windows 和 Linux 全平台，采用 Apache-2.0 开源许可。
