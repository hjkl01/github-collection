### [iOfficeAI AionUi](https://github.com/iOfficeAI/AionUi)  ![GitHub Repo stars](https://img.shields.io/github/stars/iOfficeAI/AionUi?style=social)

AionUi 是一款支持多模型、多会话的 AI 协作工具，核心功能包括：  
1. **核心功能**  
- 支持 Gemini、OpenAI、Claude、Qwen 等主流模型及本地部署模型（如 Ollama）  
- 提供文件管理功能，支持智能分类、拖拽上传及 AI 协助整理文件夹  
- 内置 9+ 格式预览面板（PDF/Word/Excel/Markdown 等），支持实时编辑与调试  
- 集成 AI 图像生成与编辑功能（支持 Gemini 2.5 Flash 等模型）  
- 支持 WebUI 远程访问，可跨设备通过浏览器操作  

2. **使用方法**  
- 系统要求：macOS 10.15+/Windows 10+/Linux（Ubuntu 18.04+ 等），推荐 4GB 内存及 500MB 存储空间  
- 下载安装：通过 GitHub 发布页获取最新版本  
- 配置流程：安装后通过 Google 账号登录或 API Key 认证，即可使用  

3. **主要特性**  
- 多会话独立上下文，支持本地存储所有对话记录  
- 支持 CSS 自定义界面风格  
- 提供完整的配置文档与使用案例（如 Excel 数据处理、文件管理等）  
- 开源协议：Apache-2.0，支持社区贡献与问题反馈  

项目提供 GitHub 讨论区、问题提交及版本更新追踪，用户可通过 X（原 Twitter）提交视频反馈。
