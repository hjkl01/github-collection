### [ZJU-LLMs Agent-Kernel](https://github.com/ZJU-LLMs/Agent-Kernel)  ![GitHub Repo stars](https://img.shields.io/github/stars/ZJU-LLMs/Agent-Kernel?style=social)

Agent-Kernel 是一个用户友好的大规模社会仿真多智能体系统开发框架，旨在支持基于大语言模型（LLM）的智能体以探索集体智能的扩展定律。

核心功能与优势包括：
1. **动态管理**：支持在运行时动态添加或移除智能体。
2. **无限扩展**：支持智能体规模的无限扩展。
3. **实时干预**：支持仿真过程中的参数调整与事件干预。
4. **可信验证**：具备严格的系统级验证机制，确保智能体行为符合物理和社会规则。
5. **高复用性**：采用基于插件的模块化微内核架构，支持跨场景代码复用。

项目已成功应用于“宇宙 25"实验及浙江大学校园生活模拟等复杂场景。提供独立版和分布式版（基于 Ray）两种安装包，支持可选的 Web 服务和存储功能，并配有可视化控制界面（Society-Panel）用于配置、部署和监控仿真。
