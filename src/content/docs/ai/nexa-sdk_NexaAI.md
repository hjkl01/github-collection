### [NexaAI nexa-sdk](https://github.com/NexaAI/nexa-sdk)  ![GitHub Repo stars](https://img.shields.io/github/stars/NexaAI/nexa-sdk?style=social)

NexaSDK 是一款高性能的本地 AI 推理框架，旨在以最小能耗在端侧设备上构建智能应用。它支持跨平台（Android、Windows、Linux、macOS、iOS）运行最新的 AI 模型，核心功能如下：

1. **多端 SDK 支持**：提供 CLI、Python SDK、Android SDK、Linux Docker 和 iOS SDK，仅需少量代码即可集成。
2. **广泛硬件适配**：深度优化支持 NPU（Qualcomm、Intel、AMD）、GPU 和 CPU，实现高效推理。
3. **全模态模型覆盖**：支持 LLM、多模态、ASR、OCR、图像生成、Rerank 及 Embedding，兼容 GGUF、MLX、NEXA 格式，并提供新模型第一时间支持。
4. **开源与许可**：CPU/GPU 组件采用 Apache 2.0 协议，NPU 组件提供个人免费授权及商业许可选项。
