### [docker cagent](https://github.com/docker/cagent)  ![GitHub Repo stars](https://img.shields.io/github/stars/docker/cagent?style=social)

Docker Agent 是一个 Docker CLI 插件，用于构建、运行和分享 AI 代理。用户通过声明式的 YAML 配置文件即可定义代理的角色、指令和工具，无需编写代码即可创建智能 AI 代理协作解决复杂问题。核心功能包括：支持多代理架构实现任务自动委派；提供丰富的工具生态及 MCP 协议支持；兼容多种 AI 模型提供商；内置推理、任务管理、记忆及 RAG 检索功能；支持将代理打包至 OCI 注册表进行分享。
