### [pedramamini Maestro](https://github.com/pedramamini/Maestro)  ![GitHub Repo stars](https://img.shields.io/github/stars/pedramamini/Maestro?style=social)

Maestro 是一款跨平台桌面应用，面向键盘优先用户，用于编排和管理 AI 代理及项目。作为 AI 提供商的透传层，它复用用户现有的工具权限与认证。

核心功能包括：

1. **多代理并行管理**：支持并行运行多个 AI 代理，每个拥有独立工作区、会话历史与上下文。
2. **自动化工作流**：通过 Playbooks 批量处理 Markdown 任务清单，支持长时间无值守运行与可重复工作流。
3. **Git 深度集成**：支持 Git Worktrees 在隔离分支并行开发，具备分支管理、Diff 查看及 Git 感知功能。
4. **协作与远程**：提供多代理群聊协调，内置 Web 服务支持移动端远程控制，支持 CLI 无头操作。
5. **高效交互**：全键盘控制（AI 终端与 Shell 终端无缝切换），支持会话导入、自动保存、成本追踪及主题定制。
6. **数据分析**：提供使用量统计仪表盘与文档知识图谱，可视化项目文档关系与使用模式。

支持 Claude Code、OpenAI Codex、OpenCode 及 Factory Droid 等主流 AI 工具。
