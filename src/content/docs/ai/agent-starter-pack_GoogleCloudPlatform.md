### [GoogleCloudPlatform agent-starter-pack](https://github.com/GoogleCloudPlatform/agent-starter-pack)  ![GitHub Repo stars](https://img.shields.io/github/stars/GoogleCloudPlatform/agent-starter-pack?style=social)

Agent Starter Pack 是 Google Cloud 上用于构建生成式 AI 代理（Agent）的 Python 包，提供生产就绪模板。它自动化基础设施、CI/CD、可观测性和安全配置，让开发者专注于 Agent 逻辑。项目包含多种预置 Agent 模板（如 ReAct、RAG、多 Agent 等），支持通过命令一键创建完整项目架构及远程模板分享。核心特性包括自动化 CI/CD 流水线、RAG 数据管道、监控集成、Gemini CLI 支持，支持部署至 Cloud Run 或 Agent Engine。
