### [farion1231 cc-switch](https://github.com/farion1231/cc-switch)  ![GitHub Repo stars](https://img.shields.io/github/stars/farion1231/cc-switch?style=social)

CC Switch 是一款基于 Tauri 2 构建的跨平台桌面管理工具，用于统一管理 Claude Code、Codex、Gemini CLI、OpenCode 及 OpenClaw 五个 AI 编程 CLI 工具。

**核心功能：**
*   **配置管理**：提供 50+ 种 API 提供者预设，通过可视化界面实现一键导入与切换，无需手动编辑配置文件，数据基于 SQLite 原子写入保障安全。
*   **资源同步**：统一跨应用管理 MCP 服务器与 Skills，支持双向同步、Deep Link 导入及从 GitHub 一键安装。
*   **便捷访问**：支持系统托盘快捷切换、多云盘（Dropbox/OneDrive/WebDAV 等）配置同步、本地代理与故障转移。
*   **辅助工具**：内置使用量与成本追踪、会话历史管理与恢复、OpenClaw 工作区编辑及登录确认功能。
*   **系统支持**：原生支持 Windows、macOS 和 Linux 系统。
