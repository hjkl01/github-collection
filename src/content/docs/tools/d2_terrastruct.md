### [terrastruct d2](https://github.com/terrastruct/d2)  ![GitHub Repo stars](https://img.shields.io/github/stars/terrastruct/d2?style=social)

D2 是一款现代化的图表脚本语言，可将文本描述转换为图表。支持通过 CLI 生成 SVG、PNG、PDF 格式，也可作为 Go 库调用。具备自动格式化、语法高亮等语言工具，支持多种布局引擎（如 DAGRE、ELK、TALA）及主题自定义。提供插件系统扩展渲染管线，并集成 VSCode、Vim 等编辑器插件及第三方工具。
