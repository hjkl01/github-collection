### [mulaRahul keyviz](https://github.com/mulaRahul/keyviz)

**项目核心内容总结：**

Keyviz 是一款免费开源软件，可实时可视化键盘按键和鼠标操作（如点击、拖拽等），适用于演示、协作等场景，帮助观众直观看到操作过程中的快捷键和鼠标动作。

**主要功能与特性：**
- 支持键盘快捷键（如 Cmd+Click）与鼠标动作（如 Alt+Drag）的联合可视化；
- 可自定义可视化样式（颜色、位置、动画、图标等）；
- 提供过滤功能，仅显示快捷键或同时显示普通键；
- 跨平台支持（Windows、MacOS、Linux）。

**使用方法：**
- 下载安装包或通过 Microsoft Store、Scoop、Winget 等工具安装；
- Linux 需安装依赖库后通过 .deb 或 .rpm 包安装；
- 可从源码使用 Flutter 构建（需先安装 Flutter）。

**注意事项：**
- MacOS 需开启“输入监控”和“辅助功能”权限；
- Windows 安装时若提示缺少 .dll 文件，需安装 Visual C++ 红istributables。
