### [simulot immich-go](https://github.com/simulot/immich-go)  ![GitHub Repo stars](https://img.shields.io/github/stars/simulot/immich-go?style=social)

Immich-Go 是一款开源跨平台命令行工具，用于将大量照片和视频高效上传至自托管的 Immich 服务器。

核心特性包括：
1. **多源上传**：支持本地文件夹、Google Photos 导出包、iCloud 版本、ZIP 压缩包及跨 Immich 服务器迁移。
2. **高性能处理**：无需 NodeJS 或 Docker，可稳定处理超过 10 万张照片。
3. **智能管理**：自动进行重复检测、连拍与 RAW+JPEG 堆栈、跳过系统垃圾文件。
4. **元数据同步**：完整保留照片元数据，自动同步源端相册结构与标签。
5. **跨平台支持**：兼容 Windows、macOS、Linux 及 FreeBSD 系统。

通过提供灵活的 API 交互与多种上传指令，该项目实现了照片的批量导入与结构化迁移。
