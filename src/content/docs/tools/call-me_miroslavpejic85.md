### [miroslavpejic85 call-me](https://github.com/miroslavpejic85/call-me)  ![GitHub Repo stars](https://img.shields.io/github/stars/miroslavpejic85/call-me?style=social)

Call-Me 是一个基于 WebRTC 技术的网页端一对一视频通话工具。支持端到端加密，用户可通过用户名登录并设置语言。核心功能包括发起视频通话、切换音视频设备、实时聊天、静音/关闭摄像头、屏幕共享及文件共享。提供主机保护模式，支持通过 REST API 获取连接用户或发起通话。支持 Node.js 和 Docker 部署，可通过 iframe 快速集成，也支持自托管部署。
