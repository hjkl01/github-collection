### [locustio locust](https://github.com/locustio/locust)  ![GitHub Repo stars](https://img.shields.io/github/stars/locustio/locust?style=social)

Locust 是一款开源的 HTTP 及其他协议性能与负载测试工具。它允许用户使用原生 Python 代码编写测试场景，无需使用特定领域语言，测试脚本可作为普通代码版本控制。Locust 基于事件驱动架构，支持分布式运行和高并发模拟，可在一台或多台机器上测试海量用户。项目提供实时 Web 界面，用于监控吞吐量、响应时间和错误统计，并支持在测试运行时动态调整负载。此外，它支持命令行模式运行，便于集成到 CI/CD 流程，且架构灵活可扩展，可适配任意系统或协议，支持自定义负载模型、报告及扩展功能。
