### [wezterm wezterm](https://github.com/wezterm/wezterm)  ![GitHub Repo stars](https://img.shields.io/github/stars/wezterm/wezterm?style=social)

WezTerm 是一个由 @wez 开发的 GPU 加速跨平台终端模拟器和多路复用器，使用 Rust 语言实现。项目提供以下核心功能和信息：

1. **功能特性**  
   - 支持跨平台运行（Windows/macOS/Linux）  
   - GPU 加速渲染，提升性能和视觉效果  
   - 支持终端多路复用（类似 tmux 的功能）

2. **使用方法**  
   - 安装方式详见 [官方安装文档](https://wezterm.org/installation)  
   - 提供中文用户文档和截图示例（如 macOS 上运行 Vim 的界面）

3. **支持与协作**  
   - 问题反馈可通过 [GitHub Issues](https://github.com/wezterm/wezterm/issues) 或 [GitHub Discussions](https://github.com/wezterm/wezterm/discussions)  
   - 实时交流支持 [Matrix 房间](https://app.element.io/#/room/#wezterm:matrix.org)

4. **项目支持**  
   - 提供赞助渠道（GitHub Sponsors、Patreon、Ko-Fi、Liberapay）  
   - 赞助用于覆盖项目维护成本及开发者时间投入
