### [zed-industries zed](https://github.com/zed-industries/zed)  ![GitHub Repo stars](https://img.shields.io/github/stars/zed-industries/zed?style=social)

**Zed 项目核心内容总结：**

1. **项目功能**  
   Zed 是一款高性能、支持多人协作的代码编辑器，由 Atom 和 Tree-sitter 的开发团队打造，适用于实时多人协作开发场景。

2. **使用方法**  
   - **安装**：提供 macOS、Linux 和 Windows 的直接下载链接，支持通过各平台包管理器安装（如 macOS 的 Homebrew、Linux 的包管理器、Windows 的 scoop 等）。  
   - **其他平台**：Web 版暂未支持（正在开发中，跟踪问题 #5396）。

3. **主要特性**  
   - 支持跨平台（当前主要覆盖桌面系统）。  
   - 提供详细的开发文档，包含 macOS/Linux/Windows 的构建指南。  
   - 开源社区贡献机制完善，提供 CONTRIBUTING.md 文件指导参与方式，并开放招聘岗位。  
   - 通过 `cargo-about` 工具确保第三方依赖的许可证合规性，保证开源合规。

4. **开发与协作**  
   - 提供开发文档和构建流程说明。  
   - 鼓励社区贡献，支持通过代码提交或加入团队参与开发。
