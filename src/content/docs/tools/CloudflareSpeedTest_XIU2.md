### [XIU2 CloudflareSpeedTest](https://github.com/XIU2/CloudflareSpeedTest)  ![GitHub Repo stars](https://img.shields.io/github/stars/XIU2/CloudflareSpeedTest?style=social)

CloudflareSpeedTest 是一款用于测试 Cloudflare CDN 延迟与速度的工具，旨在为中国内地用户筛选最优的 IPv4 和 IPv6 IP 地址。该工具支持自定义参数进行延迟测速（TCP/HTTP 模式）、下载测速及多维度筛选（如延迟上限、丢包率、下载速度、指定地区等）。结果可直观显示并导出为 CSV 文件，帮助优化 Hosts 配置以实现本地加速。支持 Windows、Linux、macOS 及 OpenWrt 等平台，同时也可用于测试其他 CDN 或自定义网站，但不支持 Cloudflare WARP 协议。
