### [kieranklaassen compound-engineering-plugin](https://github.com/kieranklaassen/compound-engineering-plugin)  ![GitHub Repo stars](https://img.shields.io/github/stars/kieranklaassen/compound-engineering-plugin?style=social)

本项目是一个跨 AI 编程工具的插件系统，核心提供 Compound Engineering 插件，旨在通过优化工程流程减少技术债务。主要功能：

1. **多工具兼容安装**：支持将插件安装或转换为 Claude Code、Cursor、Copilot、Gemini、Windsurf 等十多种 AI 编程工具兼容的格式。
2. **配置自动同步**：自动同步 Claude Code 的个人技能、命令及 MCP 服务器配置至其他支持的工具，确保配置实时更新。
3. **标准化工作流**：提供计划、执行、审查、归档四个核心命令，贯彻“每步工程工作应使后续工作更易”的理念，通过 80% 精力投入规划与审查，沉淀知识经验。
