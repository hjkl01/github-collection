### [oauth2-proxy oauth2-proxy](https://github.com/oauth2-proxy/oauth2-proxy)  ![GitHub Repo stars](https://img.shields.io/github/stars/oauth2-proxy/oauth2-proxy?style=social)

OAuth2 Proxy 是一个灵活的开源工具，可用作独立反向代理或中间件组件，用于通过 OAuth2 / OIDC 认证保护 Web 应用程序。它支持 Google、Microsoft、GitHub 等多种认证提供商，能够拦截请求并引导用户完成身份验证。验证成功后，它可将用户详情（如用户名和组）作为 HTTP 头转发至上游应用，为多个应用程序提供统一的身份认证服务。
