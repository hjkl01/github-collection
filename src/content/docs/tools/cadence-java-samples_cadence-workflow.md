### [cadence-workflow cadence-java-samples](https://github.com/cadence-workflow/cadence-java-samples)  ![GitHub Repo stars](https://img.shields.io/github/stars/cadence-workflow/cadence-java-samples?style=social)

该项目为 Java Cadence 的示例集合，旨在演示客户端与服务器的各项功能，主要包含以下核心内容：

1. **HelloWorld 系列**：涵盖活动执行、重试、异步调用、子工作流、异常处理、查询、信号、周期性任务、搜索属性及 Cron 工作流。
2. **文件处理**：演示任务路由及活动亲和性特性。
3. **行程预订**：展示 SAGA 模式的 Cadence 实现。
4. **Spring Boot 集成**：演示如何使用 Spring Boot 框架启动 Cadence 工作器服务。

此外，项目提供构建、环境部署及运行样本的完整指南。
