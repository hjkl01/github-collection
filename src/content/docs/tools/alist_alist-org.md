### [alist-org alist](https://github.com/alist-org/alist)

**核心内容总结：**  
AList 是一个支持多种存储（如阿里云盘、OneDrive、Google Drive、FTP、S3 等）的文件列表程序，基于 Gin 和 Solidjs 开发。主要功能包括文件预览（支持 PDF、代码、Office 文档等）、暗黑模式、多语言、受保护路由（密码/认证）、WebDAV 支持、Docker 部署、Cloudflare Workers 代理、文件打包下载、网页上传、离线下载等。提供文档和 API 接口，开源且采用 AGPL-3.0 许可证。用户可通过 Docker 或 Cloudflare Workers 快速部署，适合用于文件共享和网络存储管理。
