### 项目核心内容总结

**PasteMD** 是一款用于解决从 Markdown 或网页复制内容到 Word、WPS 或 Excel 时格式混乱问题的小工具。其主要功能包括：

- **Markdown 转 Word/WPS**：从剪贴板读取 Markdown 内容，通过 Pandoc 转换为 DOCX，并自动插入到 Word 或 WPS 的光标位置。
- **Markdown 表格转 Excel**：智能识别 Markdown 表格并一键粘贴到 Excel。
- **HTML 内容支持**：支持从网页复制 HTML 内容，直接粘贴到 Word/WPS。
- **全局热键**：默认使用 `Ctrl+B` 快捷键一键粘贴。
- **托盘菜单**：可设置热键、通知、保留文件、查看日志等。

**使用方法**：

1. 下载便携版（需自行安装 Pandoc）或一体化安装包（自带 Pandoc）。
2. 打开 Word、WPS 或 Excel，并将光标放在需要插入的位置。
3. 复制 Markdown 或网页内容到剪贴板。
4. 按下热键 `Ctrl+B`，内容将自动插入到文档中。

**主要特性**：

- 支持多种 AI 网站复制内容的兼容性。
- 智能识别应用类型（Word、WPS、Excel）并自动打开。
- 支持格式保留、光标移动、通知提醒等功能。
- 提供配置文件，支持自定义热键、保存路径、是否保留文件等设置。
