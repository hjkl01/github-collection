### [NationalSecurityAgency ghidra](https://github.com/NationalSecurityAgency/ghidra)  ![GitHub Repo stars](https://img.shields.io/github/stars/NationalSecurityAgency/ghidra?style=social)

Ghidra 是由美国国家安全局（NSA）研发并维护的软件逆向工程（SRE）框架。它提供了一系列功能齐全的分析工具，支持在 Windows、macOS 和 Linux 平台上分析编译代码。核心功能包括反汇编、反编译、绘图和脚本编写，支持广泛的处理器指令集和可执行格式，可运行于用户交互及自动化模式。用户可通过 Java 或 Python 开发自定义扩展。该项目旨在解决复杂逆向工程中的扩展与协作问题，用于分析恶意代码、理解系统漏洞，并提供可定制的逆向工程研究平台。
