### [xmake-io xmake](https://github.com/xmake-io/xmake)  ![GitHub Repo stars](https://img.shields.io/github/stars/xmake-io/xmake?style=social)

xmake 是一个基于 Lua 脚本的跨平台构建工具，专注于现代 C/C++ 构建，具有简单、快速和强大的依赖集成特性。它集构建后端、项目生成器、包管理器、远程分布式构建及缓存功能于一体。xmake 使用 xmake.lua 文件配置项目，支持直接编译或生成工程文件。它支持广泛的操作系统、架构及多种编程语言（C、C++、Rust、Go、Swift 等），并提供内置包管理以自动安装依赖。此外，xmake 还支持跨平台交叉编译、多语言混合编译、分布式编译加速、增量编译以及丰富的 IDE 和 CI 集成，且安装轻量无外部依赖。
