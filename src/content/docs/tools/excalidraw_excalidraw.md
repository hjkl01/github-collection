### [excalidraw excalidraw](https://github.com/excalidraw/excalidraw)  ![GitHub Repo stars](https://img.shields.io/github/stars/excalidraw/excalidraw?style=social)

Excalidraw 是一款免费开源的虚拟手绘风格白板工具，支持多人实时协作与端到端加密。具备无限画布、手绘样式、丰富绘图工具（形状、箭头、自由绘制等）、图像及形状库支持，涵盖多语言、暗黑模式等功能。支持导出 PNG/SVG/JSON 格式，提供撤销/重做、缩放平移、PWA 离线使用、本地自动保存及分享链接能力。此外，提供 npm 包与 VSCode 扩展，便于集成至其他应用。
