### [gtsteffaniak filebrowser](https://github.com/gtsteffaniak/filebrowser)

FileBrowser Quantum 是一款免费、开源的自托管式 Web 文件管理工具，提供现代化界面和丰富的功能。核心特性包括：支持多源文件管理、多种登录方式（OIDC、密码+2FA）、实时搜索与索引、可配置的分享权限、目录级访问控制、API 接口等。用户可通过 Docker 或二进制文件部署，配置通过 `config.yaml` 文件完成。系统内存需求取决于文件源复杂度，支持跨平台运行（Linux/Windows/Mac）。相比同类工具，其优势在于更高效的索引机制、更灵活的分享设置及更轻量的 Docker 镜像（180MB）。
