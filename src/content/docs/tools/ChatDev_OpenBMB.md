### [OpenBMB ChatDev](https://github.com/OpenBMB/ChatDev)  ![GitHub Repo stars](https://img.shields.io/github/stars/OpenBMB/ChatDev?style=social)

ChatDev 2.0 (DevAll) 是一个零代码多智能体编排平台，致力于实现“万物开发”。用户无需编写代码，仅需通过简单配置即可快速构建和运行定制的多智能体系统，支持定义智能体、工作流及任务。平台可应用于数据可视化、3D 生成、深度研究、游戏开发等多种复杂场景。提供 Web 控制台进行可视化工作流设计与监控，支持 Python SDK 实现编程化执行，并兼容 Docker 部署。
