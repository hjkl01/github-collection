### [HunxByts GhostTrack](https://github.com/HunxByts/GhostTrack)  ![GitHub Repo stars](https://img.shields.io/github/stars/HunxByts/GhostTrack?style=social)

**GhostTrack** 是一款用于追踪位置或手机号的信息收集工具（OSINT），支持以下功能：  
- **IP追踪**：结合 Seeker 工具获取目标 IP 信息；  
- **电话号码追踪**：搜索目标手机号相关信息；  
- **用户名追踪**：在社交媒体上搜索目标用户名信息。  

**使用方法**：  
1. 安装依赖（Linux/Debian 或 Termux）：  
   ```bash  
   sudo apt-get install git python3（Linux）  
   pkg install git python3（Termux）  
   ```  
2. 克隆仓库并运行：  
   ```bash  
   git clone https://github.com/HunxByts/GhostTrack.git  
   cd GhostTrack  
   pip3 install -r requirements.txt  
   python3 GhostTR.py  
   ```
