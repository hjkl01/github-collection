### [krau SaveAny-Bot](https://github.com/krau/SaveAny-Bot)  ![GitHub Repo stars](https://img.shields.io/github/stars/krau/SaveAny-Bot?style=social)

Save Any Bot 是一个支持 Telegram 及多网络源文件保存的机器人。它允许用户将 Telegram 媒体（含受限内容）、URL、磁力链接及 1000+ 网站资源下载并保存至多种存储后端（如 S3、WebDAV、Alist、本地文件系统等）。项目支持批量处理、流传输、多用户管理、文件自动整理及频道监控自动保存，并可通过 JS 插件扩展自定义网站保存功能。
