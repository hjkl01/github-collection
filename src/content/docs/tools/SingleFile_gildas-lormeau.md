### [gildas-lormeau SingleFile](https://github.com/gildas-lormeau/SingleFile)

SingleFile 是一款浏览器扩展和命令行工具，可将网页保存为单个 HTML 文件，支持 Chrome、Firefox、Edge、Safari 等主流浏览器。其核心功能包括：通过扩展工具栏按钮保存当前页面、取消保存操作、自动保存标签页、注释和编辑内容（如高亮文本、添加笔记）。支持多种保存格式（HTML、自解压 ZIP、MHTML 等），可压缩文件、移除冗余内容，并兼容多个第三方工具（如 ArchiveBox、Trilium 等）。用户可通过浏览器商店安装，或手动下载 ZIP 包进行安装。CLI 工具支持命令行操作，还提供上传至 Google Drive 或 GitHub 的功能，以及通过区块链验证文件完整性的选项。
