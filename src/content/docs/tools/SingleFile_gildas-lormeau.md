### [gildas-lormeau SingleFile](https://github.com/gildas-lormeau/SingleFile)  ![GitHub Repo stars](https://img.shields.io/github/stars/gildas-lormeau/SingleFile?style=social)

SingleFile 是一款将完整网页保存为单个 HTML 文件的工具，提供浏览器扩展及命令行接口，兼容多种主流浏览器。其核心功能是将网页内容、样式和资源打包成独立文件，支持离线查看且无需扩展，同时优化文件大小并移除无用代码。用户可保存当前页、选中内容或多标签页，支持页面注释、自动保存、用户脚本集成，并可选择上传至云存储或进行区块链存证。
