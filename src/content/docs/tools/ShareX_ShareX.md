### [ShareX ShareX](https://github.com/ShareX/ShareX)

ShareX 是一款集屏幕捕获、文件共享和生产力工具于一体的软件，支持截图、录屏、图像处理（如添加特效）、OCR文字识别、自定义上传器等功能。用户可通过官网或 GitHub 下载安装，使用快捷键或命令行参数快速操作。主要特性包括：多种截图模式（如滚动截图）、兼容主流云存储（Amazon S3/Google Cloud Storage/Cloudflare R2）、支持多平台分发（Steam/Microsoft Store），并提供详细的使用文档和社区支持。
