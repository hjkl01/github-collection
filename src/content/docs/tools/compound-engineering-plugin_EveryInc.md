### [EveryInc compound-engineering-plugin](https://github.com/EveryInc/compound-engineering-plugin)  ![GitHub Repo stars](https://img.shields.io/github/stars/EveryInc/compound-engineering-plugin?style=social)

Compound Marketplace 提供 Compound Engineering 插件及跨工具配置管理，旨在通过规范流程降低技术债务，提升后续开发效率。主要功能：
1. **多工具兼容**：支持 Claude Code、Cursor、Copilot 等数十种 AI 编码工具的插件安装与格式转换。
2. **配置同步**：可将个人 Claude Code 技能、命令及 MCP 配置同步至其他支持工具。
3. **工作流指令**：提供规划、执行、审查、总结指令，通过重规划与复盘沉淀经验，降低技术债，优化代码库质量。
