### [EveryInc compound-engineering-plugin](https://github.com/EveryInc/compound-engineering-plugin)  ![GitHub Repo stars](https://img.shields.io/github/stars/EveryInc/compound-engineering-plugin?style=social)

**Compound Marketplace** 是一个提供工程工具的插件市场，核心是 **Compound Engineering Plugin**，旨在通过系统化流程让工程任务逐步简化。

**功能**  
- 提供规划（Plan）、执行（Work）、审查（Review）、知识复用（Compound）的完整开发工作流。  
- 通过多代理审查、任务追踪、知识文档化等工具提升开发效率与质量。

**使用方法**  
1. 安装插件：  
   ```bash  
   /plugin marketplace add https://github.com/kieranklaassen/compound-engineering-plugin  
   /plugin install compound-engineering  
   ```  
2. 使用命令启动工作流：  
   - `/workflows:plan`：将需求转化为详细计划  
   - `/workflows:work`：执行计划并追踪任务  
   - `/workflows:review`：多代理代码审查  
   - `/workflows:compound`：文档化经验用于后续优化  

**特性**  
- **复利式工程**：通过计划与审查减少技术债务，使后续开发更高效。  
- **知识沉淀**：将经验转化为可复用的文档，持续优化流程。  
- **质量优先**：80%精力投入规划与审查，20%用于执行，确保长期可维护性。
