### [jgraph drawio](https://github.com/jgraph/drawio)

**核心内容总结：**  
draw.io 是一款可配置的图表与白板可视化工具，支持创建和编辑图表、白板内容，但不提供源代码，不适合用作开发框架。其主要功能包括：  
1. **功能**：基于 Web 的图表编辑，支持多种主题（图表/白板），SVG 导出仅用于网页嵌入，不支持进一步编辑。  
2. **使用方式**：可通过 GitHub Pages 发布项目、使用 Docker 镜像、下载桌面版（[diagrams.net](https://get.diagrams.net)）或通过发布页面获取完整 WAR 文件运行。  
3. **特性**：不支持协作编辑；不提供源代码，不接受 PR；支持 Chrome、Firefox、Safari 等主流浏览器最新版本。  
4. **限制**：项目不开放源代码，部分资源（如图标）使用 CC BY 4.0 许可，需遵守商标使用规范，禁止将 draw.io 名称或标志用于商业用途或误导性场景。  

**注意事项**：  
- 项目非开源，不支持协作编辑，不提供源代码。  
- 使用需遵守商标和许可条款，避免与 JGraph 混淆。  
- 推荐替代方案：需协作功能时选择 [Tldraw](https://github.com/tldraw/tldraw) 或 [Excalidraw](https://github.com/excalidraw/excalidraw)。
