### [alam00000 bentopdf](https://github.com/alam00000/bentopdf)  ![GitHub Repo stars](https://img.shields.io/github/stars/alam00000/bentopdf?style=social)

BentoPDF 是一款隐私优先的客户端 PDF 工具包，支持自托管。所有文件处理均在浏览器本地完成，无需上传至服务器，确保数据安全与隐私。

主要功能涵盖：
1. **组织管理**：合并、拆分、提取、旋转、重排、多页合一及书签管理。
2. **编辑修改**：表单创建与填写、添加水印/页眉页脚、裁剪、去斜、红标、内容提取及书签编辑。
3. **格式转换**：支持图像、Office 文档、Markdown 等转为 PDF，以及 PDF 转为图像、文本、Excel、OCR 识别等。
4. **安全优化**：压缩、加密/解密、数字签名验证、权限设置、元数据清理及修复损坏 PDF。
5. **自动化工作流**：提供可视化节点编辑器构建自定义 PDF 处理管道。

项目基于现代 Web 技术构建，支持 Docker、静态托管及离线部署，采用 AGPL 和商业双重许可。
