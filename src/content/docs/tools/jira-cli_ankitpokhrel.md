### [ankitpokhrel jira-cli](https://github.com/ankitpokhrel/jira-cli)  ![GitHub Repo stars](https://img.shields.io/github/stars/ankitpokhrel/jira-cli?style=social)

JiraCLI 是一款用于 Atlassian Jira 的交互式命令行工具，旨在减少对 Jira 网页界面的依赖。它支持 Linux、macOS、Windows 等操作系统，兼容 Jira Cloud 和 Server 部署环境。功能涵盖问题全生命周期管理（包括搜索、导航、创建、编辑、克隆、链接、状态转换、删除、评论及工作日志）以及 Epics 和 Sprints 的查看与管理。工具提供交互式终端界面，支持 POSIX 兼容命令行参数和脚本化输出（如 Plain、JSON、CSV 格式），支持 Basic、Bearer、mTLS 多种认证方式，并提供多项目配置功能。
