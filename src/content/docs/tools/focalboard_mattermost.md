### [mattermost focalboard](https://github.com/mattermost/focalboard)  ![GitHub Repo stars](https://img.shields.io/github/stars/mattermost/focalboard?style=social)

Focalboard 是一款开源、多语言且支持自托管的独立项目管理工具，可作为 Trello、Notion 和 Asana 的替代方案，用于定义、组织、跟踪和管理个人或团队的工作。

项目提供两个版本：
1. **个人桌面版**：单用户独立应用，支持 macOS、Windows 和 Linux，用于管理个人待办事项及项目。
2. **个人服务器版**：多用户独立服务器，支持 Ubuntu，适用于开发和私人用途。

此外，项目支持 Docker 部署，提供 API 文档及完整的开发构建指南。
