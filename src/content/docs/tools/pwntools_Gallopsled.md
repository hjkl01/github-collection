### [Gallopsled pwntools](https://github.com/Gallopsled/pwntools)  ![GitHub Repo stars](https://img.shields.io/github/stars/Gallopsled/pwntools?style=social)

Pwntools 是一个基于 Python 的 CTF 框架和漏洞利用开发库，旨在简化漏洞利用编写并支持快速原型开发。它具备远程连接、汇编 Shellcode 生成、架构配置及交互式调试等核心功能，用于高效完成 CTF 挑战及漏洞利用任务。
