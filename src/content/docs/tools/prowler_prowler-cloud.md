### [prowler-cloud prowler](https://github.com/prowler-cloud/prowler)  ![GitHub Repo stars](https://img.shields.io/github/stars/prowler-cloud/prowler?style=social)

Prowler 是一款开源云安全平台，旨在自动化跨任何云环境的安全与合规性检查。它提供数百个内置安全检查和合规框架（支持 CIS、NIST、GDPR 等标准），涵盖 AWS、Azure、GCP、Kubernetes 等多种云服务。项目包含命令行工具（CLI）和 Web 应用（Prowler App），具备 AI 驱动的评估、威胁评分（ThreatScore）及攻击路径分析能力。支持 Docker、Pip 等多种部署方式，致力于提供简单、可扩展且具成本效益的云安全解决方案。
