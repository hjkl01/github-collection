### [koreader koreader](https://github.com/koreader/koreader)  ![GitHub Repo stars](https://img.shields.io/github/stars/koreader/koreader?style=social)

KOReader 是一款主要面向电子墨水屏设备的跨平台文档查看器。

*   **跨平台支持**：兼容 Kindle、Kobo、PocketBook、reMarkable 等嵌入式设备，以及 Android 和 Linux 系统。
*   **多格式兼容**：支持 PDF、DjVu 等固定页格式及 EPUB、FB2 等流式排版格式，内置工具可重排扫描文档。
*   **个性化阅读**：提供多语言界面，支持自定义页边距、行距、字体及断词字典，界面针对电子墨水屏优化（无动画、分页菜单）。
*   **功能集成与扩展**：与 Calibre、Wikipedia、Google Translate 等服务集成，支持插件扩展、在线目录订阅、查词及 SSH/FTP 工具。
*   **高性能**：运行速度快，翻页延迟低于部分设备自带阅读软件。
