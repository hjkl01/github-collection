### [koreader koreader](https://github.com/koreader/koreader)

**项目核心内容总结：**  
KOReader 是一款专为电子墨水屏阅读器设计的文档阅读器，支持多种格式（如 PDF、EPUB、MOBI、TXT 等），可在 Kindle、Kobo、Android 等设备及电脑上运行。其主要特性包括：  
- **多格式兼容**：支持固定版式（PDF、CBZ）和可重排格式（EPUB、FB2），扫描文档可借助内置工具重新排版。  
- **高度自定义**：支持多语言界面、自定义字体样式、页边距和行距，内置多语言断词词典。  
- **集成功能**：与 Calibre、Wallabag、Wikipedia 等服务联动，支持无线传输、OPDS 目录浏览。  
- **优化体验**：针对电子墨水屏优化，无动画 UI、可调节文本对比度、快速翻页。  
- **可扩展性**：支持插件扩展，提供 FTP、SSH 等实用工具。  

**使用方法**：  
根据不同设备（如 Android、Kindle、Kobo 等）的安装指南进行部署，或从 GitHub 下载最新版本。

提供用户指南、开发文档及社区支持，用户可通过翻译、提交问题或捐赠参与项目。
