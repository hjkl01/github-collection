### [cloudflare vibesdk](https://github.com/cloudflare/vibesdk)  ![GitHub Repo stars](https://img.shields.io/github/stars/cloudflare/vibesdk?style=social)

Cloudflare VibeSDK 是一款开源的全栈 AI Web 应用生成器，允许用户部署并自定义运行在 Cloudflare 平台上的 AI 编码环境。用户通过自然语言描述构建需求，AI 代理即可自动生成代码、在沙箱容器中提供实时预览，并一键部署至 Cloudflare Workers。该项目基于 Cloudflare 全栈生态（Workers、Durable Objects、D1、R2 等）构建，默认生成 React + TypeScript + Tailwind 技术栈的应用。主要功能包括分阶段 AI 代码生成、交互式聊天开发、GitHub 集成及 TypeScript SDK 程序化访问。适用于希望构建 AI 应用开发平台的企业、非技术团队内部提效及 SaaS 平台功能扩展。
