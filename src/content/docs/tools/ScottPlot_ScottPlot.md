### [ScottPlot ScottPlot](https://github.com/ScottPlot/ScottPlot)  ![GitHub Repo stars](https://img.shields.io/github/stars/ScottPlot/ScottPlot?style=social)

ScottPlot 是一个面向 .NET 的免费开源绘图库，能够轻松交互式地展示大型数据集。它支持创建折线图、柱状图、饼图、散点图等多种图表类型，并兼容 Windows Forms、WPF、控制台、Blazor、Avalonia、Uno Platform 及 Notebook 等多种运行环境。
