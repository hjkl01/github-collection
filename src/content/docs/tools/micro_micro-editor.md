### [micro-editor micro](https://github.com/micro-editor/micro)  ![GitHub Repo stars](https://img.shields.io/github/stars/micro-editor/micro?style=social)

micro 是一款基于终端的文本编辑器，旨在成为 nano 的易用继任者，以无依赖（单静态二进制文件）、开箱即用和跨平台为特点。

核心功能包括：
1. **编辑能力**：支持多光标、分屏、标签页、宏、自动补全、持久化撤销、软换行及 Unicode 支持。
2. **交互体验**：常用快捷键（可自定义）、优秀的鼠标支持（拖拽、双击、三击选词/行）、nano 风格菜单提示及内置帮助系统。
3. **可视化**：支持 130 多种语言语法高亮、16/256/真彩色主题、智能空格与 Tab 错误提示、内置 diff 边栏。
4. **扩展与集成**：基于 Lua 的插件系统（内置管理器）、自动 linting 和错误通知、系统剪贴板集成。
5. **使用方式**：支持通过命令行打开文件、终端标准输入创建缓冲区。

提供多种安装方式（预编译二进制、源码编译、各类包管理器）。
