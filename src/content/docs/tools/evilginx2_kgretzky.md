### [kgretzky evilginx2](https://github.com/kgretzky/evilginx2)  ![GitHub Repo stars](https://img.shields.io/github/stars/kgretzky/evilginx2?style=social)

Evilginx 是一个基于 Go 语言开发的中间人（MITM）网络钓鱼框架，用于窃取登录凭证和会话 Cookie，从而绕过双因素认证（2FA）保护。该工具作为独立应用运行，内置 HTTP 和 DNS 服务器，部署便捷。

其高级版本（Evilginx Pro）提供反钓鱼检测、Bot 防护、网站伪装、内容混淆、通配符 TLS 证书及自动化部署等功能，并支持与 Gophish 集成。项目声明仅应用于获得授权的渗透测试任务。
