### [danielmiessler Fabric](https://github.com/danielmiessler/Fabric)  ![GitHub Repo stars](https://img.shields.io/github/stars/danielmiessler/Fabric?style=social)

Fabric 是一个开源框架，旨在通过 AI 增强人类能力，解决 AI 工具集成困难的问题。它通过将提示词组织为基于任务的“模式”（Patterns），让用户统一管理 AI 解决方案。

核心功能包括：
1. **多供应商支持**：兼容 OpenAI、Anthropic、Ollama 等主流及兼容厂商。
2. **Pattern 系统**：内置大量预设模式（如摘要、分析、代码审查），支持创建自定义私有模式。
3. **多接口支持**：提供命令行（CLI）、REST API 服务及 Web 图形界面。
4. **扩展能力**：支持语音/文本转换、图像生成、YouTube 内容分析、多语言（i18n）及 Shell 补全。
5. **辅助工具**：集成 to_pdf、code2context、generate_changelog 等独立工具。

项目致力于让 AI 便捷地融入工作流，提升人类创造力与效率。
