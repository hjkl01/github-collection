### [espanso espanso](https://github.com/espanso/espanso)  ![GitHub Repo stars](https://img.shields.io/github/stars/espanso/espanso?style=social)

Espanso 是一款基于 Rust 的跨平台文本扩展器，支持 Windows、macOS 和 Linux 系统。其核心功能是自动将输入的特定关键字替换为预设内容，旨在节省打字并支持系统级代码片段、自定义脚本、Shell 命令、表情符号、图片及日期展开。项目采用隐私优先设计，100% 本地运行，兼容几乎所有程序。支持特定应用配置、包管理、正则触发及文件化配置，并提供实验性 Wayland 支持。
