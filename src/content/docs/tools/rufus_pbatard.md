### [pbatard rufus](https://github.com/pbatard/rufus)  ![GitHub Repo stars](https://img.shields.io/github/stars/pbatard/rufus?style=social)

Rufus 是一款可靠、无需安装的便携式 USB 格式化和可引导启动盘制作工具。主要功能包括：支持将 USB、闪存卡及虚拟驱动器格式化为 FAT32、NTFS 等多种文件系统；能够基于 ISO 或磁盘镜像创建 DOS、BIOS 或 UEFI 可引导盘，支持 Windows 和 Linux 系统安装，包括绕过 TPM/安全启动的 Windows 11 安装盘及 Windows To Go 驱动器；支持创建 VHD/DD 等镜像、持久化 Linux 分区、校验和计算、坏块检测及假冒闪存盘识别；提供官方 Windows 和 UEFI Shell ISO 下载。软件支持多国语言、兼容安全启动，采用 GPLv3 开源协议。
