### [web-infra-dev rspack](https://github.com/web-infra-dev/rspack)  ![GitHub Repo stars](https://img.shields.io/github/stars/web-infra-dev/rspack?style=social)

Rspack 是基于 Rust 编写的高性能 JavaScript 打包工具，拥有极速启动和构建能力，支持增量编译与极速热更新（HMR）。它高度兼容 webpack 生态，可无缝替换 webpack 并支持其插件与加载器，同时提供模块化联邦（Module Federation）支持和内置生产优化策略（如树摇、压缩）。Rspack 框架无关，作为 Rstack 统一工具链的核心，旨在提供一致且高性能的构建架构。
