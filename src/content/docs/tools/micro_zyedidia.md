### [zyedidia micro](https://github.com/zyedidia/micro)  ![GitHub Repo stars](https://img.shields.io/github/stars/zyedidia/micro?style=social)

Micro 是一款基于终端的文本编辑器，旨在提供直观易用的编辑体验，适合作为 nano 的现代替代品。它是一个无依赖的静态二进制文件，支持跨平台运行。

主要功能包括：
1. **基础编辑**：支持多光标、分屏、标签页、常用快捷键（可自定义）及完善的鼠标支持。
2. **语法与显示**：提供 130 多种语言的语法高亮、真彩支持、自动补全及错误提示。
3. **扩展功能**：内置 Lua 插件系统及管理器，支持宏录制、持久化撤销和自动 Lint 检测。
4. **系统交互**：支持系统剪贴板复制粘贴，易于配置，适合终端或 SSH 环境下的日常编辑工作。
