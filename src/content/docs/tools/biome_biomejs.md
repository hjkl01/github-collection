### [biomejs biome](https://github.com/biomejs/biome)  ![GitHub Repo stars](https://img.shields.io/github/stars/biomejs/biome?style=social)

Biome 是一款面向 Web 项目的高性能工具链，主要用于代码格式化和检查（Linter）。

**支持语言**
JavaScript、TypeScript、JSX、JSON、CSS、GraphQL。

**核心功能**
1. **格式化**：速度快，与 Prettier 兼容性达 97%。
2. **代码检查**：内置 340+ 条规则，输出详细诊断信息。
3. **编辑器集成**：支持 LSP，可在编写代码时实时格式化和检查。

**主要特点**
- 无需配置，开箱即用。
- 无需 Node.js 环境（编译为 WebAssembly）。
- 统一工具链，整合了格式化、检查等功能，提供一致的配置、缓存及并行工作流。
