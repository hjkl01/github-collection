### [biomejs biome](https://github.com/biomejs/biome)  ![GitHub Repo stars](https://img.shields.io/github/stars/biomejs/biome?style=social)

Biome 是一个用于 Web 开发的高性能代码格式化和静态检查工具，支持 JavaScript、TypeScript、JSX、JSON、CSS 和 GraphQL 等语言。其核心功能包括：  
1. **代码格式化**：兼容 Prettier，提供高效、一致的代码美化能力；  
2. **静态检查**：内置 340+ 条规则（来自 ESLint 等），支持实时代码质量检测；  
3. **编辑器集成**：支持主流编辑器实时校验，无需依赖 Node.js；  
4. **性能优化**：通过语言服务器协议（LSP）提升处理速度，减少资源占用；  
5. **开源授权**：采用 MIT 和 Apache 双许可证，支持企业定制化需求。  

**使用方法**：通过包管理器安装后，使用命令行执行格式化（`biome format`）或检测（`biome check`）操作，也可集成到编辑器中实时生效。项目由社区和企业赞助支持，包含多个层级的赞助商。
