### [sinelaw fresh](https://github.com/sinelaw/fresh)  ![GitHub Repo stars](https://img.shields.io/github/stars/sinelaw/fresh?style=social)

Fresh 是一款现代化的全功能终端文本编辑器，支持零配置开箱即用。它提供类似 VS Code 和 Sublime Text 的直观体验，具备标准键位、鼠标支持及 IDE 级功能。编辑器针对高性能优化，可处理多 GB 大文件并保持低内存占用与低延迟输入。核心功能包括命令面板与模糊查找、多光标编辑、主题与自定义、文件管理、基础编辑、搜索替换、多视图布局、LSP 语言服务器支持、生产力工具、TypeScript 插件扩展及多语言支持。
