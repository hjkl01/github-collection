### [trustedsec social-engineer-toolkit](https://github.com/trustedsec/social-engineer-toolkit)  ![GitHub Repo stars](https://img.shields.io/github/stars/trustedsec/social-engineer-toolkit?style=social)

SET（Social-Engineer Toolkit）是一款开源的社会工程学渗透测试框架，提供多种攻击向量以快速构建逼真的攻击场景。该工具仅限获得严格授权的安全测试使用，支持 Linux 和 Mac OS X（实验性）平台。
