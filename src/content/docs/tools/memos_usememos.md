### [usememos memos](https://github.com/usememos/memos)  ![GitHub Repo stars](https://img.shields.io/github/stars/usememos/memos?style=social)

Memos 是一款开源、自托管的隐私优先笔记服务，适用于个人笔记、团队 Wiki 及知识管理。该项目基于 Go 和 React 构建，以单二进制文件运行，资源占用低且启动迅速。支持原生 Markdown 编辑，数据以纯文本存储确保可移植性。部署灵活，支持 Docker 及多种数据库（SQLite、MySQL、PostgreSQL），并提供 REST 和 gRPC API 便于集成。界面简洁，支持深色模式和移动端，无追踪、无广告、无订阅费，所有数据由用户自行掌控。
