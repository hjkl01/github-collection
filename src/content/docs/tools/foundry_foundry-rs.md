### [foundry-rs foundry](https://github.com/foundry-rs/foundry)  ![GitHub Repo stars](https://img.shields.io/github/stars/foundry-rs/foundry?style=social)

Foundry 是一个使用 Rust 编写的快速、便携且模块化的以太坊应用开发工具包。核心功能包括：Forge 用于构建、测试、模糊测试、调试和部署 Solidity 合约；Cast 用于与 EVM 智能合约交互、发送交易及获取链数据；Anvil 提供快速的本地以太坊开发节点；Chisel 为快速实用的 Solidity 交互式命令行工具（REPL）。
