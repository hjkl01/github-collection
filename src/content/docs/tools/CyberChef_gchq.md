### [gchq CyberChef](https://github.com/gchq/CyberChef)  ![GitHub Repo stars](https://img.shields.io/github/stars/gchq/CyberChef?style=social)

CyberChef 是一款基于 Web 浏览器的多功能网络安全工具，被称为“网络瑞士军刀”，用于在浏览器中对数据进行复杂处理。

**主要功能：**
*   **数据编码与解码**：支持 Base64、XOR 等多种格式。
*   **加密与解密**：涵盖 AES、DES、Blowfish 等算法。
*   **数据处理**：支持二进制/十六进制转换、压缩解压、哈希计算与校验和。
*   **协议解析**：支持 IPv6、X.509 及日期时间转换。
*   **自动化分析**：具备编码自动检测（Magic）、Shellcode 解密反汇编及断点调试功能。

**核心特性：**
*   **纯客户端运行**：所有数据处理在浏览器本地完成，不上传服务器，确保数据隐私。
*   **操作便捷**：支持拖拽式配方组合、实时自动处理（Auto Bake）、配方保存及 URL 分享。
