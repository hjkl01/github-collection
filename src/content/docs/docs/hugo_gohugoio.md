### [gohugoio hugo](https://github.com/gohugoio/hugo)  ![GitHub Repo stars](https://img.shields.io/github/stars/gohugoio/hugo?style=social)

Hugo 是一款使用 Go 语言构建的快速、灵活的静态站点生成器。它专为速度优化，具有先进的模板系统和高效资产管道，能在几秒内渲染完整站点。支持多语言、强大分类体系及内置开发服务器，适用于企业官网、文档、博客等多种场景。核心功能涵盖图像处理、JavaScript 打包、Sass 和 Tailwind CSS 处理，以及通过 Hugo 模块实现内容、主题和配置的共享。提供标准版、扩展版及扩展部署版，支持源码或二进制安装。
