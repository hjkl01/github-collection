claude-hud ¶

Claude HUD 是 Claude Code 的插件，在输入框下方常驻显示会话状态。它通过状态栏实时展示项目路径与 Git 分支、上下文占用与用量条、工具读写搜索活动、子代理运行状态和待办进度，并支持配置布局、显示项、颜色与阈值。使用原生状态栏 API 获取真实 token 数据，解析会话记录并高频更新，帮助用户直观掌握当前会话情况。
