nodejs node ¶

Node.js 是一个开源、跨平台的 JavaScript 运行时环境。项目提供当前开发版、长期支持版（LTS）和夜间构建版三种发布类型，支持源码构建、二进制下载及 API 文档获取。Node.js 遵循开放治理模式与 MIT 协议，提供安全验证机制，并鼓励社区协作贡献。
