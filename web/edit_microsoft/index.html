microsoft edit ¶

Edit 是一款面向简单需求的文本编辑器，致敬 MS-DOS 经典编辑器，具备现代化界面和 VS Code 风格输入控件，便于终端新手使用。支持通过 ICU 库实现搜索替换，并提供多国语言国际化支持。
